2005-07-28 09:08:22 • stw
The ideal is always to use minimal number of Bezier points needed, both for economic reasons (the font file becomes smaller) and visual reasons (simpler curves tend to look "smoother"). I guess the reason why Freehand showed more Bezierpoints than necessary was that you used a True Type version of Arial, while in Freehand curves are described with the same polynomials as Type 1 fonts, not True Type. The extra points were probably added during the process of translating the latter into the former, but they're useless. oribendor

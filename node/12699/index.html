Usually the problem with Chinese fonts is the encoding, wich is different if you’re using fonts for the Hong Kong/Taiwan market or the mainland China one, and it can get quite nasty with old font files.

Did you try to put the font file directly into the system fonts folder? Since it’s only two of them it shouldn’t slow down too much your machine’s performances and you can remove them when you’re done with the book.

Song is one of the system fonts that come with OSX, so you could install the additional asian fonts package from the installation disk, but I don’t know how you’re going to make color separation so it might not be a good idea to use system fonts.

I hope it helps, Jacopo

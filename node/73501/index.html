To do that:

  1. Remove fonts in question from Fonts folder
  2. Quit InDesign
  3. Search for “fnt” and trash all “AdobeFnt07.lst” files BUT NOT the “AdobeFnt.db” files
  4. Install revised fonts (drag to “User” > Library > Fonts)
  5. Reboot InDesign

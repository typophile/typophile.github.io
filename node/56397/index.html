Command G Name your glyphs “paragraph.alt” and Q.alt"

They will now be available to users in the glyph palette.

Also:

Create a Stylistic Set.

sub paragraph by paragraph.alt; sub Q by Q.alt;

This will work if you generate “.otf” fonts, and be available to users in the OT features palette.

Missing punctuation

May 26, 2004 • michaelt

I edit a charity magazine and have to send pdf's of the layout to various managers for approval. The layout is done in Quark 6.1. The main font is Gill Sans. When I export the layout as a pdf all of my punctuation marks go missing! , ' " etc all disappear. This makes proof reading a nightmare and doesn't make them look kindly on poor little ol' me! Any other font is fine. Any suggestions on how to fix?
I've run into this problem on our Macs w/ QX 4.1 when printing. It happens with only one face: Times New Roman. We have the good fortune of being able to use Times instead, so I can't offer a fix. My guess is that your font is bad/ corrupt and needs to be replaced. I'm sure others have a more specific diagnosis for you. Does this happen with Gill in other apps?
2004-05-26 16:51:53 • Chris Rugen
This is almost certainly not a corrupt font. It is likely an encoding problem caused by having two different versions of the same font in the workflow in ways that conflict. First, make sure you're embedding the font. Always embed all fonts. Second, check your system really thoroughly for multiple versions of Gill Sans. Is there an OS X system dfont for Gill Sans as well as a Type 1 version, both on your system? If so, uninstall one of them. Regards, T

What I do:

Go to System preferences > Language & Text > Input Sources > activate Unicode Hex Input.

Now you should see some little flag at the top of the bar in Finder (which flag you see depends on what language your keyboard is). Click it and change it to Unicode Six Input.

Now, in InDesign: press alt (also called option) while you press any Unicode number you want. It should work everywhere, for example here:

alt 2018: ‘ alt 2019: ’ alt 201c: “ alt 201d: ”

How to directly enter Unicode on a Windows platform, with InDesign:

  1. make sure Num Lock is on.

  2. Whilst holding down the “alt” key, hit (on the numeric keypad) plus then the Unicode numbers. For an alpha character, use the regular keyboard, of course.

That’s all there is to it. No change to the System preferences or anything else.

2010-10-25 01:44:46 • charles ellertson
Pre-Unicode legacy. This is the encoding of Latin-1 native to Windows. First of all, the code is in decimal so that it can be entered on the keypad: To get é you type Alt+0233. In Unicode é is Ux00e9, which is hexadecimal for two hundred and thirty-three. Secondly, the quotation marks are in the two rows of characters which don’t match up with Unicode at all. In the history of computing there have been dozens of various encodings and changing standards, so in terms of the final fit it could have been a lot worse.

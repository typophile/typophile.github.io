Ambicase Fatface

January 7, 2011 • eliason

As a follow-up to [[http://typophile.com/node/75182|Ambicase Modern]] (the development of which was greatly assisted by its [[http://www.typophile.com/node/60316|Typophile critique thread]]), I have created Ambicase Fatface. It is considerably bolder version of that single-case font which strove to combine upper- and lowercase forms for each letter. Character set and OpenType features are much the same as Ambicase Modern, though I have added discretionary ligatures for FT, CT, and ST. [img:sites/default/files/old-images/AmbicaseFatface01_5059.png] PDF specimen below. I welcome any constructive feedback you have!

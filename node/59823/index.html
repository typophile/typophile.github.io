2009-07-14 12:38:08 • Jan
Janson Text Roman Std contains 407 glyphs, while the Pro version has 590 glyphs. As Dan from Linotype has explained, the difference is characters that are needed for Central and Eastern European languages. Hence, the Pro is more versatile, but also a bit more pricey. If you don’t need those extra glyphs, you’re better off with the Std version. Both OpenType versions contain small caps and the various figure styles. F

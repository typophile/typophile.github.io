2011-11-07 16:19:23 • oldnick

Text editing in Acrobat is really just intended for small tweaks, like fixing a typo or changing a number. There is no automatic text reflow, so things get ugly real quick when you make changes. An alternative is to make an interactive PDF with multi-line text fields, but that’s really intended for “fill in the blank” spots in forms, not a way to create an entire document.

I believe there are some programs intended for the prepress departments at print shops that allow more sophisticated revisions to PDFs, but I think they’re pretty expensive and are not intended for use by non-professionals.

Regarding the font issues, some fonts can be embedded in a PDF but to avoid licensing issues you’d probably want to stick to a system font that you knew everyone already had (Arial, etc).

If the client insists on the ability to make major changes on their own, I’d probably just grit my teeth and do the project in Word.

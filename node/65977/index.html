Brandegoris

The typeface is “Erbar Initialen” (Erbar Initials) see http://www.klingspor-museum.de/KlingsporKuenstler/Schriftdesigner/Erbar/JErbar.pdf

There are two digitisations I know of, Dave Nalle’s Brandegoris (with alternative caps in the lowercase), look at the shareware version http://www.dafont.com/brandegoris.font

and

DeRoos Caps (by Dave Fabik) http://www.dafont.com/search.php?psize=m&q=deroos (free for non-commercial use. IMHO this is the cleaner digitisation - from the Dover specimen book, the readme file says.

January 5, 2018 • robmcconnell

Which typeface or type family has the largest language support that you know of? How many languages does it support? The best I’ve been able to find is Source Sans Pro which supports 104 languages (Cyrillic and Latin)

Google’s Noto family of fonts support about 800 languages in about 100 scripts. It is not a single font but a family weighting about 1gb though.

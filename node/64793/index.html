This is more complex and richer than finger smears (which don’t evoke that much). For one thing, it generates depth. Also, you have to time it with the scanner’s movement*; and you can’t have upstrokes for example. It takes a lot of skill - as you can see with that “R” it’s pretty hard to get good results… So I’ve removed the Flickr set (the two photos are still there though) until I can take a class in dedography to improve my skills. ;-)

  • It might seem artificial for the results to depend so much on a “tool”, but it’s the same with using a broad-nib pen for example.

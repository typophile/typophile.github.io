2004-07-14 22:55:26 • Mark Simonson
Dieter Steffmann digitized a set of 3 Parsons Bold fonts under the name Puritan. The set contains the base font as well as an Alternates and a Swash variant. None of these include the G in the Gracie image, but Puritan Regular contains that M, which the Alphabets version doesn't. Herr Steffmann digitized Parsons Heavy and, for some reason, allowed it to keep its name. This also includes an Alternates font. Finally, for completeness sake, there's an old Compuworks font called Churchman. This is Parsons Light. None of these fonts are terribly well made and the Compuworks font is missing some punctuation and all extended characters. They all look lousy on screen at even medium small sizes.

Unicode 5.2: Andron Mega now features new Old Cyrillic block

February 26, 2010 • Andreas Stötzner

Andron Mega has yesterday been released in version 1.2. Among various other character and glyph additions (plus about 300) the complete coverage of the [[http://www.signographie.de/cms/front_content.php?idart=223|new block A640–Old Cyrillic]] in Regular is worth noticing.

The Andron Mega fonts package now contains altogether 13.590 glyphs in five fonts.

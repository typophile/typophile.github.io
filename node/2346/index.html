Hrant, There's a way round this, but not foolproof. You could set the 'open options' (file > document properties > open options) to make the file open at full screen mode, at 100%. Set the permission as viewing only. Then every time the file is opened it will display the way you intended. But of course the users can always get out of the full screen mode and do whatever they want with it… Putting navigation on the actual pages would help, I guess, making the PDF a self-contained 'application'.

> Check this out: http://www.selios.org/UK/index.php I see only two type families excluding the logotype, Underware's Dolly and a sans. I think it's best to accept the 2 families limit as a guide, not as an absolute rule. There are situations where you feel the need to break this ‘rule’. But really, I think that we need to know the ‘rules’ (or the basics) before we break them otherwise we’re not designing with conscious thought.

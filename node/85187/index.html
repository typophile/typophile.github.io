ID this chunky, sharp serif

September 6, 2011 • lathen.kamas

Very familiar but can’t put my finger on it. Thoughts?

After searching with the Serif Font ID Guide, I think this is a modification of the late Bob Alonso’s Shangrala. The letters have been widened/flattened and the serifs have been modified with triangles, but the underlying font seems extremely close (see especially the A, which was not modified much). The serifs in this sample look like they have been pasted on to me.

At least, if I was re-creating this, this would be a good starting point.

I can see now why the Serif Guide missed Amerigo, which Kent recognized. The E foot serif for Amerigo was classed as angled and I searched for vertical.

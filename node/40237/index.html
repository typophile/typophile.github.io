We used a few different faces on the site, Hoeffler text is used for a lot of the copy in the forms, Gotham is used for the little hint text below the dotted lines, Helvetica Ultra Compressed for the form headlines. It’s all over the place typographically. I was trying to make it seem old and yet new, like it was trying to be futuristic when made in 1933.

The design is still a bit rough around the edges. You’ll notice that a lot of the forms change the metrics of things, particularly the areas on the right side.

Any suggestions would be appreciated. Additionally, I’d be curious to hear suggestions for how to add more variety to the different forms. My fear is that they’ll all start to look the same in time.

Enjoy!

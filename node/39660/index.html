I find hairline weights are much more invisible at text sizes. All joking aside, I suspect it is mainly a function of what we are used to seeing as readers. Because texts in bolder or black weights (or hairline weights for that matter) are used mainly for emphasis, we just don't see them as often. When we do, we are distracted, so to speak, by the difference in texture from what we are used to seeing. We pick up, either consciously or not, that we are seeing a special type treatment. On the other hand, medium weights are defaults and we tend not to notice the type treatment as much, and go straight to seeing the message. For someone used to seeing texts set predominantly in a blacker texture, as in blackletter to use a historical example, would text set in a relatively lighter colour corresponding to the medium weight be as invisible? I have my doubts.

2007-12-02 05:09:16 • Wesley.Bancroft
…why do you think this occurs? I think that this goes back to the old notion that people read best what they read most. Typographic hierarchies aren’t as variable as we think they are—they’re as much convention as they are a design tool. Type designers take advantage of this and create heavier weights to draw attention to themselves within the context they’ll most likely be used.

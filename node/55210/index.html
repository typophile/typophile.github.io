Word 2007 can save files in the old .doc format. It can be done individually for each file by using Save As, or [[http://office.microsoft.com/en-us/word/HA102406831033.aspx|there’s a setting]] that makes Word always use .doc instead of .docx. Or you can download the [[http://www.microsoft.com/downloads/details.aspx?FamilyID=6b9238e1-cf69-48c4-bf2d-c4a8aceee520&DisplayLang=en|Open XML File Format Converter for Mac 1.0]].

So my mission here is to find a soul out there with a much wider, dare I say "geekery" of type knowledge to help me select a similar face with an equal amount if not more beauty than that of Melior. Any takers?

I believe that the received wisdom on logos is that it is best in anycase to draw variations on the letters to suit that particular logo. So if you are going to draw anyway, having the basic model should be enough, no?

Doyald Young's Fonts and Logos contains many brilliant examples of such variations & creations inspired by existing typefaces. He always uses pencil first, but you could also draw in a computer program, and get a good result.

Since this is pre-digital, I started scanning the Photo-Lettering catalogs and there are scores of brush scripts with every degree of thickness, slope, and letter styles. Those of you with Solotype Catalogs can see a small sample of these in pages 104-106. The names are not the same as in the P-L books, but the lettering sure is. Some like Hostess Script or Hemlock Script (Solotype names) are pretty similar looking, but we can't see enough letters.

I guess my point is that I think you are looking for something that probably doesn't exist in the digital font world (yet). Now that House owns the Photo-Lettering collection there's a chance they will digitize some of these 50's-era workhorses. Which reminds me House has a few fonts like this in their current Sign Painter collection.

2008-06-19 16:22:17 • Florian Hardwig
You can edit the PDF file in Illustrator or similar programs assuming it's not locked - or if it is, you have the password - but first off you'd will definitely need the fonts in question - and that's assuming the text wasn't converted to outlines at some point before making the PDF. So, the obvious thing you're going to have to do s purchase those fonts again. I've looked on Linotype and if you just need the two weights, it's not that expensive at all. The only other option if you don't want to buy the fonts is to try and find a free similar looking font, but off hand I can't suggest an alternative, sorry. JH

what to use with Avante Garde

February 26, 2009 • dl

I’m doing a website for a company whose logo uses Avante Garde. The overall style is very clean.

I’d like to use the Avante Garde in callout headings, to be created as images and placed in side columns on the page . The rest of the content, headings, menu, etc. will be in web based Verdana.

I’m not feeling confident in the mix. Verdana, when bolded, looks pretty clunky along side the Avante Garde.

Thoughts?

i would choose Georgia, it’s a serif with a nice open style. I think any combination of Avant Garde with a different sans-serif will look bad. Good luck!

Deleting glyphs in FL Studio

October 11, 2010 • Stephen Rapp

I’m curious if there is some easy way to delete glyphs properly in FontLab Studio. When I change the name of a glyph FL Studio asks if I would like to have it changed in all the classes and feature code. That’s quick and easy. When I delete a glyph, however, all it allows is deleting that cell. Then I have to wade through feature code, kern classes, OT classes, kern feature and so forth to remove each occurrence of that glyph as well. That is time consuming.

My plan was to delete all of the swash caps in a font that I am restructuring for a client. Then I need to add in 4 sets of swash caps and ligatures with the swash caps for stylistic sets. I don’t really want to spend a week just wading through all this stuff. Any ideas?

Dump your classes, OT code, etc to text files. Then clean out all the references to deleted characters with TextWrangler. It has a “Process lines containing” feature that can delete every line containing a word. Alternately you could run sed in a loop to just strip out all the lines in all the files at once.

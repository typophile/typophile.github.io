Someone at my studio modified this font a few years ago and changed the name from the original. We are trying to track down the original so the client can buy it, but nobody can even find it now… If anyone has any pointers, it would be much appreciated!
It looks like one of the 300 Handwritten Fonts that were issued by ClickArt in 1995 on a CD collection, but looking through that catalog didn't show the same font. There are some similar fonts issued by various foundries. I have collected roughly 500 fonts of this hand printed style (including the ClickArt fonts) in Part 11 of my Script Font ID Guide, but you should realize that this font style can be easily and cheaply created. There are more freeware and one-of-a-kind handprinted fonts than you can start to hunt through, so my advice is usually just find something similar, and go with that.

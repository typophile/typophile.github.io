Indie Fonts 2

August 23, 2007 • James Grieshaber

[[Indices]] : [[Books]] : Indie Fonts 2

Indie Fonts 2, a companion to the award winning [[Indie Fonts]], brings together an additional 19 independent font foundries from around the world. This book is an invaluable resource of over 1600 fonts for designers, art directors and typographers.

This book is divided up by foundry so that the user can get a feel for the work produced by each group. This also makes it convenient to order fonts directly from their source. There is a visual font index provided in the front for quick scanning and a fully cross referenced index in the back containing font names, designers, and original designers.

Contributing foundries include [[Atomic Media]], [[Feliciano]], [[Galápagos]], [[Holland Fonts]], [[Identikal]], [[ingoFonts]], [[Jukebox]], [[MVB Fonts]], [[Neufville Digital]], [[Nick’s Fonts]], [[No Bodoni]], [[Parkinson]], [[Sherwood]], [[Mark Simonson Studio]], [[Storm]], [[Terminal Design]], [[Underware]], [[Union Fonts]] and [[YouWorkForThem]].

Accompanying this book is a Bonus Fonts CD-ROM that contains 38 indie fonts on an unlocked disc located in a sealed package in the back of this book. The fonts are licensed from each contributing foundry for use solely by the owner of this book.

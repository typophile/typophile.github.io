Is it possible to name a font so that it shows in the family name menu on Illustrator as “FONTNAME REGULAR” and “FONTNAME BOLD” but without the “regular” appearing in the weight window?

Yes, I think so. But then the family name had to be “FONTNAME REGULAR” and the full name would be either “FONTNAME REGULAR” or “FONTNAME REGULAR NORMAL” or “FONTNAME REGULAR UPRIGHT” or “FONTNAME REGULAR ROMAN”.

For more information have a look into Adam’s naming tutorial and the replies.

Crit personal web-site

July 31, 2009 • totocaster

Finally, I’ve launched my personal web-site (http://totocaster.com). I wanted it to be super light, simple, focused on front page and to have great typography and readability without any non-web-standard font. I’ve used only three fonts from which only 2 are used mainly, those are Georgia and Helvetica (Neue for ultralights), for super small texts and photo descriptions I’ve used Apple standard Lucida Grande which looks perfect form me when used in a very small size.

All major browsers are supported except IE, best viewed in Safari. Sorry Windows users of Internet Explorer, text shadows are not supported by your browsers yet, and typography looks ugly if you have turned off screen smoothing.

Share your thoughts, what could be done better?

Thanks in advance.

2009-08-04 16:55:08 • dtw

I find it interesting that you chose to use letterspacing:-.1em on the headers. I can see how it is interesting to merge characters, but not all the characters merge and, given the differences in font rendering between browsers, more or less of them are actually touching.

As mk2 says, having a clickable header is extremely common, so requiring users to click the “back” button is a little counter intuitive. It should also have a rollover, similar to the nav system to indicate that it is clickable. Ahh, I see. You’ve set it up so that the back button goes back to the previous page rather than to the home page. A persistent navigation might help resolve this.

I really like the way you’ve overlaid your images with the text and underline. It feels rather nice, though I worry if you don’t have the text shadow (like on the about page).

Also, on the “Software” page, the “ft” is begging for a “ft” ligature (I think it is ﬅ), though that might break your letter spacing intent.

In any case, that’s the main stuff I saw. Overall I like the feel of the site & especially the hit of color in the navigation area, though I wish you’d use those colors more throughout the site.

Nice stuff so far!

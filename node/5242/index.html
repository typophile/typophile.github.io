Kent is on the right track, but I think it will work better if you set the First Baseline Offset to "Leading". Also, make sure the leading is the same for all three text boxes.

BTW, I wouldn't call this an error so much as an inconvenient default setting. Quark has a similar setting (I seem to recall) which used to default to the same thing. They must have changed it at some point, or perhaps your Quark default setting is different.

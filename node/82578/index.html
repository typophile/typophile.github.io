> I'm beginning to think it will need a lot of kerning and ligatures Only if you want it very tight, and that will backfire when setting strings of letters that simply cannot be too tight without colliding*. Look at it this way: you only need very tight spacing for display setting - for text** you can/should go looser; and if it's really for display you can/should expect the discerning type user to manually tighten & tweak the spacing based on the string at hand anyway. * And if you don't mind collisions then spacing is very easy. :-) ** I don't mean a book of course - this font isn't for that. hhp

Antique Georgia license plate fonts

May 27, 2008 • guiddruid

Can anyone please help identifying the fonts used on Georgia license plates, from 1957-1970? I have attached an example from 1969, more examples may be found here; http://www.plateshack.com/georgia/georgia3.html http://www.15q.net/ga.html

I am trying to create a replica plate for a ‘69 Pontiac GTO, and I would like the font to be as close to the original as possible.

Thanks, Simon

I’d recommend looking for the small letters among the Techno freebies at Dafont. As for the large numerals/letters, look at [[http://www.myfonts.com/fonts/fontboutique/noga/|Noga]] from Font Boutique. It seemed to be the closest for all the characters. Many others were similar, but would have needed some serifs added or removed to match more closely.

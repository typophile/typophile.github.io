Hi,

This looks like its cross-hatching done by etching.

You could get a similar look if you had two parallel lines, ones ticker than the other in Adobe illustrator then use the blend tool to set ‘specific steps’ between the two strokes. Then repeat the process on a different angle.

I hope this helps.

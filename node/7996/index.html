2004-12-04 21:52:30 • James Gareth
James,

I've been there, and after several frustrations I've managed to get the problem solved with a strange technique:

1. You will need Stuffit Expander 7 or 7.5 (I use 7.5) The recent versions of this PC software can't be used cause you end up with the zero size.

2. Right click > Expand with options > Cross Platform > Save files in MacBinary Format > Always

3. Use Trantype 2 (best font converter) or the shareware Acute Systems CrossFont 3.3.

This work for me.

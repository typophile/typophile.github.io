To restore all preferences and default settings Do one of the following: (Windows) Start InDesign, and then press Shift+Ctrl+Alt. Click Yes when asked if you want to delete your preference files. (Mac OS) While pressing Shift+Option+Command+Control, start InDesign. Click Yes when asked if you want to delete your preference files.

That’s from the Help Function, BTW ; )

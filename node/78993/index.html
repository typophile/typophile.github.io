Floral script initials on 1970s UK football shirts

February 11, 2011 • Storyteller

I’m trying to identify the floral script font that was used on these and other 1970s UK football shirts. Any ideas? http://myfonts-wtf.s3.amazonaws.com/99/99bebb6db21e78b648902a3b557f762b.159276.png

I think that in this time period (which is pre-digital) they were probably using some sort of lettering template for the embroidery machine, or had built in lettering that the machine could stitch. I also noticed that there are two different styles of A, so possibly a different machine was used for the last sample.

The first A is rather unusual, and I can’t find a font that has the same characteristics as these letters. I was hoping maybe to find a digital font based on a typeface that may have been the model for this stitching, but I didn’t see one, so far.

I am trying to work out how to make an animation with the word "construction" where it gets built with small blocks. Can anyone suggest the best program for me to use or the best way to approach it?
Draw a storyboard. Build "construction" first in a vector format in something like Illustrator or FreeHand. You can use a bold face and cut it into squares. You can also use a typeface if you want the blocks to bee seen in "construction" after the animation is completed but if you do it by hand that's better. There are typefaces built with rectangles but I can't remember the names. If the output destination is web, you can use Flash, if it's TV or a kind of video format, you can use After Effects. Take "construction" into Flash or AE and scatter rectangles one by one to your taste. Then play the animation reversed. This way you can have full control over the animation and it will look the way you want. There should be plugins to do it in AE but I don't know any. If you use a plugin, it will look default and will be similar to other animations out there.

A nice trick might be to use Underware's Unibody pixelfont, since its letters are made up of distinct blocks*, which makes it easy to individually remove the blocks. So you'd go "backwards" from a full rendering, removing the blocks step-by-step. And it's free. * Which is frankly a poor way to construct a pixelfont. hhp

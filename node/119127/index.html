For a font with the straight portions of the inline check Core Circus 2d In [img:sites/default/files/old-images/CoreCircus2dInInUse_3767.gif] https://www.myfonts.com/fonts/s-core/core-circus/ Use as is or extend the inline to make the curved portions of the Kreisler _K_ logo. The weight of Core Circus is a little lighter than the Kreisler _K_ logo. So if you are editing you could make the _K_ bolder. Or you could do all of the inline from scratch, using a base like ITC Avant Garde Gothic Bold, as suggested by Fivos. Don

I’ve make logo for my site called “OrigamiArabia” and i’ve problem with the typeface of the logo and im loss beacuse i tried many types and i cant find a type consistent with the mark.

There are a few typefaces I would like for this mark… If taking the serif route I think a Hoefler Text Roman is always a good choice.

If considering a san serif, Sudtipos makes some great faces to match the angularity of this mark. Mr. Giacco and Politica are great faces, check them out.

http://sudtipos.com/fonts/36 http://sudtipos.com/fonts/42

There is also an interesting serif from adobe called Belwe

Good luck!

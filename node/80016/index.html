InDesign: annotation figures in main text

March 14, 2011 • Andreas Stötzner

I don’t know wether this has been ventilated before.

A Word document gets placed in InDesign, the text running over many pages. It is science, there are many annotations handled as generic footnotes; InDesign does import this correctly so that the footnotes take place dynamically at bottom of each page. But now to the main text, where the annotation numbers are interspersed in the lines. These numbers employ the basic figures (0030 … 0039 by encoding) and renders them diminished and vertically raised. – That latter fact is my problem. In a body text of 11pt a figure diminished to 5…6pt looks famished, looks bad. I would like to set the genuin superscript figures of the font instead (00B2, 00B3, 00B9 and 2074 … 2079 by encoding) because that looks far better. But if I change the character in the main text – the link breaks and the footnote disappears.

– What can be done?

Some time ago I gave up trying to tame the automatic management of footnotes by InDesign (I transform them in endnotes in Word, so I have the numbers, but not the link). However, have you tried, instead of replacing the figure, to apply the Character > Unicode > Superior directly to the reference?

This might be done most easily by applying a character style to the footnote numbers, which InDesign can do automatically. In English (sorry), Type > Document Footnote Options shows a tab “Numbering and Formatting” where “Character Style” is a choice (Windows short-cut Alt-C).

Over on the Adobe’s InDesign forum there is much griping about the inadequacies of InDesign’s automated footnotes, and many people use various workarounds, several of which are described by Peter Kahrel.

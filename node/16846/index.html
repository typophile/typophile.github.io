[[Indices]] : [[Terminology]] : Letter

The set of shapes upon which we all loosely agree on that carry certain [[alphabet|alphabetic]] meanings and cues, which can be expressed with different [[glyph|glyphs]] and [[typeface|typefaces]], as long as it shares enough of the commonly recognized shapes that it is recognized as a meaningful symbol.

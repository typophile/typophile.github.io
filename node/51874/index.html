Do you have 'allow footnotes to break' on? If not, and you have a really long footnote here, I can imagine InDesign chokes. InDesign doesn't break footnotes at will, only if the reference is at the bottom of the text and the note still can't fit under it. You can even use the Keep settings to specify 2 or more lines to hold together. Sometimes the amount of space between the last text line and the first note seems a bit wide. Fiddling a bit with 'space before', or line heights of the text or in the footnote, or (last resort) make the text frame just a bit higher will allow another line inbetween.

I don't know about that moving the topic (moderators?). You cannot set "Allow footnotes to break" per note. But, as I said, ID will not break *all* notes, just the ones that don't fit entirely on one page. (Logically.) But -- given the constraints the page imposes -- you can select a note that you don't want to get broken and set its text Keep options to "Keep together", that will move that entire note plus the text line its reference is in to the next page. However, I don't see a problem. Do you really need every complete note to appear on the same page? With "Given the constraints of the page" I mean, obviously, that the note and its line have to be able to fit on one page. A point to keep in mind with notes that long.

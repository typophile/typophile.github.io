Hi!
I've been following the discussions on this forum for a while now - and it has been very inspiring to see the work in progress here.
I've been working on this font (working title: A1) for a couple of weeks and would like to have some critique. The missing glyphs of the l/c plus all of the u/c glyphs still need some (lots of) work, but I would like to hear your comments before I move along.
The font is partly based on the superellipse (thanx to Herman Zapf and this discussion board for introducing me to the superellipse) and its stems and hairlines has the same width as Helvetica Neue 55 Roman - so the uploaded image is an example of A1 roman (regular). Other weights will perhaps follow.

I think instead of trying to match an exact font you need to do some other things: make sure your tattoo artist can handle text - it’s a pretty difficult thing to get right; and find a font that simply has the overall proportions and feeling that you’d like. Tattooing is not a high-fidelity medium, so it’s not like you can actually reproduce fine details in a font. So it’s better to start with something relatively plain so you don’t end up with a bad surprise. Considering you like the tattoo above, maybe something like this: https://www.fontfont.com/fonts/videtur/regular

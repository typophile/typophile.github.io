I think they left this one for me because I have a whole set of 'speeches' about this font style.

I think it is a custom font, because this style is so easy and cheap to make. The lettering looks perfectly ordinary, probably because it was done by someone with no training. That's all the skill you need to have such a font made – that and about $9.

The bad news is that it's almost impossible to find this exact font, even if it's out on the web somewhere, because there are thousands like it. The good news is that there are thousands like it, so just find one with most of the characteristics you like and use that.

If you want a concentrated place to see samples of over 500 similar fonts, have a look at Part 11 of the Script Font ID Guide.

It may sound a bit compulsive, but it's actually a pretty cool feature. You can set a default for a whole document and/or individual text boxes. You have to figure that when you open a new text box the style will have to default to something... if you're used to using other Adobe products then you probably expect that something to be the most recently used style. In Indesign's case it will be whatever style settings were selected prior to the document being open. Here's the explanation from Indesign help: If you change settings when no documents are open, your changes set the defaults for new documents. If a document is open when you change settings, the changes affect only that document. Similarly, if you change settings when no objects are selected, your changes set the defaults for new objects. And defaults are fully scriptable, so everyone working on a project can be using the same defaults and, trust me, that's a beautiful thing :)

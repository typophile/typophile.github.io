If your font is to be used for setting text, the caps should be spaced to work with lower-case letters, punctuation, and especially the default word space. The caps have to look correct in a normal sentence, which can include initials such as "H. L. Mencken had a wicked wit", as well as other constructions. Moreover, the amount you letterspace full or small caps depends on how many words are involved, and the wordspacing values too will vary, depending on the number of words. Try it -- set CHAPTER ONE & get it feeing correct. Then set something longer, such as THE INTERMITTENT CHATTERING OF SMALL FURRY ANIMALS, AND BIRDS. I believe you will find that you pick different values. The type designer cannot function as the compositor, for the simple reason that s/he cannot specify the text to be set.

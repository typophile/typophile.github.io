2005-05-08 21:45:01 • nadine_chahine

I’m glad you touch that subject. It crossed my mind while I was implementing the Greek. Actually it would be very interesting and challenging to add a right-to-left script to [[adhesiontext]].

I believe that the way [[adhesiontext]] is setup allows me to add any Latin language, like German or Italian for example (and maybe even a central-European, like Czech), within a couple of hours. A non-Latin LTR script would need a bit more time, but surely not more than a day’s work. A RTL script is a bit more tricky, but still doable. I can’t predict how long that would take.

Nevertheless, if you can provide me two Arabic word-lists (one in Unicode and the other in Arabic-Windows-1256) and a string of the characters (in Unicode) to be included in the “Insert characters” palette, I can try to prepare an Arabic version of [[adhesiontext]] for you to test.

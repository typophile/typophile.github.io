As Mark Simonson said, #3 it looks like Lateinisch from Berthold. I want to add that this font was digitized by a Russian foundry, ParaType (www.paratype.com). They named it Literaturnaya. The font was donated to public domain, but only for use in TeX applications. It can be found in various russian TeX archives, both PFB and metrics, but the encoding is for TeX. Be sure to read the licence, cause it is not quite a free font.

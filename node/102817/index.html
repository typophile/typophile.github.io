2013-05-12 05:37:22 • Ken Krugh
Charles: It would be a hack -- not the intended purpose -- but what if you put the f-ligatures into an rlig feature? If Word 2010 sets Arabic, I'd bet it honors anything in rlig. Word, and other Uniscribe apps, make desicisions about which layout engine to use for a given text string based on the characters involved. So Arabic characters are passed to an Arabic layout engine, which processes {rlig}, among other things. But that doesn't mean that characters from other scripts will be passed to a layout engine that processes {rlig}.

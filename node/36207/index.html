I don't know about the MM version, but Bickham Script Pro, the latest incarnation of Bickham Script has 20+ alternates of the lowercase 'l'. The default form is indeed the one without the loop. The variant forms are either contextually inserted (if the OpenType "Contextual Alternates" feature is active), or can be manually inserted through the Glyph palette (which is available in a few applications).

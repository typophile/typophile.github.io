2007-06-27 15:39:47 • nevelle flynn
Actually I've used blending in CorelDRAW: 1. I've drawn an upright ellipse and a 90 degree rotated one that's wider 2. Since I didn't want to play too long with blends over paths (that would actually be the best way with less work to get a more complex gullioche pattern) I created a curve from top to right 3. I used this curve for the margin as it's set on the image 4. Did the blending and adjusted number of steps 5. visually adjusted each ellipse's size to go to the margin (YOu wouldn't have to do this if using paths) 6. copied and mirrored the result For the special effect I also put a gradient in the back (blueish colour from centre to the outside)

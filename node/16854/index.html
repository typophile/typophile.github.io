Saritza: When I wrote 'font formats' I was referring to the trio: TrueType, Type1 and OpenType. Notice that I didn't make any remarks to PC or Mac (although the font files might not be interchangeable between both platforms, in some cases). So, what I would suggest is: decide which font format you want/can use (TT, T1 or OT), and then generate the font files, in that format, for both platforms. I believe that, once you have the same font (same format, but not necessarily the same files) installed in both platforms, PC and Mac, InDesign will not alert that the fonts are missing. Notes: 1. If you decide to go OT, the same files can be used in both platforms. 2. If you decide to go TT, you just need to generate the files for the PC (extension .ttf). These can then be used on the Mac (only OS X). 3. If you decide to go T1, that's probably not the best choice, as the format has been phased out » Read about it Miguel

What's the format of the fonts you're opening? If it's OpenType (either TT or CFF), then the best way to change the fonts' name is: 1. In FontLab's preferences, under "Opening OpenType & TrueType" select "Read all OpenType name records" 2. In Font Info window, under "Names and Copyright" edit the entries in the panel "Additional OpenType names" 3. In FontLab's preferences, under "Generating OpenType & TrueType" select "Export only OpenType name records - ignore default names" 4. Generate the OpenType font (TT or CFF)

The Image Upload function usually works for most people if the file size is less than 130K. As Jared says, this restrictive size limitation is a known glitch that the Punchcut guys have been working to resolve.

If your file size is less than 130K and it still doesn’t work, then you can use the html <img> tag as Jared suggests.

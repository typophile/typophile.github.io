I agree with tylor. Make that logo about 1/4" and see what you get. If you absolutely want all those lines, then I would consider two versions with a more simple version for certain uses. If you do that, be sure and address in your usage guide precisely when each version gets used. Better yet, I would simplify it to work in all circumstances or you will wind up with the wrong one all the time. It is just human nature for clients to get it wrong.

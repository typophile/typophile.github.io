2005-05-28 19:38:29 • theorosendorf

Listed below are a few usability websites out of my bookmarks. Hope you find them useful.

Usability Expert Jakob Nielsen’s website: http://www.useit.com/

Usability Expert Bruce Tognazzini’s website: http://www.asktog.com/

Outdated but nevertheless comprehensive database of some very useful websites on user interface design and information architecture: http://www.usableweb.com/

An association for Usability professionals worldwide. You’ll need membership but I say its worth it. http://www.upassoc.org/

Resource for techniques on usability testing and heuristic evaluation: http://jthom.best.vwh.net/usability/

An HCI (human computer interaction) portal: http://www.hcibib.org/

Resources for user interface design http://gui-designers.co.uk/resources/resources.htm

A very useful (and well designed too!) webzine on UI design: http://www.uigarden.net/english/

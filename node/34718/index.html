Fontographer could always easily import Illustrator files as long as they were measured correctly, each letter being the same height. A good easy intro on how to do this is here Chank was one of the original guys who introduced fonts to the Internet world and is still working. There are of course many other places to look for info. If I were you I would, if you haven't already, download from FontLab the manuals and or demo programs for Fontographer or FontLab or better yet, for the beginner, TypeTool 3, also from FontLab, an inexpensive yet very powerful program. Without seeing an example of the font you are attempting to create it is difficult to understand where the problem is but I suspect that the outlines of the glyphs are very complex, which can cause problems. If you can, put up a gif or pdf file of the work you are doing so readers can get an idea of what is going on. Not familiar with Font Creator 5. And given the tools available in the font creation programs I use, TypeTool 3, Fontog, FontLab Studio I don't see why you couldn't create the fonts directly in those programs. But for clarity the contributors to Typophle probably need to see something. Good luck Jordan

2007-03-27 14:57:14 • dan_reynolds
Basically, the prefix shows the weight and the suffix shows the character width. Wikipedia explains it : http://en.wikipedia.org/wiki/Univers
2007-03-27 14:59:45 • Andreas Krautwald
Weight: 3… = Thin weight 4… = Light weight 5… = Normal weight 6… = Bold weight 7… = Black weight 8… = Extra Black weight Width: …3 = Expanded width …5 = Normal width …6 = Normal Oblique width …7 = Condensed width …8 = Condensed Oblique width …9 = Ultra Condensed width
2007-03-27 15:01:14 • dan_reynolds
So: 39 is Thin Ultra Condensed 45 is Light 46 is Light Oblique 47 is Light Condensed 48 is Light Oblque Condensed 49 is Light Ultra Condensed 53 is Expanded 55 is Roman 56 is Oblique 57 is Condensed 58 is Condensed Oblique 59 is Ultra Condensed 63 is Bold Expanded 65 is Bold 66 is Bold Oblique 67 is Bold Condensed 68 is Bold Condensed Oblique 69 is Bold Ultra Condensed etc… Get it?

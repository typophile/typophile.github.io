I don't think it's Neutraface, because the counter of the R is more normally proportioned than Neutraface, which has a very large counter. Neutraface also has pointed apexes (apices?) on the M, which this sample doesn't.

Granby is one font I can find that has the high-crotch M, and that style R (which Gill Sans doesn't have). Nobel, which also has that M, also has a different R. TF Forever, is also a possibility, but may be a bit more condensed.

I will stick with Granby as my guess, since this image is rather hard to see clearly.

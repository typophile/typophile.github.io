Web fonts: small caps and ornaments

April 3, 2010 • Frode Bo Helland

The “font-variant: small-caps” property doesn’t actually map to the small caps, instead the browser scales the uppercase. The obvious solution for web fonts is keeping the small caps in a separate font file (sc instead of the lc). This causes a problem: someone using a browser that doesn’t support @font-face will see lowercase characters where you intended uppercase characters. How would you solve it?

Another issue: I’m thinking about building a font with a set of ornaments for a web site project. A less updated user might end up with a system font instead, leaving the ornaments useless. One option might be using Wingdings or Webdings as fallbacks, but that restricts me to their character set. I’m not really sure how “web safe” those are anyway. Do I have other options?

Btw: This article seems to indicate the glyph order of Wingdings is patented by Microsoft.
2010-04-03 17:34:48 • Frode Bo Helland

Frank,

the obvious solution is to use a webfont that includes smallcap glyphs in both uppercase and lowercase positions.

Also, please note that the [[http://dev.w3.org/csswg/css3-fonts/#small-caps|CSS3 Fonts proposal]] introduces the font-variant-caps: small-caps and font-variant-caps: all-small-caps properties to access OpenType Layout smallcaps, and also modifies the description of font-variant: small-caps to try OpenType Layout smallcaps first, and only apply uppercase downscaling as a fallback. Of course, it will take considerable time before this gets introduced.

Regards, Adam

Adam: ..There are no Unicode small caps ..

Frode: FYI, there are a number of small caps characters defined in Unicode, but these describe the form, rather than the function. The function is for phonetic transliteration of certain speech forms, and these are not to be confused with “regular characters in a small caps form”.

Unicode does not define any font appearance at all – italics, bold, underline, outline, and so on. It has the advantage that this, combined with a Small Caps feature, will copy a small caps text from a PDF in its original ‘lowercase’ form. If the font has a c2sc (Caps to Small Caps) feature, the original case of the text will be preserved.

2010-02-01 18:23:22 • Mark Simonson

You’re right Mark. In the metal typeface (shown in Mac McGrew’s book), Mandate is shown as having a character that is an extender that can be added to the end of the letters. That was not included in the set of glyphs for the font.

The metal type was designed in 1934 by Robert H. Middleton. So it could have been used in early Monopoly sets.

The Mandate look was pretty long running, apparently from 1937 to 2008. In 2008, Hasbro standardized the game internationally, using a computer graphic image of Mr. Monopoly (formerly Rich Uncle Pennybags) on the cards - and using the same image on several similar cards, instead of a different image on each card.

I had not formerly known of this, and am now deeply shocked and distressed. Yes, even a long-running classic must be modernized, but it ought to be an improvement when it is done.

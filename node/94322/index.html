The download location for Dieter’s version of Prisma (which was a memorial to Rudolph Koch) is at: [[http://moorstation.org/koch_memorial/dl/dl_02.htm|Steffmann Prisma]]

Like much of his work, that one is probably a clone with many added characters/glyphs. Might have been based on any of quite a few versions of this 5-line Prisma (e.g., Opti, Williams) All have that question mark that’s different than the OP’s sample.

Is it possible to get an identification on the font that says "The industry standard for over 15 years?"

Joseph, the odds are great that this is a custom (home made) font, because this type of font is so easy and cheap to produce. There is even a website (Fontifier) that will do it for free with your own printed letter samples.

However, there are a lot of these kinds of fonts already available, or in circulation. If you have the patience to scan through about 500 samples, see my Script Font ID Guide, Part 11. Those with "HW" in the names are from the ClickArt Handwritten Fonts CD (no longer being sold), but there is an HW Alias List that tells you the re-named versions that you might be able to find on the web.

I'm sorry, but I don't have time to do it myself right now.

My advice in cases like this is to find the closest match and go with that, because you may never be able to find the exact font. It's not worth the effort, in my opinion.

I'm trying to figure out how to type a Hungarian character called an ogronek that resembles an "O" but has a double accent over it. Similar to an umlaut, I guess, but has the double accent rather than a double period over it.

(This is on a Mac, by the way.)
for mac - type: "option" "u" then "o". The "option u" entry establishes that the umlaut will be used; the "o" entry establishes that the umlaut will be placed over an "o".
for windows (incidently) - type: alt 0246 (using the numeric keypad at the right of your keyboard for the numerals.)

The Hungarian umlaut is a part of the standard
Macintosh character set and is accessed through the
shift-option-g keystroke.

None of the composite characters with this accent are
part of a standard font, however.

You can set the 'o' followed by the accent and then
kern the accent back over the o. If you are using Quark
with the Kern-Track Editor xtension loaded, then you
can build your own custom kern pair that will alleviate
the tedium of manually kerning each time.

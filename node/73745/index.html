There’s no fast or simple way to do it because most of the time that goes into distressed type is spent cleaning up damage done during distressing to make letters legible and fixing the errors introduced by autotracing. If you do the distressing by hand you’ll also have to spend time reshaping the letters because the distressing process tends to warp and stretch the paper. Distressing manually with drybrushing and sandpaper will average 10–20 minutes per character. If you use Photoshop, the Mr. Retro Machine Wash filters, do cleanup with a Wacom Tablet, and autotrace with ScanFont you can get this process down to about five minutes of work per character. I still have all this stuff set up from when I did Black Monday if you’re interested in contracting this out.

September 20, 2007 • Krushie

'ello all I was curious as to what type gurus like you use as their main typesetting/layout program? I have been using Adobe's Indesign for like 4 years now and i'm pretty comfy and settled with it as it gives total control over type features. I cannot see any other software atm that matches it, even newest of Quarks, imho. And if anyone has made the upgrade to version CS3 of Indesign, is it that much of a revolution over CS2?, considering that i need to upgrade to an intel-based Mac for the program to run smoothly or whatever...
Since I work on InDesign, I'm not going to give my subjective opinions. However, I can say that version CS3 is basically the same speed as CS2 on the same machine. The Intel Macs do run faster than PPC Macs, so it appears as if InDesign CS3 runs "smoother" on them, but you won't see much difference on the same machine. The new features are listed at http://www.adobe.com/products/indesign/upgrade/ . Some text-related details that aren't called out are: CS3 handles OpenType mark positioning and justification of marks better than CS2 and composed Unicode works better -- these help Vietnamese and Thai. We also improved our handling of some obscure polytonic Greek details.

For those who haven't upgraded from InDesign CS2 to CS3, the major new feature -- IMO -- is the Table and Cell Styles. They're still not perfect, but they add a whole new layer to production of table-heavy documents. For documents that I need to keep in CS2 format for some reason, I often build all the tables in CS3, then break the links to the styles and export the document to INX, then re-import into CS2: perfect tables in much less time. CS3 runs peachy on my PPC G5, btw. In my experience, it's CS2 on Intel Macs that is the "problem child."

Helvetica World is only available in those four styles.

If you buy the OpenType “Com” versions of the regular Helvetica, which has a more styles, they include support for 48 European languages (basically everything used in Europe except Greek or the languages that use Cyrillic… ALL CE languages are supported).

You can see which styles have OT format fonts available by looking at the font format icons on each font display: there is one for OT that uses the familiar round O OpenType icon.

Here is a list of those languages.

Neue Helvetica has OpenType “Com” fonts as well. Or you can by “Pro” versions of Neue Helvetica. These are a little cheaper because they support less languages (just the Pro languages… <a href=http://www.linotype.com/16453/33languagessupportedinopentypeprofonts-doc.html">here is a list of them… it is shorter than Com but it still includes all of the CE languages).

Note that there is a package of all 51 Neue Helvetica fonts available in OT format for $749. These are just “Pro” fonts, though. Not “Com” fonts.

Unfortuantely the only OT Helveticas that have all the CE fonts plus Greek and Cyrillic in them are the four Helvetica World styles. Sorry :(

FYI – Kaufmann Bold is the very first typeface I used. It was for a business card that was a class assignment in Print Shop during Jr. High (7th grade).

We had to learn the California Job Case, set the card using a composing stick, lock it up using furniture, set the pins to hold the card stock in place, then print on a hand press.

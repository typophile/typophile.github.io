2004-03-22 15:31:31 • Stephen Coles
Coles, going PC? You must really be in a pinch. Here's what you do (in Fontlab) 1.) Open both files. 2.) Select the characters from from the file whose sidebearings you want to copy. 3.) Then Edit > Copy 4.) Switch to the open file to which you'd like to apply the sidebearings and select the same characters (this is key). 5.) Then Edit > Paste Special 6.) Select the left and right sidebearing options and hit ok. This should do the trick. There are others ways to do this but this is the fastest and smoothest.

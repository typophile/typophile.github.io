Why not mess with the point size? As long as it's within reading range it's fine. Think of fractional increments though.

Frankly I don't have much experience copyfitting large amounts of text in tight spaces, but in terms of typeface integrity I'd say that tracking is the worst thing you can do. Second worse is horizontal scaling. Changing the word-space is not too bad. Same for leading. Both of those can actually take more abuse than most designers realize.

A good type user can conceivably have a better appreciation for the balance of economy versus readability than some type designers. So if you have reason to believe that the spacing of the font is too lose for example (like in Mrs Eaves), apply negative tracking (just keep an eye on "boundary" glyphs, like the lc "r" - you might need to add some custom kerning pairs). Helvetica for example is too tight for text (assuming you have to gall to use it for text anyway).

Two things I would add:
1) Watch the paragraph breaks for places you can save entire lines.
2) Try to use a MM font with a width axis.

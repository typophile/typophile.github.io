[[Indicies]] : [[Typefaces]] : Neue Helvetica

In 1983, Neue Helvetica was released as an updated version of [[Linotype]]’s [[Helvetica]]. It includes more weights and styles, making the classic type family more consistant.

Helvetica is still one of the most widley used typefaces of all time; although it is loved by many, it is hated by some. There is no shortage of commentary on the typeface, and there has even been a book published by [[Lars Muller Publishers]] entitled “Helvetica: Homage to a Typeface” dedicated to the famous type family.

Linotype’s website includes an overview of all of the original Helvetica font weights, and all of the Neue Helvetica font weights as well.

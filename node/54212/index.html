Thoughts on the logo concept (Pingstrom) ?

January 30, 2009 • apankrat

I'm putting my thoughts together for the logo for a software product called "Pingstrom". The name is play off the [[http://en.wikipedia.org/wiki/Angstrom|Ångström]] word, which is metric unit equal to 1/10 of a nanometer. The software deals with (computer) network measurement, which primarily revolves around a technique called 'pinging' - repeatedly sending small pieces of data to the target computer and then expecting the target to return this data back. Think submarine sonar pings for a more conventional analogy. In any case, I'd like to get a general idea if either of the following concepts is worth pursuing. Serifed type is used to try and capture the precision aspect of the software, which is implied by the angstrom reference. (A) This on is a straight forward reference to Ångström. It also maps nicely to a small application icon, which is obviously pretty important. [img:sites/default/files/old-images/pingstrom_5611.png] (B) This one tries to capture the nature of the actual pinging process. [img:sites/default/files/old-images/pingstrom-2_4985.png] . Thoughts or comments ? Thanks.

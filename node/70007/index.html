Rendering experiment

April 30, 2010 • Frode Bo Helland

The image above shows a bold sans serif on the left, and a grid fitted version of the same typeface on the right. The latter is adjusted so that every stem lands on a whole pixel when displayed at 8px size. (One pixel = 125 units)

Below, I’ve attached an image showing the rendering in OSX and Windows.

At these sizes you have very little control over spacing and the shapes will look crude if the web user for some reason decides to resize the text. However, I can imagine designing for a bigger size might work quite well. You’ll get improved rendering at the intended size, and with hinting you won’t lose much compared to other fonts if the user zooms in or out.

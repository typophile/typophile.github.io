It is a pretty distressed typeface...so maybe that is part of the problem. - That's definitely the problem. I think you should try optimizing the glyphs - you've obviously got WAY to many nodes per glyph. If it's a pretty distressed face anyway a little (or a lot) simplification won't make that much difference. You can do a global optimize to start, then open each glyph and remove even more nodes manually until you get a manageable file size. Nick Cooke

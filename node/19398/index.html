I wonder if any one can help me. I have a bit of a typographic design problem. I have to lay out a design that is going to be carved onto Portland stone by a stonemason. Hope the following makes sense. There are blocks of portland stone. Please see image.

The panels are marked Type A1. Names are going to be carved into the stone. The first Type A1 column from the right is the first column to have names on. These people are more important. These will have two names on each panel within the column. The biggest sized font. The second Type A1 column will have three names on each panel within the column. The medium sized font. The third Type A1 column will have five names on each panel within the column. The smallest sized font. This isn’t the way I would have done it. I would have had the font size all the same!! But no… As you can see the layout of the stone panels makes a strong grid. I have decided because the strong grid and its a modern building that the text should be right aligned. Fine. Names might have to be added later on so this means that the names should be aligned to the top of the panel (theres a margin). OK. But here is my biggest problem. Is it right to align the fonts of each of the three different type sizes to the top of the font rather than the baseline? If I align to the baseline there will be more of a gap at the top of the panel with the smallest font. What do you think I should do? Hope this all makes sense. This is giving me a headache!

My take would be to align the baselines because they are the most constant element. I would also use the baseline grid set by the largest point size to use for the next two columns. e.g. (these are hypothetical sizes) 1st 24/30, 2nd 18/36, 3rd 12/42 [point size/leading].

REKORD is slowly growing up...

July 5, 2010 • hn2o

I’ve done a lot of work on [[http://typophile.com/node/40757|Rekord]] recently, so I thought I’d share it with you. Since it has developed so much, I thought I’d start a new thread in the sans-serif section. The new Rekord will, if all goes well, be a six-weight family with italics, and be available later this year (after I’ve finished [[http://vllg.com/Incubator/Tram#panel=poster|Tram]], which is more than overdue).

Also, Rekord is the new typeface used by [[http://rpho.nl/|Rotterdam’s Philharmonic Orchestra]], which I’m very happy about. More can be seen here: [[http://www.gergievfestival.nl/|Gergiev Festival]]

There’s still a lot to do and refine, but so far I’m happy with the development… and curious to hear what you think.

Cheers, Hanno

I had to do this a few times a year or so ago. I would place the equation in an empty Word document (legal-sized landscape worked well) and then resize that to about 3x the print size (Word still treats it all as vector data, so the resize is no problem). Export that to PDF (used Adobe PDF exporter for Word, but PDFCreator works well, too). Save a copy of the Word doc in case you need to make edits/corrections later, and then just place the PDF into InDesign as you would any other image. Since it's oversized, it'll look perfectly fine when you scale it down to regular size.

I’ve never had problems with Scala and don’t use either Fontbook or Suitcase (I use Font Reserve under Windows). However, I’ve had all sorts of mysterious problems with fonts that either cannot be deactivated or are supposedly activated but when you select it in an application, what you get is actually Arial or something. Never thought to ask the supplier for help, but here’s what works for me:

  1. Copy the font files to a safe place.
  2. Delete the offending font files (or shortcuts to the actual files) from c:\windows\fonts.
  3. Re-start Font Reserve, add the now missing fonts again to the font database.
  4. Activate the newly-added fonts.

Have no idea if this will work on a Mac, but it’s worth a try.

Is it possible to move all glyphs (a one go) a few spaces to the right.

July 17, 2014 • djnippa

I need to move all glyphs 8 points to the right for improved spacing. Is there any easy way of doing this across the whole font at once, without having to move each one individually?

Sorted it. It was in TOOLS > ACTION SET > All glyphs in Font (drop menu) > Available Actions > Contours > Shift > Click the -> arrow to add to the right side menu > Horizontal Shift - Type 8 > Vertical Shift -Type 0.

Then RUN

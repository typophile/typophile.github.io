Tim, you’re welcome to do it your way; however, The Chicago Manual of Style recommends: “Common Latin abbreviations such as ‘etc,’ ‘eg,’ ‘ie,’ are set in roman type without punctuation.” This includes c for circa; c. is commonly used as an abbreviation for century. So, “circa 16th century” would translate as “c 16th c.” Or not…

In CMS, possibly, however I would use punctuation for etc. e.g. i.e., the abbreviation for century commonly used in UK is C. The Oxford house style and Hart’s Rules suggest any abbreviation that has the first and last letter needn’t be punctuated whereas any other lowercase abbreviation should be.

c. 16th C (or c. C16) is how I would set circa 16th century.

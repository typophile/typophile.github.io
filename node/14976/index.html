As always, I should preface this by mentioning I’m blundering my way through this learning process and thus will likely sound like an idiot in my attempts to explain my current query.

I’ve been given the opportunity to take a close look at a pre-release version of an upcoming font (let’s call it PRERELEASE) and one of the tasks still to be done is re-ordering the glyph index order. What I’ve gathered is that the glyph index order designates the order glyphs are listed in InDesign’s glyph palette (I’m sure index numbers serve other purposes as well, so I suppose that’s part of my query here).

What I began to be curious about was how to set up pop-up menus in InDesign’s glyph palette; that is, to set up the small arrow in the bottom-right corner so that when you mouse-over the glyph a menu pops up with the alternates for that glyph. For instance, mousing-over lowercase e pops up a menu with e, smallcap e, superscript e and scientific inferior e, which is great. However, the PRERELEASE font has over 10 alternate ampersands, and half a dozen alternate Qs, yet only the small cap versions of each of these is showing up in the pop-up menu. What I’m wondering, then, is where should I place the alternate ampersands and Qs, etc.?

I should mention that I’ve only re-ordered about 200 of the near 2000 glyphs in the font, but I thought I’d post this query now rather than get too far ahead of myself.

If this makes any sense at all to anyone, I’d appreciate any insight you might have to offer.

Thanks in advance…

The glyph index order may best be thought of as the physical order of the glyphs in the font file. That’s the order the glyphs are displayed in InDesign’s glyph palette.

Ordering has no effect at all on what alternates are shown for a given glyph. This is instead controlled by OpenType layout features. The ‘aalt’ feature in particular fives “all alternates” of a given glyph. I’m not sure if InDesign will also show alternates that are not in ‘aalt’ - perhaps.

Simple recipe for reordering: right-click on the Font Window, choose Sort Glyphs > Names, then once more, but this time choose Sort Glyphs > Unicode.

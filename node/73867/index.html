OT - changing last glyph in word

September 5, 2010 • tourdeforce

Can someone help me and tell me is it possible to have OT feature that will change last glyph in word?

For example: America into AmericA.

Thanks

@LETTERS = [A-Z a-z]; feature calt { ignore sub a' @LETTERS; sub a' by A; } calt; feature fina { sub a by A; } fina;

‘fina’ feature is script specific (Arabic and Syriac AFAIK), and requires certain knowledge about the respective script in the engine itself, since final position is not necessary at the end of the word and can be triggered by certain letter sequences even in the middle of the word. The same is true for “init”, “medi”, “isol” and other similar features. Actually, I don’t think any of those features will work with Latin script.
2010-09-05 19:06:39 • Khaled Hosny

Khaled,

isol, init, medi and fina work in Latin fonts, e.g. in InDesign CS4/CS5, using the “Positional Forms” submenu in the OpenType flyout menu. This allows users of Latin fonts to have a fine-grain control over which forms should appear (typically, in calt, you’d have code to deal with all forms at the same time).

You’re right, though: in calt, the font designer needs to specify the context, i.e. identify the end of a word or a line. In the positional forms features, it is the application (i.e. the OT Layout engine) that “knows” where to apply those features if the user enables them. This is why in my example, the calt feature uses a contextual substitution while the fina feature uses a simple substitution.

I have a trouble building a pixel font using fontographer 4.1 PC.. 2 letters are not good.. I’ve tried to check them in many ways (redesign of shape, remove overlap, cleaning up…)

The block size is 75X75 I want the font to fit 8px in apps. like photoshop or flash.

Setup : Ascent =600 Descent = 150 Em =600 (since 8x75=600)

The result is the same with Ascent+Descent=Em ie: Ascent =450 Descent = 150 Em =600

The only way I get it to work (with block still 75x75) is with that setup : Ascent =850 Descent = 150 Em =1000 … but the font fit 14px and not 8 like I wish..

some help would be great.. i’ve been wandering hours and hours trying setups..for only an “/” and a “@”…

Do not include hints in the font. This can cause problems in pixel fonts like the one you are having.
2005-12-28 20:10:56 • Mark Simonson
I think you have to make the ascenders 250 and the descenders 750 em units. Put gridspacing (preferences) to 125en units, and align points to grid. Now your characters are 8 pixels high. This has to work. Succes. Pieter

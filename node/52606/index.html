If I understand what you are asking, you are looking for a distressed face, but not one that is overly effected. I would not look for a distressed slab. I would choose an appropriate slab, set it in InDesign or Illustrator and then distress it in Photoshop. This one looks like it was distressed inside Photoshop. Note that the L’s and the A’s are differently distressed. There are many tutorials online that can help you get started. One technique is to use a photograph, destroy it through any variety of means, levels, adjustments, filters, threshold settings, until you whittle it down into an abstract pattern and then apply the pattern to the type. Another which was recently discussed in the tutorial section of The Font Feed recommends using filters on the type itself to distress it.

There is always Yves Peters' Worn, Weathered Stamped Look that can be used on any typeface. This will get you part of the way there anyway.

You could also use the Machine Wash Filters from Mister Retro.

For those of you naysayers who think it’s lazy to start with a filter I’ll just add that these are great ways to start for anyone. For some they are good enough as they are, but others are always welcome to continue the tweakage.

Note: I think Yves tutorial above had a rough draft version here on Typophile first.

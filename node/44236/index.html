@David: It’d be sweet if one could paste a glyph into the metric window’s glyph cells, as opposed to the text line, but that’d cause work somewhere.

Actually, you can, though the way it works is not very obvious (big surprise): Switch to the “text mode” in the Metrics window. Then you can copy glyphs from the Font window and paste them into the Metrics glyph display area (as opposed to the text line). You switch to “text mode” in the Metrics window using the button just before the “preview mode” button. Unfortunately, you need to switch back to “metrics mode” or “kerning mode” to make adjustments, so it’s not quite as seamless as the Fontographer implementation. Although it does allow you to insert a glyph when you paste, rather than replacing.

2008-04-15 15:59:34 • Mark Simonson
There is another way that is often simpler, and doesn’t require changing modes: Select a glyph (or range of glyphs) in the Font window and drag it (them) to the glyph area of the Metrics window. In this case, you can only insert, not replace. An insertion point will appear in the window to let you know where the glyph(s) will be inserted.

1939 Studebaker Clock Face

March 18, 2015 • Misfitchild

This is my recreation of the font from a 1939 Studebaker clocker face. The original image is very distorted, though, so I was wondering if anyone recognized this font or knew what font Studebaker might have used. Any help is so appreciated. Thanks!

I doubt this was ever a typeface. Seems more likely to be a custom set of numbers inspired by something like Paul Renner’s Futura Black (1929) or similar typefaces:

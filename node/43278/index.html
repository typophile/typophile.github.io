That doesn't work everywhere in Windows Si (unless there's some spiffy new feature in Vista) only in Word and WordPad. Any monospaced fonts that are used in terminal emulators are likely to contain these characters. If you are getting just question marks, that would indicate that you are entering Unicode 16bit codes into an 8bit environment. If you are getting question marks inside boxes, that probably indicates that you are using a font that does not contain glyphs for these characters.

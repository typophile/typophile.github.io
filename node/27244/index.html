I suspect that Günter Gerhard Lange supervised the digitization of Standard for the PostScript version of Akzidenz Grotesk, as he was only born in 1921, while Standard/Akzidenz was introduced by Berthold in their 1896 catalog. I've found no information about the original designer. More information about Günter Gerhard Lange can be found at MyFonts.com: http://www.myfonts.com/Person205.html Information about the H.

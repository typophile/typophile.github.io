2007-07-26 19:44:30 • Bert Vanderveen

If I understand correctly, you want to place a big text file and automatically have it flow from page to page without having to keep selecting the out arrow, right?

If so, what you do is, on the master page, let’s say you have 2 columns. If you shift+select the out anchor point (lower right) and then click inside the next empty column, you have now strung them together.

Now, when you’re in your page view, you should be able to place a text file in the first column and it will automatically fill all the pages.

I slightly misspoke earlier about linking and auto text flow.

You do link text frames together by clicking the out box and then inside the next frame.

But when in your page view, if you place a file, you hold down on the shift key WHILE placing the text. Not like I said earlier.

This should automatically fill all the text boxes and create new pages till the doc is completely placed.

But again, you have to link the text boxes together on the master page.

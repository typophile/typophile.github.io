body: "<strong>[[Indices]] : [[Books]] : Indie Fonts 2</strong>\r\n\r\n[img:sites/default/files/old-images/IndieFonts2Book_4995.jpg]\r\n\r\n<em>Indie
  Fonts 2</em>, a companion to the award winning <em>[[Indie Fonts]]</em>, brings
  together an additional 19 independent font foundries from around the world. This
  book is an invaluable resource of over 1600 fonts for designers, art directors and
  typographers.\r\n\r\nThis book is divided up by foundry so that the user can get
  a feel for the work produced by each group. This also makes it convenient to order
  fonts directly from their source. There is a visual font index provided in the front
  for quick scanning and a fully cross referenced index in the back containing font
  names, designers, and original designers.\r\n\r\nContributing foundries include
  [[Atomic Media]], [[Feliciano]], [[Gal\xE1pagos]], [[Holland Fonts]], [[Identikal]],

Word must think it’s a symbol font. To prevent illegible font names for symbol fonts, it uses Arial (or something else) – check “Symbol”, “WingDings” etc., these should display the same.

If all you did was convert the font, its encoding probably was SymbolEncoding or some custom job. Consult your convert tools to change it.

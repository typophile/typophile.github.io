Hi Ramiro,

as Jason has suggested, you can perform a site-wide search via your favourite search engine.

Grant Hutchinson has made a little favelet, for searching Typophile via Google:

javascript:void(q=prompt(‘Search%20Typophile:’,"));if(q)void(location.href=‘http://google.com/search?q=site:typophile.com'+'%20'+escape(q))

Copy the code into your address bar and bookmark it. (In some browsers, that doesn’t work: You have to create some random bookmark first, and then edit it. Set its name to something like ‘Search Typophile’ and copy the code into its adress/location/URL field)

Joe, Zara, …: As long as Typophile’s internal search doesn’t work and can’t easily be fixed/replaced, I’d suggest removing the search box from the template. It doesn’t offer anything but a lot of confusion. F

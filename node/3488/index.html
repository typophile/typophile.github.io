I'm affraid there isn't one all-encompassing list. Different type
families have different weight names, and weights with the
identical names might come on a different spot in the "weight
scale" across type families. For example the "Book" weight is
usually lighter than the "Regular" or "Normal" weight but bolder
in FF Meta. Same goes for "Heavy" and "Semibold".

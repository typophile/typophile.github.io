2005-06-02 00:03:11 • Stephen Coles
This typeface has always interested me, so I took some trouble with this answer. The type the New Yorker uses is called 'Irvin' and you can see it on Page 184 of McGrew's book on "American Metal Typefaces of the 20th Century". I know of three digital versions (there were at least two film type versions): FF New Yorker, which has been mentioned; 'Manhattan' which was sold by Franklin Type Founders (now available only at Atomic Type); and David Rakowski's digitization called 'Upper West Side' (free at typOasis). The last is the only one of the three that faithfully recreates the true 'Q' from Irvin (in version 1.3 which is at typOasis. Rakowski's earlier version also had a bogus Q). I have made and posted an image with the posted sample for this thread, shown with samples of each of the three digital versions. See http://bowfinprintworks.com/FaithFordType.html (Yves, when is image posting coming back? Soon, I hope!) Based on the W, I would say the Faith Ford card uses Manhattan. - Mike Yanega

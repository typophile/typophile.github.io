I have never encountered a program that didn't support positive kerning if it supported kerning at all. Positive kerning has been used in many early T1 fonts. Monotype's Bembo and Bell are examples, as well as all the Berthold PS fonts. Berthold fonts are unusable without kerning because the Berthold spacing system depends on positive kerns between all possible o-o shape strings.

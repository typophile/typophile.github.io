Hi Tommy I think you'll find that one (and maybe others like it) in Nicolette Gray's Nineteenth Century Ornamented Type Faces, although you might have to hit up your local library for a copy. Offhand I couldn't say which foundry was responsible; Caslon, Figgins, Thorne, Thorowgood, Stephenson Blake and others all showed 19th C display types that were ahead of their time. If you're curious about it as a typo genre, there are excellent essays by Emily King (at typotheque) and Keith Tam (at typeculture) that go into a lot more detail about the 19th C.

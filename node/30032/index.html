OT language tags: IDCS2 support?!

December 11, 2006 • Rhythmus.be

Now this is really unbelievable! InDesign CS2 does not support OpenType language tags…

What’s all the fuss about “saving the i-dot in Turkish”, the Servo-Bulgarian ghe’s, excepting ligatures in German, Moldavo-Romanian S’s with comma-accents, Polish diacritics, and so on and so on, if the industry’s major typesetting program doesn’t support language system tags?

How on earth is it possible Adobe ships full-featured OpenType fonts that claim to have multilingual support, but at the same time doesn’t manage to implement this more than indispensable OT spec in their state-of-the-art layout program, and so for over a decade since the OT format was defined?

Lack of support for LookupTypes is one thing—- but this?

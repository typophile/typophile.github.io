  1. A lot of times, things that align metrically can actually not appear to align visually. I suspect that this was due to the rounded edge and the color of that box. Personally, I’m leaning more toward the visually aligned solution; so I would reduce the margin a little bit, and then use H&J to compensate for the lost words. Franklin Gothic’s counters are open, too. I think they could withstand some wordspacing adjustments.

  2. I think that the rule on widow is not hard-and-fast. In this case, it is fine, especially since harrasment is quite long of a word. Speaking about which, the Other woman’s issues section got a short wives. Now that’s a widow allright.

  3. I would use the existing Triplex serif, set at visually the same x-height as Franklin Gothic. It is distinctive enough to be noticeable among the line of text, yet would still provide some continuity with the context of the book.

  4. I don’t think so. One or two visual marker to distinguish between one entry and another (in this case, resources set in Bold and leading between entries) are usually enough.

Best of luck!

2006-08-19 15:27:43 • noftus

It seems you have several ways of treating lists:

  1. outdented first line, ranged left, ragged right
  2. indented first line, justified
  3. ranged left, ragged right, line spaced The consistency seems to come from having the first word(s) in bold. My taste would be to use the bold as the indicator and range left with ragged right on a slightly narrower measure (taken from left and right), not being justified you have greater flexibilty to takeover words to reduce the incidence of widows and not being indented means that the shorter definitions would decrease the stepping effect on the left. You might feel that it needs a little extra linespacing after each entry and if you can do this without adding too many pages it is worth considering. Tim

2006-08-22 03:54:36 • pattyfab

  1. Indents should be equal to the leading.

  2. A word on a line by itself is NOT a widow. It is a widow ony when it jumps to the next page. You should NEVER track type in text. Adjust your word (not letter) spacing if necessary, but it always looks bad if you if you change your constants – size, leading, letter spacing – to fit text.

  3. Use Franklin Gothic for your URLs. Make them bold if you want them to stand out.

  4. There is no need to indent the resource section. You have already set off each entry with a blank line.

Etiquette...and type design...

September 6, 2005 • vaaaska

I recently presented a client with some initial concepts which were quite well received. However, we quickly got into the disucssion about 'appropriateness' regarding a particular typeface. It's an ultra-simple grid-based font. We agreed that it would probably be more appropriate if the edges of the elements were not sharp...instead they should all be rounded. And so, I'm in a position to build a font that is clearly based upon something else, yet I'm making some small global changes. I don't even own the face in question so there is no basis for me to copy their actual work...but I wonder if there is some etiquette regarding these kinds of things? Any tips? Thanks...
1. The client licenses the original foundries type properly, i.e. for the number of machines and locations as specified by the eula. 2. you customize it as the client wishes. 3. the client replaces the customized font into the same licensing "footprint" as the original font. 4. You don't license the font to anyone else unless they have the same deal as the first client, and it's not exclusive to that client, or you totally recreate the font from scratch, remaking all the contours, all spacing and tables as if it was new.

You say “small changes”, what does that mean? To me, the first thing is to look at the EULA. If you don’t own the font, ask the foundry about it. If you are just doing a few words and not an actual font and the result will not be taken as the original you are probably OK. If you are making an entire font for sale competing with the original, it should be clearly enough different to not be taken as a rip off. If it must look just like the original with a few minor changes, ask the designer to do it for you or give you permission to do it your self. ChrisL

I personally don't think you should share your source for inspiration. 1. The client could end up liking the source and wanting you to round the corners of the source and you'll be left with a typeface you've created and hope someone else wants it. 2. Or I mean how were you able to draw inspiration from it?

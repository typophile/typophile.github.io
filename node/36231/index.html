Try this: Export the document in InDesign Interchange Format (it’s under Export — a pop up). Next: open the resulting document. InDesign will rebuild it. (Quite a few corrupt ID-docs can be saved this way.)

If this does not work, try erasing font caches (use a tool like FontFinagler — I assume you’re on a Mac).

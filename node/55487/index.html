2009-03-06 15:18:57 • Thomas Levine

I remember there being a kind of “gaming” keyboard that let you move modules around for different configurations, but I think you were still limited to the basic staggered grid typewriter keyboard arrangement.

Most keyboards you can yank the key caps out of and rearrange them, but you still have that same problem of the staggered grid arrangement, and that many keyboards have the keys contoured differently depending on their row.

Then there is the [[http://www.artlebedev.com/everything/optimus/|Optimus Maximus]] keyboard with a tiny 48x48 pixel monitor in each key. It’s been coming soon since 2005, but the guy who invented it is also a type designer so I’ll cut him some slack.

But if you really want to move things around, perhaps the [[http://www.ergodex.com/mainpage.htm|Ergodex DX1 Input System]] is for you. Still not the steampunk typecase you’re thinking of, but you could easily prototype one with that… Well you might need a few of them, since each only comes with 25 keys. It looks more useful for developing custom controller layouts.

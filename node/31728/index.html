2007-02-23 23:21:13 • timd

I have one right here. It’s a nice idea, but like Tim says, not very useful in practice. The swatch binding works well for comparing paper or color swatches because you only need to see a small segment of each page. But type is different. You need to see more than a few characters to properly compare typefaces, which means comparing fewer at one time. So the fan format ends up being more of a gimmick than a truly helpful.

That said, the editor picked made some good typeface choices (though limited) and the organization is sensible.

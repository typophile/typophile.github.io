Superscript for chemical settings

May 21, 2010 • Frode Bo Helland

I just came across a superscripted minus in some material I’m designing for a client. The company is working with nano technology and the technical specifications can be rather advanced.

Example: 158 [superscript]-6[end superscript]

Are there standard character sets developed for writing chemical equations?

Technically there isn’t a standard set for superscripts but a good font with superscripts figures should also have superscript basic math symbols for these kind of basic math settings. If it has a superscript alphabet, it should also have basic punctuation. Every designer has to draw the line somewhere though and supporting mathematical/scientific typesetting can be a very slippery slope.

The Keyboard is a Tool Kit

June 21, 2004 • MaxK

The recent News and Events posting for Bruno Maag's talk Type is a design tool at St Bride Institute in London on July 13, states that "Changes in technology over the last 20 years have meant that type is now not only regarded as a carrier of information but a design tool. In particular, its importance within branding/corporate identity will be assessed and the practical issues in creating type for such an environment demonstrated." It's essentially a keyboard-like base which can accomodate interchangeable faces that use application-specific labels (and even keys). So, you can have a Photoshop keyboard and a Medal of Honor (a videogame) keyboard, etc. The shapes of the buttons themselves also vary, based on what the applications' keyboard usage techniques involve.

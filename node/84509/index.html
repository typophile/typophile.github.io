I'm on Windows 7 and use suitcase fusion 3 to manage my fonts and have had success with this method: Install fonts via Suitcase Fusion. When I need to update the font, I delete the current version from Suitcase. Close the program. Close FMCore.exe (via Task Manager) Create the new font. Reopen suitcase and reinstall the font. This is the best method I have found that doesn't require me restarting my computer.

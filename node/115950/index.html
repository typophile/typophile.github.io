I didn't find this one. If it's a font, it's well hidden from the casual searcher. If I had to make it from a font, rather than from scratch, I would start with Impact Bold Condensed http://www.myfonts.com/fonts/urw/impact/ and make the words as follows: _r_.... use n, shift the top curved portion of the interior contour up so horizontal stroke is about half as thick, cut off the bottom 60% of the leg _e_.... shift the top curved portion of the counter up so horizontal stroke is about half as thick, same procedure for bottom stroke, drop bottom of counter to thin horizontal stroke. _s_.... use o, same procedure as for e to make top and bottom strokes thinner, cut openings in sides and add horizontal stroke. _t_.... use U, delete rhs, same procedure as for e to make bottom stroke thinner, thin vertical stroke to match lower case width, add rhs crossbar _a_.... use o, same procedure as for e to thin top and bottom strokes, cut opening in lhs and add horizontal stroke, square rhs bottom corner. _u_.... use u, same procedure as for e to thin bottom stroke _r_.... same as first r _a_.... same as first a _n_.... use n, same procedure as for r, but don't cut the leg. _t_.... same as first t _h_.... use h, same procedure as for n _o_.... use o, same procedure as for e to thin top and bottom strokes. _t_.... same as first t _e_.... same as first e _l_.... same as t, no crossbar _c_.... use o, same procedure as for e to thin top and bottom strokes, cut opening in rhs _a_.... same as first a _n_.... same as first n _e_.... same as first e _t_.... same as first t Don

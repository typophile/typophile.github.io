2011-02-06 21:16:55 • John Nolan

Why don’t you want to use tables? They would work fine.

If you’re not using tables, and IF the tag lines fit on one line, a simple method is just type the tag line, tab over to the paragraph, and then insert an invisible “indent to here” character (Command-Backslash) at the start of the paragraph. That will cause subsequent lines of the paragraph to automatically left-align with the first one. But if your tag lines are more than one line deep, this method won’t work.

2011-02-06 22:36:59 • oldnick

There is another way. Create a text box for the body text and another for a tagline, then copy and paste the taglines text box inside the the body text text box. These are called anchored objects. Go to Object-> Anchored Objects -> Options and Customise the position to suit. Saving these settings as an object style means that they can be applied to all taglines. The increasing line length can be accomplished by adding in a right side indent. These can be saved a paragraph styles and applied to subsequent paragraphs

Drop me an email if you need more help, I know this may not be clear.

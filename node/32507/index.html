Help! I've set up two paragraph styles (headline and body text). In the initial text frame, everything looks fantastic... [img:sites/default/files/old-images/a_5210.png] When I flow text into a second frame, the body text loses it's vertical alignment to the baseline grid... [img:sites/default/files/old-images/b_4485.png] Funny thing is, when I create a headline at the beginning of the second frame, things line up again... [img:sites/default/files/old-images/c_6532.png] Help! Any clues to avoid the shift in vertical alignment? Thanks in advance! --new2me
In the paragraph style for the body text, make sure to select the option 'Align to Baseline Grid' (must be set to 'first line only' or 'align all lines'). Make sure to set the leading to the same amount as the baseline grid spacing. You can experiment with this is you wish to use a leading of 1/3 as well obviously, or any fraction. Just make sure to at least align the first line.

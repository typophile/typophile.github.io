The new ClearType type families were specifically designed for Longhorn (the next version of Windows). As the book explains, the designs take advantage of impressive new features of the version of the ClearType renderer that will ship with Longhorn: subpixel positioning and y-direction anti-aliasing. That said, they look pretty darn fine with the current version of the renderer: I've been using Gary Munch's Candara as my e-mail font for several months now, Luc's Consolas for all monospaced text, my own Constantia as default web font, and Jeremy's Corbel as a UI font.

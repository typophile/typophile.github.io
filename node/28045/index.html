2006-09-10 12:33:51 • Goran Soderstrom
I totally couldn’t resist this one, so therefor I did a font inspired by this logo. Details are not exactly the same and I had to lower the Caps height for it to work and also take some decisions on which way to go, round or more squareish shapes, since it was hard to keep the original letterforms. Here’s a small preview for those who is interested: [img:sites/default/files/old-images/upcoming_flieger_5405.gif] This font will have a large language support and a lot of OpenType-features, ligatures, swashes, alternate letters etc. Anyway, the font is an upcoming Fountain release, just polishing some final details right now. It will contain three weights, light, regular and bold. http://www.autodidakt.se/index.php?page=news http://www.fountain.nu/catalogue/fresh_upcoming.asp Thanks to Helena for showing it :)

This is a modern typeface. (modern as in style, like oldstyle, didone, etc.) Mostly used in the late 19th and early (i.e. first half) of the 20th century. I would not be surprised, if your book was from this time.

Surprisingly (or not so much if one considers the effect of cultural cycles) little of them have been digitalised.

You might want to checkout the “Old Standard” typeface by Alexey Kryukov http://www.thessalonica.org.ru/en/fonts.html which is a very nice rendering of that style! (Digitalised original print).

btw. I would not call the above example expanded. It’s the optical size that makes it look expanded, I guess. Which size is that annotation in in the original? 6pt? (Alexey’s digitalisation is base on something like 10pt originals).

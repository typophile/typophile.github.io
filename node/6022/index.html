At Hrant's request, I've scanned the specimen page for ATF Garamond from their 1923 specimen book. The 200dpi preview can be found here:

http://levien.com/type/atf_1923/p0018_sm.jpg

Apologies for the JPEG artifacts in that image. I tried Hrant's recipe for making a GIF, but for this particular scan the paper texture posterized into something resembling a skin rash, so no thanks.

The full-resolution, 2400, 37M image is attainable by removing the _sm from the above URL. By all means, download if you're really interested, but please don't hammer the server.

The scan is fairly raw; the grayscale axis is calibrated to sRGB (which is a bit lighter in gamma than most of the scans I've done so far), but there is no sharpening or other image processing. It would probably benefit from a modest sharpen.

For those who are merely curious, hopefully the following illustration will be of some interest. It shows all four sizes from the smaller lowercase a pattern plate, normalized to 400 pixels/em.



I'm toying with the idea of making these hi-res scans more generally available, perhaps on a DVD-R. Would there be interest in such a thing?

WILD GUESS: To gain access to the Adobe Substitution fonts, you can typeset the words you need with a similar typeface but don't embed the fonts into the PDF. Close the font. Open the file into acrobat. If it looks right, and you have Acrobat Pro, Go to pull-down: TOOLS>PRINT PRODUCTION>PDF OPTIMIZER. Under TRANSPARENCY, "turn all type to outlines." It will then ask you to save the file again. If this works or doesn't work could you report back? I'd be curious/surpised. Although, this doesn't look spindly enough to be the Adobe fonts.

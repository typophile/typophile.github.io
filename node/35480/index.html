2007-07-30 21:40:29 • jdusek

Many color printers use an emulation method instead of true Adobe PostScript. For the color prints, I would recommend you open the PDFs in PhotoShop and let it rasterize it as a Photoshop PDF. Save it then as a rasterized PhotoShop file but at a high enough resolution to get good output from the color printer you are using. Having done this, you will not need the fonts anymore since it will all be pixels. Find out the resolution your service bureau needs for the color printer first.

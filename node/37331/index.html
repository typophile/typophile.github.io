Type Supply News - September 2007

September 21, 2007 • Tal Leming

Hello Typophile,

I have a few announcements to make…

I have decided to open source a bunch of code that I have written for various type development tools. You can learn more at [[http://code.typesupply.com|code.typesupply.com]] You might find the [[http://code.typesupply.com/wiki/FontLabScripts|FontLab Scripts]] and the [[http://code.typesupply.com/wiki/FeatureProof|FeatureProof application]] to be particularly useful.

In other news, I launched a new Type Supply website at [[http://typesupply.com|typesupply.com]] and started a blog at [[http://talleming.com|talleming.com]].

That’s all for now. Have fun!

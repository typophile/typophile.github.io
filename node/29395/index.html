Inputting color swatch names in Indesign—bug or feature?

November 9, 2006 • blank

Is there some trick to being able to actually type a color’s Pantone number into the “Add Color Swatch” dialog box in Indesign? All I can ever type in is the first digit, from there I still have to click through the color swatches to find the match.

Indesign is great, except for the color pallet; which I feel was treated like an after thought, I mean having the color picker set in RGB mode? come on. Sorry off topic…

To answer your question, yes the ability to punch in a number and have it go to the right swatch is buggy. If you type all the numbers in really fast it will jump to it, if you mess it up and try with the mistaken numbers it won’t go to the correct swatch. I’ve found hitting one and letting it go to 100 then going back to the start of the numbers and trying again will yield good results.

OpenType - Is it possible to take it to far?

February 21, 2007 • rosem

Hey everyone,

I’m a huge fan of all the features OpenType has to offer (swashes, discretionary ligatures, ordinals, etc…) but is it possible to take it to far?

For me, when I use a pro font the first thing I want to do is turn on all of these features and restyle my text in all the available style sets, etc…

So the question I guess I have is if it’s ever a bad thing to use all of these features. I know there is the right tool for the right job - but I guess I don’t know when it’s appropriate to use them all. An example would be, is it right to set all my text with discretionary ligatures?

I need some advice :)

Thanks, Mike

Depends on the piece. It could be thought of like color. For some projects lots of psychedelic colors are appropriate, but for others simple black and white is right.
2007-02-21 00:54:49 • Stephen Coles

Along Stewf’s lines, think of opentype features as a toolbox. Some jobs require a few tools, some many. When you take your toolbox to work, you don’t know which you will need so you pack the whole kit to be prepared for all situations. A type designer might pack his opentype font toolkit with lots of features just to have them available for whatever job is needed. The craftsman who uses the tools should be skilled enough to use them wisely.

So, to sum up, the simple answer is “yes” it is possible to take things too far (at least, on most jobs).

Discretionary ligatures are off by default because they would be inappropriate in most jobs.

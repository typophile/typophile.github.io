The Emigre Fonts license does allow the user to alter the characters in a font as long as the user has purchased the appropriate user license and the altered fonts continue to be used in accordance with this license. In addition, the altered fonts must retain the original Emigre copyright notice.

When you alter a font from the Emigre library, the resulting font(s) will remain the property of Emigre.

2007-08-30 02:00:40 • paul d hunt
Or, you could get someone to design from scratch a custom font with just the Greek symbols you need to match the Helvetica you are using. This would avoid the EULA issues associated with trying to modify an existing version of Helvetica. There is no reason to start from existing Greek versions of Helvetica, which are aimed at recreating the shapes of the Latin letters of Helvetica as much as possible, since the Greek letters would be used as scientific symbols and it is actually desirable to distinguish them from the Latin letters of Helvetica.

I don’t think that the x-bar (or any other letter-bar) is included in a standard font set. Normally, the typesetting program will add the bar itself by writing the letter and then drawing the line over the top. If you want a symbol like this in Word, you should be able to use the Microsoft Equation plugin (something like Insert > Object > Microsoft Equation x.x). You can use the dialog that appears to create the x-bar (and a whole equation that includes it). If you’re looking to use mathematics symbols seriously, consider the LaTeX typesetting system.

As an alternative to Word’s equation editor,[[http://sbarnhill.mvps.org/WordFAQs/Overbar.htm| this]] page explains how to construct an overstrike character and assign a shortcut key.

But for doing statistics or other math, wouldn’t a math pi font be a better solution in the long run? The American Mathematical Society has free Postscript distributions of Computer Modern and AMSFonts at [[http://www.ams.org/tex/type1-fonts.html]]. TrueType versions are on the Mozilla website at [[http://www.mozilla.org/projects/mathml/fonts/]]. Wolfram’s downloadable Mathematica 6.0 fonts are at [[http://support.wolfram.com/mathematica/systems/windows/general/latestfonts.html]], and MathType fonts are at [[http://www.dessci.com/en/dl/fonts/]].

(These are all properly licensed or donated fonts, so I don’t believe the links violate any Typophile rules.)

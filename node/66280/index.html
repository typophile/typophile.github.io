I still think this is a sort of constructed grid lettering, using the dots like pegs to attach either vertical, horizontal, or diagonal transparent bars. There are two sizes of bars. That's it. Not a font, I think. Most similar font idea is LED-style fonts that use fixed elements to create crude letterforms. See [[http://www.dafont.com/led-real.font?nb_ppp=50&text=BLOG|LED Real]], as an example. [edit: I am not saying this is fabricated lettering, but I thinks it's a graphic image made to look like plastic pieces assembled to make letters] - Mike Yanega

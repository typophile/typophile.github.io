2011-06-01 22:34:01 • Michel Boyer

There is only one font in iOS that provides Armenian glyphs. It’s called .PhoneFallback:

The description reads:

»The .PhoneFallback font is used by the operating system to display Unicode data when the font requested does not have the capability to handle the requested Unicode. PhoneFallback is based upon the Open Source font Bitstream Vera and the public domain font Deja Vu. Bitstream Vera is a trademark of Bitstream, Inc., designed by Jim Lyles. See complete license details at http://dejavu-fonts.org/wiki/index.php?title=License«

Indeed the Armenian in .PhoneFallback is identical to that in DejaVu Sans.

2014-10-21 18:21:43 • Martin Silvertant

Have you checked the sidebearings you want to be active for the kerning class? Have you remembered to click the “Accept” button after making any changes or additions to a class? It might be helpful if you posted a shot of your classes panel, showing the _a class.

To your second question, click the third-last icon at the top of the Metric window (“Expand the window”) it will reveal a table the sidebearings and kerning info for each glyph in whatever text you have active in the window. You can select the figures in the table to change them.

To change sidebearings and advance widths you need to set up metrics classes separate from the kerning classes.

The FontLab manual can be daunting but it is your friend, you just need to dig into it. Check the sections on Class-based Kerning and Metrics Assistance.

You may have inadvertently created a kerning exception for the aacute. You can get a list of all kerning pairs by choosing “Table” from the “Options” pop-up menu at the bottom of the Metrics window. It displays a list of kerning pairs in a spreadsheet-like table on the right side of the Metrics window. You can select and delete kerning pairs from that table.

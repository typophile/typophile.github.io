Immediately below the comment text input box, you should see the “Insert image” link. That will bring up a file dialog from which you can locate the image file on your computer. It will upload it and insert the appropriate tags into your comment.

The “attach file” allows you to upload any file type, but that feature is only available to the originator of the thread (in the case of this one, that’s you chopshop), and they all group in a list below the first post. you can add more files later with the “edit” link on the top of the thread.

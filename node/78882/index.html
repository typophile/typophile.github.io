2011-02-09 15:19:20 • Té Rowan

Supporting OpenType features may require a lot of programming, and, of course, since it’s open source, it’s not as if money coming in supports it.

But if OpenType support is not trivial - so that even, say, WordPad would fully support all OpenType features - then it may be reasonably asked whether the features in OpenType are really useful to people. If a font can “do fancy things”, then it ought to be possible to make it do those things everywhere the font is used.

Which means that the mechanism currently used to get at things like variant characters, small caps, and so on and so forth, is deeply flawed.

The “right” way for a font to behave on a computer, “of course” (and I realize what I’m proposing won’t happen, since it’s wildly impractical for many reasons) is for it to be simple and intuitive to control all its features, i.e. requesting a long s, disabling ligatures, and so on…

when you’re generating your text this way

      WRITE (6,11)    11 FORMAT (11H Some text.)

Maybe like this:

      WRITE (6,33)       WRITE (6,11)       WRITE (6,44)       WRITE (6,22)    33 FORMAT (51H*                                                 =)    11 FORMAT (53HA word like official is printed with the ligature ffi)    44 FORMAT (18H*                         -)    22 FORMAT (20H representing the letters ffi.) C 3456789 123456789 123456789 123456789 123456789 123456789 123456789 12

assuming that * as a printer control character indicates a line containing font behavior characters, and in such a line, = means “force ligature” and - means “prohibit ligature”.

“Redirecting”? Nothing to re-direct, really.

Fake Small Caps: make all lowercase character caps and decrease font size by 80%.

Opentype Small Caps:

  • Parse entire Opentype font
  • Look up current script
  • Look up current language in current script list
  • Look up ‘smcp’ feature in current language list
  • For each character o Check if it has a GSUB entry o If so, do GSUB lookup for this character
    • Take care to handle each Lookup Type correctly (there are eight of ‘em) o Display the glyph

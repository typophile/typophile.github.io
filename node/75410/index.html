2010-10-20 08:58:29 • P_P
Tjena Petter. It sounds like the issues you’re experiencing are most likely just the product of font rendering with different engines. Browsers and OS’s vary wildly and the only way to really get the same result across all platforms is using images. The font’s appearance is also effected by something called hints, clues that help the rendering engine turn curves into pixels. You’ll probably never again come across a typeface as well hinted as Georgia: Even with Windows’s latest rendering most of the fonts on Typekit doesn’t hold up, and if you happen to use older versions I’m not sure if there is anything readable left. Tom Rickner and Matthew Carter spent years perfecting Georgia for this task. My point is: If Georgia’s rendering is your problem, Typekit is not going to offer any better solutions.

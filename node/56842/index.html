April 9, 2009 • shielddesign

Okay so I am starting my stationary/wedding invite company that specializes in letterpress and silkscreen as well as pen and ink illustration.

The name is Anavi Ink. I have a logo that I like which combines the i from Anavi and the i from ink so there is just one i with an ink splotch for the dot on the i.

However people have been reading it as Anav Ink which I don’t want since that is NOT my last name.

I need suggestions for how I can do the same basic idea, but with people somehow reading the name with two (ii)s.

thanks!

It’s hard to give feedback without seeing an image but it sounds like the ii confusion is going to be unavoidable in the format you have in mind. The problem is that Anavi is an unfamiliar name so readers are going to have a hard time separating AVANINK the two words. You could try using a color build on the I to demonstrate the combination eg. AVAN in cyan, NK in magenta, and I in an overprinted cyan/magenta. Otherwise, I think it’s time to start sketching two word solutions.

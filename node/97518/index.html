This is probably [[http://www.typography.com/fonts/font_overview.php?productLineID=100035|Tungsten]] with some post-treatments (solid shade, inline and outline stroke). Most of the rest is set in Tungsten too. Some inline: [[http://www.typography.com/fonts/font_overview.php?productLineID=100003|Cyclone]], [[http://uppertype.net/fonts/smart.htm|Smart]], [[http://www.myfonts.com/fonts/matchandkerosene/detroit|Detroit]] Some shaded: [[http://www.myfonts.com/fonts/jurizaech/frontage/|Frontage]], [[http://www.myfonts.com/fonts/matchandkerosene/detroit/|Detroit]] Shade x Inline: [[http://www.myfonts.com/fonts/matchandkerosene/detroit/|Detroit]] [img:sites/default/files/old-images/fhm_4645.jpg]

Side bearing Composite ?

March 11, 2007 • RachelR

I have most of my glyphs in the correct places, and then use Glyph - Create Glyphs, to fill in the European characters with accents , which works fine, but if I change the side bearing on the master character it’s not updated in the glyphs that use this character. For example if I have the “a” glyph and the grave glyph in place and use Create Glyph when selecting the agrave glyph, the agrave glyph is created. If I then change the sidebearing on the “a” glyph it doesn’t seem to be updated in the agrave. Is this normal or is there a way to update the sidebearings on all glyphs.

R ; )

Open the key glyph in Metrics window. Benath the letter there is a little arrow – click it and select “copy to composites”. Then you get the same sidebaerings.

You could also do metrics classes of course. Either via the class-panel or the Metrics Assistance.

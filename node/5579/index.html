Ricardo, install the Cyrillic language kit from the MacOS 9 CD. Then you can use the "CY" versions of system fonts. If your version of Word is 2000 or over, the text will open correctly, albeit with a wrong font. Reassign the font to any of the "CY" ones, then you can cut and paste to any layout application. However, unless you have some professional Cyrrillic fonts (you can buy quite a few at www.paratype.ru), you'll be stuck with these mediocre "CY" system ones.

Step 1: Open up your generated font, open the glyph window, enable FontAudit, and step through the glyphs one-by-one looking for any serious outline problems Step 2: Download and run the FontQA macro. Run it and review the results. Fix the problems. Step 3: Download and run Microsoft’s font validator. Run it and review the results. Fix the problems.' Step 4: Validate the font with Apple’s FontBook tool. But be careful, FontBook is a buggy POS and problems it flags might not actually be problems.

Sorry for getting back to this post so late. I forgot about it after I found a solution.

I HAVE FOUND THE SOLUTION! I threw away the Microsoft Word Font Preferences file on my mac, and then it loaded into Word just fine! It was that corrupted font preferences file that was causing it. A very important lesson to all!

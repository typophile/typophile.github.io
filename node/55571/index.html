Legato & Avance

March 6, 2009 • nina

Has anyone here paired Legato with Avance? Seems to me like they should compliment each other well; I'd be curious to hear some experiences/opinions. Also, would it be considered sacrilegious to use Legato (presumably the Regular) for a large (~96pt) poster headline? I've used it for text (and in a logo) so far, but it seems like a great fit here – especially since the headline should feel a bit "uncomfortable", or "unusual" (although very "quietly" so), rather than "pretty". So if I go with Legato in the headline, I'm wondering if I should use a heavier weight of Legato for the smaller text, but I've been wondering if Avance might not be the perfect companion to Legato here; this is science/education-related, so I have a vague feeling that not having all text in a sans face might appeal to the client more. Plus, I admit I'm sorta in love with both of them!
Not a soul? I think the question just answered itself; a second project started smelling like Avance :-) so I just tested this myself. I love it (any objections?). 2009-03-10 16:00:28 • Jan
Evert might not have wanted you to use Avance - by the time he made Legato he didn't like it very much. On the other hand it's pretty safe to ignore the wishes of type designers when it come to actually using fonts! :-) I personally still like Avance, and I think it can work for longish text just as good as Legato, thanks to its [half] serifs*. I've seen Legato used in two books, one of them only for titles (and that was intended as an "unsettling" book too) the other for titles and text. I think Legato looks quite nice large, so I think you should try using it exclusively for this project - it should be able to carry the whole weight by itself. The only problem is if the project is supposed to be serious (in addition to unsettling) in which case Legato might be too friendly; except that would disqualify Avance even more. If that's the case, and you want to stick with Bloemsma, try out Balance (preferably remove the trapping though). * A serif incarnation of Legato, now that would be the highest peak in the craft.

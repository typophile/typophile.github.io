body: "Dear Typophiles,\r\nOne of the most frequently asked Questions by non-Arabs
  is how to configure Windows Operating system for Arabic. Here is how and more...\r\n_________________Windows
  VISTA_________________\r\nStart>Control Panel > Regional+Language Options> Keyboard+languages
  > Change keyboards.\r\nIn the Text Services and Input Languages>General tab>Add.\r\nIn
  the Input Language menu choose Arabic.\r\nIn the Keyboard layout menu choose the
  keyboard you want to add, click the plus sign.\r\n_________________Windows XP_________________\r\nStart>
  Control Panel > Regional+Language Options > Languages tab.\r\no Check Install files
  for complex script and right-to-left languages > Apply\r\nAfter the files are installed,
  you must restart your computer. Then,\r\nStart > Control Panel > Regional and Languages
  Options> Add other languages > Details.\r\no Click on the Settings tab and then
  click Add.\r\no In the Input Language menu choose Arabic\r\no In the Keyboard layout
  menu choose the keyboard you want to add.\r\nClick OK and then Apply.\r\n_________________In
  Either Windows_________________\r\no You can change the language toolbar settings
  by right-clicking on the EN button in the system tray at the bottom right of your
  desktop > Settings...\r\no You can change languages by left-clicking on the EN and
  choosing Arabic.\r\n\r\n_________________Personally_________________\r\nI press
  Shift+Alt to toggle between Arabic + Latin, and\r\nPress Left (Ctrl+Shift) to type
  English from Left to right.\r\nPress Right(Ctrl+Shift) to type Arabic from Right
  to left.\r\n===========================================================\r\n____________Virtual
  Arabic On-Screen Keyboard____________\r\nIn XP, Start > All Programs> Accessories>
  Accessibility> On-Screen Keyboard.\r\nIn Vista, Start > All Programs>Accessories
  > Ease of Access > On-Screen Keyboard. \r\nYou can click the on-screen keys to type
  text.\r\nUse Standard keyboard (Enhanced keyboard - Number Keys). \r\nUse Always
  on Top to keep your keyboard displayed when you switch programs or windows.\r\nThe
  program in which you want to type characters must be active while you are using
  On-Screen Keyboard. \r\n\r\nFor Single Arabic Marks:\r\npress Shift + Q to get Fatha\r\npress
  Shift + W to get Tanween fath\r\npress Shift + E to get Zamma (9)\r\npress Shift
  + R to get Tanween Zam (69)\r\npress Shift + A to get Kasra\r\npress Shift + S to
  get Tanween Kasr\r\npress Shift + X to get Sukoon\r\npress Shift + ~ to get Shadda\r\n\r\nFor
  Double Arabic Marks:\r\npress Shift + ~ then Q to get Shadda + Fatha\r\npress Shift
  + ~ then W to get Shadda + Tanween fath\r\npress Shift + ~ then E to get Shadda
  + Zamma (9)\r\npress Shift + ~ then R to get Shadda + Tanween Zam (69)\r\npress
  Shift + ~ then A to get Shadda + Kasra\r\npress Shift + ~ then S to get Shadda +
  Tanween Kasr\r\n===========================================================\r\nIn
  M$ word, short-cut keys can be added to insert symbols as follows:\r\nShift + Alt
  + Q to get Dagger-like Fatha\r\nShift + Alt + A to get Dagger-like Kasra \r\nAnd,\r\nShift
  + Alt + any Arabic letter to get the corresponding Farsi/Jawi letter.\r\nExample
  = Shit + Alt + kaf to get the Jawi dotted Kaf or the Persian Gaf. \r\n===========================================================\r\nHappy
  Toggling with Flowers\r\n\r\n\r\n"

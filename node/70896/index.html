I’ve contacted Eduard Manso about this, with no response yet. Another issue is that I hope to license some inhouse fonts for the client, but most type designers don’t prepare their fonts for use in Microsoft Office and it’s likes. I’ll need a basic style linked set, diacritics that doesn’t disappear and preferably some hinting for Windows users. I’ve seen FF’s Offc library, but none of those resembles Relato’s style.

Any suggestions?

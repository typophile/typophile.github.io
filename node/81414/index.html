Unipasta, a Unicode browser

April 20, 2011 • quentin.t

Hey! I’m French, and frequently looking for some characters I can’t type with my keyboard (like œ) so I needed a way to quickly access them. I used to use copypastecharacter.com but some of the ones I need are not there and I thought it was lacking features.

So I wrote [[http://toki-woki.net/p/Unipasta/|Unipasta]]! You’ll find more [[http://toki-woki.net/blog/p2560-unipasta-a-unicode-browser|details here]] but really it’s pretty easy to understand. I think and hope this tool can be useful to you too!

Also, as I’m in the process of [[http://typophile.com/node/80680|creating my first typeface]] I think it should help for that.

Drop feedback/suggestions if you have any!

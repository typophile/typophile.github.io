Thanks to the person below for helping me solve my initial problem. Regretfully I am faced with another challenge, but I didn’t want to create a new topic.

When I generated the truetype file for the initial font I’ve been working on; the kerning settings weren’t saved with it. I tried using several programs and saving the font multiple times to no avail. When I open the metrics window my settings work. How might I fix this problem?

This is a sort of bug (or feature with unintended consequences) that has been in Fontographer forever. It often happens when you paste in a path from Illustrator. You can fix it by doing this:

  1. Select the affected characters in the main window.

  2. Choose “Selection Info…” from the Element menu.

  3. Make sure that “Fill outline” is checked and that the “Tint” is set to 100%. (Also make sure that “Stroke outline” is unchecked.)

This should fix the problem.

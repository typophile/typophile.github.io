For OpenType or TrueType fonts the PDF creation process should verify the embedding permissions encoded within each font file. Although these fonts would be set to embeddable, an individual EULA might still restrict posting the document on the Web or may claim that the PDF copy of the paper is a 'commercial product' which requires an additional license.

As for Type 1 which has no embedding bits, the responsibility for checking the EULA lies with the publisher.

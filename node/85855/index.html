2011-09-28 01:39:25 • Michel Boyer
Usually I like to keep the brand name font as the only instance of that font, and use a complimentary font for secondary text. It is also a great idea to use customized lettering for a logo. In terms of the legal stuff, it is ok to modify the letters to create a logo that is just vector shapes (using Adobe Illustrator or Inkscape for example), but in this case not ok to modify the letters to create a slightly different font. Anyway, here are a few other ideas: http://new.myfonts.com/fonts/agfa/prosper/ http://new.myfonts.com/fonts/4thfebruary/bladi-two-4f/ http://new.myfonts.com/fonts/bogusky2/squircle/ http://new.myfonts.com/fonts/gravitart/badona/ Hope that helps!

On what platform do you work, Sita? MS Windows or MacOS?

OpenType is bound to become the new standard and it's cross-platform,
so you can't lose. Unfortunately QuarkXPress still doesn't support them,
so that could be a problem. They work perfectly with Adobe software.

The typefaces that come with your Adobe applications are licensed
for any work, including commercial, but that license is limited to the
computer the software is installed on (and probably some output
device). So basically you can't install them on any other computer than
the one the software they are bundled with is installed on.
You should be able to find more details about the extent of the license
in the documentation that comes with the software package(s).

Apparently, Quark 7 will come out later this year, and support the full range of OpenType features. So you can continue to work in the page layout program you're comfortable with, and not miss out on special OpenType features.

Of course, like any software, fonts can have bugs; and in that, no format or foundry is immune.
2005-04-24 17:52:53 • Nick Shinn
Before OpenType fonts became available, for print work I preferred working with Type 1 PostScript fonts rather than TrueType… They were more reliable than most of the TT fonts I was familiar with.

Briefly, the differences are that PostScript fonts have two components: a suitcase containing the screen fonts and separate files for the printer fonts, while TT fonts combine these into one scalable font file… Also, PostScript fonts use the same page-description technology as laser printers.

As for OpenType, it is a cross-platform format, meaning you can use the same font file on a Macintosh or Windows machine. OpenType fonts have room for many more characters than the older font formats.

The dividing line between Windows TrueType fonts and OpenType is remarkably thin.

Personally, I use about 80-90% OpenType fonts, and the remainder almost all TrueType. I keep a few Type 1 fonts around for specific compatibility needs.

But then again, that's predictable: I work for Adobe (inventors of Type 1, and co-developers of OpenType), and we stopped making new Type 1 fonts in 1999, to switch to OpenType.

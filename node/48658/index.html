Macro Request (or feature for Fontlab Studio)

August 21, 2008 • scannerlicker

Here’s a thing that I’ve been wondering (and needing) for a long time:

  • Some vector drawing applications like Illustrator or CorelDraw have a stroke feature, where you draw a line and can specify a width to it, even with calligraphic simulation. Fontlab hasn’t. I know it has Expand Stroke thing, but in many, many cases I have the need to undo what I’ve done, change what I need and do it again. And it works nasty, sometimes.

So, my suggestion is: please develop a macro (or a tool) that can give the expand stroke without removing the inital line and that changes when we change the line.

This seems essencial when working with different weights, even for italics.

It would be a lovely and practical way of seeing Gerrit Noordzij’s theories applied.

What do you think, guys?

Or can anyone help me developing this?

Cheers!

Not strictly type related, but I figure we get enough graphic design people here that someone’ll know the answer…

We get a lot of ads in PDF format. They’re supposed to be in greyscale as they’re for b/w journals. From time to time they come in CMYK or with spot colours in ’em. I’ve got Acrobat Professional v7, which has a lot of colour management tools in it, only some of which I really understand. What I’d like to be able to do is just convert these files to greyscale; seems like it ought to be easy (like picking a different colour mode in Photoshop). Is it? If so, what do I do?

I’m not an expert here, but in version 9 you can go Advanced>Print Production>Convert Colors and set Output intent to Grayscale at the dot gain desired.

A viable solution; Use the ColorSync Utility that ships with OS X.

In CSU open the PDF and find the menu for Export…. Two pop ups. Top one should read PDF. Bottom one is Quartz-filter. Choose Gray Tone. Save. Ready.

2012-01-24 00:53:17 • Joshua Langman
Kerning data is typically built into a font indeed, but it’s not a requirement in any way neither for the font to provide it nor for the app to use it. Typesetting apps usually use font’s kerning data by default if available, but can of course also impose any user tweaks or even run its own kerning algorithms. But that’s for the typesetting apps, text processors are entirely different story…
2012-01-24 01:25:47 • Grzegorz Rolek

Kerning has been supported in the Windows version of Word (and probably the Mac version too), since like forever, select “Fonts”, “Advanced”, and select the size at which kerning turns on. Kerning has been “on” by default in PowerPoint for at least the last two releases.

Changing the kerning values built into a font is a different story. If there’s a letter combination that you don’t like you can turn off kerning and use tracking (called spacing in the same Word menu) to adjust the space between a pair or group of characters.

2014-10-21 20:45:23 • riccard0

That makes sense. I can see the solution for /m, but for the other characters I don’t think such a solution would work. M/N/W/AE/OE/&/@/Ø are particularly too dark. For OE I would simply increase the counter a bit. For Ø perhaps the slash within the counter should be lighter than outside it. I would prefer to remove the slash from the counter completely, but given the point size at which the typeface seems optimal, I don’t think the slash will be obvious enough if you remove the part within the counter.

It seems quite impossible to fix all the dark spots using a pixel-based solution. But I guess since the design is inspired by pixel typefaces, it may not be necessary to remove all dark spots.

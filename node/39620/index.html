Hi everyone, Sorry I don't have an image yet of the typestyle I'm looking for, but I'll try to describe it as best I can. It's a caps only, tall, narrow, somewhat ornate sans serif font with nicks and notches in it like Papyrus. It vaguely resembles Galahad, but more condensed. This font is used for the hardcover dustjackets of the tenth anniversary re-release of the "His Dark Materials" books (i.e., "The Golden Compass," etc.), paired with P22's Cezanne. I remember this font was also used for some of Panera Bread's marketing a couple of years ago. I'll try to get an image of this as soon as I can, but if anyone knows this font in the meantime, I'd be ever so grateful! Cheers, John the Fontaholic
If it's the one shown on this site, then it's ITC Cancione. - Mike Yanega

Get a font manager program. I’m a Mac person myself, but I think the problem is much the same. There is a manager where I work (also named Richard) who has the same problem on his Mac. Every time he opens FreeHand it runs slow as can be. His fonts are all installed in his Home/Library/Fonts folder which makes them all activated all the time. If you put them somewhere where they aren’t automatically activated and use a font manager program you should be able to solve that problem.

Linotype has a free one you can try called Font Explorer. Most font managers will detect duplicates and other problem fonts for you and allow you to create sets of fonts to activate that you frequently use. The only fonts you really need to have activated all the time are your system fonts.

Stephen

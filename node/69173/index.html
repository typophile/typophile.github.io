… 16 x 16 pixels for use on the web ..

You are looking for the term “favicon”.

Try downscaling the image to about 64 x 64. Clean up as much antialiasing as you can – it’s of no use for anything this small. You can even try downscaling without resampling, and check if there is enough recognizable detail left to work with. From that version, scale down to 16 x 16 and clean it up one pixel at a time.

2009-03-11 03:51:36 • johio

Here is example cycling code based on CALT, originally cooked up by Tom Phinney:

feature calt { lookup rotate { sub @default @default' by @alt1; sub @alt1 @default' by @alt2; } rotate; lookup rotate; } calt;

Silas Dilworth and I recently released Black Monday, a font that cycles through three glyph sets. It turned out that while CALT is nice, it is not widely implemented. Making matters worse, it’s not always on by default. And it’s flaky—for example, in Indesign long strings of nonstop CALTs occasionally result in CALT turning off (although for display faces this won’t usually be an issue). Silas dealt with this by writing a program that generates a massive list of LIGA ligatures to handle the cycling; it works very well.

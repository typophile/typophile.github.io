Mads, when working in Illustrator and FontLab, it's better work in points rather than mm because 1 pt in Illustrator = 1 font unit in FontLab. Please read http://groups.msn.com/fontlab/tipsandtricks.msnw?action=get_message&mview=0&ID_Message=2675 You can set up any grid subdivisions in FontLab you want. Regards, Adam Twardoch Fontlab Ltd.

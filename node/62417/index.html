Importing/linking a psd or ai files into indesign are asking for trouble.

a less headache-y way would be

Go to the top drop-down menu and select ‘object’ then ‘path’ then ‘outline stroke’. Once your text are properly outlined (and make sure color is in cmyk for print). Save the ai file into eps (uncheck embedded fonts).

If you insist on using ai, again you should outlining the text.

If you really insist on using ai and choose to not outline the text. Make sure you are login as the administrator of the computer. And perform all task as the admin. But note that once the indesign and the illustrator files leave your computer or different login. Fonts will be replaced again with the default serif/Sans-serif

*please note that once you outlined the text, you can’t go back and edit them as text. so you might want to save a copy of the master file

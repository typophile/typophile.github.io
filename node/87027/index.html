Currently, the hand-hinted webfont versions of Benton Sans are only offered for web licensing from Webtype. If you’re planning to set smaller body type online, you may also be interested in Benton Sans RE, as Frode mentioned. The RE (“Reading Edge”) fonts were designed from scratch and hand-hinted specifically for use at smaller sizes on the screen.

The rulers are in em units. An em unit is relative to the em size of the font. Typically, PostScript fonts (OpenType CFF or Type 1) have 1000 em units per Em. So, in this case, the Em units are 1/1000 of an em. TrueType fonts are usually 2048 units per em, so an em unit is 1/2048 of an em. Note that the em size of a font can be other values besides 1000 or 2048. These are just the most common ones.

To determine how an em unit relates to, say, points, you need to know what type size the font is set at and how many units are in the font’s em size. Basically, the size of an em unit is the point size divided by the em size. Another way to think of it is that the em size is scaled to the type size being used. For example, if you set a PostScript font (with a 1000 unit em) at 100 point, each em unit will be 1/10 of a point. So, in the real world, the em unit is always relative to the type size being used.

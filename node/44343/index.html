Hi Kris. In some respects it’s like Neo Sans, which I designed,
with the curves taken out. Not all the curves obviously. That would be hideous, but enough to make the design a bit more versatile I think.

So the same things that influenced Neo. Gerard Unger’s excellent open designs like Argo, Dave Farey’s Cachet, etc. It’s more industrial looking than the former and less curvy than the latter.

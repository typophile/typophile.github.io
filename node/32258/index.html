Not sure why you bought a Postscript version for Mac when the Opentype version works on both Windows and Mac.

Meta is available at fontshop.com. All you purchased is one weight, the roman. The Bold and Italic are separate weights because the designer(s) have to redraw all the characters for each weight, requiring extra work and thus justifying charging separate prices for each weight.

The $229 price is for a package of weights, typically between 4-8 weights. Usually the package is cheaper than buying all the weights separately (buying in bulk if you will). I would recommend the Opentype version because you will have a huge amount of characters in one font (oldstyle characters, lining figures, smallcaps). Even if Framemaker doesn’t support all the Opentype features that Adobe CS2 programs do, it still makes sense for when you eventually upgrade.

As for the client specifying “Meta Roman,” I think they just mean Meta and that most of the text be set in Roman.

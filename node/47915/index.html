I’m pretty sure Adobe’s glyph complement PDFs are not locked to prevent copy/paste. You can go to the Adobe web store, find the specific family, and there’s a downloadable PDF right there.

For example, http://store1.adobe.com/type/browser/pdfs/1737.pdf

I’ve been working on rationalizing and extending Adobe’s various extended Latin character sets, and will be publishing some “final draft” specs later this month on my blog, for review and comment.

In Adobe’s nomenclature, a western font with “Pro” in the name does have a certain minimum language coverage. See: http://www.adobe.com/type/browser/info/charsets.html

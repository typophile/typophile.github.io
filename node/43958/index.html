Is it possible to use (and have access to) the reserved 8-bit slots (especially decimal 129, 141, 142, 143, 144, 157) in a PS Type 1 flavored OTF font?

For “reserved slots” see http://www.sanskritweb.net/fonts/howto.pdf

I would prefer here PS Type 1 flavored OTF (instead of TrueType) for ease and fastness of hinting of the Devanagari glyphs.

I got an ID request for my least favorite type style - Techno - and the best I can come up with to match this is a mix of Handel Gothic and Ray Larabie's Zekton. Then I would have to add gaps in the E from Zekton, and find a new M and R, because neither of them matches, etc.

Not a good solution. Occam's Razor, or something, suggests this is too complex. It's probably a font, not a cobbled up mixture. The small lettering looks like the same font, so there are a fair number of letters to see.

Can anyone recognize this one, or is it custom? (They use Bauhaus as the other font on their home page, so I don't think they get a lot of custom design done, if I had to guess.)

Thanks for helping me out.

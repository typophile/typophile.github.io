Suitcase Fusion

January 9, 2006 • canderson

http://www.extensis.com/en/products/font_management/product_information.jsp?locale=en_US&id=1060 Extensis has released a new font manager, Suitcase Fusion. It incorporates some of the most popular features from both the Font Reserve and Suitcase font managers. In particular, it uses FontSense technology for auto-activation to distinguish between Postscript name duplicates. It also supports face-level activation of Macintosh font suitcases. There is also an flash animation on the main Extensis page showing the products being "fused". http://www.extensis.com

2006-01-10 18:31:44 • AndrewSipe
You may have noticed this thread recently on a similar topic. http://typophile.com/node/12563 Other font managers for Windows, such as Suitcase 9.2 haven't been updated recently. This is partly because a new version of Windows hasn't been released in several years. Also, Windows doesn't really provide a mechanism to make system-wide font auto-activation possible. On the Mac this is handled by a specific process called ATS server. I'm hoping that Microsoft makes something like ATS functionality possible in Vista. It's also possible that plug-in based font auto-activation will emerge under Windows. Currently, Suitcase Fusion's plug-in based auto-activation is much more sophisticated in terms of figuring out what font was used in a particular document. It would be useful for Windows users to have access to that functionality as well.

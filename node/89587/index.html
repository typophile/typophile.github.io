The smaller text looks a lot like Chronicle by Hoefler & Frere-Jones, and they do sell it to magazines and newspapers. The larger type is some sort of Egyptian or Clarendon, maybe. Both styles are rather common and are hard to pinpoint with only the letters in this sample. If you have more samples available toyou, you could also try using the Serif Font ID Guide to narrow the search. Key letters in your sample were a, b, e, g, y and K. If you can find E, J, M, R, U and W that can help with the Key. With these 6 letters I got 119 possibilities, many of which were easy to discard at a glance. -Mike Yanega

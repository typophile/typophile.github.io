Expand spacing in FLS5?

July 27, 2006 • paul d hunt

Is there a simple way (that doesn’t involve python) to expand the spacing of a range of characters in FLS5? Say I wanted to add 10 units to either side of the capital letters. Is there a (semi)automatic way to do this within FLS5?
Tools –> Actions –> Metrics –> Set Sidebearings or Adjust Metrics
2006-07-27 19:57:35 • Miguel Sousa

Also, in action set you can make a set of actions for sidebearings and save it so you can change the numbers if needed. For example: select round letters and add 10 to sidebearings on both sides; select straight glyphs and add 20 to both sides; select round right straight left glyphs (p, b, etc) and add 20 to left and 10 to right side bearings.

If you have made metrics classes, you can make selections from those. You can also fix one glph of a given type, copy it from the full font screen, then “paste special” to a larger number of glyphs and choose paste sidebearings and or kerning.

It's very web 2.0, which seems to be what you're going for, so good for you on that count. Did you create this as a raster graphic? The only reason I ask is that "FUNK" seems blurry. You should create logos as vectors so they will scale better. The star is a little too far into the 'K' for my liking. Also, you may lose details in the phone buttons in small or low-resolution environments (such as on a phone). - Lex

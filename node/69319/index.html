2010-04-11 20:40:14 • kevintheophile

You are trying to modify an existing font to create a unicase font, then?

Do you have a font that you are free to modify, either because it’s your own creation, or because the licence permits it? Then again, if you’re already experienced in designing letters, you don’t need much help in designing unicase…

The easiest would be starting off with a multi-weight font family so you can match lighter lowercase letters with heavier uppercase letters when scaled to the same height. You may need to generate intermediate weights for this. Not all fonts will permit this type of modification in their licence but it may be fine if you don’t need to generate a font but just use some letters for a logo.

2010-04-12 16:41:44 • kevintheophile
Fontlab Studio should have a blend option so you can quickly interpolate intermediate weights between a regular and a bold. Paste the same glyph of a different weight in the background layer and fiddle with the blend amount. Then scale to see if you’ve matched the stem widths. The result may need some cleaning up, but it might be good enough for personal use. There is even a ‘blend fonts’ option, I think, which is even simpler. There are more sophisticated ways to interpolate, but they may be difficult to set up. I am not aware of ways to interpolate in FontCreator, but maybe others can prove me wrong.

2004-02-25 17:02:47 • Miss Tiffany
The official reply from Peter Lofting at Apple:

There are no built-in supports in OSX at present, i.e. nothing
equivalent to the double-click on a font suitcase that exists in OS9.

However most classic font tools will run in Classic emulation
mode. e.g. you can run ResEdit and delete the NFNTs with that.

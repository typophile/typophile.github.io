The first one is indeed the best, in my opinion. And the reason for that is the way the photograph gets the message across. It is an interesting image which makes you wonder - always a good thing.

Since this image works well, I think you shouldn't let typography get in the way too much. As it is, all kinds of fonts and typographic treatments compete for attention, and the end result lacks uniformity because of this.

So some suggestions:
- Try using one font for all the lettering; this may feel slightly boring, but isn't that exactly what this cover is about?
- Try moving 'not with' lines to the bottom, to maximize the effect of the photograph
- Treat the title of the magazine more like a logo instead of two fonts next to each other.

Good luck!

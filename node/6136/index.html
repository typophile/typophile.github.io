It seems the slab serif font is called "Archer" and the serif font is "Surveyor" both created by Tobias Frere-Jones (http://www.typography.com/) for Barbara de Wilde, the former art director of Martha Stewart Living. Surveyor (which must be the italics on this cover) is based on old map fonts. See this article.

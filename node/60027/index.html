It never had a resize grip, that’s why you can’t change it back.

As soon as I drag ways the attached panel it snaps back to default width.

If that doesn’t work, you can always try resetting the program preferences.

Here’s a way to fix it.

  1. Open the Character panel and attach it to the side of the column of minimized panels
  2. Attach a panel that can be resized (e.g. Table Styles) below the Character panel
  3. Use the grip above the Character panel (see highlighted in red in the image below) to change the width of the column of panels

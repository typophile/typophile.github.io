TransType Pro

TransType Pro converts fonts between PostScript Type 1, TrueType, Multiple Master, and OpenType. In seconds you can convert practically any font into any other format on Mac and PC.

TransType supports batch conversion so users can convert hundreds of fonts at a time, even directly from Stuffit (.sit) archives. TransType Pro automatically adds some OpenType typographic features such as ligatures or fractions and supports Python scripting.

Macintosh/Windows $179.00

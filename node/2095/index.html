Regardless of what else is going on, I'll note that ATM Deluxe is officially *not* compatible with OS X. (ATM Light can still be run in Classic, and is pretty useful there.)

Some people have reported success using ATM Deluxe in OS X as long as they turn off auto-activation. Of course, it's still a Classic app, so you have to run Classic to run it. But you're taking your chances if you choose to go this route.

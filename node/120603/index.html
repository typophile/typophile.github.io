2014-10-19 23:01:31 • Bert Vanderveen

I should add two caveats:

  • Folks using some Wacom tablets may find that they need old drivers for the tablets to work properly with FontLab Studio (and with assorted other apps), but that the old drivers are not compatible with Yosemite.

  • We have one report that while they still work, UFO export scripts (both from Robofab and UFO Central) seem to be surprisingly slow under Yosemite. I’d be curious to hear if more people have experienced this slowdown.

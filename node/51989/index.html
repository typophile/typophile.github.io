I like the one on the bottom left. Nice ampersand. The right leg of the “R” is a bit funky — looks stretched to me. Although I like the idea of the ampersand leading into the “flight streak” of the plane, I don’t really like how it tapers down towards the plane. Try making the plane bigger so that it matches up better with the thickness of the ampersand (and you wouldn’t need to taper the trail). Also try it without the trail through the “D”.

Good start here.

2008-11-23 23:19:16 • Ratbaggy

The plane is almost certainly too detailed compared to the rest of the logo. So simplify it a bit. Ampersand adds quite a substantial amount of detail into the logo but it’s too sans serif to complement the details of the plane.

To me the most promising logo was #4 in the first batch of logos. It connects both letters and the plane and doesn’t leave R outside of the mark.

Logos with only half the plane don’t work with me.

One more thing. If you’ll combine outlines with flats (as in logos in the second batch), be very careful about it because it usually tends to complicate application if not done properly.

But other wise type choice is very good for R&D lab.

Modestly priced workhorses

August 11, 2008 • nitrospectide

I've run across a few threads on here where people have recommended some excellent choices for serif and sans 'workhorse' test/headline faces. Stuff with some flavor, but eminently usable and not likely to date soon. Being a one person studio, I tend to be a bit more budget minded than some of the beautiful faces I've seen would fit with. Can anyone make recommendations for type families, and foundries where one could find real killer values like Stefan Hattenbach's beautiful Tarocco Opentype (7 faces) for a mere US$109. http://www.psyops.com/html/spec_tarocco.html Any other jewels worth checking out?
This kind of thing is always subjective but I have some fonts I find I can use pretty often. I think [[http://www.vllg.com/Thirstype/ApexNew/mudTyper+Weights/|Apex]] is a great buy. I have gotten a lot of use from it. I also use [[http://www.underware.nl/site2/index.php|Dolly]] regularly. I have used it for a client's website and have used it in my photography portfolio. I'm working on a website now using [[http://www.fontshop.com/search/?q=bulldog&x=0&y=0|Bulldog]]. I love this font and it is reasonable. I know it is going to be very useful in other projects. Sharon

The ITC Stone superfamily is $179 (56 fonts in four styles) at stonetypefoundry.com; his Magma family is $99 (24 fonts). Fontin and Fontin serif are excellent and free. And of course you can always buy a used Corel Draw on eBay and get dozens of workhorse families cheap; at least half a dozen of them will always be in style!

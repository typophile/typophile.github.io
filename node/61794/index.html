For my Latin fonts I use a self-made list of characters imported as .enc-file into Fontlab. This list orders all characters and accented combinations needed to cover all European languages set in Latin. *All*. The list is a reliable compiling tool and is far more better than all codepages and encoding schemes all the big players have created in three decades. If interested, go to [[http://www.signographie.de/cms/front_content.php?idart=214&changelang=2|this site]] and there find “character repertoire”-pdfs for download.

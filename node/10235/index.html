Hold on. The Ultrafonts stuff is still outlines: they're pixelfonts, outline fonts designed to render out crisp anti-aliased type (the best stuff to read onscreen) at a single point size. I doubt they will render out faster (although their lack of curves might make a slight difference). So I think Robert might be talking about one of the relatively new features of Flash: the ability to render out an internal actual bitmap font (from an outline font) and then use that instead; the problem is I think that doesn't work with grayscale - although I think it might depend on platform -not sure though.

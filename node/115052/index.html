Lincoln Vintage Cars Logo typeface

March 18, 2014 • duosista

Lincoln has long used logoscripts. The originals were done by designers familiar with the Palmer style of penmanship. Note that the slope is very inclined from the vertical. For a font in that style see Filmotype Yukon http://www.myfonts.com/fonts/filmotype/yukon/ as well as fonts descended from Bickham Script http://www.myfonts.com/fonts/adobe/bickham-script-mm/
I am not certain whether this version of the Continental logoscript is still based on an original handwritten effort or wheter it is taken direct from a font. In any event, it is fairly close to Profont’s Sonata http://www.myfonts.com/fonts/profonts/sonata-pro/ which is not quite as deeply inclined as the logoscript.

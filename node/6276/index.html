Can anyone suggest a heavy slab serif which is contemporary and a bit different?
Ooooh, many, but I'll restrict myself to a couple of lesser known options. Oxtail is subtly different, a contemporary take on Clarendon. Fantabular is monospaced, which opens up a lot of design possibilities. Silica is an often overlooked little gem by Sumner Stone. ApexSerif is so contemporary it hurts. Charifa is very geometric, inspired by Bauhaus. Aminta is a re-interpretation of the typewriter style. Barbera is nice'n'quirky. Aurea Ultra has only one weight, but with a killer italic. Tell me if you need more options.

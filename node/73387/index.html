One question is what are you going to do with the font? Even if you can update it, I seriously doubt you’d be allowed to sell it.

On the other hand, if you’re setting type, esp. for print, there could be solutions. Starting with this one: if the accents themselves are already in the font, how, from reading a print edition of a text, could anyone tell whether you had added glyphs or simply done a lot kerning with existing glyphs? Or used a layout program that supported floating accents, such as TeX?

Scott,

I’m the creator of TypeShow. Please contact me directly (there is a tiny contact link in the site footer at [[http://www.typeshow.net]]).

TypeShow is still a little rough around the edges and not quite ready for release, but it works as advertised – check it out on these websites: [[http://www.janfromm.de/typeshow/]] [[http://www.lucasfonts.com/typeshow/]] [[http://dehac.de/]]

Ben, you’re right – LetterSetter is a great product with great rendering quality, it supports OpenType features and has been tested for years.

The basic difference is: LetterSetter is a service, TypeShow is a software package to install on your own server. Both ways have pros and cons.

One thing that is an important fact to me as an interaction designer: Right now, LetterSetter explicitly comes without an interface, where TypeShow brings a user-friendly, very responsive interface that I have spent a lot of time on, optimizing and tweaking the user experience. TypeShow’s front-end runs on all browsers, even Internet Explorer 6. TypeShow is extremely fast, which is also important for the user’s interaction with your site.

Don't delete - these

Marlett
Microsoft Sans

As there are all fonts used by the user interface. Marlett and MicroSS can't easily be deleted as they are 'system file protected'.

All of the others, except Webdings are not fonts that come with Windows, so removing them may break other apps (unlikely but possible).

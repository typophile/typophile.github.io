2014-12-08 19:22:51 • Nick Shinn

Bank of America used a custom typeface called, ironically, Higher Standards, but I believe they—like NatWest—stopped using it within the last five years or so (I last worked for them in 2009).

BoA’s identity typeface is Franklin Gothic.

Higher Standards was a Clarendon (numeral)/Century (letters) hybrid that I thought worked quite well with Franklin Gothic, but it was eventually relegated to being used only to set long-form copy before it was dropped completely.

2014-12-10 03:47:56 • kthomps5

I’ve done graphic design work for several regional banks. Some used to use custom fonts but they’ve all gone to standard fonts.

The reason I was told was that with many branches, employees, vendors, etc. their graphics departments got sick of constantly sending the custom font to people, and also seeing pieces done in the wrong font when the vendor didn’t have the right one.

It’s easier to say to a vendor “just use [common font]”.

You’d think that employees could send it without bothering the graphics department, but most employees can’t even pick the right color, let alone the right font.

Ubuntu font by Dalton Maag

September 29, 2010 • apankrat

http://www.omgubuntu.co.uk/2010/09/new-ubuntu-font-lands-in-maverick-for-all-users The new Ubuntu font that Canonical commissioned typeface designers Dalton Maag to develop for the Ubuntu operating system has landed in Ubuntu 10.10 for all users. The font, which has been under development for almost 9 months, was only previously accessible through a limited beta program which saw Ubuntu members and Canonical employees gain access to the font for exclusive testing.

–

Though I don’t think it’s complete yet, at least the kerning tables aren’t.

Also - http://news.ycombinator.com/item?id=1736630 - if you don’t run Ubuntu, but want to access the TTFs.

Indeed, this first release of the fonts covers Latin A+B Extended, Greek Polytonic and Cyrillic Extended; and is four font styles only. The project is still in progress with a number of other font styles to come with the same glyph coverage. The four core fonts that have just been released will also be extended with Arabic and Hebrew.

The kern tables are as complete as you can get. The fonts contain both ‘GPOS’ kerning as well as a traditional ‘kern’ table. The ‘kern’ table is incomplete in that we had to drop accented kern combinations due to technical limitations. Appls that support OpenType will have access to the ‘GPOS’ and hence will find the full range of kerning, including accents.

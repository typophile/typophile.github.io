Below is an historically interesting settings of Zapfino. Historically interesting because I made it during production of the first AAT version of Zapfino, which shipped with the first release of Mac OS X. Between this version and the version that shipped with later releases of the operating system, Hermann Zapf redrew every single glyph in the font, working with Akira Kobayashi. So when we came to make the second version of the Zapfino AAT font, we were dealing with a completely new and very much improved design and some really lovely, clean outlines. The version in the illustration below is the earlier, less good version, but even it gives some sense, I think, of the kind of lively display setting that can be achieved with careful selection of alternate forms.

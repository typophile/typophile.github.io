Indesign/PDF output question

November 21, 2007 • glencc

My book designer has placed each of my forty Indesign-produced chapters in separate files along with pertinent graphics files for each chapter. In preparation to go to prepress with a PDF, I’ve had the following questions arise:

  1. Must I create one master Indesign file with all of the chapters using the book option in Indesign? Or can I create a pdf file for each chapter and then merge all of the individual pdf chapters into one master PDF? Are there any potential output snags in the latter option?

  2. Do the graphics files that are linked to each chapter automatically embed when Indesign prints to PDF?

Thanks, Jim

Hm, where and how is this being printed? Usually pre-press departments (in my experience) prefer packaged indesign files set up in reader spreads rather than a PDF, just so they can handle (with more flexibility) the setup on the press sheet.

to answer question 2 yes all images and fonts should embed (be sure to use the “Press Quality” output option)

I have limited experience with book production, but I do know that printers prefer to work with a section at a time, to facilitate their workflow. So your designer is right on track with using separate files for each chapter. It’s also a lot easier on editors and proofreaders.

And then you always have the option of putting the chapterfiles in an InDesign Book-file. Just do some reading up on the Book-feature… Information vcan be found under Help/InDesign help…

BTW: The best way to get a sense of flow is to print everything at 100% (spreads!) and place or paste or pin them in sequence someplace where you can look everything over in peace. Making a dummy also works ; )

Must I create one master Indesign file with all of the chapters using the book option in Indesign? Or can I create a pdf file for each chapter and then merge all of the individual pdf chapters into one master PDF?

I agree with Abi… You can use the Preflight and Package features to collect each chapter’s files for output from within InDesign… That way you will be able to give your printer the InDesign files and the image and font files that belong with each chapter.

If you do choose to go the PDF route, make sure you select Press quality, as stated above. If you have any images that bleed off the sides of the pages, you will need to check the appropriate boxes for bleeds and printer’s marks… But in my opinion, it’s safer to send them the packaged INDD files.

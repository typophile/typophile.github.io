You should remember that the Heinz 57 brand name is much older than computers, so finding digital fonts that match is more coincidence and/or luck than design. Tempo is just similar. I doubt if you can match the 57 without a lot of outline manipulation to an existing font. You are better off scanning it. The script is pretty similar to an old Berthold script called Houston BQ (see below), which I cannot find at Berthold any more, or anywhere else on the web right now.

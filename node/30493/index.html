The differences in pronunciation and vocabulary between different regions for Castilian (i.e., “Spanish”) are substantial, but this doesn’t affect typography. “Voces” might be pronounced “vothesh”, “vos’s”, or “vose”, but it’s spelled the same everywhere, and any rules concerning syllable division, accent marking, digraphs, capitalization, etc. are also the same everywhere. So far as I know, all the countries concerned follow the orthographic standards set by the Real Academia Española, so even when the RAE changes something that might affect OT behavior - like the changed alphabetization rule of 1994 - it is adopted everywhere. Of course, this could always change in the future, at which time a new tag might be needed.

Fuss over cyrillic version of the EURO

October 15, 2007 • Andreas Stötzner

Bulgaria recently claimed that the proper Cyrillic spelling EBPO is to be adopted by the European Union. The argument is about the European Central Bank’s preference for EYPO in Cyrillic, which does not match the Cyrillic spelling of “Europe” in Bulgarian and Russian either. – Bulgaria is the only state within the EU which uses Cyrillic. But Banknotes are uniform, and in the Latin script a unification of spelling has been adopted on the banknotes as well. While in documents the Euro is referred to as euro, euras, euró depending on language (and some others), on the banknotes it only appears in the “euro” spelling.

So a unification within the cyrillic has a precedent. What should not happen, however, is, if all cyrillic countries were to use ЕВРО, ЕУРО (a mere transliteration of EURO) should not be considered.

The names used for the euro in various European languages using cyrillic, at least according to Wikipedia:

Belarusian: ЕЎРА Bulgarian: ЕВРО Macedonian: ЕВРО Russian: ЕВРО Serbian: ЕВРО Ukrainian: ЄВРО, ЕВРО

From the article: “In standard literary Ukrainian ‘euro’ is spelt евро (‘evro’), although Russian-influenced євро (pronounced ‘yevro’) is also possible and more common in large cities across the country, South-Eastern areas, and among Russophones.”

Belarusian is the odd one out, it seems. I think the Belarusian pronunciation is basically the same as the pronunciation used by at least some Russian speakers, but the difference is that Belarusian uses phonetic spelling. It’s as if some hypothetical quasi-anglophone language with phonetic spelling used “yooro” for euro.

2008-04-23 20:28:40 • SuperUltraFabulous
Monotype Garamond in digital format is based on (probably digitized from) the 12-point (and under) size master drawings from the metal range of Monotype Garamond. Those originals (the smaller sizes) were very crusty and "antiqued" looking, and the rustic quality is carried through in the digital version. It won't clear up if you get a PostScript version because that is the way it was designed. Note: this is one of the 'Jannon' Garamonds, so if you want something smooth for display with this same feel, be careful about what you select. If you want a real Garamond, and it has to be smooth, straight and regular, I'd suggest Adobe Garamond. Get the original, not Premier Pro, because the new Premier Pro has some of this rustic edge quality as well, which I imagine would be obvious at large sizes. An option if you want a very clean, crisp Garalde is Galliard, which was drawn and digitized very cleanly. Scales up very well.

Yeah, that Newspaper page is ARMSCII-8 encoded, not Unicode. (That’s like pages still encoded Latin-1, Latin-2, etc. instead of Unicode: very common).

You have to change the Encoding in your browser, something like: View > Encodings > [Other > SE and SW Asia] > Armenian (ARMSCII-8).

Then it works fine. I’m such in love with the looks of the Armenian script!

While all the rest of the (granted, rather limited) Armenian fonts I have tend to have quite varied metrics, Mshtakan shares more than a few similarities with another font that I presume is well distributed (if not, as already noted by hrant, well designed). Mshtakan is black, Arial Unicode is blue.

[img:sites/default/files/old-images/armenian_comparison_5905.png]

The metrics are nearly identical, although Mshtakan (which as far as I know was designed to be more of a screen font) is bolder. But the x-heights, ascender and descender height are nearly identical. I’m not saying he copied it of course, but he has a very clear influence (although, to his credit, he did go for what, if I’ve learned well from hrant and I’m sure he’ll correct me if I be wrong, is a far better and more authentic հ and ն, and actually got the ք correct).

2012-02-18 17:53:04 • 15levels

Reuben

I could find no documentation (in a language I can read) on the subject. Using the Unix find command, I got that Mshtakan appears in two property lists in a subfolder of /System/Library/Frameworks, namely AppKit.framework/Versions/Current/Resources/NSFontFallbacks.plist and CoreText.framework/Versions/Current/Resources/DefaultFontFallbacks.plist in the subfolder ApplicationServices.framework/Versions/Current/Frameworks/ of /System/Library/Frameworks

The file DefaultFontFallbacks.plist is easy to read with a text editor and well commented. I googled for those file names and obtained only posts for Chinese, Japanese and Korean (for which the default font fallback mechanism takes the language into account), including a complete installation script (download MacOSXDefaultFontFallbacksChanger.command).

For Armenian, it seems to me that all you need to do is to replace Mshtakan with the postscript name of the regular you prefer (using sudo vi on them, for instance) and then reboot. I tried on OS X 10.6 with DejaVuSans (just to check) and it worked fine (so far as I can tell) but don’t forget to take a backup before making any modification. The font fallback files also allow choosing a different font for a sans, a serif, etc.

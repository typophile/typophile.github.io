1. To get localized font name, you should add entry to 'name' table for corresponding language. It will be displayed based on system langauge settings. 2. When displaying sample string, Office is first looking at font signature in OS/2 table. If it appears to be single-charset font, string for this charset is displayed. For other fonts, string is chosen based on current language. 3. Unfortunately, you can't customize this string. Thanks, Sergey

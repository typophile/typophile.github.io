2005-02-01 03:03:30 • andi emery
We use a water release inkjet printable decal to transfer designs onto shell as a template for cutting inlays. I've never used it as a decorative component, but it might be worth experimenting. You can buy small quanties from Bel.

I did once experiment with an ALPS printer and a gold transfer foil on water release, that looked nice, but the printer finally died and it was discontinued many moons ago.

As far as thermography, if the design is done right, I've seen amzing things done with rubber stamps and embossing powders.

Make sure you check any method for interactions with the particular finish on a test.

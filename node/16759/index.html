Among sans faces, Simian is in the same sort of space as Section, and very stylish. The display versions are a bit weird, but the text versions might work for you. More self-conscious but very well done is Cholla, which I saw used here in Washington DC at the Hirschhorn Gallery. Also a bit along the self-conscious line, but which I like better is Amplitude, which, like Simian is by Christian Schwartz. Jeremy Tankard's Aspect is both both architectural and avant-garde. If you like stuff that is somehow both modern and Baroque you will go nuts over Storm's faces

February 25, 2013 • laurent

Hello, I need to modify an existing font (OFL) to add a few characters (ç and Ç), to use it in french. I opened the .ttf font file in Fontlab Studio 5, duplicated the c with copy/paste, and added the tail. The generated file is 36ko while the original is 17ko. What did I do wrong ?
I just did something similar, just opening a font in Fontlab and then generating it again, resulting in a size increase from 70kb to 116kb. I then compared the two, using MS Font Validator, and found that some tables were added and some were deleted. Specifically: Added: LTSH, VDMX, hdmx Deleted: FFTM, GPOS In addition, the contents of some tables were changed. So, no, you did nothing wrong. - Herb

It seems that it will take some time for all the capabilities of Unicode to make it into common use. What's needed involves (a) operating systems, (b) applications program, (c) and fonts. I do know that what you want to do is easy enough with Windows Vista, running either Wordpad, Notepad, Open Office 3.01 (free) or MS Word in the newer releases. I do not know about Macs, nor have I tested Windows XP -- at least, not lately. OK, that's operating systems and applications programs. The font people have dragged their feet. Fortunately, SIL Charis (free & a pretty good typeface for writing manuscripts) has all the combining diacritics encoded. The combining diacritics really are needed with the Unicode system. So, I know the following will work, with Windows Vista, Open Office (& Wordpad & Notepad & some releases of Word), and the font Charis: Type a "c." (Make sure the "num lock" is "on.") Now, hold down the "Alt" key, and with it held down, using the number keypad, enter the string 0300 (Unicode for combining grave accent). release the "Alt key." The above paragraph is simply one way of directly entering a Unicode character from the keyboard with a Windows program. Likely there is a corresponding way to do so with a Mac. If you're using a Mac, you can test things out as well. It is easier if you can eliminate one of the potential problem areas, so start with a font that has the combining accents completed, and maybe even good *mark* and *mkmk* features, such as Charis. In sum, the font Charis is free (search for "download Charis") The Open Office products are free, and can even write off a .doc format. (Search for Open Office). For operating system support & upgrade, you're on your own. Good Luck, Charles

On the macintosh, I use the IPA-SIL keyboard that can be downloaded from SIL IPA Unicode Keyboards (IPA = International Phonetic Alphabet). The keyboard is also available for Windows and Linux but I have never tried it on those platforms. To type c with a grave accent with the IPA-SIL keyboard, just type c and then SHIFT 4 (c with acute is c SHIFT 2). You can switch fast between the American keyboard and the IPA keyboard by just typing "apple space". With Word 2008 for the macintosh, what I just described works fine with the Microsoft Calibri and Cambria fonts but not with Times New Roman; SIL fonts Charis and Gentium also have those accents (and others) and what is necessary to position them above the previous character (that always needs to be in the same font). To return to the young fellow mentioned in the first post: Croatian names typically end in -ć (e.g. Jagić, Maretić, Ivšić), i.e. c with an acute (not a grave). There is no such letter as c+grave in Croatian. Thomas

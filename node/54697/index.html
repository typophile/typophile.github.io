I scanned the book cover. I bought this book at the Spui book market in Amsterdam, and it’s written by Herbert Ihering. Here’s from the colophon;

© Rembrandt Verlag Berlin 1959 Zweite Auflage 1962 Klischees: Lemke, Graphische Kunstanstalt Druck: Franz Spiller Printed in Germany (West-Berlin)

So I guess that makes it older than Arial! This edition’s insides is set in Futura, don’t know about the first edition.

Thanks for replies, though! Personally, I think it’s a custom type inspired by the likes of Akzidenz Grotesk and Breite Grotesk (thanks Dag Henning).

Font Remix Tools update and videos

January 29, 2008 • Tim Ahrens

We have a new version of the RMX Tools! Have a look for yourself at [[http://www.remix-tools.com|www.remix-tools.com]]. The Remix Tools got some new features and a few extra tools. Also, I have produced some screencasts so you can have a look at what they do before you install them. The Font Remix Tools are a set of plug-ins for FontLab Studio that allow scaling of glyphs without affecting the stroke weight. You can use them to tune the width, height and weight of single letters interactively, automatically generate small caps, superiors, inferiors, numerators and denominators, create true condensed and extended versions, generate tabular figures with only a couple of clicks, and slant glyphs while keeping vertical tangents straight. Enjoy!
Not being a type designer, I can't fully judge the usefulness of these tools, but the results are very convincing. It seems you've added a wealth of time saving bits to the FontLab toolkit. And the videos prove it. Really nice work, Tim.

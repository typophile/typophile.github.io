There was a NovelFonts typeface from the 1994 Expresiv Ornate Font Collection called 'Carolissa' that looks like the medium weight of whatever font this is (see below). Of course, those fonts did not use the original names, and my notes mention a resemblance to Technotyp (p.215 in Jaspert's 'Encyclopaedia of Type Faces'), although neither 'Carolissa' nor Technotyp show the flat-bottom 'd'.

Except for the flat bottom on the d, this looks almost exactly like an old Headliners face called Cardiff. You might check with Joe Treacy at Treacyfaces/Headliners ( http://www.treacyfaces.com ). Treacyfaces aquired the rights to the old Headliners type library a while ago. Some of the old Headliners faces have been digitized, but not Cardiff as far as I know.

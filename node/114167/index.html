The capitals are set in [[http://www.myfonts.com/fonts/linotype/conference-lt/|Conference]] by Martin Wait MyFonts.com say that Conference's design date is 2004 but there is also an older [very close similar if not exact clone] font called [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=URW&ffam=URWColetteD%20-%20Regular&fstyle=&fsize=60&fid=d015a77c234bc4f0c8e9df71c4a41e59&wrap=2&text=ALBERT%20HEIJN%20BLOKKER%20C%26A|URW Colette D]] by URW with a copyright date of 1996 - I don't know which is the actual font origin. The ampersand /& and "abc" looks like Helvetica Bold to me.

Commercial embroidery of clothing has been done since long before other digital computer fonts. Script styles that were developed independently of computer fonts. See for example a font often called Embroidery Athletic Flair or which looks similar to the Harvard embroidery script except for the H, M and o. Variations such as Automotive Workwear Sport Script have the same o as the Harvard letters. There is probably a variation for medical uniforms. Modern embroidery machines can also use conventional retail fonts converted to a format usable by the machine. Don

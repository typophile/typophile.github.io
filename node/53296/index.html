Not a typeface per se, but some letters I’m working on for a client. I’d love some crit! (Weight, curves, widths and such) [img:sites/default/files/old-images/test_6417.png]

For a start: S and R are too wide. The head of R is too big and too wide. There is too much weight difference on the S, make it more monoline. The upper curve could be shallower. A is too narow. E too wide. K too narrow, the upward stroke should sit higher. What happened to the H, man? Wasn’t there enough space? It’s the narrowest letter in there, very awkward. The spacing is very tight. Not a problem at this size, but if it’s going to be set smaller.

Best, Jürgen

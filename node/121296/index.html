Type light might be what you are looking for. It is freeware for Windows and Mac (and an older version is also available for Linux), and is generally considered ‘easy-to-use’ (as far as font editors go). The freeware version has only basic drawing tools though.

FontForge is acutally as mighty as FontLab, but certain builds of it are very crashy. I work in it on both Mac and Windows, works good. If you have X11 or XQuartz installed, you can get a premade build here : [[http://fuuko.libferris.com/osx/packages/]]. Then just open X11, then open FontForge, should work nice (but not like a native app).

On Windows you can find a build here: [[http://fontforgebuilds.sourceforge.net/]]

I created the logo for the UEFA corporate body.

Ryuk is correct it was custom drawn from a sketch, with the modification of the E reflecting the U curve. The letter forms were very generic to start with, and whilst is was easier to create the straight version, the curved versions (which are the primary versions) involved a bit of optical correction. The F was modified later on the development to help harmonise the letterforms on the curve.

The Euro 2012 logomark and guidelines was created by another branding agency. All the fonts mentioned above do look as though they fit the style though.

Custom job to me. Only 4 letters so could be easily reproduced using any generic sans (you could even start with Helvetica). Some similars: [[http://www.myfonts.com/fonts/parachute/pf-beau-sans-pro/|Beau Sans]] (officially used during UEFA Euro 2012, very easy job with this one, only needs to recreate /U using the Beau Sans /U left half part and mirroring it then cutting /A horizontal bar and making /E top left corner squared), [[http://www.myfonts.com/fonts/layarbahtera/layar-bahtera-doomsday/|Doomsday]] (almost same work plus making all finials squared), [[http://www.myfonts.com/fonts/insigne/ashemore/|Ashmore]], [[http://www.fontshop.com/fonts/downloads/ars_type/ars_district_ot/|Ars District]], [[http://www.fontshop.com/search/?q=Sansa|Sansa]], [[http://www.houseind.com/fonts/chalet|Chalet New York 1970]], [[http://www.myfonts.com/fonts/typodermic/domyouji/|Domyouji]]

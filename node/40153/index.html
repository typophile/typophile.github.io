My esteemed colleague Richard Rutter did some research into the fonts installed by default in OS X and Windows systems and by applications like MS Office and Adobe Creative Suite. (Linux wasn’t included.) I know from my own attempts to compile such lists that they are error-prone and incomplete, but this is a good addition to the list of lists. If you can’t find a font you need for a general-interest Web site there, are you really trying? (Then again, you might go with Arial.)

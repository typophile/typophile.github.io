Can’t agree more with Dick, nothing really distinctive, could even have been done from scratch as it’s composed of the same basic graphical elements, a circle and a vertical stroke: ‘gaga’ = (/g + /a) x 2 = [(circle + stroke + circle bottom half) + (circle + stroke)] x 2 A standard alternative would probably be [[http://www.myfonts.com/fonts/itc/avant-garde-gothic|Avant-Garde Gothic]] ITC but probably with probably less amateurish/naive feeling.

Since you did day similar would be OK, and since I can't offhand name what they used, I'd suggest Bellini Bold Condensed for the I SPY part. Maybe some 'squooshing' would be in order for fitting, the way the book title was done. All the black lettering below that looks like good old Bembo. The orange type might be Bembo Bold, but it's hard to see details well in the sample. Another I SPY replacement might be LaBamba, which is a bit more 'fun' looking. "Spumoni" or "Elroy", would also be 'fun' fonts for the main title. - Mike Yanega

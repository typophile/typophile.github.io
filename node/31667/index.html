2007-02-22 13:52:21 • Bert Vanderveen
There are charts of restricted exchanges; one option would be to use a 411, 611, or 911- number, as those are restricted exchanges in most area codes in order to free up emergency and/or service features for the telephone company. eg. (204) 911-1234 would be a fake number because 911 in Manitoba is a strictly emergency-only number. Alternately, the form at http://www.telcodata.us/telcodata/telco allows you to enter an area code and see which exchanges are currently in use; pick one that isn't listed and you're guaranteed there's no numbers on that exchange. EDIT: I notice you're in Australia, and I'm not sure how the phone system works there, but my answer was good for a terribly North-American-centric answer to the problem.

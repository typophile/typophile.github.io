Use BBT Font on your Blogs - Press Release

August 24, 2010 • javascript

New standard being installed on latest browsers (some now and later), for example Mozilla Firefox and IE allows for fonts to be easily used on Blogs and Websites. This is the Web Open Font Format or WOFF see http://en.wikipedia.org/wiki/Web_Open_Font_Format

BBT font introduced in 2008 for altering the k/K letter shapes for spirituality has been updated for this new standard. Now you can use it on Blogs with easy code snippet called @font-face and some changes to the Blog CSS stylesheet.

This is a blog using the BBT font as an example: http://BBTfont.blogspot.com

2010-05-28 23:30:53 • Si_Daniels

Oldnick: Since the QWERTY keyboard was designed to SLOW DOWN typists (in order to avoid mechanical jams)

This is a myth. The keyboard was designed to avoid jamming at speed, not to slow down the typist. The layout enabled the user of a mechanical typebar system to type English text at high speeds by avoiding adjacent typebars moving in close succession. This is possible through analysis of pair and triplet letter sequences, and ensuring that these letters come from different parts of the typebar array.

In a mechanical keyboard, the arms connecting the keys to the lever mechanism need to be offset, so that they don’t interfere with each other; hence, the keys are offset. But note too that the typist’s hands tend to be slightly angled inward, not perpendicular to the keyboard, so the offset keys are also ergonomically superior. For example, my right index finger resting over the J extends in a straight line to strike the U key.

Most mobile device keyboards are two small and not responsive enough for touch-typing, so it probably doesn’t matter much whether the keys are offset or not.

2010-06-01 22:16:47 • Bert Vanderveen

Re. ergonomically designed anything:

Keyboards included.

There can be too much of a good thing. Hands are highly adjustable and — “A change is as good as a rest”. But the keyboard isn’t infinitely adjustable, and if it is designed to be exactly right, it will only be exactly right from one position and for people with the right size of hands.

We have (for example) an ergonomically designed set of pruning shears. They are very comfortable to hold when held as intended, but uncomfortable, difficult to use and possibly dangerous if not held that way. When working in the garden, as you can imagine, you have to reach over, under and around things, so it is quite inconvenient and tiring not to be able to change your grip on the shears. Other ergonomically design tools have the same problem. The designers have imagined there is one right way to hold them and and that is that.

2010-06-02 15:58:29 • Si_Daniels

The rows of keys are indeed offset on the normal typewriter keyboard to reproduce how mechanical typewriters are made.

The numeral and QWERTY rows are offset to one another so that the Q key is exactly halfway between the 1 and the 2 keys. Similarly, the ASDF and ZXC rows are offset so that Z is exactly halfway between A and S.

Those two pairs of rows are then offset so that A is halfway between Q and 2.

This way, since on a typewriter each key is connected to a lever that continues forwards, these levers, starting under the center of each key, are separated by a constant 3/16ths of an inch, each one from the next.

Starting with Q (as there is one unused position after the 1 key), the order is

QA2ZWS3XED4C and so on.

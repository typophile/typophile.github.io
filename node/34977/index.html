Simon,

it’s not only Apple that replaced sputnik with Euro in their codepages (at least most of them), Microsoft did so as well. The difference was that in Mac OS 8.5, Apple did some additional silly things like patch the printer driver to always throw in the generic constructed Euro logo whenever a particular codepoint from the codepage came across. Microsoft only changed the internal codepage definitions, so the changes affected non-Unicode apps in a stronger way (i.e. whenever the sputnik codepoint was used an Euro appeared), while Unicode apps noticed the change in a smoother way (simply the new Unicode codepoint U+20AC was noticed/added).

I think you’ll find this incorrect. There were some hack fonts made for a Win 95 euro support beta, but these were quickly replaced with a patch that updated codepages.

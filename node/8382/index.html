You have to go to the InDesign (Mac) or Edit (PC) menu, then choose Preferences->Grids. There you can set the baseline grid setting (used for the whole document).

You can define a starting point and an interval ("Increment Every"). InDesign "stretches" the value of your leading to the next factor of the interval. Say, if you set an interval of 13 points, 12 point leading will get you 13 points (13 x 1), but 14 points will become 26 (13 x 2). You should adjust the increment to your copy and use it for headlines or captions only if it makes sense (e. g. the caption is in the same size but italic).

Try a bit with the options available, it's worth it! :-)

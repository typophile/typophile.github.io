Borderlands 2 Hyperion Corporation Signage

October 29, 2012 • e_Rac

This is a compilation of glyphs used on signs related to the Hyperion Corporation in the game Borderlands 2. These are slanted oddly due to perspective of the viewer in the reference photos, and because many of the signs use slanted text that changes slant across each line. Differing sizes of text are because I left all text at the resolution they appeared in the references. The signs only use capitals, and the period is a square. Most of the font is pretty unremarkable, but hopefully the G is distinctive enough to be recognized.

The attached file is the unmodified image that these were crudely ripped from.

You are probably looking for [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=Dafont-1&ffam=Ash%20-%20Regular&fid=4f86452be532863658d4da5ad0bcc46b&fsize=36&text=EMERGENCY%20SUPPLY%20CACHE%20E.S.C.&wrap=2|Ash]] free font by Billy-Joe Lee [Cherrylane, 2007]

Word is word. You should be able to just open it in the Mac version of MS word (provided they have saved it in a format older than Office 2007)

Now, the issue might be that they are using a font that you don’t have installed on your machine. In that case, you either need to get the font, or they need to send it in a format like PDF with fonts embedded (or converted to outline).

You don’t say what “foreign languge information” you are talking about. Word for Mac does not support as many languages as Word for Windows. For some languages the process you are using is one of the best and most reliable ways. For others, it is completely unnecessary and simply opening the Word file on Mac OS and changing the text to a font that supports the relevant language will be quite sufficient.

The languages that Word for Mac does not support most prominently include complex writing systems such as Arabic and the Indic languages. But if you name the particular languages or writing systems, it should be easy to determine.

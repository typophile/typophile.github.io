2008-08-29 09:55:01 • Stephen Coles

If Neutraface is the display face, a modern (didone) would be a good text complement. Or possibly a non-geo slab serif, e.g. Archer or Bodoni Egyptian.

If Neutraface is the text, you could try matching a modern where the hairline corresponds to the weight of the text monoline. To see this effect, set up some 72 pt text in Scotch Modern Display in the type tester, and put the window next to a showing of Neutraface Light

But really, any true (“optically sized”) display face with lots of fine detail would complement the strong personality and sparkle of Neutraface text.

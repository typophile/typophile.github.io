Canadian Flag remix

So I got this commish … to remix our Canadian flag!! Going up on tees and whatnot…

…I think it’s strong and kinda funky as is. My deadline is in 3 days (lol). This version you see here is done with simple vector mono-thick line work but will get a lot more gestural from here … including drip style which I’m really really looking forward to doing!! But, there’s only one aspect of this design that keeps nagging at me, prolly something to do about nothing because I’ve been working at this for too long now.

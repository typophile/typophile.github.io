Interesting. If I was going to use a font to recreate the logo I would start with Mark Simonson’s Coquette Bold. Edit the o; using a horizontal mirrored lhs to combine with the existing lhs, and thereby create a new o without the loop. Use the result as the basis for the new c and a. Extend tails of all letters and reduce spacing so there is overlap. Trim top and cross bar of t. Reduce size of dot on i. Should be pretty close. Don

There’s a lot of great stencil typefaces out there, so you need to be more descriptive. You say “modern,” but that has so many senses to it.

Eames Century Modern has a stencil, but that might not be the modern you mean: http://www.houseind.com/fonts/eames

Contrived is also a bit vague. A stencil font is a contrivance. However, Dala Floda seems to avoid contrivance by making the thins so thin they disappear: http://commercialtype.com/typefaces/dala_floda

And then there’s all these stencils, which are very nice too: https://ourtype.com/#/all-fonts/ (Search Couteau and Bery Script)

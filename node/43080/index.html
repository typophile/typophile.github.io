OpenType website against Microsoft guidlines

March 12, 2008 • malcolm

I have just notice FontBolaget have a new website: http://www.opentype.se

Using OpenType in the domain name goes against the Microsoft trademark guidelines for OpenType. http://www.microsoft.com/about/legal/intellectualproperty/trademarks/usage/opentype.mspx

I wonder how long they will be able to keep that website going for.

What about opentype.com forwarding to Monotype!? ;-)

All the top level domains for Opentype.something are taken. I guess the main reason for Microsoft to register such a trademark is, that no one else can do it. I don’t expect Microsoft to sue all of the domain owners.

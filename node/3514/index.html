Super-/Sub-script in MS Word with Palatino Linotype

March 9, 2004 • ind

Hi, I need to let you know firstly that I'm a chemical engineer with a non-professional interest in typography. Dealing with equations, as I do, I often have need to use superscript (esp. for exponents) and subscript (esp. for indices). For purely subjective reasons I want to use the OpenType "Palatino Linotype" font, rather than the default in Windows ("Times New Roman"). I'm hoping that you're not cursing MS by this stage, because I curse it enough as it is. I need to use MS Word because I need to use a plug-in which (I think) is only compatible with that. Back to the problem. Formatting in any of the other fonts works perfectly. It is exclusively in "Palatino Linotype" that vertical alignments are off (for my purposes). I need the bottom of the subscript to be below the line of the main text, and likewise I need the superscript to rise above the capital letter height (sorry, I don't know the jargon). This happens in every other font, but with "Palatino Linotype" the superscipt and subscript in MS Word are set up to be correct for fractions, so that they neither drop below nor rise above e.g. the letter "X". (The easiest way is to try this out yourself.) I have contacted MS, who indicated that this is due to the fact the "Palatino Linotype" is their 'flagship' OpenType font, packed with all the features - and, of course, MS Word (or Windows?) is not yet compatible with all of them. It seems this requires MS to make a major overhaul of their software, and is not expected in a patch, or even in the next full version (but perhaps the one after!). My questions to you are twofold: (1) Given that I only need the plug-in while I am composing my prose/equations, can any of you recommend a (Word Processing) programme that will correctly handle equation-style superscripts and subscripts in this font? E.g. OpenOffice, Star Office? (2) Can any of you contact MS and/or start a petition to get the OpenType features implemented as soon as possible?! I realise that a lot of you may not regularly use MS products, but I do believe you would be doing a service to typography in general (and to me at the very least) by making these features more widely available to people in the 'real world' (no offence!). Thank-you, DIV

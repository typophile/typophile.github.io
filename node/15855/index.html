2005-11-03 01:29:12 • Hector Muñoz
It’s pretty easy to do, if you’re using Adobe Illustrator. First, set the text that will serve as the “background.” Select, copy and lock the selection. Next, Paste In Front, placing the second set of type directly over the first. Using the pen tool, draw the area you want to emphasize. Select the pasted in front text, and change it’s weight to something contrasting (for instance, if the locked background text is light, change the second set to bold, or italic, or bold italic, etc., so long as it contrasts with the first). Select the area shape and the second set of text at the same time, go to “OBJECT”, pull down to “Clipping Mask,” and choose “Make.” You will be amazed and delighted. Impress your friends.

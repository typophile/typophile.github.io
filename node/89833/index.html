Try using one of the sans serif (Kaku Gothic = 角ゴ) Hiragino (ヒラギノ) Japanese fonts that is bundled with Mac OS X. In fact, if you use these with the Simplified Chinese Hiragino fonts, the style should match closely. U+20B1D is mapped to Adobe-Japan1-5 CID+17340, which is included in all Hiragino “Pro” Japanese fonts.

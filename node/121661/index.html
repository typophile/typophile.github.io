Tal - a numbers font I'm working on

November 29, 2014 • andreaslarsen

I'm creating a new font because I needed some numbers for an app and couldn't find any free font that I was satisfied with. It's inspired by Handbook, DIN and the like. The letters all have same the width as I need that for the interface I'm creating:) Will probably open source it when done. Hoping to hear your feedback so I can improve it. I've created the thinnest, regular and black from scratch - the rest are just interpolated. [img:sites/default/files/old-images/minfontdemo_5415.png]
I think it’s looking good for the purpose. A couple of annotations: - Even if it’s a geometric design, I think you should address the top-heaviness of the |8|, since you already made other optical adjustments like overshot. - Personally I find the heaviest weight a bit jarring in the open glyphs. - Not a fan of the bump in |6| and |9| either, where the stem joins the loop.

Hi, I'm reviewing a font with a lot of styles and I have to remove In Point BCPs. I don't want to apply the Remove Overlap action to the entire font because I don't know what FontLab could do (I really want to be sure of every single action working with FL). Anybody know a script that recognizes glyphs with such problems so that I could work on them directly? Or something that solves the problem only where it is necessary? Best, Michele
Hi Michele, this uses RoboFab and calls a "Merge Contour" command to clean the glyph. Optionally the script can color the glyph instead of calling the Merge Contour so you can edit the contour by yourself. #FLM: Sanitize BCP # scriptype - Paolo Mazzetti - 2009 from robofab.world import CurrentGlyph from FL import * g = CurrentGlyph() for contour in g: try: for p in contour.bPoints: if p.bcpIn == p.bcpOut and p.bcpIn == (0,0): fl.CallCommand(fl_cmd.ActionRemoveOverlap) #g.mark = 1 except: pass g.update() (Indentation won't probably make it) Cheers!

Norwegian Type designers

February 1, 2010 • breckon

I’m compiling a list of norwegian/scandinavian type designers, (particularly contemporary) - any suggestions/favourites much appreciated…

Magnus Rakeng has made typefaces for Telenor and [[http://www.melkeveien.no/portfolio/index.php?pid=696&cat=portfolio&offset=0&sstrn=&scat=15&custid=|Statoil]]. Torbjørn Eng has made a revival of Gerhard Munthe’s great but forgotten [[http://www.identifont.com/show?MRF|Frisianus]], Jacob Øvergaard has made a [[http://www.identifont.com/show?MRT|bunch of typefaces]], and Magnus Bjørk has released a [[http://www.identifont.com/show?MRV|typeface]] too.

I really don’t know any other contemporary Norwegian type designers. Norwegians don’t understand type. There’s still a weird monopoly on setting type in Norway, which results in typographers knowing very little about design (and these days, very little about typography too), and designers knowing very little about type.

Sweden has a lot more type going on, but I’ll leave that to a Swede to elaborate upon.

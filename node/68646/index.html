I’m trying to determine the manufacturer of a font I’ve used in a particular project on behalf of a client. They would like to purchase a license and use the same exact font in designing materials in-house. Assuming you don’t have font editing software, if you’re using Windows XP+, you can right-click the font and get the copyright information under the “Details” tab." That should set you in the right direction…

…and if you’re on a Mac computer, select the font file and press the “Command” and “I” keys to “Get info”… You should see the manufacturer’s name listed under “Version.”

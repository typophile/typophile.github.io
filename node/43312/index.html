2008-03-26 18:58:38 • Typography.Guru

I wonder if there are tables that can be removed from the fonts to make them less usable if extracted?»> I could imagine browsers making use of fonts without using the OS/2 table

Missing an OS/2 table will not prevent Macs or Linux boxes from using the font. But it probably will make Windows unhappy.

Depending on how the glyphs are rendered, you might also be able to do a similar trick with the maxp table This table contains important information for truetype instructions. The glyph count can be figured out from the size of the ‘loca’ table, but the other information cannot be. I don’t think the font would work in a browser even. If it’s got a CFF table rather than glyf then ‘maxp’ could probably be omitted.

Tables: ‘CFF ' – Can’t be omitted, contains outlines. ‘loca’/‘glyf’ – Can’t be omitted, contains outlines.

‘cmap’ – If you omit this you’d have to supply a GID stream on your web-page rather than text. I don’t think that’s a good idea (you’d have to extend html/xml somehow, and the w3c wouldn’t go for it because you could only display with a particular version of a given font – no speach synthesis possible).

‘head’ – Supplies unitsPerEm, without this table there would be no way to figure out how to scale the font.

‘hhea’/‘hmtx’ – Omitting these would break a truetype font, but might be ok in a CFF font (the advance width is stored in the ‘CFF ' table as well as the ‘hmtx’ table.

‘maxp’ – Omitting would probably break a truetype font, might be ok on a CFF font.

‘name’ – This could be omitted without breaking the font. But since that’s where license info is stored it might not be a good idea. (FreeType will display a font without a ‘name’ table though)

‘OS/2’ – Omitting might make Windows unhappy, but is ok for Macs and FreeType

‘post’ – This could be omitted without breaking the font. (Again, FreeType will display a font without a ‘post’ table).

So the ‘name’ table seems about the only thing that contains non-essential information and might still break the font as far as the OS was concerned.

I expect both were custom work. THOR is a classic example of “stovepipe” or “gaspipe” lettering, which was usually worked out in a modular form on squared paper. Letterhead fonts have one called [[http://www.letterheadfonts.com/fonts/unionthug.php|Union Thug]], but it is currently only available as a free gift for spending $99 or more! Have you tried drawing this on a grid in Illustrator? [[http://new.myfonts.com/fonts/adobe/rockwell/|Rockwell]] was a fairly common slab serif at this time. It would be quite easy to modify that to get the TWIN MOTOR.

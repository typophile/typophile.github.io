December 17, 2006 • Pierre-Luc Auclair

Hi,

I’ve just updated an old logo for a project I’m in.

[img:sites/default/files/old-images/zap_logo_2_5568.png]

The aim of the project is to provide free wireless access in Québec city.

The look I’m aiming at is a mix between 40’s illustration style and web 2.0, so something humanist with a modernist twist, or the other way around. Actually the less the modulation, the better I guess, but I still want some. ;)

The type in the logo is handmade, and I’m looking for a font to go with it. So far there are my choices after a bit of research:

• ITC Johnston — Like it but unsure. • URW Grotesk — Dinka like this one too but I just dislike the “a” and I’m not sure it would read well. • FF Din — It starts being a bit far from the logo but it would be interesting to use it.

So far my favourite is Johnston.

If you have any ideas, go for it.

The tail of the ‘Q’ could/should be more visible.

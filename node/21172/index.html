Script companion for Adobe Jenson

July 8, 2006 • andrewh

Hello to all! I'd like to use a good companion script typeface to pair with Adobe Jenson (main text face). It's for an invitation. Any suggestions are greatly appreciated. Thanks, Andrew
LTC Metropolitan Italic Alt was based on Beatrice Warde's 1925 Arrighi design -- you know, the one that was 'tamed' and then paired with Roger's Centaur? -- and therefore, can be set alongside the likes of Adobe Jenson. You might, however, want to use the same roman for setting the main text. Or P22 Operina if you like to play rough. It was also based on Arrighi's italic, by the way.

If LTC Metropolitan is not available in an opentype format, or if you don't intend to use Metropolitan Roman to set the text, I would totally go for Poetica -- simply because pairing two Venetian faces, one roman and one italic, is one tough act to pull. By the way, what was Poetica based on? It definitely have sharper hooks and more complicated (but rather conservative, in comparison to Arrighi's) ligatures and swashes. Was it based on a more contemporary italic?

Looking for a way to automatically organize my fonts into sans serif and serif sections

September 6, 2011 • nhood

Hello, I'd like to be able to look through all serif or all sans serif in my font management software. Is there a way to separate my fonts into these two categories other than manually separating several thousand fonts into two sets? Ideally, id like to be able to do this with Font Explorer X pro 3. Any input is much appreciated. -nick
I separated 1700 fonts just the other day by hand in that same program. Took me maybe 30 minutes. I just went to Classify Fonts>Categories>made custom categories and started clicking away. It shouldn't take THAT long. When that was all done I created Smart Sets in the left sidebar and it added everything to them. Worked wonders. Though, if there is a faster way I'd love to know as well.

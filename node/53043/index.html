Hello, I currently design a Wordpress-based, a bit old-fashioned yet classy looking theme for my blog and would like to use a font for the headings which carries and extends this feeling. I used Georgia rather extensively so far and would like to use a different type for the headings which should look a bit more stronger and firmer. I remembered that the $- and the €-signs have their extra-lines for representing just these characteristics and vaguely remembered these types which contain capitals with a second stem. I attached a demonstration (actually modified Georgia-letters) of what I mean. I am not sure whether this is the right forum (Pardon me, if it's wrong...); but I would like to know if there is a particular term for this characteristic and would be glad if you could list some types providing these second stems. Thanks in advance! RoyBasch PS: Don't mind the internet's restrictions, I would use this type to generate an image for each blog entry.
These kinds of faces are known by many different names: openface, inline, engraved, handtooled, and others I'm forgetting. ITC, for example, issued a series of "handtooled" versions of some of their popular faces (Goudy, Century, Garamond). http://www.fontshop.com/search/?q=handtooled&x=0&y=0 There are more contemporary faces with the same treatment, of course. FF Scala has an engraved option, for example: http://www.fontshop.com/fonts/downloads/fontfont/ff_scala_jewel/ Font Bureau's beautiful Narcissus uses this effect: http://www.fontshop.com/search/?q=narcissus&x=0&y=0 And there are dozens, if not hundreds more...

For “AM/FM” you can try [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=T-26&ffam=CorpusGothicAlternative%20-%20Italic&fid=4411bfd0747b1ca4cef2477e0d96a7cf&fsize=72&text=AM%20FM&wrap=2|Corpus Gothic Alternative (Italic)]] by T-26

for the numbers, the best I can find is [[http://www.p22.com/lanston/obelysk.html|Obelysk Grotesk]] by P22, but I don’t know if this is close enough for you

November 25, 2009 • lgndsgn

Hello, I have set all text in a text box to Align Right yet it is not cooperating. I know the trick of fixing the spaces if it is set to Align Left (add space before and decrease kerning, or drop cap), but I don’t know of a trick to fix this if set to the right. I would hope that I wouldn’t have to seperate each line into a seperate text box. Perhaps tabs? I must be overlooking something. Any help will be much appreciated.

Here is a screenshot of what I’m trying to do: http://imgur.com/TuDa9.png

As you can see the ’m' and ‘h’ are not aligned exactly to the right.

One option is to fiddle around with the optical margin alignment (Type > Story).

If you need absolute control, you can insert tabs (1 at the beginning of each line) and set the tabulator to ‘align right’ (the text box itself has to be left-aligned). Now you can adjust the tab position for each line, if necessary.

There are a lot of solutions. F

> On the other hand, hmm, would it by chance be possible that the Adobe Serif Substitution Font as you mention, would have been chosen “deliberately”? Not possible because it won't show up in any font selection menu. It virtually is a 'ghost font'. > Or not at all? In other words, are these Adobe Master Susbstitution Fonts exclusively for being “called up” when the original font in a document is missing, (…) Correct > or can themselves as well be chosen to be the original font from the start, just like any other font, or?... No, because this is not 'any other font' -- it's an automatic substitution font whose characters and letter widths are squooshed and stretched to perfectly fit another font's widths. > Also, would there be any website you could please refer me to, where I could view the charmap of the Adobe Serif Substitution Font, and as well the italic version of it? No, there isn't, for the reasons mentioned above. This font was never intended to be used on purpose.

yeah i came across a similar problem when i worked on an online literary journal. i find saving postscript files and converting them with distilller helps a lot. you can even create your own compression script (called .joboptions i think) in distiller to specify just where you want to 'squeeze' the compression rates and whatever. just go into distiller > settings > edit adobe pdf settings and a dialog box comes up with options. then just make sure when you save as you call it something different to the joboption file you based it on. if you've never used distiller before it's not very complicated. after you've saved the ps file, open distiller, choose the setting from the drop down box (which you should've made a new setting for, or else pick one of the defaults) and drag the ps file into distiller. voila. i garauntee it'll be smaller than the same file as an exported pdf.

Saving to the smallest filesize as Quincunx suggested is a pretty good start. You could also change the compatibility to Acrobat 6, it makes the file size a bit smaller, and most people have Acrobat 6 by now. Changing compression options at the Export pdf dialogue window helps, too (from the small file default 150 to 72 dpi, for example) It all depends what your needs are, the Photoshop save for web option isn't a bad option, either.

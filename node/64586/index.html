2009-11-29 23:55:09 • Dash_X

No.2 is a techno style that is similar to [[http://new.myfonts.com/fonts/t26/21-st/|21st]] from Identikal, and [[http://new.myfonts.com/fonts/dsgnhaus/saturn/|Saturn]] from 2Rebels. Saturn is more rounded, like Eurostile with the stem extensions removed.

[[http://new.myfonts.com/fonts/fw-identikal/seize/|Seize]] from Identikal is also similar, but would need some stem amputation to get closer.

When a techno style doesn’t show up readily at MyFonts, I usually check the Techno>Sci-fi category at Dafont.

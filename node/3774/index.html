2004-03-25 08:45:12 • andrew_fall
  1. Allocate equal space for each language, but always allow for the longer (French)

    2. Use a different (though harmonious) type style for each of the two languages

    3. Don't be afraid to use rules to organize and separate items, and background tints to suggest hierarchy: although the tendency with modernism is to remove non-typographic elements, that's not the best strategy for complex forms.

    4. You won't find this kind of work in awards shows/annuals (it's not Hollywood enough) However, it is amongst the purest and most demanding aspects of graphic design. Take a look at some of Erik Spiekermann's forms design – he obviously loves and respects this kind of work, and even manages to bring some style to it. I really don't see how Shannon is whining.

–

I think there are two possible strategies: make the two languages visually as close as possible; or apply some cultural sensitivity and risk the wrath of the cross-eyed Modernists.

The first option is pretty easy. But it might result in the functional quicksand like in Emil Ruder's monolithic (in contentually great) textbook.

The second option might benefit from using totally different fonts for each language. Ladislas Mandel for example insists that French type needs to have a smaller x-height. This would actually help in the technical realm too, since French will generally run longer (like Nick says). You could also be more conservative and simply apply width (friendliness versus elegance) by using an MM font with a width axis, and making the French font slightly narrower, which would also help with balancing the space usage. Note however that smaller x-heights and narrowness will make the French have a smaller apparent size, and some people might interpret this as "less important" (although others might see it as "discrete", which fortunately happens to be in cultural character). One thing you could do to counter this is put the French first, like on the left or on top. You could also use a bit more leading to make it seem "larger" in a different way.

BTW, there are some great articles in old issues of the journal Visible Language about multilingual typography.

2004-03-27 06:45:38 • Nick Shinn
Although [literal] translation adds about 10-15%, some languages are still more verbose than others. English is generally very terse, while Latin languages (except Latin itself!) are more verbose.

It's rare to find good benchmarks of language economy, but a handy one is Zhukov and Sadek's "Typographia polyglotta", which shows official, very tight, but non-literal versions of the UN Charter in 20+ languages. It shows that the main Latin languages run about 10-15% longer than English. So when you couple that to sloppy translation you can get a huge expansion. When Jorge de Buen translated my "Improving the Tool" essay (~2,500 words) into Spanish and got a gain of only 14%, that was pretty impressive - the UN Charter is 17% longer.

BTW, an Armenian from Lebanon has four "unofficial" languages! :-/

2004-03-28 04:12:12 • Hector Muñoz
Nick's recommendations are excellent and that's
how I usually go about it (Belgium has two main
official languages, Dutch and French, so we have
to do bilingual stuff quite regularly). I had to do a
newsletter in four languages once (Dutch, French,
English and German) and I got away with it by using
ITC Officina, as it has four distinct variations (even
to the non-typographic eye): Sans, Sans Italic, Serif,
Serif Italic. I designated the styles according to
"verbosity": German was set in Sans Italic whereas
on the other side of the spectrum Serif was used
for English. By keeping identical positioning of the
languages in relation to each other (they always
were grouped in blocks of four) and by using subtle
colour coding it worked out all right.

The thing I'm most proud of is (was) a bilingual
magazine about theater, not the artistic but the
technical aspects.
All pages throughout the magazine were cut in half
horizontally. Top half Dutch, body copy set in FF Scala;
lower half French, with a tinted background, set in
FF Scala Sans (it was the pre- FF Seria days). All
titles in semi-deconstructed setting (to allow
bilingual setting) along the division line. Images
clustered as much as possible along the divison
line as well in order to emphasize the horizontal
forward motion of the magazine.

Unfortunately I never got the time nor budget to
fully explore this, but when it worked, it worked.
It got copied by at least two other magazines.

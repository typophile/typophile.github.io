The TrueType format has been integrated into OpenType. Each Windows TrueType (.ttf) font is already an OpenType font, more specifically, an OpenType TT font (OpenType font with TrueType outlines). All such fonts work in both Windows and Mac OS X.

2010-02-16 16:19:07 • FontGear
TTX can be a help, when you want to avoid damages during a conversion from TrueType to OT. First convert it to PS OTF (with FontLab for example), then decompile the tables of the TTF and the generated OTF. In the decompiled TTF you have to make a few changes, namely in the name records, because there is a slight difference in the specification. And you have to remove a few tables, that are not in OTF, and you have to add the CFF-table. Then you can recompile it.

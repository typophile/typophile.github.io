Hi Zekeman,

this is a school handwriting typeface, American ‘Print’ style. You can get this one from schoolhousefonts.com. Choose ‘Schoolhouse Set A (Zaner-Bloser style)’.

There are several free fonts of similar style: Print Clearly from BlueVinyl, or Primer Print from [[Ray Larabie]].

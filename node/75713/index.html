AFDKO Autohinter: 'Error: font has neither StemSnapV nor StdVW!'

October 28, 2010 • brianskywalker

So I am trying to use the AFDKO Autohinter…

I generated an OTF with Fontforge and tried to use autohint, which gave me this error:

‘Error: font has neither StemSnapV nor StdVW!’

I looked in PS Private and StemSnapV was already there in my source file, but many of the fields were removed in the OTF. I think I might try generating a ps file in the morning instead and see if that fixes that problem.

But what goes in StdVW? Fontforge doesn’t automatically put the info in there for me. (:

StdVW is the first value of StemSnapV. Basically, StemSnapV is a list and StdVW is the first, most important value.

Also, there is a commandline switch (-nb) in AFDKO’s autohint to disable the requirement for those values.

Non-Continuous Line Segments (Illustrator CS2)

May 3, 2006 • AndrewSipe

I’m working with an AI file I downloaded off the Adobe Exchange Server, it’s a map of the United States. Very accurate with every state individually outlined, but as I work I’m noticing a few of the states don’t have a continuous outline stroke. Because of the intricate nature of the state’s shapes there are easily a thousand points used to create this outline stroke. That’s a thousand points a state (give or take a few hundred).

Is there a way that I can automatically highlight the beginning and end of a continuous line segment to make it easy to see where the breaks are in hopes of completing the path?

Try adding a fill along with the stroke or using the direct selection tool.

Ignore the above, this is easier…

just apply a brush style from the arrows selection. The start and end points will become obvious as they’ll be at the head and tail of the arrow . Remember where the gap is and remove the brush style using the appearance palette before joining the gap.

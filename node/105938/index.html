Isn’t that font, Vilna? There is an open-source Unicode font that emulates Vilna that you can learn from – Mekorot-Vilna. The Hebrew range in the Linux Libertine font also is a Vilna style font.

You’ll find both of them in the Open Siddur open source and Unicode Hebrew font pack.

Note there are some issues with diacritic placement with this font which needs fixing.

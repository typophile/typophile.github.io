What fonts are required in THE Adobe APPLICATION SUPPORT FONT FOLDER for Indesign to function?

I would like to be able to move Opentype fonts into my System Font Library so they can be used by other non-Adobe applications.

If you are talking about the fonts here:

Library –> Application Support –> Adobe –> Fonts

You need to leave all the fonts in the “Reqrd” folder alone and in that location. All the others you can safely move. (Or at least I safely moved them.)

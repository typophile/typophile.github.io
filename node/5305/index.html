2004-06-22 17:24:44 • typotect
I think Souvenir was a well-designed font with a strong flavor. But it got so popular for a while that designers just got tired of looking at it. Now that you don't see it so much anymore, I can see it making a comeback, but it will probably never be as big as it once was (at least as long as boomers are around). It's funny that you mention "technology inspired design." Souvenir, which was originally released as a foundry face in the 1920s but never caught on, was revived in the 1970s because it held up well on phototypesetting equipment, which tended to soften corners and such.

Remapping of enter/return key on PC

January 20, 2005 • antiphrasis

I received my new HP laptop this weekend which features a widescreen and a full-sized keyboard (incluing a numerical keypad), and I've run into and interesting/annoying quirk. It seems that the return/enter key works exactly like the enter key on the numerical keyboard. On most applications this doesn't matter too much, but using Adobe applications I've run into some problems. In Adobe inDesign CS when I hit return I get a column break [num enter] instead of paragraph break. I also get column break when I hit enter on the numerical keypad. According to the menu the paragraph break should be [enter]. I do get the paragraph break if I hold down the FN key while hitting enter, but it would be nice if I didn't have to do that every time I want a paragraph break. So my question is: is there a way to remap the return/enter key so that I don't have hold down the FN key to get a paragraph break? I'm using Win XP home, and the computer is a HP Pavialion zd7000.
Try SharpKeys. It sounds like remapping "Num: Enter" to "Special: Enter" would solve your problem.

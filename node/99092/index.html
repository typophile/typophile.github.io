One common trick is to type/paste your string into the OpenType Features tab of the Preview panel, activate the feature (in this case SMCP), then copy the slash-coded string from the bottom of the panel into the Metrics window. If it's something you're likely to reuse, you can also add it to the list of preview strings (the Options '...' button in the Metrics window or Preview panel). Or, if you happen to be using Windows, the [[http://www.fontlab.com/front-page/featured-product/introducing-fontlab-studio-5-2-for-windows/|FontLab 5.2 Public Preview version]] supports OpenType features in the Metrics window.

what helps a lot when doing metrics of small caps etc. is to setup a basic string like /H.sc/H.sc/H.sc/H.sc/H.sc/A.sc/H.sc/H.sc/H.sc/H.sc/H.sc/ then click the glyph you want to change, and use CRTL+mousewheel to switch to next glyph. No tedious typing involved. There are also keyboard shortcuts for it, but they seem to differ in platform, version and language of your keyboard ...

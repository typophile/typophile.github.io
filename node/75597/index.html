conventions for crediting typography in a book

October 24, 2010 • joslog

does anyone know if there are conventions for how a font should be credited at the back of a book? (ie, This book is typeset in xxxxxxx).

I’m wondering if it’s common to list the digital foundry of the version used (if there are multiple similar versions out there) and also, if the original foundry should be included (ie, should Akzidenz Grotesk be credited as “Berthold Akzidenz Grotesk”…)

Such listings are called colophons…

A lot of times, the fonts will be listed on the CIP page, along with the book designer, the typesetter, and the printer (or just the line “Manufactured in the United States of America,” for example). When fonts are listed here, usually nothing much is said about type – occasionally, it will include the foundry, e.g.,

Set in Carter & Cone Galliard

When more detailed information is given, as Nick says, it would be in a colophon at the back of the book. Sadly, that’s a disposable page if required to make even forms. The CIP page is always printed.

2010-10-24 20:11:39 • charles ellertson

I think that the foundry name should be included when it’s a key part of identifying the typeface. For example, one should never write in such a place simply “Garamond” or “Caslon” but include the foundry to identify which version one used, because they are often quite different.

When it’s not critical in distinguishing the typeface, I think it’s still a pleasant courtesy, as is identifying the typeface designer.

Also, any book that identifies the book designer should also identify the type designer(s), however, just as a matter of fairness. Also the jacket designer and/or artist….

Is it for a letterhead? To make the image a background image, you can do View > Header and Footer then Insert > Picture > from file . . . choose the logo file you want to use (pngs work best for word I think) and then click ok.

Then to get the image to go behind text you right click on the image in the document and click Format Picture . . .. That will bring up a window where you can edit how the file is displayed in the word doc — size, layout, etc.

Hope that info helps.

JPG does not support transparency. There is an updated JPG-2000 which does, but it’s barely adopted. GIF supports transparency, but lacks quality. So, the choice is to use PNG.

In IL CS4 you have the option File > Save for MS Office. It saves your artwork as a 24-bit PNG with transparent background. The same could be achieved through the command penn.and.ink indicated. Just check if the transparency is enabled and be sure your background is removed:

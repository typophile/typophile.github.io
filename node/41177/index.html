From my history there are these: http://typophile.com/node/30432 and http://typophile.com/node/20517 As far as Tutorials: the best I can suggest at this point is, using simple lines and the Object>Blend tool, you can achieve some rather basic but similar effects. layering these effects can create a more complicated looking design. Also, you might be able to use the pattern brush to create a border as well. It'll just take some experimenting to get the desired effect. Also, remember lines in a blend can be manipulated (either adding points, curves or even changing the position.) Hope that helps.

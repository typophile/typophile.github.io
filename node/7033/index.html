I've read of people doing it, there have been programs written to do it, so yes, it's possible. But I have never done it, and it sounds…somewhat painful. If I understand it correctly, you need to translate a binary font metric file into ASCII (PFM to AFM).

I've heard that you can find the AFM file for almost any Postscript printer font at Adobe's FTP site, though I've never been there to confirm it. It would probably take some serious digging to find, but it's out there somewhere.

If you have a p2p application, do a filename search.

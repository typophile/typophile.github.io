A little detective work: In this font, the Bounding Box goes 700 units below baseline, 1002 above. That's 702 units more than the 1000 unit em. The height accommodates accents above capitals, and is fairly normal. The depth accommodates some ornaments -- vertical border elements. The lowest typographic elements (commaaccent) are maximum 277 below baseline. So the ornaments in the font are responsible for its large line spacing.

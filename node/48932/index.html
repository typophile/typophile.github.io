Hi, As far as the plots go, it might be an idea to use gnuplot instead of MATLAB, and use the epslatex option terminal to produce the output, which renders all text (including number on the axis) natively in LaTeX. As for the PDF conversion, your LaTeX distribution might have come with pdflatex, which converts latex directly to PDF. Are you using MiKTeX? You might also look at http://www.toolscenter.org/ for a pretty good IDE that should automate the process for you. I hope this is useful.

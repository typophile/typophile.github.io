Hi Team Type, would anyone hazard a guess at this font? Thanks in advance for taking a look. Gracie [img:sites/default/files/old-images/castle_5549.jpg]
Looks like an old (1992) Image Club Graphics typeface called Aurea Titling. Sadly, it doesn't seem to be commercially available. It was based on/a blatant copy of Aldo Novarese's Augustea (c. 1951), which only seems to exist in an outline version in digital format: http://www.myfonts.com/fonts/efscangraphic/augustea-open-sb/alternate_cuts.html Note: Augustea BQ (Berthold Foundry) is not the same typeface. A classic font that comes very close--dagger serifs and all--is Vendome: http://www.myfonts.com/fonts/linotype/vendome/

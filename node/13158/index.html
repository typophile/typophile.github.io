I found my CD. its been through hell and back, so bare with the image. According to last thread (Funky Courier Similar to Rough) many users thought the type was hand lettered.

I don’t think the type is hand lettered because the type is used in the liner notes fairly frequently. So lets take another stab at it.

This time with better photo reference.

The reason people think it’s hand-lettered (and I still do) is that the characters are not consistent. It is possible that the entire thing was done by hand.

Look at the two l’s in collection for example.

It’s also possible that someone typeset this initially using an existing font and then traced over it by hand to get the rough effect.

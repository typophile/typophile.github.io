I take it what you are looking for is something like this: [img:sites/default/files/old-images/CAMPGROUND_5319.jpg] I have not seen a legitimate font with this treatment of Koch’s masterpiece. Nor have I come across a hacked version with it. Easy to make. Purchase an inline font that permits editing, use the parallel lines feature of a font editor and some minor tinkering, and voila you have it. Don’t know if this one permits editing, but it’s a fair inline version: NewtonInline. Don

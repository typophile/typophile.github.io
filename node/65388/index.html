David: Maybe I need to be more specific… Since I don’t read Hebrew, I feel that I need to be cautious in picking a font. Something that looks elegant to me, might look like a silly letterform to someone that is familiar with Hebrew. It’s not really a problem, that I can’t read it, but more of an issue with cultural nuance.

Hrant: “Hebrew wave” is definitely not an exact term. I use it to describe the stroke flourishes that cap the letterforms and indicate forward movement. It seems that the more traditional calligraphic forms have wavier forms and that the modern calligraphic forms are more subtle. See Margaliot as an example of the wavy forms. All this being said, maybe the traditional calligraphy would be a better match to Centaur.

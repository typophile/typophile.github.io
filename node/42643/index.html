Is there a name for this style of typeface? I see it a lot lately in magazines like idn. I’m wondering where I can get it aside from some of the shareware ones I see that have a few problems.

Ladies and gentlemen, may we present, all new in 2007, it’s the Fat Font Fad!

An ’80s throwback, counter-filled, constructivist, geometric, sausage revue, starring:

Black Boton Bold Frank Deuce FS Pele Mostra Black Blocks Alpha Bloc Sculpture FF Extra Unovis Neutrino Toch NB Block Acton Bebit (Cyrillic version of Milton Glaser’s Baby Teeth) Parkitecture Wagner Silhouette Rody and Lazar Cooter Douff Chumbitos Mostra Black Kink Ultrafat Pound P22 Constructivist DF Strand Doll

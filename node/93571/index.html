Jon, not an ID, but: This has a lot of interesting character traits, but it’s also very sloppy in terms of curve quality (and this has probable implications like incomplete character set, poor onscreen rendering, etc.). Unless you actually like it for this “fauve” quality I might recommend finding a more polished font that has a comparable style; and if you go with it you might have to have it worked on to make it adequately usable for such a prestigious project.

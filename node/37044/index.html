It’s easy. The source code on similicio.us references a css style sheet in this line:

link href=“sim.css” rel=“stylesheet” type=“text/css”

So, simply add sim.css to the end of the url, like this:

http://similicio.us/sim.css

Then, find the “h1” style attributed to the font you’re looking for.

Viola!

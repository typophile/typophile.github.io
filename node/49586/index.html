There are out of date and more web focused, but may help you get oriented with the basics.

Some quick tips:

Almost all body text is justified.

You can set up composite fonts in InDesign to use Western and Japanese fonts inline together, with InDesign making automatic sizing/alignment adjustments based on rules you define.

There are rules for widows, maybe too much to explain here, but if you search around on the web you should be able to find them (maybe search for something about japanese writing rather than design)

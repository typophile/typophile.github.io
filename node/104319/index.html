GURU A new release by Novo Typo

July 3, 2013 • Novotypo

GURU NT Guru is a multiple layered typeface especially designed for display use. To create beautiful typographic designs the possibilities are endless by using layers and colors. A beautiful set of arrows and multiple language support makes Guru complete for any unique and ornamental design. Available on July 15th 2013 via the website. www.novotypo.nl Nt Guru is a set of 5 fonts designed by Novo Typo (typo)graphic designers from Amsterdam - The Netherlands.

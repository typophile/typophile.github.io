Nick,

The measurement line sometimes goes far away to the space on both platforms. To make it visible again use the Python script below. Open your font, open the Edit Macro panel, put this small program there and click on the Run Macro button.

fl.font.measurement = 400 fl.UpdateFont()

These two strings will set the Measurement line for the current font to 400 units.

Regards, Johnych

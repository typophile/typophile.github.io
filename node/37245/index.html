Font for ghost stories

September 19, 2007 • ebensorkin

I am probably going to be setting some text for ghost/paranormal stories some time in the next 3 months in a new Lit journal so I thought I would start looking for a suitable text face for it. What I have come up with so far is Stormtype’s Walbaum. It feels victorian and the words ‘ghost’ & ‘scream’ look great in it.

http://www.stormtype.com/typefaces-fonts-shop/families-52-walbaum-text

It’s early days yet and I feel far from stumped - but still, I thought it would be a fun thread: What would you use & why?

You get to set ghost stores in Storm Walbaum? I am so jealous.

It would be a grave mistake to overlook Lestatic as a display type. And of course, the old House Monster fonts collection is to die for.

If they’re good ghost stories, they’ll be plenty scary without any help from the typeface—so I think it’s best to keep the effect of the text face subtle. Caslon, Storm’s Walbaum, Trump Mediaeval, I think they’d all work. Baskerville too. I might even try Elysium to see the effect. It’s a little heavier and edgier than many text faces. I once used it to set a novel about the ultimate solution to bad drivers: nail them to crosses overlooking the Don Valley Parkway.

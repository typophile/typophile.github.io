I don’t believe it’s possible to do that in any of those programs. Fireworks is the most capable of the bunch, and unless they just added the feature, it doesn’t let you do type around a curve like that. Visio has a circular arrow shape. Go to File > New > Charts and Graphs > Marketing Charts and Diagrams. If you just click on the shape to add text, Visio puts the text in the middle of the circle. To put text on each individual arrow, you have to ungroup the shape; the text then sorta-kinda curves inside the arrow.

2009-05-15 16:19:24 • Gina Ko
  1. Make a circle.
  2. Cut the path of the circle into as many sections as you need arrows.
  3. Set the appropriate stroke size
  4. On each path section, use the “Filter-> Stylize-> Add Arrowheads” to add your arrow shapes
  5. Optional: Use “Expand Appearance” to turn each section into it’s own object.

Hope this helps.

Type faces with multiple/varying glyphs for each symbol?

January 8, 2012 • tfischer

Computer fonts have one glyph per symbol. This is too static for organic/informal type faces. In some situations, repeating symbols in an organic font should not look exactly the same but slightly different. Are there type faces with multiple glyphs per symbol? Or with generative means to vary glyphs? What is the technical name for such type faces? Can you name examples?

Thanks so much! –Tom

FF Kosmik (Flipper) — Random Font is the term devised by creators Van Blokland and Van Rossum for a font with multiple instances for each glyph.

Example: http://www.letterror.com/foundry/kosmik/index.html

Info: http://www.letterror.com/foundry/kosmik/technote.html

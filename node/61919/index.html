2009-09-14 20:46:44 • Bert Vanderveen

Be VERY careful when translating Japanese names.

Often, the Kanji are pronounced very differently from identical Kanji that appear in other Japanese words. Maybe sometimes there is even more than one possible pronounciation. There are special dictionaries for Japanese names, and if it’s important to get it right, I’d recommend to consult one of these dictionaries.

On the other hand, if it’s a film, maybe you can find the romanizised names somewhere on the internet, then you don’t need to translate them.

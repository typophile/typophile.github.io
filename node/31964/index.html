Well, the only one I can answer offhand is #3. [[x-height|X-height]] is a comparative measure of the height of the lowercase letters (e.g. a letter x) to the height of the capitals or ascenders. Impact has a high x-height because there is not a huge difference between the height of the lowercase and uppercase. Something like Nicolas Cochin has an unusually small x-height.

Bruce Rogers - World Bible Chapter Initials

December 5, 2006 • Greg Stanton

I’m in the process of digitizing this type (see attached). I know [[Bruce Rogers]] used these letters as chapter initials in the World Bible of 1949, but I also recently discovered that he used them on the title page of “The Wind in the Willows in 1940.” If he designed these letters, it was most certainly not for the World Bible as I previously thought. Also, this type could have been an available font that he simply used for these jobs.

Anyone know anything about this?

I looked in three books I had that have information about Rogers' career, and a set of initials like these was not mentioned or shown in any of the books. Some examples of initials he used were shown, and in one case they were Cloister Initials, by Frederic Goudy.

The type for the World Bible was a version of Goudy Newstyle that Rogers did in collaboration with Sol Hess.

Since the World Bible was basically set in a version of one of Goudy’s typefaces, I looked in a book about Goudy’s works to see if these initials were also his, but they were not shown in the collection that was supposed to show all of Goudy’s type designs.

The first one is seems handwritten. Very few script fonts match it’s sharp aggressive character, i.e. angular, in a monoline connecting script. Of the fonts suggested so far, Måns Grebäck’s Notera and Signerica [thanks Ryuk] are part way there. A few more partial matches to the character of the script are
Roland Hörmann’s Neo Noir Slim http://www.myfonts.com/fonts/phospho/neonoir-bold/ PT Script Barguzin http://www.myfonts.com/fonts/paratype/pt-script-barguzin/ Robert Slimbach’s Caflisch Script http://www.myfonts.com/fonts/adobe/caflisch-script/ Mistral is sufficiently aggressive but is much thicker with too much variation in height and angle, so I did not include it.

For fonts on MacOS X: STXihei literally means "ST Hei Light". Do you have any other font named like "STHeiti"? Anyway, I heard STXihei is the standard font in Windows Phone, so it might need to be there. Perhaps you can consider giving Hiragino Sans GB a go? It should be in any MacOS X installation, I think. For fonts on Windows: You can try SimHei/黑体. You can even try SimYou/幼圆 if you dare to use a rounded sans-serif here... However, I think Microsoft Yahei is already the best you can get for Windows. And to be fair, it's manually hinted to details, and looks quite good in Windows with ClearType on.

Stop FontLab from shifting accents?

March 10, 2006 • paul d hunt

Is there a way to keep FontLab from shifting accents from their default positions when making composites using the “Generate Glyphs” feature?

To prevent the accents from shifting up, you can specify “~” as in A+~acute

But to put the component always at 0,0, you need to either use anchors (just set them onto the 0,0 position at both glyphs so the components will snap there) or set the position of the second component to 0,0 per Python after generating the glyph as is:

fl.glyph.components[1].delta = Point(0,0) fl.UpdateGlyph(-1)

A.

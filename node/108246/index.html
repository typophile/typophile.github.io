HUNGER THIRST: most probably custom and possibly hand-drawn from scratch as [[http://dribbble.com/shots/608462-Hunger-N-Thirst-Identity|Derek Hollister]], one of the designers in charge of this identity at [[http://www.theinfantree.com/work/hunger-n-thirst|The Infantree]] has a type work on his dribbble called ‘[[http://dribbble.com/shots/986127-Absolute-Finest-Type|Absolute Finest Type]] which looks pretty close to the one for ‘HUNGER N THIRST’. For what it’s worth, if I have to recreate it, I’d probably start with some font like [[http://www.myfonts.com/fonts/ef/diamante|Diamante EF]] adding those latin (triangle) serifs. More ‘Diamante’: [[http://www.myfonts.com/fonts/solotype/five-star-final|Five Star Final]], [[http://www.myfonts.com/fonts/fontbureau/herald-gothic|Herald Gothic FB]], [[http://www.myfonts.com/fonts/redrooster/chamfer-gothic-rr|Chamfer Gothic RR]], [[http://www.myfonts.com/fonts/nicksfonts/uncle-sam-slim-nf|Uncle Sam Slim NF]], [[http://www.myfonts.com/fonts/loshaj-foundry/pillar|Pillar]], [[http://elibrumbaugh.com/Dayta-Typeface|Dayta]] “Out-of-the-box” alternatives but still some customization needed: [[http://www.myfonts.com/fonts/btn/supremaganda-btn|Supremaganda BTN]], [[http://www.myfonts.com/fonts/hold-fast-foundry/bourbon|Bourbon]], [[http://www.myfonts.com/fonts/flat-it/sheepman|Sheepman]], [[http://www.myfonts.com/fonts/redrooster/yeoman-gothic-rr|Yeoman Gothic RR]] Starting with a slab and (mainly) reworking serifs to latin ones: [[http://www.myfonts.com/fonts/canadatype/player|Player]]

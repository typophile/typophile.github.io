A free font that might help you recreate the look (straighten out the upper portion of the E, create some connections between the letters, and elongate the crossbar on the t): http://www.dafont.com/search.php?q=Porongo&text=ElEncanto EDIT: Per Ryuk (see below), Porongo bears a striking resemblance to Sudtipos' Mr. Stalwart, listed below. Now, they both may be based on the same source, but if you are looking for a font for commercial use, better safe than sorry—go with the Sudtipos version. Sudtipos, an Argentinian type foundry, does amazing script faces. Regarding Porongo - Intellecta had done many revival fonts and I'm sure this one is no different - the source being lettering alphabets in an old ink company catalog. I have one myself that I found in an antique store years ago - before I had any idea what it was. Full alphabets extrapolated from signatures - each with a very distinct personality. Sudtipos offering is truer to the original - and likely better made - but I really don't believe Intellecta's offering should be disparaged as dubious.

2003-10-28 19:57:31 • Dan Weaver
Yes IP addresses can be traced.

Further, if you follow the RIAA (Recording Industry Association of America) suits, they're tracing IP addresses to the respective internet service providers (like AT&T or SBC). The ISPs are then subpoenaed to fork over the address/info of the offending end user.

In the end, Carracho is not much different from Kazaa or Limewire, except that Carracho has a central server somewhere and is usually password protected (as opposed to the others which are peer to peer networks, without a server and without passwords.)

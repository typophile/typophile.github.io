2007-04-17 18:04:59 • Sharon Van Lieu
Yes, it's both (although mostly the size). A bitmap font will not only not help, but if it's an "orthonormal" design (with a lot of things lining up and being perpendicular) like Garamond (or really any conventional face) it will typically look [even] worse than an outline font. For curved bitmap text you either have to go to grayscaling (even more non-trivial) or you have to use a "casual" design (like a roundish cursive face) where the "grid-busting" is covered up. hhp

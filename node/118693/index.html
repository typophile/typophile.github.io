2014-08-13 14:25:16 • Core0
When strange things like this occur in FL the best thing to do is to trash Preferences as well as the Workspace file and do a system restart. To access the preferences on OSX hold the Option key as you click on the GO menu to reveal the Library. In Library/Preferences/ find and trash: com.fontlab.studio5.plist In that same Library go to Library/Application Support/FontLab/Studio5/Workspaces/ and trash: CurrentWorkspace51.rc

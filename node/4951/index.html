This is almost certainly a custom designed logo. My Sans Serif Font ID Guide (page5) has the sans families I have collected with these kinds of letterforms, and the closest match was Linotype's Freytag (12-57), but there were numerous differences.

You should know that probably about 90% of logos and trademarks are custom work – to make imitation more difficult, as well as to have an exclusive 'look'.

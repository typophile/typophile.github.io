Looking for a particular version of Britannic

September 20, 2011 • Oliver Wehn

Hi. I’m currently dealing with a licensing problem regarding an old PS1 version of typeface Britannic (see image below) I got from a new client. As I decided to stick to the font, I of course tried to find out where I can get a license for me (and as well for the client!) and maybe an updated otf version of this particular font. The file name BritaBQReg as well as the file info tell, it’s a Berthold font from 1992. Berthold – as it seems – doesn’t offer Britannic anymore. I found three versions of Britannic on myfonts by [[http://new.myfonts.com/search/britannic/fonts/|Linotype, URW++ and Elsner+Flake]]. But none of those alternate fonts contain a regular weight. The medium weight is the most similar to it. But the really important thing about the particular font I’m looking for, is the higher contrast and the more distinct endings like in the C or S. I’d be happy if anyone had an idea where to look or ask. Thanks in advance! [img:sites/default/files/old-images/britannicBQ_4782.png]

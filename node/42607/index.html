My math is a little backward, but anyway:

Your number lines in a column, minus the number of horizontal lines between the rows, must be a number divisible by the number of rows. For example, if you have 51 lines and 4 rows, you’ll need to subtract 3 extra rows to get 48 lines, which is divisible by 4. If you have 53 lines and subtract 3 extra rows you’re left with 51 lines, which will not divide evenly, so you will need to either increase or decrease column height to make it work.

Muller-Brockman explains it better than I do.

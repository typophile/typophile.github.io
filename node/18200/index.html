2006-02-25 23:43:35 • Si_Daniels
Note that there are two levels to the WGL4 spec: required characters and optional characters. There are quite a lot of fonts with all the required characters, but not so many with the full complement including e.g. linedraw and boxdraw characters. If you want the latter, Helvetica Linotype is a good option (and you get the added benefit of Arabic and Hebrew).

Newbie post. I’ve been asked by a client about the Kindle format so I thought I’d query the collective.

I’ve read that only .TXT and .AZW are readable on the Kindle. That seems odd, but understandable. Has anyone read anything which they’ve bookmarked which explains the conversion process and/or how it looks?

Update: I am aware of the Digital Text Platform area on Amazon.

The Kindle can read MOBI, PRC, TXT, HTML, and DOC, in addition to protected AZW files from the Amazon store:

http://en.wikipedia.org/wiki/Amazon_Kindle#File_formats

With some tinkering, you can convert protected MOBI/PRC files into protected AZW files; they’re essentially the same format:

http://igorsk.blogspot.com/2007/12/mobipocket-books-on-kindle.html

My favorite hidden Kindle feature is turning off the justification with J when viewing the font size list.

I guess none of that really answers your question, but if you’d like to send me files to test on the Kindle I’d be happy to.

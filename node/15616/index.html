2005-10-13 15:28:17 • paul d hunt

You can export the kerning and metrics from Fontographer as an AFM file. The AFM file can be imported into FontLab when you are in the Metrics window with kerning enabled. FontLab will ask if you want to do with the imported metrics, whether they are compatible, etc.

Alternatively, you can export the font from Fontographer as a Type 1 PostScript font. When you import the suitcase file, the kerning and other metrics will come along with everything else.

Yet another alternative is FontLab’s new FogLamp utility which converts Fontographer files directly to FontLab VFB files, as Paul suggests.

Lot’s of ways to go.

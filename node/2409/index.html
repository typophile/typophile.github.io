In the past, each time I have tried to package a file for back-up storage or transfer, it will only include partial font files in the output folder
InDesign should include all the fonts, unless they have a no embedding tag, I don't think it will include those. But I think that is only available in truetype fonts anyway.


If you do want a list of fonts in the doco, however, you can find it at TYPE>FIND FONT or FILE>PREFLIGHT

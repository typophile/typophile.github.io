Well, you would need to run actions/macros to slant the font, narrow the strokes, and narrow the width of the letters and their metrics. All of this can be done pretty quickly using Karsten Lucke’s KLTF Glyph Transformer. I just tried it out and it took me about thirty seconds to figure out how to do it. Of course, you’ll still need to redraw characters like a, e, g, and f that have different italic forms.

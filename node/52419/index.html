This font is [[http://es.letrag.com/tipografia.php?id=93|“Gallaecia Castelao”]], and it is a digitalized version of a classic galician typeface which can be seen engraved in stone castles, churches or tombstones. This font was distributed for free along with the galician newspaper “La Voz de Galicia”, but now, I think you can get it sending an email to the designer Carlos Nuñez, from the studio [[http://www.variografik.com/|“Variografik”]]. There are some more free fonts like this, like [[http://www.dafont.com/gallaecia.font|“Gallaecia”]], but I recommend you the first one, as it is the most similar to the original engraved forms.

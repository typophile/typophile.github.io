2007-08-14 20:54:55 • zzi

Bob: honestly, I think you could do with some CSS tutorial, the Web is full of them and they will give you more information than what I can possibly can in a simple forum post.

As for font: .7em/1.3em Lucida Grande; that’s the shorthand CSS version for font size/leading/face.

.7em/1.3em means type of 0.7em in size with 1.3em of leading between the lines, set in Lucida Grande, which is a humanist sans-serif available on Mac OS X machines. You can experiment with these values, change the numbers and the measurement types, like putting 12px/18px, although using px for font sizing is generally discouraged because Internet Explorer 6 on Windows users will not be able to resize them at will.

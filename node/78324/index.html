I use Cobian for Windows. I’ve set it to make incremental backups of all my work files every hour. The interface is very simple and it has never crashed in the 2 years I’ve been using it. The backups are compressed and you can specify how often a full backup is done, how many full backups to keep etc. Open source: send a donation if you like it. http://www.educ.umu.se/~cobian/cobianbackup.htm

Under OPTIONS > General Options > Open and Save, fill in Auto-Save Fonts Every 10 minutes, and check Create Backup File. Then, manually save the file at regular intervals, which will keep the .bak file up-to-date.

2010-01-16 13:37:01 • nina

You can see what classes a given glyph is in by selecting the glyph in the Font window and then click the “Classes…” button on the “Glyph Properties” panel. A window pops up containing a list of all classes on the left and a list of the classes in which the glyph is used on the right. There are buttons that allow you to add or delete the glyph to or from any class.

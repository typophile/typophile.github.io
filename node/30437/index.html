There are no rules that govern how you might shape a curve when drawing a sans. The 1000 unit grid is part of the construction used by software to render a glyph. If you use FontLab, it defaults to 1000 units but this grid is not visible. It is used for scaling. If you mean the proportions of stroke to counter, the world is open to interpretation. Why don't you just start drawing glyphs and see what works best for you? ChrisL

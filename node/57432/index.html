The really annoying thing is that fonts are often named differently across different platforms. I couldn’t tell you how many times I have had problems with Helvetica Neue and it not being recognized by photoshop because I had a differently named version of it from the creative director. We finally standardized to the Opentype version, which has the same name everywhere (which might help solve your problem Dan), but while it went on, it was rather frustrating.

If you are willing to let go of some control, using font stacks would be a decently good idea. It would allow you to specify which fonts you want to use on windows and macs. It isn’t an ideal solution (indeed @font-face would help resolve many an issue), but at least it is something that works now. I’m using it on a site to identify Lucida Sans in its Windows version and Mac version (as they are named differently).

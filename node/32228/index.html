It looks as if the Griffith Gothic font you are using in the PDF has overlapping paths. This should not appear in a production quality font, so something tells me that you are not using the release version? [img:sites/default/files/old-images/12-13-03_6226.gif] I have seen the phenomenon of the thinning characters in some PDF's with specimen of my typefaces in development, and as far as I remember it was caused by a wrong pathdirection in the affected glyphs. It was visible on screen in Preview in OS X, but not on a laserprint.

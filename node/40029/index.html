Illustrator can get you most of the way there, especially if you use a little Gradient Mesh to take the place of some of the standard gradient fills. Photoshop can be used to enhance the Illustrator lettering. Or Photoshop can be used to do all of the colouring, once you have paths created in vector software.

Both methods require ‘handwork’ though, which can take quite a bit of time depending on the complexity of what you’re aiming for. Since this is a special effect more than a stock solution I would think this suitable myself but there are some plugins for both Illustrator and Photoshop I think that can generate 3-dimensional shapes on the fly if you don’t want to go the whole hog with something like Maya.

It’s been such a long time since I’ve done this. Extreme 3D had a handy function of importing EPS files and using the splines to built its geometry. I think many more modern 3D programs have similar functions, either importing some form of Illustrator curves or handling text as geometry directly, but I don’t know which ones.

One thing you might find you have to do is to add points to your curves before exporting The curves generated by the 3D software are not always as supple as in drawing programs, especially when rendering is set in a faster mode. The extra points will give stability to the shapes.

2007-12-15 00:02:08 • cuttlefish

You might want to look into POV-Ray, a free open-source 3D rendering application that’s available for Windows, Macintosh, and Linux.

POV-Ray is unusual in that you create scenes not by modeling them in a GUI application, but instead by describing them in a text file using a scene description language (SDL). POV-Ray has direct support for TrueType fonts, so you can render extruded 3D text pretty easily.

For example, here’s a scene I threw together in a couple of minutes:

#include “colors.inc” camera { location <0, 2, -18> look_at <0, 0.25, 0> angle 15 } light_source { <500, 500, -1000> White } text { ttf “frutbd.ttf” “Typophile” 1, 0 pigment { SlateBlue } translate -2.3 * x }

This produces the following image:

Note the full alpha transparency. This makes it a snap to drop rendered scenes into Photoshop layers.

Don’t be intimidated by the whole SDL thing. If you all you want is simple extruded text, you don’t even really have to learn the language. You can just play around with the parameters in the script I posted and see what comes out. Later on, if you’re feeling adventurous, you can start to read the documentation at your own pace; eventually, you’ll be able to come up with a lot of different cool effects.

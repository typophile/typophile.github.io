Humanist sanserif companion for Minion Pro

June 10, 2008 • olander

Hi Typos, I think of investing in a humanist sanserif font which I would use as a companion for Minion Pro (among other things), mainly for publications about linguistics. Apart from basic and extended Latin character set I also need Cyrillic and Greek, which limits the possibilities substantially. I am a big fan of [[http://www.fontfont.com/shop/view_product.ep?id=13665|Scala Sans]] (I know, it would be just imitating Bringhurst, but it really is a great font). Scala Sans doesn’t include neither Cyrillic nor Greek, however. Right now my favourite choice is [[http://www.fontfont.com/shop/view_product.ep?id=10110|Kievit Pro]] (I wouldn’t know whether to pick [[http://www.fontfont.com/shop/view_product.ep?id=10994|Regular]] or [[http://www.fontfont.com/shop/view_product.ep?id=11011|Book]] as my point of departure, though). Another option is [[http://www.stormtype.com/typefaces-fonts-shop/families-84-anselm-sans-pro|Stormtype Anselm Sans Pro]]. Bringhurst recommends Syntax with Minion - but as far as I can see, the Cyrillic version of Syntax, [[http://www.paratype.com/pstore/fonts/Humanist-531.htm|Paratype Humanist 531]], does not include an italic, which is crucial to my needs. And I don’t know if there is a Greek version. Suggestions are welcome! Cheers, Thomas
Some might stone me, but I think Syntax reeks of the ’80s. Kievit and Anselm are fine choices. Myriad is the obvious cmpanion and not necessarily the best, but it does include Greek and Cyrillic.

Looking for a light Serif font

June 29, 2009 • lugray

I am an architect designing my resume and portfolio for job applications. I want to use a serf font for the body text with Helvetica Bold for the project titles.

Any suggestion for a very light weight serif font? I tried rockwell but felt it was too heavy.

Thanks in advance for the advice!

Well, if you like Rockwell, it is what is called a slab serif. If you do a search on this site for “slab serifs” you will find many a discussion that mention some good ones. the other thing to consider is that you may want it to match with a neo grotesque, which is what Helvetica is.

I hope that helps.

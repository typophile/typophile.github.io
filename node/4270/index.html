2004-04-16 16:03:32 • Diner
Illustrator's autotrace function isn't really useful for accurately tracing stuff, but it gets cool-looking results. Not useful, but interesting, especially when they're outlines with no fill.

As far as vectorizing images (besides fonts), I start with Photoshop. Go to Filter > Artistic > Cutout, make the image as detailed as you want it, then select certain areas (usually I select similarly colored areas together), and run them through Streamline. If you do this for each set of shapes, you can put them together in Illustrator. I can never get Streamline to make things as detailed as I want them, so I use this method, which is actually quicker than it sounds. I probably just don't know how to use Streamline correctly. Laugh if you must.

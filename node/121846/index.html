Tom Swift and His Outpost in Space

December 7, 2014 • quadibloc

This typeface looked a lot like Century Expanded, but the capital J that goes below the line eliminates that.

The italic capital Q eliminates Baskerville or Bulmer. I’m pretty sure this isn’t Imprint, because it doesn’t look at all like Caslon. (Similarly, it isn’t the Phemister Old Style.)

I think I should be able to recognize this typeface immediately, but I’m embarassed to admit that I can’t.

Also, I see that it is not Number Twenty-One from Linotype; that doesn’t have a descending J, and it has a different type of Q in the italic.

Looking at the Baskerville types in Mac McGrew’s book “American Metal Typefaces of the Twentieth Century”, this looks like the Linotype version of Baskerville. The italic ‘k’ is I think the key to separate it from the Monotype and ATF versions. It also has that unusually long J, and the italic Q seems to match.

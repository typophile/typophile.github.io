I have yet to see Verdana used well off line.

It is too wide for a "regular" sans-serif. (By "regular" meaning that it is not an extended version). So it wastes a lot of space. Characters like a, e, j, I, J look very odd when printed. Some of the nuances that translate so well on screen become, well, nuisances.

Unlike its serifed sibling, Georgia, Verdana should stick to HTML.

Kerning - difference between optical and metric?

June 20, 2006 • JonnySchneider

Can someone help me out? Whats the difference between these two types of kerning?

I know that metrics are instructions built into a face that gives software programs instructions about how much space to put between different combinations of glyphs. Arent these ‘metrics’ set-up so that faces have correct optical spacing depending on the shape of each glyph? If so, the what is ‘optical’ kerning.

Specifically, i’m talking about type controls in Adobe CS programs.

Thanks - JonnySchneider

I’m not sure of the difference. While optical kerning may perhaps be a handy tool for large bodies of text, for display type (and wherever possible) one should always manually kern. I guess metric kerning would be based on the kerning info from the font file, where as optical is an approximation/adjustment based on the visual character. Still never going to be as good as adjusting by eye.

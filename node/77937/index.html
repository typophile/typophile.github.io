Flash and ClearType

January 11, 2011 • apankrat

Does anyone know if the FlashPlayer is capable of using native Windows ClearType for rendering text or does it always use its own rendering engine?

Passing on a quote from one of our engineers here:

Generally speaking, for desktop systems, “device text” is rendered by the OS, which can include ClearType rendering. “Embedded text” is always rendered by Flash with its own internal rendering engines (of which there are several).

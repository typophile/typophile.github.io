Does anyone know how to make the o-breve (ǒ) in QuarkXPress? I am using a Macintosh, OSX.

Any help would be greatly appreciatated.

Thanks much!

Use your OS SX “Character Palette” to find the breve accent, then insert it. Then type “o”, and kern the o underneath the breve.

Unless you are using a glyph-rich OpenType font, which should have that glyph. You can insert it using Character Palette, and Quark should understand it. (I honestly have not used Quark in years for the same reason that I don’t use a horse-and-buggy to get around.)

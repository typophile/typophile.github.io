2008-02-06 04:02:53 • Si_Daniels
To input off-keyboard characters on Mac OS X: Open your System Preferences from the Apple menu Select International Select the Input Menu tab check the Character Palette, Japanese Kana Palette, and Keyboard Viewer to ON Check on Show input menu in menu bar This will bring a flag icon into your menu bar set according to your local keyboard encoding. The palettes can now be selected from a menu under that flag. International preferences can also activate alternate keyboard encodings for other languages, if you want to. Be advised these palettes only work fully in Cocoa (OS X native) apps. They may cause problems in older Carbon applications, but if you have a fresh new Mac with fresh new software, you should not have many of those.

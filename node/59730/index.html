2009-07-15 02:57:02 • dezcom

That Adobe page is an outdated version (probably v1.4) of the OpenType spec. The DHV tag has been deprecated. In OpenType coding, the correct language tag for Dhivehi is DIV.

The current version of the OT spec provides correspondence with the ISO 639 language IDs: http://www.microsoft.com/typography/otspec/languagetags.htm

By the way, the OT language system tags are actually four characters long, not three. But almost all of them have only three letters and a padding space character, e.g. ‘DIV ‘. Most tools allow you to use only the three letters and add the padding automatically. There are a couple of four-letter language system codes defined though, and probably more will be defined in future.

Database of Type in Use

August 28, 2004 • nwe44

Hello all,

I am in the process of developing an online database of type in use. By this I mean photographs of typography in the real world (book covers, posters, food packaging, you name it) categorised by font, weight, (rough) location, media, subject (Personal hygene, sport, electronics etc.) and so on. This was initialy envisaged as a student tool, to help first year students answer the perennial question
Peter Karrow has been working on a database of magazine advertising (he talked about it at ICTVC2) it would be super cool to add the names of the typefaces used in each ad to the database - even automating the process by some super-advanced version of MyFonts 'what the font' identifier. Add the names of the ad agency that created the ad and a database of font licensees and you could even include automatic invoicing if unlicensed fonts were being used. Then again maybe not.

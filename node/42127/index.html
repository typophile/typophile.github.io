2008-02-20 01:23:30 • ipseity
I think the problem is that I cannot get my file size down below 24 Kb. I’ve made the image extremely small (30x30) but the file size will not get smaller than 52 Kb. [...] Any suggestions? Hi, Michael. This worked for me: 1. In Adobe Photoshop, start with a new image that is 60 pixels wide x 60 pixels tall, and has a resolution of 300 pixels per inch. Alternately, in Adobe Illustrator, make a square that is 60 points x 60 points. This will equal 60 x 60 pixels later, when you save your file for the Web. 2. Make sure you are in RGB mode (in either Photoshop or Illustrator), and that you have the color palette set to Web-safe colors (to make sure your icon's colors will look the same on a Mac or on a PC). 3. When your icon is ready, go to File > Save for Web. This will open a window that gives you a preview of your icon, and lets you choose different image formats (JPEG is best for color, GIF is best for black and white). This window will also show you what size the final file will be, and let you choose different variables for image compression and quality. Once you hit the Save button, you will get a chance to change the name of your file, if you wish. I hope this helps!

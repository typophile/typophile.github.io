Does anyone have any insight as to what typeface this original Kodachome logo is? If not exactly (as it might be custom) something that is close? Knockout and Nobel are pretty similar, but looking for something closer still.

I think it’s likely that it was custom hand-drawn lettering.

Notice how the “A” tucks under the “D” and “C”. It’s possible to do this in metal type with kerns (elements that extend beyond the edges type body into the body of adjacent letters), but not practical to have them on these particular letters. They would crash too often with other common letter combinations, requiring extra spacing material to be added between the letters.

With lettering, tight spacing like this would not be an issue.

Also note that the two “O"s are not quite the same, another indication that it was hand-drawn.

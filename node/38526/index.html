Bitstream Font Equivalencies List

October 28, 2007 • ryanholmes

My employer has a site license for the Bitstream Type Odyssey Mac CD, and I wanted to compare some of the fonts to the Adobe/ITC/Linotype/MT versions I am more familiar with, since the numerical names from Bitstream aren’t always that helpful… I found some matches from a forum search, others from myfonts, and yet others from google, and thought someone else might find such a list useful. Some of the Bitstream fonts below appear to be clones of the more familiar versions; others are only loosely based on the matched font. The usual accuracy disclaimer applies, I may have made some search mistakes, YMMV, etc….

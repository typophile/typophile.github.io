Attachment and bookmark container

November 28, 2010 • Arno Enslin

It seems to be impossible here, to edit or attach (small) files (feature files or batch files for example) to a post, that is not the first one in a thread. So I will use this blog entry as attachment container. If I write a post in another thread, in which I want to post a file, I will post a direct link to the attachment. And in this blog entry, I will add links to the posts, to which the attachments are related.

If the name of a file is *extension_.txt, *zip_.txt for example, the filetype is zip and you have to rename the file to *.zip, because uploading zip files is also impossible.

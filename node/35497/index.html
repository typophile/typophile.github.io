Sorry about my late reply; I lost track of this thread.

The Adobe Sans Substitution Font cannot be used as such. It is an automatic replacement by the now defunct Adobe Type Manager Deluxe Professional Überversion Whatever when the intended font is absent on the computer used to output a file. In short, it shows up when the correct font is not there. So it was never intended to be used on purpose. You can’t purchase it nor can you specify it in your font menu.

2008-02-27 10:38:58 • Pieter van Rosmalen

it’s a reference point. it comes in handy when you’re using the transformations panel, you can set this point where you want and then perform transformations that key off of this point.

for example, to draw a 5-lobed asterisk: draw one lobe, place the reference point toward the bottom of the lobe, copy the lobe and rotate 72 degrees, setting the center as the reference point, repeat until you have all 5 lobes…

Hi Pieter,

If you want to remove the “thing”,

  1. Click the horizontal guideline that the “thing” on holding down the SHIFT and COMMAND keys, so you can see the Vertical metrics palette.

  2. Change the “From” pull down menu into “current Position.”

  3. Input zero into the dialogue.

2008-02-28 16:13:51 • Pieter van Rosmalen

If your vfb data have the things in every glyph, I don’t know the way to remove them at once, but try this method.

1)Duplicate your vfb data (make A and B). 2)Open the “A” vfb data and generate metrics data. 3)Open the “B” vfb data and delete all glyphs. Select all glyph and do command Glyph>Create glyph If empty, then import outlines using command Mask>Assign Font Mask from the “A” vfb. Swap outline with Mask. 4)Import the metrics data of the “A” vfb into the “B” vfb.

I think it is a better and easy way than the way that you remove the “thing” in all glyph individually.

2008-02-29 08:49:44 • Pieter van Rosmalen

I found out another method. How about this one?

1)Open Preferences…>Generating OpenType & TrueType>OpenType PS(.otf) Check off(inactive) Decompose all components

2)Generate OTF(PS). (I could remove all “things” at this moment.)

3)Open OTF font that you generated. And then you can copy and paste some glyphs that include a component into the original or import some data that you need such as metrics and class into this vfb.

In case your vfb have any MultipleMaster axis, once you copy and paste the glyph that include a component into the original, you can keep the component in every glyph. Memo: You need to copy and paste not in the Glyph window but in the Font window.

This seems to be a simple way than the previous one. But there might be any other problems.

Best regards.

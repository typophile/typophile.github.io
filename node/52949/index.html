Achieving true shared baselines is very hard. You can never be sure which typefaces are used to display a webpage & most browsers use the top offset of a font to render type, and not the baseline like most page design programs. The different x-heights mess up sharing baselines.

Alistapart published an informative article about this:

http://www.alistapart.com/articles/settingtypeontheweb

Rob Goodlatte built a small bookmarklet to help check whether you are on the right track:

http://robgoodlatte.com/2007/07/31/syncotype-your-baselines/

Hope this helps.

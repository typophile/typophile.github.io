2014-11-08 20:14:11 • George Thomas
Since you’ve modified an already existing font, you’ve created a derivative work. Unless you used an open-source font, it’s extremely unlikely that you can sell this commercially. You should have started from scratch, not from altering someone else’s font. However, for the final say, check the end user licence agreement or contact the originating foundry.

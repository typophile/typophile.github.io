‘CAST IRON HOUSE’: slightly customized, mainly artificially thinned with an outline stroke and some triangles added to angles; base fonts: [[http://www.myfonts.com/fonts/profonts/euro-sans-pro|Euro Sans]], [[http://www.myfonts.com/fonts/castletype/carisma|Carisma]], [[http://www.myfonts.com/fonts/andrijtype/seaside|Seaside]], [[http://www.dafont.com/balham.font|Balham]], [[http://www.myfonts.com/fonts/greyscale-type/bodoni-sans-display|Bodoni Sans]] (/R leg off), [[http://www.myfonts.com/fonts/t26/minerva-modern|Minerva Modern]] (flare serifs), [[http://typophile.com/node/101748|latin counter-part of some non-latin language fonts]], [[http://e-daylight.jp/design/fonts/type/s/stark-debonair.html|Stark Debonair]], [[http://www.myfonts.com/fonts/typeinnovations/contax-sans|Contax Sans]] ‘SHIGERU BAN’: part of /S, looks like [[http://lineto.com/The%20Fonts/Font%20Categories/Text%20Fonts/Brown|Brown LL]] by Aurèle Sack

EDIT: the more I look at it, the more I think it has been started with Brown and they created a contrasted version version of it.

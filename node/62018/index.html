Just released 5 weight font, Motown at YouWorkForThem.

[[http://www.youworkforthem.com/product.php?sku=T0315|Motown]]

Motown is a geometric slab serif with 5 weights. Although it was intended to be used as a display face, Motown’s heavier weights can be used effectively for text as well. The angled crossbars and vertically symmetrical slabs give Motown a unique flair for a font of its class. The implied energy in these features also help give the font an athletic, youthful vibe. And it can also look quite classic when set appropriately for headlines, titles, and identities. Motown is available in Open Type format.

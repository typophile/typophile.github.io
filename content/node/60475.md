    should only be at the end of a word, so in \"\u03C2\u03B9\u03C9\" it should be
    replaced by a normal sigma (\u03C3).\r\n\r\nIn the first word \"Dos\" you appear
    to have used a <A HREF=\"http://en.wikipedia.org/wiki/Stigma_(letter)\">stigma</A>
    instead of a final sigma; it looks very slightly different and has a different
    codepoint. Stigma isn't normally used in writing."

2007-10-18 21:22:30 • Goran Soderstrom
In general, in InDesign, you can set your defaults by choosing them when nothing is selected – with a document open you’re setting the document defaults, or you can choose OpenType features to be on by default in a no-documents-open state which will copy these defaults to all new documents. However, I would recommend a style-sheet based workflow. There is a related [[http://typophile.com/node/38181|thread about OpenType features]] going today as well.

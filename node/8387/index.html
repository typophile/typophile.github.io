The typical place to start is Robert Bringhurst's The elements of typographic style. This book is mainly concerned with typography (i.e. design with type), particularly book typography, rather than type design (design of type), but the type specimen section is quite good.

I guess it depends the reason that you want to build your collection. If it is to learn about the basics of designing type, the books are quite different from learning about the history of type, punchcutting and design. I have just finished an excellent book called 'TYPE the secret history of letters' by Simon Loxley. Which I felt gave a brief, easy to read history of type (from a British point of view). If you are looking for books which will help you to understand type design, I would suggest LetterLetter : http://www.typotheque.com/products/letterletter1.html Edward Johnston - Writing and Illuminating and Lettering http://www.amazon.com/exec/obidos/tg/detail/-/0486285340/103-3770796-1332632?v=glance Both of these books approach type design from a calligraphic angle, which I personally have found to give great insight in to the rules of lettering.

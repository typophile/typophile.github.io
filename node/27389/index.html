Hi, i was browsing the linotype font library and found an explanation about XSF fonts, first of all do anyone of you have any experience with these type of fonts? what i could make out that it's only intended for Windows, what i really want to know if these fonts can be used for example say web sites or are they just especially made for a windows environment (and do you have to manually select the XSF font if you work with small type). any info is appreciated.
XSF fonts have a higher level of TrueType hinting than most of the other fonts that Linotype distributes. They are primarially intended for use in corporate office environments, where so much reading is being done on screen now (Microsoft Word, e-mail, etc.). Hope that helps!

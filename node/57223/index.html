Hi,

You can try with this:

from robofab.world import CurrentFont from robofab.tools.accentBuilder import AccentTools

font = CurrentFont()

accents = [‘ccaron’,‘amacron’,‘Cacute’,‘Cdotaccent’,‘Abreve’,‘cdotaccent’] accentTool = AccentTools(font, accents)

accentTool.buildAnchors(ucXOffset=30, ucYOffset=70, lcXOffset=20, lcYOffset=50, doProgress=True)

accentTool.printAnchorErrors() accentTool.buildAccents(doProgress=True) accentTool.printAccentErrors() font.autoUnicodes()

#font.update()

This glyph update isn’t the best way, but it works…

for glyph in font: glyph.update()

If you use standard names, you shouldn’t even need a script. In the “Generate glyphs” dialog, you can just paste glyph names or link to a text file with the names of the glyphs you want to generate. I’m actually not really sure why people bother to use scripts. There is a FL system file which specifies how base glyphs and accents combine, so you can modify this if you want to use case specific accents.

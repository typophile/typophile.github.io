Hi Michelle. If you go to Element > Correct Path Direction, that should solve your problem. Essentially, the outside path of your 'B' and the 2 inside paths need to move in opposite directions. Every letter with a counter needs this same fix applied.

http://www.fontifier.com/ I think Dan is referring to this website. The size issue is to do with the way you import the streamline drawings into Fontographer as Elliot mention the best way to do this is to open all the streamline files in Illustrator (or similar vector graphics software) and create a box the same height (or slightly more) as the combined height of your ascender and descender and position each glyph in it on separate layers, adjusting the width to suit each glyph. Then cut and paste the glyph and the box into Fontographer then delete the box, which leaves the glyphs at the same scale, a rather long job but ultimately satisfactory. You might decide that speed dictates you spend the $9 to get it done by Fontifier, but it is worth doing to learn the basics of Fontographer. Tim

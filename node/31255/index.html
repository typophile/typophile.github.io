I was able to download it then, but if your question goes beyond merely interest in who was able to download it and you're asking if someone can send it to you, I don't think that most around here would consider that to be ethical. Foundries like FontShop make fonts available for free at their own discretion, so if they decide to only allow a font to be available for download for a limited time, then it is not up to one of us to decide that it should be downloadable in any other context. Sorry if that's what you're asking; maybe you can scrounge up the money somewhere to buy it, as there's nothing better than buying the perfect font for a project.

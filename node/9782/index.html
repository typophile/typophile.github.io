There is bound to be some confusion with names, especially with 'shareware/freeware' fonts. David Rakowski digitized a lot of old type designs and re-named them. In turn, other folks may have re-named his fonts.

This is the name used in the 1995 Precision Type Font Reference Guide for the Intecsas font, but I wouldn't bet much that it is the original name.

> What font creation program to use? [[http://www.fontlab.com/font-editor/fontlab-studio/|FontLab Studio]] or [[http://www.fontlab.com/font-editor/typetool/|TypeTool]]. > What file formats (or type of font) to generate? OpenType > Any web sites that describe the process in detail? Not really, since there are too many details to describe. There are websites that show the basics; see Miss Tiffany's post. > Any books that are a must read for font creation? Not must reads, but can be useful: [[http://www.amazon.com/Designing-Type-Karen-Cheng/dp/0300111509|Designing Type by Karen Cheng]] and [[http://www.amazon.com/Learn-FontLab-Fast-Leslie-Cabarga/dp/0965762858|Learn FontLab Fast by Leslie Cabarga]] and I found [[http://www.amazon.com/Stroke-Theory-Writing-Gerrit-Noordzij/dp/0907259308|The Stroke by Gerrit Noordzij]] an interesting read. > What problems will I expect to encounter, and how can I avoid these common pitfalls. Too many to even start. :) > A basic breakdown of the process, i.e. draw it on paper first, or start with another font first and edit it... Whatever you find that works the best. Personally I prefer drawing on paper first, then scan these and digitize them; but only as a starting point. I never draw perfect glyphs. I would avoid the editing of existing fonts. > I also need to know how to document my process so I can establish my intellectual property and not have issues with claims of ownership. Keep everything you make.

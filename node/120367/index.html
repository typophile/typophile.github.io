Hi All, I'm looking for a display face similar to HANDI-CUT in the reference attached. 1950s hand script type font... Have looked at House industries to no avail - can anyone suggest a myfonts solution? Thanks! A. [img:sites/default/files/old-images/Dan-River-Fabric-Roll_3456.jpg]
[[http://www.myfonts.com/search/brush+script|Brush Script]] came to mind first but mainly for /H and /C/u/t part. A decent solution to consider it as 2 different letterings, a brush script for /H+/C/u/t and a different script for /a/n/d/i. FT [[http://www.myfonts.com/fonts/filmotype/keynote|Keynote]] might work well for the /a/n/d/i part but that's a total different story for /H/C/u/t. Some more interesting ones: Filmotype [[http://www.myfonts.com/fonts/filmotype/lasalle|Lasalle]], [[http://www.myfonts.com/fonts/filmotype/filmotype-leader|Leader]], [[http://www.myfonts.com/fonts/filmotype/lucky|Lucky]] and [[http://www.myfonts.com/fonts/filmotype/kitten|Kitten]] (I like this one even if too skinny), [[http://www.myfonts.com/fonts/mika-melvas/sanelma|Sanelma]], [[http://www.myfonts.com/fonts/typejockeys/sauber-script|Sauber]] (works well for /a/n/d/i), [[http://www.myfonts.com/fonts/sideshow/coffee-service|Coffee Service]], [[http://www.myfonts.com/fonts/sudtipos/calgary-script|Calgary]], [[http://www.myfonts.com/fonts/sudtipos/mr-stalwart|Mr Stalwart]], [[http://www.myfonts.com/fonts/profonts/sonora|Sonora]], [[http://www.myfonts.com/fonts/ihof/p22-slogan|Slogan]]

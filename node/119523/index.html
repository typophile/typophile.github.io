2014-09-14 14:58:01 • kthomps5
The key to this lettering is the “gaspipe” or “stovepipe” corners. Start with a font with about the right height to width ratio and similar weight that has fairly tight curves outside AND inside, such as Ryoichi Tsunekawa’s Design System A 700 [img:sites/default/files/old-images/DesignSystemA700InUse1_6471.gif][img:sites/default/files/old-images/DesignSystemA700InUse2_5322.gif][img:sites/default/files/old-images/DesignSystemA700InUse3_5194.gif][img:sites/default/files/old-images/DesignSystemA700InUse4_3549.gif][img:sites/default/files/old-images/DesignSystemA700InUse5_6690.gif] http://www.myfonts.com/fonts/flat-it/design-system/ Clip and paste stovepipe elbows as required. Note the final row. These are the glyphs 3, 4 and l. For E you could start by flipping 3. For Y you could start with 4. For I you could start with l. For N ignore the round-top version and make a simple straight line N. Only the lhs of A will take some curve-bending. Don

2014-09-16 19:07:32 • donshottype
I mentioned Ryoichi Tsunekawa’s Design System A 700 in a previous post and suggested a recipe for using it’s corners to make stovepipe elbows to approximate the letters. Unfortunately the result is very approximate. The curves are too gradual and the weight is too light. I created my own stovepipe elbows from scratch and constructed these letters and the rest of an alphabet based on my conjecture as to what I thought might work with this concept. I have no idea how the the designer of the Knopf cover would have made the rest of the alphabet. So here is a Tsar Poutine alphabet in use [img:sites/default/files/old-images/TTSS_SM2_6621.jpg] The picture of poutine was contributed by Jonathunder to Wikipedia http://en.wikipedia.org/wiki/Poutine#mediaviewer/File:Poutine.JPG The letters might be hard to digest and are definitely not part of a healthy diet. I might release Tsar Poutine as a font someday, but it does not seem like a very remunerative project. Don

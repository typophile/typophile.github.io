“SHOP…” looks like Comix Regular from WSI-Font Collection (1993). Note thought, that WSI is a well known font forgery corporation that was shut down and the font above is probably published without authorization of the original font designer. You can find more info on the subject here: http://www.sanskritweb.net/forgers/wsi.pdf

I concur in your find. So, how legitimate is WSI’s Comix Regular? I flipped though the Comicraft and Blambot collections but did not spot a match. But perhaps on closer examination? Another comic book/strip font publisher? Here is internal font info:

Font Name : Comix Regular Font ID : 1865 Family Name : Comix Sub Family Name: Regular Version : Publisher’s Paradise – Media Graphics International Inc. PostScript Name : ComixRegular Font Type : True Type Copyright : From the WSI-Font Collection. Copyright (c)1993 W.S.I. All Rights Reserved.

As for “School Spirit” it’s the overused Comic Sans, which is installed as a core font on Microsoft windows.

By the way if you want an alternative to Comic Sans, you might try Comicraft’s Soliloquous and Comicrazy, Ron P. Dunant’s Architect Small Block, or Robbie de Villiers' Saycheez. Don

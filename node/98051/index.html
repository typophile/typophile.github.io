2012-11-23 20:03:22 • cuttlefish

Steps to install fontforge, the short version:

In Linux: Install dependencies, configure, make, make install. Most distros have a package called “fontforge”, anyway.

In Mac: Install Macports/Homebrew/Fink, use whatever the package manager to install a package called “fontforge”.

In Windows: Download the fontforge-cygwin from somewhere, unzip and use.

It’s not all that hard to install fontforge. But working with it without experiencing crashes is a lot harder.

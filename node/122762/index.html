Ways of simulating legibility

January 23, 2015 • sstaccato

Hello–

Wondering if anyone has any ideas or recommendations of ways to test the legibility of a typeface. Especially for signage typefaces, I’ve seen specimens that simulate the glow of headlights or the blur of long-distance reading. I suppose I’m wondering if there is a standard of performing such tests/displaying the results or if it’s just a blur filter in Photoshop (if so which one would most accurately simulate long-distance?). Thanks for any help.

For signage just make a full-size mockup on paper and hang it temporarily in place. I’ve done it many times.

If it isn’t huge you may be able to print or Xerox it to size. Or project the image on cardboard (or plywood painted the appropriate color) and use marker pens or paint. It doesn’t need to match exactly but should be reasonably close.

The advantage of a full-size mockup is you can view it realistically from different distances, different angles, while walking, while driving, etc.

Also your client can view it along with you and sign off on the sizing.

Yes it’s some work, but you don’t want a client buying expensive signs and then blaming you they aren’t legible.

I think you can get quite far along the same styles (although in a more cumbersome way) in Photoshop, just using Gaussian blur for distance, outer glow (in layer style, set it to black for black text and darken) for overglow, and a contrast layer for contrast.

If it’s a huge highway sign I can see using simulations. But for ordinary signs, why not make a mockup? Graphic designers do it routinely. And for some large signs, it’s possible to mockup small portions of them to test legibility.

Also legibility will vary not only with the weather but also with the viewer. Eyesight gets worse as we age. If it’s a sign that’s meant to be read by older folks, use some of them in the test.

[img:sites/default/files/old-images/info_type_4549.jpg]

Sometimes when no searches are getting anywhere it means something has been customized. This might be a common sans serif font with one letter redesigned and replaced, such as the R. Even if that is not the case, you might be able to duplicate it that way. Use the P to create a new R. Sometimes letter ‘transplants’ might be used to create a custom look. Depends on what your purposes are.

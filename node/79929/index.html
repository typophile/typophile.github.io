Actually, the jagged edges seem the result of an impact/thermal printer. These machines have their own proprietary typefaces which reside inside the hardware. So it’s very unlikely you will be able to find an exact match. You could search for bold monospaced fonts and for license plate fonts, which share some characteristics with the samples. You could also try to recreate it with Fontstruct (fontstruct.com).

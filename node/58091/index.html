Does anybody buy chromatic fonts? Do users have a clue?

May 17, 2009 • blank

I’m working on a series of fonts that could be broken into layers to make a great set of chromatic fonts. But I’m wondering if it would be worth the effort. The only chromatic font I see used often is Rosewood. I’m also wary that if they did sell, I would end up getting a lot of support requests from users who don’t have a clue how to set type in layers with Indesign, or worse, want them to work in Word. Does anybody have any experience selling chromatic fonts?

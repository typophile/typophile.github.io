Using Fontlab 5.0.4 and adding an onum feature to a font that’s generated as OTF.

I’ve added just the onum feature but InDesign CS6 has activated both the “Proportional Oldtyle” menu as well as the “Tabular Oldstyle” menu.

Is that something InD just does or is there something I’ve done wrong?

Feature is pretty simple, I’ve created an onum1 and onum2 class, the feature is simply: feature onum {

Latin (default)

sub @onum1 by @onum2;
script grek;
sub @onum1 by @onum2;

} onum;

I use the same method as John describes here: 1 — Create a custom .enc file in which you define the order of the glyphs the way you would like to see them in your fontwindow. 2 — In the fontwindow's Names mode you select your custom encoding 3 — Then: Tools > Sort Glyphs > By Encoding 4 — Switch the fontwindow to Index mode to check if the index order now equals your encoding order. Best, Artur

2009-02-03 12:52:18 • charles ellertson
I have a suggestion for you. 1. Open your font 2. Save an encoding based on the glyphs you have made. Just like that. By doing this you get the names of all the glyphs in a text document. [img:sites/default/files/old-images/saveeenc_5887.png] 3. Open that enc-file in a text editor, and sort the glyph names the way you want them. You can take away the numbers before the names, no problem. In the top of the enc-text document, you can name the encoding as you wish. [img:sites/default/files/old-images/encod_4303.png] Then choose this encoding in FontLab and sort the glyphs by that (sort by encoding). You can now have this encoding file as a base and then further on add more glyphs, or make changes in the order you want the glyphs to appear. Make all these changes in a text editor and then re-select it in FontLab for it to update.

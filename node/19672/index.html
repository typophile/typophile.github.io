If I can start off by asking a question regarding type used in magazines. If a certain typeface is used for a masthead, should it be reused for all other headers and title pages within the magazine, but like in differnet fonts such as bold, condensed etc? Add to that, should the same typeface be used throughout the whole magazine (for headers, title pages, body copy etc) or should they vary depending on the article subject?

Anyway, I would be thankful if you could give out a few suggestions for a typeface/s to be used for a food magazine. Masthead, title pages, body copy etc. I’m hoping to achieve a modern/in-style comtemporary look with the magazine, and would like the type to really help it shine. The photography will also be nicely done.

It’s for a student project, and so any help/advice given would be much appreciated. Thanks.

In my opinion The masterhead should a unique piece… treated pretty much like a logotype. Also I would never use the same typeface for both headlines and body copy. Better then to pick some typeface suited best for its purpose. That would give an optimal use and display of both typefaces, I think. Some suggestions:

Headlines: SOPHISTO - STALEMATE - LUMINANCE - REMONTOIRE - OXTAIL

Body copy: DELICATO - ANZIANO - TAROCCO

See for yourself; http://www.macrhino.com

Good luck!

Hi, s_c, and welcome to the forums. I don’t think that there are any “right” or “wrong” answers to your first two questions… I’m sure that there are (or have been, or will be!) real-life examples of the different options you are asking about… These are design decisions, and they in turn should be informed by your research and documentation on other magazines, not to mention trying different things out for yourself to see what works for you, etc. For example, take some dummy text and make a spread for your proposed magazine, using the same face for everything, then duplicate it and try varying the weights, or the fonts… Print everything out and compare the results…

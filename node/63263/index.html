I’m in the same situation as Bezier Abuser.

I want to make one of these flourishy, embellished, calligraphic marks. Sorry to crowd the bandwagon.

Here are some of the options I’m considering.

  1. RENOVATE. As mentioned, building off an existing font seems like it would be efficient. But that assumes that one has the right font. Even then, you gotta tweak the letters and add the flourishes and tails.

  2. SCRATCH. Personally, I think the best way to go is to work from a sketch and scan. And it doesn’t necessarily preclude the use of an existing font. It can be used with method #1.

  3. TABLET. I’ve never used a tablet for calligraphy. Anyone out there use a tablet for this type of work?

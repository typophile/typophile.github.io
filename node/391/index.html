2003-06-19 22:27:22 • typotect
Yeah, it works. Give all of your text a custom class, maybe like this:

<font class="dontLookLikeCrap">This is the text</font>

Link a stylesheet, or just add the following to your existing styles:

.dontLookLikeCrap {width: 600px}

And voila, readable text that will break at 600px. You could probably make the changes in 5 minutes. You just have to make sure that you only apply the style to text, not images within the post. With a little work, you should be able to parse the code to avoid applying this style to <img> tags.

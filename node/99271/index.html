Font needed for Neon Clock face

January 1, 2013 • neon clock

[img:sites/default/files/old-images/American Clock Company Face_4713.jpg]

This is probably a custom font/lettering of the clock manufacturer. You could try to use a similar font like [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=Dafont-1&ffam=3x5%20-%20Regular&fid=9b9ed06776df35e7dfcb1508f471bc68&fsize=60&text=1234567890&fit=1|3x5]] by K-Type (free for personal use). If you make it a bit bold (to hide the intermediate lines) and modify |1| & |2| I think it will be close enough.

I found it using Find my Font - http://www.findmyfont.com

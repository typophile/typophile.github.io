W uses Garamond 3 for body copy (and some display type)… http://www.myfonts.com/fonts/atf/garamond-no-3/familytree.html

…and the aforementioned W Macro (slab serif) by Joshua Darden… http://joshuadarden.com/custom/behind_wmagazine.php

…and both NewJune and NewLibris (sans serifs) by Hubert Jocham http://www.hubertjocham.de/item.php/type/sans/NewLibris/ http://www.hubertjocham.de/item.php/type/sans/NewJune/

I didn’t find this one. If it’s a font, it’s well hidden from the casual searcher. If I had to make it from a font, rather than from scratch, I would start with Impact Bold Condensed http://www.myfonts.com/fonts/urw/impact/ and make the words as follows: r…. use n, shift the top curved portion of the interior contour up so horizontal stroke is about half as thick, cut off the bottom 60% of the leg e…. shift the top curved portion of the counter up so horizontal stroke is about half as thick, same procedure for bottom stroke, drop bottom of counter to thin horizontal stroke. s…. use o, same procedure as for e to make top and bottom strokes thinner, cut openings in sides and add horizontal stroke. t…. use U, delete rhs, same procedure as for e to make bottom stroke thinner, thin vertical stroke to match lower case width, add rhs crossbar a…. use o, same procedure as for e to thin top and bottom strokes, cut opening in lhs and add horizontal stroke, square rhs bottom corner. u…. use u, same procedure as for e to thin bottom stroke r…. same as first r a…. same as first a n…. use n, same procedure as for r, but don’t cut the leg. t…. same as first t h…. use h, same procedure as for n o…. use o, same procedure as for e to thin top and bottom strokes. t…. same as first t e…. same as first e l…. same as t, no crossbar c…. use o, same procedure as for e to thin top and bottom strokes, cut opening in rhs a…. same as first a n…. same as first n e…. same as first e t…. same as first t Don

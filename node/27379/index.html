Technically, an Open Type font can contain either Postscript outlines or True Type outlines. From a standpoint of practicality, any True Type font that is able to load on a Windows PC can also be loaded on a modern Mac running OSX -provided there's nothing really wrong with the generation of the font. Postscript Type 1 and Type 1 Multiple Master fonts are very platform dependent. Even still, some fonts that are sold as True Type are still sold for separate Mac and PC platforms. I recall this when I purchased a license for Clearview Highway at Terminal Design's website. I received a folder of True Type fonts, but they were also called "Open Type" for some reason on the website. The site insisted that PC and Mac users download their purchases from the specific Mac and PC links. Perhaps there is something that gets lost in platform shifts concerning letter spacing and other properties in the fonts.

When would a “Windows” TT font not be an OpenType font? When it doesn't contain a dsig table. That is the only official difference between TrueType and OpenType, because all the other OpenType-specific tables are optional tables.

2006-08-02 14:45:36 • dezcom
Individual font quality is indeed the real key regardless of format. I deal with a lot of customer-provided artwork, which sometimes involves really wierd off-the-wall freebie type fonts gained from sites like DaFont and others. Even from the early 1990s it was easy for anyone to tell not all fonts are created equal. Buy some little "Key Fonts" disc for chump change and then try to use them. Convert the type to outlines and see all the errant control points all over the place -as if the type were scanned and auto-traced. Then compare it to the good type you had at the time, such as a Garamond BE weight bundled in with Illustrator 4.0. Wow. It has a lot less anchor points and more smooth, gradual curves. When I rout it out of a 4' X 8' sheet of aluminum I have no bumps or divits on it from poorly placed nodes! Egads! TTF, T1, T1 MM or OTF in TT and T1 flavors -people can generate really good or really terrible quality fonts in any of those formats. Observation of quality really does need to proceed in a case by case basis.

Bobby, The ClearviewHwy fonts have a few OpenType features (numerator, denominator, arbitrary fractions) with TrueType outlines. Since they do not contain digital signatures, they technically are not OpenType fonts, but they function as OpenType fonts. The reason there are Mac and PC versions of the font and that we ask users to download their appropriate flavor is that when all of this went up a few years ago there were still several issues with Windows PC running on Macs (OSX was not as common then). When asked now, we recommend our Mac OSX users to download the Windows TrueType versions of the fonts. There are, however, a lot of sign shops out there running some pretty old software both on Mac and PC. More than once we've had to provide PS Type1 versions of the font software to clients who were still running Illustrator 8 on OS 9.
2006-08-05 23:34:52 • .00
One advantage of Type 1 fonts that no one ever discusses -- mostly because it affects very few of us -- is that Type 1 outlines are much easier to edit. I know, TrueType outlines can be converted to Postscript-style vectors. They do change a bit when this is done. For this reason I always use a Type 1 version of a font. Hinting is easier to edit too. I edit most fonts I use, but I'm one of a few. T1 and TT look the same on paper, and if that's all that matters to you, either format will do. The only advantage of TT fonts is that they can look a lot better on a computer screen, if the designer has taken the trouble to hint them thoroughly. This is not always the case, and I've seen some TT fonts look bad on-screen as well.

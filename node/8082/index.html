Now that I can see it better, it does indeed have a lot in common with Binderstyle, but differs as well in significant ways. The unusual symetrical lowercase y and the lowercase g fit Binderstyle. The cap R and the lowercase t and f don't. Overall it's very Binderstyle-like, though.

I'm thinking now that it might be a modified version of Binderstyle. Hmmm.

I think the font for Port Magazine (their masthead/logo) is a custom font, can anyone confirm this? If it’s a custom font, can anyone suggest a few fonts that are similar which are easier to obtain?

Right, Port magazine masthead is a bespoke typeface designed by [[http://www.mattwilley.co.uk/Port-One-Typeface|Matt Willey]] (formerly Studio8) called ‘Port One’. For what it’s worth, if I have to reproduce it, I’d probably start with [[http://www.myfonts.com/fonts/urw/torino|Torino URW++]]. Some ready-to-go suggestions: [[http://www.myfonts.com/fonts/tipografiaramis/compass-trf-stencil|Compass Stencil TRF]], [[http://www.myfonts.com/fonts/behaviour/xree|Xree]], [[http://www.playtype.com/font/q-stencil|Q Stencil]], [[http://www.myfonts.com/fonts/blankisthenewblack/volterra|Volterra]] (not serif), [[https://ourtype.com/#/try/pro-fonts/bery-roman|Bery Roman]], [[http://lineto.com/The%20Fonts/Font%20Categories/Headline%20Fonts/Le%20Corbusier|Le Corbusier]], [[http://commercialtype.com/typefaces/dala_floda/dala_floda|Dala Floda]] (Roman)

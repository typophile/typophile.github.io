Text sampler software for Type web site?

June 21, 2006 • dezcom

Does anyone know if there is a text sample creating software off the shelf that can be implemented on a web site so that potential users can try mall bits of text in various typefaces? If so, how much does it cost and what is involved in implementation? ChrisL
I know of three: 1. At Typecon01, Don from SynFonts demoed a Flash-based built to order package that I believe P22 used for their IHOF site. Perhaps it's still available and would probably be the cheapest solution, though not the most elegant because it uses Flash (which wrecks havoc on character shapes, particularly curves). 2. Letterror built the LetterSetter for House Industries (and someone else?). 3. Village's mudTyper was developed by Takashi Okamoto of MudCorp. It's the best of these three, in my opinion, as it sticks to PNGs and still renders instantly.

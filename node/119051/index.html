I'm not sure if font editors support that format, so I suggest you save/export the AutoCAD file as Encapsulated PostScript file (.eps) as that format is supported by several font editors. [[http://www.high-logic.com/font-editor/fontcreator.html|FontCreator]] is one of them. This article might help: [[http://forum.high-logic.com/viewtopic.php?f=4&t=2846|From Vector (e.g. an Adobe Illustrator image) to Font]]

By which keyboard, I mean which keyboard layout. For instance, here is the portuguese keyboard layout on the macintosh. When you type the key for the tilde on that keyboard layout, you are asking for the character 007E TILDE. On the other hand, if you are using the US extended keyboard layout and type alt n to get the tilde, it is 02DC SMALL TILDE that is asked for. When the character the keyboard is expecting is missing, then TextEdit or Word switch to a fallback font, normally Lucida Grande.

2011-10-13 15:55:46 • TypZ

You are welcome.

If you test your font in Indesign or XeTeX, missing characters are clearly visible. Else, if you want to see clearly that a character is not defined when displaying some text in TextEdit or Word or some browser (especially if there are just a few characters replaced by the fallback font), then it may be useful to put some dummy and clearly visible glyph in the characters you have not yet designed. Otherwise, I would personally leave slots in grey except for those characters that may serve for “dead keys” in keyboard layouts (that are displayed in orange in the keyboard viewer).

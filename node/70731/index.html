Hi Sebastian, this is not a font, but handwriting. This style is called Kurrentschrift, it was popular in Germany until the 1940s. Your sample reads ‘Junger Fliegenschwamm’ [young fly agaric].

See these older threads for more information: German script on “Das weiße Band” movie poster Beautiful handwritten cookbook

There are not a lot digital Kurrent fonts. DS-Kurrent, the font mentioned in the first thread, will bring you only somewhat close. If you need something more commonly readable, try [[Viktor Solt-Bittner]]’s Voluta Script. F

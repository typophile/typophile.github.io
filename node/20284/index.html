Font Reseller Websites as Graphs

May 28, 2006 • Stephen Coles

The home pages of the largest font resellers as viewed using http://www.aharef.info/static/htmlgraph/

blue: links (a tags) red: tables (table, tr, td tags) green: divs violet: images (img tags) yellow: forms (form, input, textarea, select, option tags) orange: block formatting (br, p, blockquote tags) black: root (html tag) gray: all other tags

I’ll let someone with more web development experience analyze these, but I think we can assume a few things:

  • They are in order from most to least complex tag structure
  • The folks at MyFonts know how to make a clean page -- goofy graphics notwithstanding.
  • Veer wins for the most CSS-based layout -- it's table-free.

In simplest terms, a character is the smallest, basic piece of an alphabet: a letter. Glyphs are the various visible forms that letter can take. Example character: lowercase 'a'. Example glyphs: lowercase 'a', small cap 'a', alternate/swash 'a'. [img:sites/default/files/old-images/FinderScreenSnapz001_6491.jpg] So: many glyphs can represent one character. Also, one glyph can contain more than one character, as in a ligature.

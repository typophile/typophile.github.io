I presume these OpenType fonts you want to convert are PostScript-flavored (.otf extension), right? If so, I believe you can use [[TransType]].

If the OpenType fonts are TrueType-flavored (.ttf extension), then you shouldn’t have any problems, as these are supposed to be backward compatible.

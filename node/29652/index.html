I’m confused, if you hard returned each line then you need to treat each line as a separate paragraph. Therefore you can set the first line/paragraph with a left indent of 0 and subsequent lines with a left indent of, say 1 pica - or whatever you decide.

I would have recommended soft returns instead. If you do it that way, then in the paragraph palette you set the first line at whatever your indent is, again let’s say 1 pica, then in the window below it you type in -1p. (In Quark this was a little more clear) This is called a hanging indent.

As pointed out in that other thread you can also either do “indent to here” or command-backslash to set up your hanging indent. Lazy but easier if it’s a one off. If you want consistency throughout a longer text better to create the indents you want in the paragraph palette and make style sheets from them.

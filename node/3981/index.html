FOG problem - Quotes don't work

March 27, 2004 • Rene Verkaart

Hi guys, I have a problem with Fontographer. My quotesingle and quotedbl don't work in Quark. They work in FOG, but when I generate the font, they don't work. The fill colour is OK and the font is generated as a macintosh postscript 1 with a NFNT bitmap output. Has anyone else had this problem before? Thanks in advance,
Sounds like you have "smart quotes" checked in the Quark preferences, which causes the "stupid" quotesingle and quotedbl to be replaced by the curly quote marks. If you have no glyph in these character positions, it will seem as if the "stupid" quote marks aren't working.

James,

if you're on Windows 2000 or XP, just drag the accompanying PFM file to your Windows Fonts folder. If there is no PFM file but there are AFM and INF files, download ATM 4.1 Light and use it to install the font.

If you're on Windows 98/ME, download ATM 4.1 Light and use it to install the font.

http://www.adobe.com/products/atmlight/

Otherwise, you can use TransType 2 by Fontlab Ltd. to convert the font to Windows TTF:

http://www.fontlab.com/html/transtype.html

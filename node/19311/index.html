2006-04-17 23:25:00 • alldaveallen

The problem with trying to use Adobe SerifMM is that it is a Multiple Master font. Adobe’s type engine, which is used in all Adobe software, sees that a font is missing, looks it up in its internal database and, if it know about it, will adjust the weight and width axes of Adobe SerifMM to mimic the weight and proportions of the missing font. (There is also Adobe SansMM for missing sans serif fonts.)

So, assuming you could get it to show up in your font menu (and don’t ask me how Tiffany did it–I don’t know), you would need a program that can manipulate MM font axes, and most of those are from the pre-OS X days (on the Mac anyway).

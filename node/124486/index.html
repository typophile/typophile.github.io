I don’t understand why you would want an authoritative reference for that. It’s pure mathematical logic and therefore undeniable. It doesn’t require any “metal type guy” saying so in a “metal type book”. The first baseline is at “top edge” (0) + x. The top edge of the second line is at 0 + point size (y). The second line’s baseline is at top edge of the second line (=point size) + x. The difference between each consecutive baselines is therefore y + x - x = y (=point size).

2015-04-12 17:03:23 • Bert Vanderveen
Well, it certainly is true from mathematical logic when the type in both lines belongs to the same font. It can be false, though, if the first line is Century Expanded (set on the Standard Line) and the second line is Cloister Lightface (set on the Art Line or Script Line). In the case of metal type, the position of the baseline within the metal type slug is variable depending on the typeface. (I see, though, you specifically said "in a given font".) In the late 19th century, for the convenience of printers, the major typefaces made their types lining by having only three standard baselines - one for classic typefaces like Caslon or Jenson, used also for script faces, one for the workhorse faces of the day with fairly short descenders, and one for titling faces. The differences between these baselines were made to be in whole point increments as far as possible - and 1/2 or 1/4 point if not. Before that, the baseline was an individual matter, depending on the typeface.

Which font is this Italic: http://www.sanskritweb.net/fontdocs/zigabonus-italic.jpg Which font is this Roman: http://www.sanskritweb.net/fontdocs/zigabonus-roman.jpg These 2 fonts were used for typesetting a commentary to the epistles. For full bibliography see http://library.brown.edu/find/Record/b1875141 The commentary was printed in 1544 in Lovanium (Leuven, Netherlands). I own a copy of this 603-page huge commentary book bound in parchment.
Despite being printed in the Netherlands, the Roman shows strong Venetian influences…

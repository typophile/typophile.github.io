Sanglier type

October 17, 2010 • jeanw

[img:sites/default/files/old-images/sanglier-typo-test_4458.jpg] I’m working on this project intermittently since some months. It is based on a blackletter but y try to construct each letter with some geometrical shape, giving it a more contemporary aspect. My first idea was a whole family for an edition project in wich each music style (it was about music festivals) was accompagnied by its own type variation. Something like Peter Bil’ak’s “History” project — in all humility! –. The structure stay the same (generally the vertical strokes), but terminals and bowls change. This was a first sketch for it.

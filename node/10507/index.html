This question seems basic but I am not able of including one space character between two words in Fontlab when I make tests of kearning (Preview / Metrics). I follow the Mr. Carbaga manual (Learn Fontlab Fast) : see page 72 : To add the Space character to a string, type\0020 (then a space…. I did but nothing happen.

Help please! Thanks
\0020 will only work if the space glyph is correctly encoded. If you have a glyph in the font called 'space', try running the Assign Unicode command from the Glyph menu. This will assign Unicode values to glyphs based on their names, following a mapping list (the default will be the STANDARD.NAM file that comes with FontLab).

Also note that in preview and metrics strings etc. you can also enter a glyph by name, using this convention: /space

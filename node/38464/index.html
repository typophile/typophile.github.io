How can I use oldstyle numerals in Plantin Expert?

October 26, 2007 • George Adams

Hello Wondering if anyone can help. I am using Plantin regular and italic for a project at the moment. I need non-aligning numerals throughout though, is there a way of achieving this without selecting them and changing them to planitn expert? Any help would be much appreciated. Thanks George
Yes, there is a way. It seems there's no "Plantin OSF" available, but you could use [[http://www.fonts.com/findfonts/detail.htm?pid=433194&|Plantin Pro]] which contains both, lining and old style figures. Then, in an OpenType-aware application like Quark XPress, you would just turn on Old Style Figures option.

1. Type some text in InDesign. 2. Select all text (Ctrl+A) 3. In the Character palette (Ctrl+T) click the upper right menu (indicated by the down arrow symbol). 4. Point to OpenType. 5. On the submenu, click Proportional Oldstyle. That's it. From that point on, any numbers you type will be oldstyle figures (provided you are using OpenType font containing also the oldstyle figures - e. g. Plantin Pro).

If, for any reason, you cannot use Plantin Pro, you can first typeset all text using regular Plantin. Then, in one manual operation, you can change (Edit > Find/Change, in InDesign) all "1" of Plantin (i. e. lining numbers) to "1" of Plantin Expert (i. e. oldstyle numbers), then all "2" of Plantin to "2" of Plantin Expert, and so on.

To all type designers:

If you had to reduce your methodology down into a set of stages what would they be? Is it even possible to suggest stages, if the process is too personal?

Lindsayy

Preface: I’m a very green type designer. In the simplest case, you go about it something like this:

Get the idea. Draft forever. Create outlines. Work on kerning. Test. Go back and kern some more. Repeat the last two or three (or four or five) steps until you’re feeling good with it.

It depends on the approach from here; some designers will create different weights at this point either by manipulating the outlines already drawn, or by starting over from the top. Depending on the complexity of an accompanying italic face, it also likely starts from the top down. Many times a single face is all that’s produced.

I would suggest that, in between “create outlines” and “Work on kerning,” you add “Refine metrics.” Although “best practice” suggests that sidebearings should be equalized, sometimes it can be counterproductive to do so.

I always make at least one pass (and, often, several passes) through every possible combination of uppercase+uppercase, uppercase+lowercase and lowercase+lowercase letters to set my metrics. Good (that is, practical) character spacing can eliminate the need for a lot of kerning later on.

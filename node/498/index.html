2002-04-26 19:14:29 • sevenfingers
I think it's pretty nice - the type seems to work well, especially if it's a "sober" operation. BTW, swooshes aren't so bad for non-internet stuff!

But I see two potentially serious problems:
1. It's really wide - so it's less versatile.
2. With the green, those look like leaves, and not at all atmospheric effects. So it immediately makes me think of an environmantal conservation organization or something. I know blue is cold, but...

There's also this: Because your logo relies on tight spaces (in this case the gap between the "leaves"), you should make a different version for smaller sizes.

BTW, if they're serious about the aurora stuff, you might consider making a wide, shimmering, irregular curtain spanning the width of the lettering. Technically much more work for sure (you'd have to do some magic with negative spaces), and you'd have to make sure it's not too "literal", but it might turn out really nice.

Suggestions: books about typography in Portuguese

June 18, 2009 • hbeier

I made a quick search and didn’t find any topic like this, only the existing [[http://typophile.com/node/33915|Advice please: on useful old books to add to Typophile’s library]], which is not the case, I think. So I’d like to suggest some books about typography to be added to Typophile’s library, all of them available in Portuguese:

Pensar Com Tipos. -Lupton, Ellen. Cosac Naify, 2006. (translated from Thinking With Type)

O que é a tipografia. -Jury, David. Gustavo Gili, 2007.

História da Tipografia no Brasil. -Semeraro, Claudia M. & Ayrosa, Christiane. Museu de Arte de São Paulo (MASP), 1979.

Thanks!

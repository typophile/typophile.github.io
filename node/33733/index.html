Hi there I'm new here, but I have had a long love affair with my font collection. But it does give me grief when I try to find *just* the right font for a given application. So I got Suitcase for Windows. The thing is, I have a lot of fonts and I was hoping that the manager would be able to sort files by category...but it can't. Does anyone know of a way to either sort the files on the disk into categories, or know of a manager that can do this? ...Mike
Most font managers are capable of collecting fonts any way you want, but unless it either refers to an included database or embedded information in the fonts themselves, you will have to make these categorizations manually, to your own specifications. This, depending on the number of fonts in your library, can be very labor intensive, but in the end more useful to you than any factory-supplied organization system. Still it would be handy to have a guide to go by. The Typowiki might be a good place to start looking for one.

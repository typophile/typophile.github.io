There’s a great book by Tamye Riggs, called Typeface: Classic Typography for Contemporary Design, which shows many classic fonts “in action”.

No disrespect to Typophile, but when I want to bombard myself with type in action, I go to the “What The Font” forum over at myfonts.com. It’s only logos, but that is a great way to see random fonts in action.

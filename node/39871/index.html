Gill Sans Infant/Futura hybrid

December 7, 2007 • RALFS

I found this font in a german cookbook from the 50ies (DR.OETKER BACKEN MACHT FREUDE); Mike Yanega thinks it might be a hybrid typeface (Gill sans Infant with l, t, u and M out of Futura/Twentieth Century) and suggested to post it here.

Has anyone another idea? Is it common to mix typefaces? Sorry, I am not a professional so please bare with my naive questioning.

I would like to use the font for a publication, I find it very elegant…

Thanks a lot for helping me!! [img:sites/default/files/old-images/oetker_0020_3428.JPG]

I’m surprised there has been no response yet on this question. Is this a modified version of Gill Sans Schoolbook (Infant), or is it truly a hybrid typeface, mixing Gill with a Futura-like type?

I suggested that Ralf post this here, becuase I was hoping someone else might know whether type mixing was a common practice to create a ‘new’ type look. I know I am one of the older folks on this forum, but my background in type is based in my books, and not on firsthand experience in the field of typography. I was an engineer, not a typographer.

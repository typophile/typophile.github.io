Yes Do the small capitals and old style numerals need to be defined here in order to generate the OpenType PS file? Yes. Well, not necessarily "defined." You need (1) a set of classes and (2) a set of instructions as to what to do with the classes. You can put both in the features file, or you can use the classes file to "define" them. I prefer to have everything in the features file. In fact, I prefer to have that file written off so as to be able to use a text editor on it. You might have to read the manual...

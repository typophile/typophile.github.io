2009-04-11 00:52:14 • blank
If you really like to see dense, sharp text, you need a laser printer. Toner cartridges are expensive, but if you’re only printing text (no large solid areas), a cartridge should last 12 months. You can also use less expensive paper.
2009-04-11 00:52:56 • Nick Shinn
“Just stay away from color lasers” In the budget/consumer market, color lasers also tend to have worse resolution than b/w lasers. So if you mainly need/want high quality in text, definitely go for a b/w laser.

WHat you describe in your first post; packaging, leaflets, etc. would all be considered business/personal use. Using the fonts to make documents. Commercial use, generally refers to including the font software in a commercial product, or having the font itself the major value of the product, such as plastic letters that are sold individually, or a t-shirt which uses a symbol from the font as its sole image.

There are two possibilities. 1) It's the case that newer versions of Windows are getting ever stricter about fonts adhering to the appropriate fonts specs (whether it's Type 1, TrueType, or OpenType). The reason for it is stability/security concerns. As a result, some fonts that worked just fine before don't work correctly in XP. 2) It is possible to "break" the built-in Type 1 support in Windows. Most commonly by installing a really old Windows 9x/ME version on Windows 2000/XP. Have you ever installed ATM 4.0 (not 4.1) on your XP machine? Regards, T Thomas Phinney Program Mgr.

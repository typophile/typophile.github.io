You don't have to make a Latin (for one thing it's certainly a ton more work) but since many people will need a Latin font to use alongside, you should choose an existing one to be compatible with in certain ways, such as color (darkness), apparent size (how big each script looks), etc. And if you've already settled on all those things, it's worth the effort finding a Latin font that harmonizes with your design, to recommend to people. hhp

2007-06-14 18:50:36 • Pieter van Rosmalen

Actually it’s not so strange, since font rendering is handled by various pieces of software, each with different limits. For example some printers might have trouble as well.

The good news in your case is that you can figure out your onscreen algorithm’s limit by comparing the glyphs that show up versus those that don’t.

On the other hand, you need to do a lot more testing, since presumably you’re not the only person who’s going to end up using the font! You need to aim for a configuration (on each platform) that makes sense for the target user of your font; as a rule of thumb, somewhat below average.

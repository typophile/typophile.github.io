javascript:void(q=prompt(‘Search%20Typophile:’,''));if(q)void(location.href=‘http://www.google.com/search?q=site:typophile.com'+'%20'+escape(q))

Or this in Google:

site:typophile.com your search term/s here

And if anyone is interested, I just created a new command on YubNub to search Typophile using the revamped search engine.

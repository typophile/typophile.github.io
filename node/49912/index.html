When I worked on creating a monospaced Thai character set OstScript font to be used for a Simth-Corona typewriter keyboard in Thailand, I researched Thai characters in non-computer related books about this interesting language.

In those days, Unicode did not exist. Today, much information about Thai is available in the official Unicode 5.0 book from Addison Wesley (is there Unicode 6.0? yet). See page 373-376 and page 657-639.

Thai is a fascinationing language as the forms of certain letters change depending upon its context in a word.

The OpenType font format addresses this requirement with its CSUB routine. Correctly applied, a ‘smart’ OpenType Thai font can produce Thai text in a way that was not possible in the computer-based typesetting until now.

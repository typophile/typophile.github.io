I'm going to guess that you're not giving makeOTF info on what font outline file (OTF, PS, PFA) to make the font from, so it is using the default of font.ps, which isn't in your directory. Use 'makeOTF -f Parallel Histories.otf' and it should work, though I would take out the space in 'Parallel Histories', as you may have to escape the space on the command line. Taking out the space would mean that your makeOTF command would be 'makeOTF -f ParallelHistories.otf'.

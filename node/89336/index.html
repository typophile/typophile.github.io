When you have an image of a font/logo you can use a number of special software tools to match the visual image of the font against a font database. One way is to use the WhatTheFont service: http://new.myfonts.com/WhatTheFont/ It requires a clean image of black letters on white background and the letters must be "disconnected" e.g. if you have a script font you have to pre-process the image and add some white space between letters. The basic advantage of WhatTheFont is their large database but it will only match fonts they actually sell - not free fonts nor any other fonts you may have in your computer. In your case I used a different tool called Find my Font - http://www.findmyfont.com (I'm the chief designer & developer of the Find my Font tool). Find my Font can match a font image against any fonts you have in your computer (installed or not) and it works directly on any color image or screen capture - no need for pre-processing. It can search e.g. 200.000 fonts in 10 secs. If you have a large collection of fonts in your computer, go and try the [[http://www.findmyfont.com/index.php/download/compare-editions|free-trial of Find my Font]] - I think you will find it interesting.

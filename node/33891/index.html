Typography in Large Type Books

I've been discussing graphic design in books with a librarian at a library I work at, and she mentioned that design within large type books could really be worked on. As far as typography goes, I found a page that mentions some type size and line spacing guidelines, but nothing about specific fonts. I know that certain fonts read better at smaller sizes - are there fonts that work better at larger sizes for greater readability? (Especially for those who have limited eyesight.) It makes sense that larger fonts automatically make it easier to read, but frequently the fonts chosen are unattractive... If anybody has any information on this or suggestions about fonts, that would be great. David

> are there fonts that work better at larger sizes for greater readability? In terms of "normal" adult readability: 1) There is an upper limit, probably around 15, above which readability drops like a rock. 2) The larger the size the smaller the x-height should be. 3) Physically very large books are meant to be held (or often placed, like on a lectern) further from the reader; so the type basically ends up looking as large as let's say 11 point type in a regularly-held book. On the other hand, for people with very poor eyesight, really large fonts with really large x-heights help. hhp

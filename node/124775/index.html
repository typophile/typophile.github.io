Creative folk art lettering. The 3 in particular is very original. But there is a common angular pattern to the letters. You might consider taking an existing font such as Diamante Demi Bold http://www.myfonts.com/fonts/ef/diamante/ and using it as a source for your missing letters. If you want to edit to approximate the flavor of your unique letters you can use some of the wedge shapes in Albertus for guidance http://www.myfonts.com/fonts/mti/albertus/ Don

Church identity and font licensing

October 14, 2007 • Frode Bo Helland

Hi.

I’m creating the visual identity for a newly founded church. The main group is about 18-35 years old, mostly students and young families. We strive to not grow old (Don’t we all?), so I’ll probably aim the identity at 18-20 year olds. Everything needs to be kept as cheap as possible since we don’t have much money, but I really want to create something unique. Church design is, in general, terrible.

Anyways, here’s my issue: We use type in many areas: Music lyrics for projector, Powerpoint presentations (yeah!) for projector, our website, interactive cd-roms and a lot of printed material. I intend to use Dolly as one of our base typefaces, but we can’t afford to buy a lot of licenses. Ex.: Everyone speaking in church is responsible for creating their own Powerpoint presentation, with the obvious dreadful result. I’m going to set specific rules for typography and composition, but I can’t just e-mail everyone a commercial font like Dolly.

Any suggestions? Should I use a “web safe” font like Georgia, since everyone probably have it, or should I look for a free font? (I know this probably isn’t a good place to ask for suggestions.) I read some place about creating styles in MS Word with specific names, and when you imported the documents into InDesign you could define different specifications for those styles. Is there something similar in Powerpoint?

Often when a corporate identity package is created, one set of rules for designers and another for everyday use is specified. For instance, you might say that Dolly should be used on all events posters, committee logos, pamphlets, etc. and Georgia should be used for daily memos, correspondence, PowerPoint presentations, etc.

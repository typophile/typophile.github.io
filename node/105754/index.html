To fix the problem with the space you need to fix a width to the character “space” 0r 0x0020. Just select it, right click and “set width”.

The kerning in Word is not activated by default. To get it you need to select “Format > Font > Advanced” and check “kerning” and specify the smallest point size for which you want kerning to be applied.

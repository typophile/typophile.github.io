You can also use Python which is more powerful and flexible. The following script sets anchors and places caps and small letters diacritics in the right position. It is very simple to make changes in the anchors positions.

With Robofab and Font Tools installed:

Building Accented Glyphs for PS1 font

from robofab.world import CurrentFont

accentBuilder lives in robofab.tools

from robofab.tools.accentBuilder import AccentTools

font = CurrentFont()

a list of accented glyphs that you want to build

myList=[‘Ntilde’,‘ntilde’,‘aacute’, ‘agrave’, ‘acircumflex’, ‘adieresis’, ‘atilde’, ‘aring’, ‘eacute’, ‘egrave’, ‘ecircumflex’, ‘edieresis’, ‘iacute’, ‘igrave’, ‘icircumflex’, ‘idieresis’, ‘oacute’, ‘ograve’, ‘ocircumflex’, ‘odieresis’, ‘otilde’, ‘uacute’, ‘ugrave’, ‘ucircumflex’, ‘udieresis’, ‘Adieresis’, ‘Aring’, ‘Agrave’, ‘Atilde’, ‘Acircumflex’, ‘Aacute’, ‘Eacute’, ‘Ecircumflex’, ‘Edieresis’, ‘Egrave’, ‘Iacute’, ‘Icircumflex’, ‘Idieresis’, ‘Igrave’, ‘Odieresis’, ‘Otilde’, ‘Oacute’, ‘Ocircumflex’, ‘Ograve’, ‘Udieresis’, ‘Uacute’, ‘Ucircumflex’, ‘Ugrave’, ‘Scaron’, ‘scaron’, ‘Yacute’, ‘yacute’, ‘Zcaron’, ‘zcaron’, ‘ydieresis’, ‘Ydieresis’]

start the class

at=AccentTools(font, myList)

clear away any anchors that exist (this is optional)

at.clearAnchors()

add necessary anchors if you want to

at.buildAnchors(ucXOffset=20, ucYOffset=40, lcXOffset=15, lcYOffset=30)

print a report of any errors that occured

at.printAnchorErrors()

build the accented glyphs if you want to

at.buildAccents()

print a report of any errors that occured

at.printAccentErrors()

I have a composite management script that Ben Kiel wrote for us. It dumps and fuses composite glyph data to and from a semicolon-separated text list, of the format

[glyph name];[width];[base component];[x offset];[y offset];[2nd component];[x offset];[y offset];…

e.g.

aacute;1465;a;0;0;acutecomb;1435;0

[I now always use zero-width ‘comb’ accents for composites, not spacing accents, as the composites then match mark postioning offsets and can be leveraged to build GPOS data.]

Being able to dump and fuse composite info in this format makes it easy to define or edit composite glyphs in a spreadsheet.

We paid Ben for the development of this script, but it is released under the GNU open source license. Contact Ben for more info: http://benkiel.com/

I’m refining kerning based on pages of words such as Kern King, printed from Indesign. Is there any reason to not make the kerning adjustments to the page, in Indesign, and then just implement the same values in Fontlab so that I don’t have to generate as many test fonts?

No reason not to, as long as the font is using a 1000-unit em (that’s what InDesign uses). Otherwise you’ll have to do some conversion.

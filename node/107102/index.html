I am interested in matching this font for a restaurant menu project. The font is serif, a book weight, slightly condensed, has a slightly taller cap and ascender height, is somewhat like Clearface (but not), has an x-height that is also slightly taller, and the loop of the “a” is more like a slab-serif face (more full round, than sloping downward. All help is very welcome. Thanks

Assuming that serifs have somehow fade in your sample, there is a number of matching fonts, but I’m afraid all of them coming from font forgery companies: Aesop - Copyright SoftMaker Software GmbH 1995 AeroDB - Copyright by Christine Mauerkirchner and Rainer Grunert AdrianoOpti by OPTI Castcraft

For the last one (AdrianoOpti) Luc Devroye mentions the original name Aeterna but I can’t find any more info about it. http://luc.devroye.org/OPTI-Castcraft-Fontnames.txt

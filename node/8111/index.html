Can anyone help me find a macro that generates all open files from FontLab? I'm sure there has to be one out there somewhere… Any help is greatly appreciated. Thnx.
Paul, 1. Save all files as .vfb files, best in the same folder. 2. Go to Tools / Transform Range / Transformation range: Fonts in the fonts list / … 3. Click on "+", add the .vfb files that you want to generate, in Destination format select the format you want, in Destination folder pick a folder to your liking. 4. Click on OK, then Run. Adam

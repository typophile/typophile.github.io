it's a signboard for a kindergarden

June 25, 2014 • Duintje

This nameboard is produced around 1965-1970 in the Netherlands. I want to know what the font is, so I can reproduce the board![](/images/old-images/Naambord BSV Selwerd_5556.jpg)

As you can see from the differences between repeating letters, this is probably handmade. I don’t know if it’s based on some old font of the era and if this font is actually digitized, so I suppose your best chance is to find something similar in style & contrast.

If nobody else finds something closer, take a look on the following similars: [[http://www.myfonts.com/fonts/flat-it/cotoris/|Cotoris (Bold)]] by Ryoichi Tsunekawa [Flat-it type, 2006] [[http://www.myfonts.com/fonts/anewmachine/holt-sans/|Holt Sans]] by Kent Swecker [A New Machine, 2012]

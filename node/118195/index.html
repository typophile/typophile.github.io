Could be wrong but as it is very neutral with no very distinctive earmarks, it could be as simple as [[http://www.myfonts.com/search/akzidenz-grotesk|Akzidenz-Grotesk]]. On their website, they’re using [[http://www.myfonts.com/fonts/dunwich/armitage|Armitage]] which comes close, main difference being on /E bar width. [[http://www.myfonts.com/fonts/linotype/neue-haas-grotesk|Neue Haas Grotesk]] using the [[http://www.myfonts.com/fonts/linotype/neue-haas-grotesk/std-display-55-roman/glyphs.html#glyphs/612006/315|alternate straight leg /R]] might work pretty well too. More alternatives with some minor differences: [[http://www.myfonts.com/fonts/garagefonts/district-pro|District]], [[http://www.myfonts.com/fonts/hubertjocham/voice|Voice]], [[http://www.myfonts.com/fonts/berthold/royal-grotesk|Royal Grotesk]] (Akzidenz renamed?), [[http://www.myfonts.com/fonts/wordshape/clobber|Clobber]]

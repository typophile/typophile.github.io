Petition Adobe for unified OpenType support!

September 14, 2006 • paul d hunt

I got fed up today (finally) of the spotty OpenType support in Adobe CS applications, so i finally looked up their feature support page and sent them a couple of product feature requests. I’d like to ask all of you who have time to do so to do the same thing. Or if this is not a big concern to you, perhaps you’ll use this form to request a glyph pallete in Photoshop. These were my two requests this morning. Hopefully if Adobe gets enough requests like this, they’ll actually do something about these shortcomings! The link to Adobe’s Product Feature Support Page is:

http://www.adobe.com/support/feature.html

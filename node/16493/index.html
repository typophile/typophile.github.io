2005-11-28 22:15:44 • Dan Weaver

Apparently, the type engine in Flash is greatly improved. There is threshhold control over anti-aliasing and the pathetic kerning controls have been refined.

Most importantly, perhaps, font rendering in .swf files is supposedly much clearer, regardless of positioning.

I have not yet tried Flash 8 yet myself, but some type-sensitive colleagues of mine think it’s the biggest improvement yet for typographers and designers.

From Macromedia’s site: FlashType—high-quality font rendering engine: A revolutionary new font rendering engine that provides clear, highest-quality font rendering, FlashType makes small fonts look amazingly clear and greatly improves readability. Choose from several options for font rendering, each optimized for different use cases. Optimize font rendering with the new custom anti-aliasing feature in Flash Professional.

As far as Dreamweaver goes, I’m not sure. I use Smultron and CSS Edit.

2005-11-29 04:52:06 • Addison Hall

I teach using MM applications and the Studio 8 version of Dreamweaver has made my life a lot easier in terms of introducing CSS to my students. Essentially Dreamweaver has absorbed TopStylePro (etc.), which sucks for TopStyle, but, as I said, makes my life a lot easier as an instructor. It’s not just the preview improvements, but the simple ability to edit already defined styles in a side-palette rather than opening a dialogue box is great. Not to mention that this upgrade makes creating and editing ID selector styles (such as “#idname p, a, h2”) much easier to work with than previous versions.

I’m not a Flash guy, so can’t comment there, and Fireworks seems pretty much the same, but I do appreciate the CSS improvements a lot.

Firefox now with webfonts support

October 25, 2008 • Typography.Guru

The latest beta and the nightly builds of Firefox 3.1 now support webfonts. Full article: http://opentype.info/blog/2008/10/25/sneak-peek-font-face-in-firefox-31/

BTW: The final release will support Access Control Headers, which can be used to control from which domains the font can be accessed - a kind of server-side URL-binding.

FYI to anyone thinking about installing this beta: I’ve been using it for a few days now and it’s not remotely stable. Only install it if you don’t mind it crashing a few times every hour.

Contact sign painter/type designer John Downer. He has put together a good presentation about these techniques in painted signage; the Type Director’s Club might have a video of him giving the presentation earlier this year. /why do people design 3d type?

I’ve designed one 3D font, as part of the Bodoni Egyptian family, in 1999. I did it because I was interested in the tools that Fontographer provided to create that sort of effect, also because I thought it would be appropriate for a design based on a type from an era, c.1800, where the genre first appeared, and because I was interested in the effect of “outline and fill” that could be achieved in Quark XPress by layering.

I was particularly interested in the way that a proper 3D typeface is not a mere mechanical offset, but has adjustments to make the dropshadow appear more uniform in thickness.

Actually, I had earlier (1980) created a 3D version of a Headliners face (Gryphon) – again because I was interested in the way that technique (at that time, photo-modification) could be used to create such effects.

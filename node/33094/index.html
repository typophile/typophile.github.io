You can create that effect using the underline options in the character palette. Just highlight the text, then choose underline options in the drop down menu in the character palette. Set the underline color to white (or whichever color you want behind the text), then play with the pt size and offset until you create your desired effect. ps: my first post! :)

When I want color under my text I set the type and then draw boxes on a layer underneath using pre-measured blocks to ensure that all the boxes are the same size relative to the text. As for Der RIng, it should relate to the specific production if for public consumption, if for actors/singers to read, just pick something legible like you would for any text.

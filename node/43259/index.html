This from Dov Isaacs over on the Adobe ID board: Unfortunately, some third party (i.e., non-Adobe) PostScript and PDF software either doesn’t implement CID-encoding or improperly does so. However, using Distiller to create PDF contrary to popular belief does not preclude CID-encoded fonts from appearing in the resultant PDF files.

Note that all PDF/X standards and Ghent PDF Workgroup best-practice PDF guidelines and standards support CID font encoding and PDF exported directly from InDesign.

If you already have FontLab Studio 5.1.x installed, you should be good to go. If you don’t, make sure it is installed before you do the upgrade to Mountain Lion, otherwise it won’t let you install it under Apple’s new code signing rules. FontLab Studio works just fine under Lion and Mountain Lion.

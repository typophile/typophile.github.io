I’d probably try to start with [[http://www.myfonts.com/fonts/marksimonson/proxima-nova|Proxima Nova]] (soft rounded version [[http://www.myfonts.com/fonts/marksimonson/proxima-nova-soft|here]]) or [[http://www.typography.com/fonts/gotham/overview|Gotham]] ([[http://www.typography.com/fonts/gotham-rounded/overview|rounded version]]), manually outlining to artificially embolden and round it.

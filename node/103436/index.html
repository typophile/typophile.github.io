2013-08-16 10:43:50 • donshottype
For a metal type this was a difficult design to produce and use. The little wedge type pieces are obviously designed to float clear of the main strokes. In use the type would soon be stressed and the wedges would be pushed into the main stroke, or pushed outwards. Where digital versions are based on different impressions, this should account for the variation in appearance. As for the R, I don’t know if it’s a redesign or a font size difference. A proper digital version would be a real treat! Don

This is the symbol Psi

Hi Christopher,

it depends: psi is a Greek character (U03A8 for uppercase), so it is included in every font with Greek language support.

Then, it could be used as a symbol in mathematical typesetting. There are a couple of codepoints, depending on its application: U1D7A7, U1D6BF, U167A7, U1D76D, U1D733, U1D6F9.

Special fonts like Cambria Math will likely contain these glyphs. F

Period before "y"

November 2, 2004 • rugi000

i'm designing my first font (actually a bastardization of helvetica) and i'm experiencing a little problem. i'm using fontlab. every time i generate the font and view it in say, illustrator, the period after the lower case 'y' moves itself before the 'y'. something like this: "honorar.y" any ideas why this is happening?
I have had somewhat the same problem with different letters though. I am guessing that you have autospaced and autokerned the font. That is what I did and weird things started happening. Certain letter pairs were kerned as much as -1000. I had to go back and change the values manually in Fontlab's kerning dialog to get this straightened out. Check the kerning values for the glyph pair "y." Magnus

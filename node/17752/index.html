[[Indices]] : [[Terminology]] : Faux Formatting Faux formatting is formatting automatically done by computers to fake [[style|styles]] of fonts such as [[bold]], [[italic]] (sloped roman), [[small capitals|small caps]], or [[squooshed|condensed]] from a single font that either is not linked to or does not contain true versions of these types of formatting.

Garamond Changes When Conterted to Outlines

August 9, 2006 • duncan

Can anyone explain to me why it is that Garamond Bold seems to bulk up when it is converted to outlines? I am working with a logo that uses Garamond Bold and when I convert it to outlines it looks like it has a 0.5 px stroke. It really changes the feel of the logo.

Thank you in advance for imparting your wisdom,

Duncan Robertson

This effect is due to the fact that hinting information embedded in the font is being lost when converted to outlines. The lower the output device’s resolution is (screen!) the stronger the effect.
2006-08-09 16:16:45 • franzheidl

Whatever software you have is using different methods to render text and outlines; e.g. it’s probably using hinting when it renders text, which obviously it cannot do for curves which may or may not be text.

If the logo’s intended for print, then just ignore it: there won’t be any visible difference on paper. If it’s intended for computer use, then the solution will probably depend on the software you’re using, but in similar situations I’ve found rendering the logo at a higher resolution than needed and then scaling it down in Photoshop often helps.

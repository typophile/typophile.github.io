It is a little off topic, but one thing you may wish to consider is to have the same setwidth for both the lining and oldstyle tabular figures. Many fonts don't. I suppose there is no real problem if you don't, but for those of us who make use of the "figure space" (U+ 200[something]), it is a help. It is important that the roman and italic tabular figures have the same setwidth -- both roman and italic figures can occur in a table. I've seen (& fixed) fonts where the roman & italic tabular figures had a different width; of course, they don't line up. Bold is less common in tables, but does happen. You can also have a different design for the tabular oldstyle figures, to help fill/fit the common body. One example would be Carter's tabular oldstyle figures for Galliard.

Rachel, in glyph names, an underscore before the dot determines a ligature conjunction, while an underscore after the dot can be freely used. In class names, underscore can be freely used though in FontLab Studio, class names that start with an underscore indicate a kerning class. A.

This smacks of custom type design.

This logo replaced an earlier version that was set in Lubalin Graph Bold: http://www.myfonts.com/fonts/itc/lubalin-graph/

The type in the new logo may be a heavily customized adaptation of Lubalin Graph Bold Italic:

Plume Ad Italic, by Dalton Maag, comes close for the |F| (and it’s not bad as a starting point for the |X|): http://www.myfonts.com/fonts/daltonmaag/plume/

New York to Las Vegas might work for the |O| (and it’s also not a bad starting point for the |X|): http://www.fontbros.com/families/new-york-to-las-vegas/styles/regular

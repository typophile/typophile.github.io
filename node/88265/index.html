2011-12-15 13:40:25 • Michel Boyer

Hmm. I’m a late arrival. To answer the question, I don’t find any font completely indispensable. Actually, I don’t find any font to be partially indispensable.

I’d assume that as a designer, you’ve purchased InDesign, which comes with a lot of fonts. So, I suppose that you’re looking for fonts not used by other people, for a sort of “self-branding” purpose? The free font world is populated by some giveaways from larger families, which might work for advertising. The open source world still has a lot of dogs, but careful prowling will turn up promising fonts, and a few real gems. That’s where a font editing program is useful.

In my word – books – useful is far more important than new, which lets judicious purchases go a long way. I understand that it is a world many find boring, and not terribly profitable. Again, I’d prowl fontsquirrel, dafont, and the TeX font resources. Open source fonts and FontLab Studio will teach you a lot, as well as expand your font library.

Try Lato, BTW. http://typophile.com/node/77283

And look at the League of Moveable type. I don’t find the Goudy clones too useful, but while he lacked a sense of space, Goudy was a good man with a curve. Some of these “based ons” might work for you.

Actually, just Google “open source fonts” and spend a couple days going to the sites.

FWIW

Concerning Minion and the Adobe Reader, it seems like the software doesn’t give a licence to use the font. The Eula says that a licence to use the software (and fonts) is issued only for the ways of use and purposes described in the documentation (help, online Adobe docs, etc…). Since there is no word in the documentation of how to install Minion, it seems that the Eula forbids it. (However it is another question, if the Eula is applicable in every country.)

In support of Ryuk’s suggestion that the FO design used Gotham as a base, I note that the website for Crown Metropol uses the Gotham Medium webfont. You can make a multiline font from an existing font – preferably a sans – in Fontlab by using the parallel lines feature. Use pull down tabs: Tools -> Action -> Effects -> Parallel. Then edit the result. Repeat as required. With proper attention to detail the results should be as good as any of the fonts mentioned. Don

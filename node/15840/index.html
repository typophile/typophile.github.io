Do you have checked the font with the “Compare Family” tool of the free [[Adobe FDK]]? It will work on single fonts too. Also MS [[Font Validator]] could give some info.

OpenType fonts should work in Quark like unkerned Type1 standard fonts. The line break is really strange. My guess, you have some glyphs missing or wrong encoded.

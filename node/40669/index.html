Hello,

I think the source of the problem are the bitmaps not the hints. The Bitmap Tool does not create any hints (or better instructions). Try to generate the font without the option “Export embedded bitmaps”. You find it in Options - Generating OpenType & TrueType - OpenType/TrueType (.ttf)

Regards Eigi

I assume you need the font for the Russian language or Cyrillic. As far as web type that has Cyrillic support Times is best choice. I don’t think Georgia has Cyrillic natively ( you can buy an OpenType version from Ascender that has Matthew Carter oft under under-appreciated Greeks and Cyrilics for Georgia). This would probably require you embed some how with flash and in that case you can use something even more prettier- like Baskerville… it one the best neoclassic faces that has Cyrillic support. Its fabulous for text and display.

Adobe allows embedding in all their font products so you don’t have to worry about extra licensing issues if you go with them.

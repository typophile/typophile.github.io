Finding the Japanese equivalent of Courier is the interesting bit. Courier is used for many reasons. Sometimes, it is used to evoke the output of typewriters. Sometimes, it is used mainly because it is a monospaced font, especially to display code. Now, because most Japanese typefaces are monospaced anyway when it comes to the kana and the kanji, it's tricky to decide what to use for the translation when in the original Courier was used only because it is a monospaced font. I might be wrong, but I have the feeling that in Japan the Courier rough equivalent is probably the monolinear, rounded ナール(Nar). It's visually distinct from the Mincho or Gothic styles, and is quite widely used. I'm having trouble finding any information on it in English, though. Here's the Japanese Wikipedia page: [[http://ja.wikipedia.org/wiki/ナール]]

It's hard to say the equivalents without seeing the design you try to make, but, If you need to set Japanese font and Univers together in the same page, I recommend you use AXIS Font from [[http://www.typeproject.com/|Type Project]]. AXIS Font covers wide range weights and has smart skeleton, each weight style of which was developed along with the one of Linotype Univers. Axis trial fonts are available to use from [[http://www.typeproject.com/trial/|here]]. The trial version font has only Regular weight and the limitation that Kanji have about 1000 characters only, but all Hiragana and Katakana characters are included, so you can check the looking of it before buying it. Of course, the retail version of AXIS Fonts have all Kanji characters (Adobe-Japan1-3). The retail site is [[http://www.axisfont.com/|here]]. This site is only Japanese language, but the customer support via Contact will answer your request in English. Apple Japan uses AXIS fonts on its web site. You can also browse them [[http://www.apple.com/jp/|here]]. Sabon goes well with Mincho style. Kozuka Mincho from Adobe might be good for Sabon. And Kozuka and AXIS, both Kanas of these have similar skeleton, I think these work well together. Kozuka family has Gothic style but it has tiny serifs on the end of strokes, so I recommend AXIS for Gothic style. Unfortunately, almost of Japanese foundry don't support the order from overseas. However, Linotype deals with [[http://www.linotype.com/289648/typebank-library.html|Typebank family]], it would be helpful for you to get Japanese fonts quickly. Typebank Gothic and Mincho would be useful to combine with Univers and Sabon. Which Courier, rounded or edged? Its hard to find the an equivalent of Courier. I can say rounded san-serif might be better, but Gothic would be also no problem for it. M+ 1m might be good for it, but it seems to be in progress now, so it doesn't have enough Kanji characters to set Japanese texts. I hope it would be helpful for you. Best regards. Kunihiko Okano Shotype.com

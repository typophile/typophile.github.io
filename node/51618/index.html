In case you are working with InDesign: Change the wordspacing values in H&J if you want tighter word spaces globally. If you want to do this on a selection (and this is straight from InDesign Help):

Adjust kerning between words With the Type tool , select a range of text and do one of the following: To add space between selected words, press Alt+Ctrl+\ (Windows) or Option+Command+\ (Mac OS). To remove space between selected words, press Alt+Ctrl+Backspace (Windows) or Option+Command+Delete (Mac OS). To multiply the kerning adjustment by 5, hold down Shift as you press the keyboard shortcut.

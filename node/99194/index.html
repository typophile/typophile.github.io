It depends on which letters you need to make “come together” with your logotype. There are mid-weight script typefaces which may be close but you’ll have to add any eventual interlocking by redrawing parts of them, so I’m not sure using a typeface is the best solution.

Maybe you could start from something like this: http://www.myfonts.com/fonts/schizotype/gelato-script/ or this http://www.myfonts.com/fonts/bluevinyl/fancier-script/ but not sure it’s the best way to go: if you need to produce a similar logo, you have to draw it freehand.

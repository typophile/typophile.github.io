2003-10-16 21:19:02 • Grant Hutchinson
You can get the ratings symbols (R, PG-13, etc) in certain pi fonts. (I'm pretty sure I've seen one, but I can't remember where.)

The "RESTRICTED" text looks like it could be Akzidenz, but I've seen the same text in any of your standard workhorse grotesques (Akzidenz, Univers, Helvetica, Trade Gothic, etc.).

The text "UNDER 17…" and "STRONG VIOLENCE…" is in Verdana (or possibly Tahoma). Once again, this is by no means standard. You commonly see this text in any of the typefaces mentioned above.

My guess is that "Columbia Pictures" is a lettering job, but that's only a guess. You can get an EPS version(s) of the logo here.

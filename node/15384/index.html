If you like Zapfino, you should try Zapfino Extra LT Pro, an expanded version of Zapfino: http://www.linotypefont.com/8-36-8-17885/zapfinoextra.html It includes a load of OpemType contextual features that produce a lively script impression by automatically switching between different alternates in an intelligent manner.

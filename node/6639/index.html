'FUEL' is set in Futura Extra Bold, not Kabel.

'Buster' and 'Made in Germany' are indeed set in Kabel, but not the ITC version as linked above. They use the Linotype version which has a square dot for the lowercase 'i' in the Black weight, rather than the diamond shaped one in ITC's ultra weight.

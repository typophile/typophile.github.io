Text font similar to Delicato

December 17, 2014 • Meloncally

I am always on the look for a nice text font similar to Delicato, which I love, but it’s unfortunately above my means to purchase at this time. Any suggestions for a similar, more affordable alternative much appreciated. Thank you!

Mathew Carter designed a face for MS Windows 8.1 called Sitka which is very close to Delicato. Seems to be only distributed with the software.

If you need an entire family of fonts (roman, italic, bold, bold italic), you may not be able to find anything more economical than Delicato. Here are a few suggestions that may work, but some only if you need a few weights:

Aster comes very close: http://www.myfonts.com/fonts/urw/aster/ http://www.myfonts.com/fonts/linotype/new-aster/

Leksa Pro: http://www.myfonts.com/fonts/alexandra-korolkova/leksa/

Berling: http://www.myfonts.com/fonts/linotype/berling/

PT Serif Pro: http://www.myfonts.com/fonts/paratype/pt-serif-pro/

even if I do purchase them, will that allow me to use them in the ebook? A “normal” license is usually a desktop license, which only allows the installation on a fixed number of local devices. It doesn’t cover any uses which “distribute” the font to other users. There are special licenses for that: web license, app license, ebook/epub license, OEM license.

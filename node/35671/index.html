I’m not sure what you’re asking about how to format things in Word, but getting a line of text between 2 horiz bars should be pretty straight forward. Since word will substitute a horiz bar anytime you type 3 hyphens in a row, enter: — [hyphen][hyphen][hyphen][enter] (your contact info here) [enter] — [hyphen][enter]

that should get you roughly what you’re asking about, I think.

The color of the text in the graphic you pointed to is black. Unless it shows in a different color while you’re editing it?

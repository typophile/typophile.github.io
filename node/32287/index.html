2007-03-16 10:05:52 • alexfjelldal

Word chooses as default whichever figure style the foundry designates by the basic names “zero, one two, etc.” Traditionally that was tabular lining, but some fonts now are published with proportional oldstyle figures as default.

In a Typophile survey, around 60% of respondents favored proportional oldstyle as the default.

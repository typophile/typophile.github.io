I completely agree that there is a need to emphasize that what’s currently termed “preferred” names should be considered the “normal” ones (possibly by omitting any qualifier) while today’s “normal” ones should be considered as “special” (serving for style-linking or “grouping” as you say). AFDKO does so since a long time. This is not a matter of words but of conception. The only thing that I do not like is the term “typographic” – everything we do is “typographic” somehow, so the term doesn’t say anything, like “black” does not help to distinguish one kind of liquorice from another.

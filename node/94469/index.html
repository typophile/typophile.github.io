Agree with fvilanakis. Looks like they distorted some generic condensed sans. It could also have been manually slanted. I’m sure it would be not so hard to reproduce the same effect starting, for instance, with some generic sans like [[http://www.myfonts.com/fonts/adobe/helvetica-neue/cond-bold-obl/|Helvetica Condensed Bold Oblique]] or [[http://www.myfonts.com/fonts/linotype/univers/UniqueName-217369-115593/|Univers Condensed Bold Oblique]]. For the rest, [[http://www.myfonts.com/fonts/fontbureau/bureau-grot/cond-medium/|Bureau Grot Condensed Medium]] suggested by fvilanakis is probably one of the best approximation you could get to me.

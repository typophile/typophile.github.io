Heather, if you're still here: A tip about using this forum. Sometimes (a lot of times, actually) it will seem like your post is not going through. Most of the time it actually does, but there is a bug in the code that prevents the original page from loading correctly. If it seems like this is happening, stop the browser, copy the text of your post (just in case) and go back to the original page. Usually, your new post will be there. If not, you've got it on the clipboard and can try again. If you keep trying to post by hitting the "post this message" button, you will instead post multiple copies of your post.

It's a stupid and annoying problem and hopefully at some point it will be fixed. But in the mean time, just consider it part of the way Typophile works. (Judging from the outage of Typophile in the last two days, not the only bug.)

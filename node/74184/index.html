Viktor, you don’t mention which software and version you are using.

For Adobe InDesign CS4 (and newer), this

I have also tried to set the prime mark in Times New Roman, using Unicode U+2032 and then change the font to Garamond, but this does not work since the glyph is missing.

can be solved by this:

  1. Create a character style that applies only a font; select a font that supports U+2032 – if you have a choice, select one that matches your mathematical font. (Although I suppose for “just a prime”, Symbol would suffice.)

  2. Create a GREP style in the paragraph style you are using for math. Have it apply the character style you just created to this text:

\x{2032}

  1. There is no point 3! The character style will automatically be applied to every occurrence of the prime character in that paragraph style.

Since GREP styles cascade down the “Based on” hierarchy of paragraph styles, you could define the GREP style in your main text style, and it’ll be active in every derived one as well.

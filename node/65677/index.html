I scanned this image from the novel Breaking Dawn by Stephenie Meyer and I’m pretty sure it’s a font of some sort. I have successfully identified every other font used in all four books in The Twilight Saga (not just those that are common knowledge among fans, like Zephyr, Pablo, Carmilla, etc.), and I’m in the process of creating a website where I give the names of these fonts since there seems to be a great demand for this and for the fonts used the movies. Speaking of, I tend to have a good chuckle whenever I see somebody mistakenly identify the Twilight logo movie font as Zephyr or a modified version thereof. It is most definitely a modified version of another font, but it is NOT Zephyr. ANYWAYS, getting back to what this post is about; the picture I include here is of two ornaments. If by chance you are able to identify which font they belong to, PLEASE let me know. Thanks in advance.

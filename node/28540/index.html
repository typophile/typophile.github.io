Sounds like you’re looking for something similar to the good old Adobe Type Reunion (ATR).

I read that Fontlab allows renaming/grouping. But, I don’t have Fontlab, nor am interested in purchasing it (the expense isn’t currently in my budget.)

If you’re willing to edit the font files (and assuming that: 1. the license allows it, and 2. the fonts are TrueType and OpenType), the only thing you’ll need is [[TTX]], which is free, and a lot of knowledge on how to juggle the various Name IDs of the ‘name’ table.

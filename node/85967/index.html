Font Bureau’s <a href=http://www.webtype.com/catalog/intended/?ts=829>Reading Edge (“RE”) series of webfonts were designed specifically for use at small sizes on screen. Personally, I’m partial to Benton Sans RE, but any real suggestion for a specific family would depend a bit on the context in which it will be used.

(Full disclosure: I am a Font Bureau employee, but have made use of the RE fonts in personal web projects, so can confidently comment on them from that point of view.)

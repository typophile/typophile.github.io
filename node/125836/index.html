Type Battle 43: Classic Font Emojis

April 3, 2017 • Jared Benson

This week’s battle: Classic Font Emojis

Select a classic font and design 3-5 emojis in that style from the currently available set. When posting, include the name of the font you chose. (Thanks to Dave Williams for the idea.)

Winner take all, no holds barred. May the best designer win.

THIS BATTLE ENDS: Sunday, 09 April 2017— Midnight PST Brush off those type drawing skills and get to work!

// REQUIREMENTS

  • Solution must contain 3-5 characters
  • Emojis must be custom drawn.
  • Image size: 1100 width x 550 height. 72 dpi
  • Color: Black and White
  • Format: Optimized PNG, GIF or JPG

// ABOUT TYPOPHILE TYPE BATTLES

This is your chance to flex your type muscles on a weekly basis. You have one week to create and submit your entry. Anyone may submit a design response to the challenge. You may enter as often as you wish. Post anytime. Critiques and comments are welcome throughout the game, from participants and spectators alike. Smack talk is encouraged.

As with any street battle, there is no panel of judges and no prize — the community decides on merit alone, and you may win the ability to call yourself the best on the block for this challenge.

If this is your first Type Battle, start by looking through some of the previous Type Battles.

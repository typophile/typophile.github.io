Well, I use "tracking" to mean that. And kerning is something you really do in a font, not in a setting. But frankly the terminology isn't so clear cut - people adapt and improvise it all the time. Move the first "R" a hair to the right. Loosen "EA" a hair. Tighten "RK" slightly. Tighten "KE" a hair. (Where "slightly" is about twice as much as "a hair". :-) BTW, if this is going to be set pretty large, I'd remove the trapping (the pulled-in curves intended to lighten joins at small sizes - like in the "armpits" of the "N"). hhp

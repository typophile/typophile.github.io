If the letters is vinyl, there is a possibility this would not be a standard TrueType or OpenType digital font. Many manufacturers of vinyl cutters and large scale printers create their own OEM fonts, which are vector fonts in a non-standard custom format. See for example the [[http://www.gspinc.com/default.asp?contentID=56|Gerber System]] which uses custom Gerber Fonts (.gsf). Most of these fonts are exact copies of standard commercial outline fonts, but some of them are modified to be more suitable for a specific application like e.g. vinyl cutting or engraving.

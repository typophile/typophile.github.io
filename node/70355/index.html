The font menus are build from information from within the font files themselves. If this is how it comes out, there is nothing you can do about it (unless you made the fonts). Technical background: in a proper font family, the entire family should have the same name, and each of the styles should have an additional unique name. If the family is called "Knockout HTF94-UltmtSumo" for one font, and "Knockout HGTF93-UltmtHeviwt" for the other, all software -- no matter how smart -- will think it's actually two different families. It's kinda sloppy of the font creator; contact them if it bothers you.

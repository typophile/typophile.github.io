ColorSync workflow. Your printer should come with ICC profiles. What you need to do is buy a spider that will create a profile for your screen (Gretag McBeth is most in favor). That should allow you to get accurate results. If you are not already doing this, you can convert your image in PShop directly from RGB to a printer specific profile if you are not using the image for traditional process printing. Good Luck.

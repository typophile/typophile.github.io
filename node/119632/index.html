Partial names with small capitals

September 17, 2014 • Physicist

I would like to use small captials for the names in my document. So it looks like this:

In the HILBERT space we have …

However, I am not sure what I would do with “Lagrangian”. The actual name is “Lagrange”. Should this be

  • “LAGRANGian” (wrong Name highlighted)
  • “LAGRANGEian” (wrong word in the end)
  • “Lagrangian” (exception of the rule, since name is no full substring)?

Similarly “fermion” from Fermi, “boson” from Bose.

In your example, “Hilbert” it’s not semantically a name, so you don’t need to apply whatever style you choose for proper names (more so for the others).
2014-09-17 14:37:31 • riccard0

LAGRANGian

If you are emphasizing a word (by using caps, bold face, or whatever) do the entire word or it may look weird. Your readers should be smart enough to figure it out. And you certainly don’t want to misspell a word, it’ll be an obvious mistake.

Another approach might be to rewrite the sentence so the name stands by itself without “…ian”.

Regarding the “Why?”: I saw it in lecture notes and found it quite nice. Then I read a bit about it and found that it is not uncommon, at least.

When I change the words in order to get the full name into them, this probably does more damage for the reader than the small caps help. So I would rather do no more small caps then, since words like “boson” and “lagrangian” would break that pattern. And consistency is key for a thing like this.

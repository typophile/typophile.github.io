Ordinal letters in an all-caps display face?

August 21, 2008 • blank

I’m doing an all-caps display face. In theory nobody will be using it to set running, so can I just leave the ordinal a and o out? And if I do need them, do I need to create minuscule ordinals, or can they be superior versions of the caps?
In theory, "in theory" motives work. In practice... I'd make the ordinals based on majuscules, just because they'd fit in. Don't just scale existing glyphs, though. Gotta look good. Imagine a headline title (in all caps) about "La 1ª Mujer que ha hecho..." (I'm assuming you're also doing numerals). That brings up another thing...would you want oldstyle numerals with an all caps font? Hrm...I could see them working if they're as tall as the capitals, with the descenders/ascenders going below/above the letters...sorta. I'd have to see it in practice to know whether or not I'm crazy here. Peace, JT

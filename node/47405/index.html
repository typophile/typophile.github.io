what kind of effect to try?

Photoshop. The most basic:

  1. Type the word etc; convert to shape.
  2. Layer Styles Blending Options dialog: Advanced Blending 35%; Blend Mode: Multiply
  3. Apply Layer -> Layer Style > Stroke / Inner Glow/ Outer Glow etc etc etc

If it does not work check the electrical wall outlet :^)

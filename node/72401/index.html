The “computing” logic is simply built into the Excel formulas. For example if the formula reads “1” for the left-turn lane then it would result in the ASCII character that yield the correct arrow within the new font. The cell is simply formated to use the new font. So let’s say the left-turn arrow is “t” in the new font, then the result of the formula would place a “t” within the cell. The cell would be formated with the new font and yield the arrow symbol.

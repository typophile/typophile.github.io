Custom work by [[http://youngjerks.com/|Dan Cassaro]] for [[http://statemottosproject.com/gallery/2011/2/8/new-york-dan-cassaro.html|The State Mottos]] project. Hard to say because it has been heavily customized, could have been done totally from scratch (try to ask directly to the designer). I don’t know any commercial font that could give this final result directly (probably some Ps effects applied and a long process to get there). Some suggestions (geometric gothic condensed, /R straight leg and round top /A), in the same vein (a lot work needed: possible slanting, manually adding tuscan-style spikes, 3D/shade effect, roughing texture): [[http://new.myfonts.com/fonts/chank/liquorstore/|Liquorstore]], [[http://new.myfonts.com/fonts/jnlevine/deerfield/|Deerfield]], [[http://new.myfonts.com/fonts/brassfonts/anorak/|Anorak]] ([[http://new.myfonts.com/fonts/brassfonts/anorak-cond/|Condensed]]), [[http://new.myfonts.com/fonts/dsgnhaus/dieselis/|Dieselis]], [[http://new.myfonts.com/fonts/letraset/mekanik/|Mekanik]], [[http://www.letterbox.net.au/berber|Berber]], [[http://new.myfonts.com/fonts/agfa/raleigh-gothic/|Raleigh Gothic]] (using some alternates), [[http://new.myfonts.com/fonts/urw/baucher-gothic/|Baucher Gothic]], [[http://new.myfonts.com/fonts/suitcase/magion/|Magion]], [[http://new.myfonts.com/fonts/suitcase/monopol/|Monopol]], [[http://new.myfonts.com/fonts/comicraft/elephantmen/|Elephantmen]] (aged style)

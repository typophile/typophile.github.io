Stephen,

Sure. The project will be a series of 6ʺ×9ʺ pages, composed as spreads (probably about 20–25 pages in total). The left side of each spread will be composed of illustrations, typographic compositions, and photographs, sometimes with captions relating them to the main text. Each of these pages will also have a one-word identifier which ties it into the text in some way. This is the largest typographic element, at about 24pt, and is where the typeface in question will likely be used. Right now I’m using Franklin Gothic Condensed for the captions- which can change, however the way the captions and layouts are set up, they tend to have a significant vertical emphasis, so I think it might be good to have the larger type contrast with that by being wide or rounded.

The right-hand side of each spread will contain the actual text, and is set up in a very old-school way, with classical proportions. Right now this is set with Sabon, which I feel pretty comfortable with. The text itself deals with a number of issues. Some concepts which I feel are relevant to type selection are the idea that we simultaneously parse a number of typographic styles at any given point in time (and are getting much more adept at doing so), and the idea that type reflects culture and can do so either on an honest, historical level or in a contrived, imagined-by-the-designer kind of way, and that these two are not always mutually exclusive.

As is probably obvious by now, this is a student project, and my budget is fairly limited, so it would be nice to license something which has the side benefit of being useful in the future. The selection of fonts available at the school is fairly good, but they don’t have anything truly contemporary.

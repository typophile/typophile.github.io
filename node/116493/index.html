Nothing free that I can find that is even in the ballpark. About as close as I can get is Jeff Levine’s Bayview JNL http://www.myfonts.com/fonts/jnlevine/bayview/ based on the condensed version of Inland’s typeface Studley from circa 1900. At US$25 it’s not a budget breaker. It can be used as is for a rough approximation. But relatively minor editing can make it very close. Don

2012-10-14 12:37:44 • aiming528-spammer
The second glyph of any pair will be colored if it is a dependent member of a kerning class (to remind you that if you’re going to alter the kerning, you should use the master glyph, unless you want to create an exception). To eliminate the distraction you can change it in Options, Metrics window colors.

As cerulean said, in Metrics windows colors* you can change the color from purple to any color you wish, including black. By doing this the dependent letters don’t change their status, only the color in which they are shown.

  • I am on a Mac, so the path is: Fontlab Studio menu > Preferences > Metric windows > Metric Window colors > Dependant pairs.

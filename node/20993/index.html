2006-06-29 18:33:56 • Goran Soderstrom

Try this;

  1. Draw a box around both objects.
  2. Select All.
  3. Select Correct Path Direction.
  4. Select Remove Overlap. This does the same thing as Illustrator’s ‘Remove Front’, but the box that was drawn around the objects remains.
  5. Delete the box

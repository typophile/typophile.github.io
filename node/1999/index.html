Basically, you need to use software that speaks Unicode. In some cases, this only helps if the OS also speaks Unicode fluently (e.g. OS X).

You cannot access these characters in any version of QuarkXPress up to and including 6. Juergen Kurz recently said that they would add Unicode to QXP 7 (at a cost of 18 engineers for 9 months).

More recent versions of Freehand reportedly do Unicode (may require OS X). Photoshop certainly does. Any version of InDesign does. Depending on the application you may need to change keyboard settings to enter these characters, or use the OS X character palette. In Illustrator CS and InDesign, you can use the built-in glyph palette.

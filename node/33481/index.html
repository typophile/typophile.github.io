Isabella, it’s very hard to know whether something ‘script’ is a font, or hand-drawn, unless there are enough letters to see some duplicates. Even then, it can be hard to be sure, because with OpenType fonts it is possible to have alternate characters and ligatures that vary the letters. Offhand I do not recognize this, although I know there are a number of Casual or Everyday Handwriting fonts that may be similar.

If you look at the Casual Flowing Scripts in Part 7 of the Script Font ID Guide you will see a large selection of this category of script. The other part of the Guide is Part 9, Everyday Handwriting. You may find this font, or something similar to it. Then use the Script Font Reference List to see where it comes from.

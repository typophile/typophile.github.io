Very close, except for the leg on R, is a new font, Sergio FY Black http://www.myfonts.com/fonts/fontyou/sergio-fy/ Three weights, italics, ornaments. For some reason this reminds me of a spaghetti western. The wedge or triangular serifs originated with a 19th century font category called “Latin.” You might also like the somewhat similar font Latin CT http://www.myfonts.com/fonts/castletype/latin-ct/ “With six widths (from Extra Condensed to Wide), Latin CT offers the most extensive collection of Latin fonts available, each with a large character set that includes support for all European languages including those that use the Cyrillic alphabet.” There almost certainly other Latin fonts that are close matches.

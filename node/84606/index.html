or

If the object is on its separate layer it could be that the layer is set to non-printing (usually the layer name would be in italics)

If I understand correctly, you’ve got other photo boxes that link to the same file and they are acting okay. So I’d suggest that you delete the photo box you’re having trouble with, then select (click on) one of the other photo boxes that’s acting properly and select Duplicate from the Edit menu to make a duplicate of that photo box, then slide the duplicate box into position and make whatever adjustments are needed to the cropping and so forth.

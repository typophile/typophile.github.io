2008-08-13 15:28:07 • christopher.martens

I like the goose and the colors you’ve chosen. I think there are a few changes that need to be made to the type though.

Is the “.com” necessary? - This is for a town isn’t it?

I wouldn’t do the glassy effect on the type as it’s been done to death and I don’t think it suits a logo for a town - to me it suggests a more programming/internet type company.

The contemporary style sans-serif font along with the effect you have on it AND the “.com” give this a very modern, cold feel. It’s too corporate. I’d suggest removing the reflection effect, removing the “.com” and perhaps changing the font to one with a little more character. (though the font selection may be fine once the glassy effect is removed).

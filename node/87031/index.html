Typomapp available as free web app

November 3, 2011 • typomapp

Typomapp features 430 typographers from Gutenberg to the present day and about 1000 typeface samples to illustrate their work.

You can access it here.

Key features:

  • toggle on and off the Points of Interest (adjusting the time slider or by category: typographers, places, events, foundries, favorites)
  • suggestive search function
  • full index
  • save favorites

We hope you’ll find it useful.

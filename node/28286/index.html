Light Light It Regualr Italic Medium Medium It Bold Bold It

It perplexes me that, after all these years, there is no way built into OS X to move fonts in and out of suitcase files. (Suitcase has a feature for doing this I think, but I don’t use Suitcase.)

The simplest way I know is to boot into OS 9 (if you can) and combine the suitcase files in the Finder by simply dragging one onto the other. If your Mac can’t boot into OS 9, move the files to an older Mac that can. (It’s handy to have one around for things like this.)

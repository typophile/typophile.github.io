CosmeticBit

September 17, 2009 • hefler

CosmeticBit is a pixel font I’m working on for the last 3 years (yeah! pretty slow). What make it different (or try to), from other pixel fonts is make it have OpenType features, such as subscripts, superscripts, ligatures and, at least, have all European alphabet characters. If you want to have a go, drop me a line: felipe [at] sugarcaned.com.

Pixelfonts are mainly used in Flash applications, and Flash does not support Opentype. That’s a bummer. It looks nice, but there’s not much to add to this genre (though the language support might be very useful to someone!). I’d rather go down the road of subpixel rendering and optical sizes for screen use – which really is some sort of hinting when I come to think of it.

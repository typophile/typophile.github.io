I compared the two fonts and found a few differences, such as the upper swash on R, but the main difference was that Palace Script connected smoothly, while Pollux Script – at least the old copies of two versions I had on file – connected with a gap. Palace Script dates from 1923 as a Stephenson Blake/Monotype font. Similar fonts from that era: Embassy Script (Caslon 1924), Marina, Imperial, Society Script (all Stephenson Blake).

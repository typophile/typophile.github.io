Bitmap font composition software

December 25, 2004 • richard_washer

Using mainly PHP-mySQL scripts, with a little Javascript thrown in, I have created an online script to render input text into images, using ligatures where appropriate. Currently only a single typeface is available, named Burgerserve, it is created by my brother Christopher Washer. Its narrow form is available, only in lowercase. No numbers, uppercase or punctuation are available, the typeface itself is still being developed. You can find out more about it and the script itself here: TypeRender 1.0. I will also add a page describing how it functions for those who have some knowledge of php.
Note that without JavaScript enabled (some recent issue with IE and/or SP2 can do this), the select boxes will not appear correctly, but due to there being only a single typeface, this has no effect on the script and isnt a problem.

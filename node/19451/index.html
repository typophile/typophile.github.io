Safari vs Firefox

April 23, 2006 • pattyfab

I've been using Safari since it came with my Mac and like it just fine, vast improvement over IE or Netscape both visually and in terms of speed and display of graphics. I notice on this forum that a lot of people use Firefox and wonder why. Are there any advantages to Firefox that should make me consider using it instead of Safari?
There are many philosophical reasons for using Firefox, but when it comes to a web browser, I'm not such an idealist. I've used Firefox and Safari extensively and will tell you that, especially on the Mac, Firefox isn't nearly as polished. IMHO, Safari feels like more of a professional product. I prefer Safari for type rendering, ColorSync, Bookmark syncing with iSync and the highly usable SnapBack feature. If you prefer the Mozilla/Gecko layout engine that Firefox uses, I recommend Camino, which is much more Mac-like and professional feeling than Firefox. Some practical reasons for using Firefox might include: 1. Better compatibility with crumby banking websites and such. 2. More extensible features with plugins. However, for every Firefox plugin that's well-written, there are at least ten that are poorly written. I'm not trying to start a war here – these are just my personal preferences.

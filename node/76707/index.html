I am curious to find if there is a style available for Photoshop that allows a blank colored layer to be ‘erased’ by type to reveal (eg. a photo in this case) layer underneath?

As far as I know you can’t leave the text as editable text for this effect. You have to either rasterise the text or just select it (CMD/Strg + Click on the text layer), then create a mask on a coloured layer.

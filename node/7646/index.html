2004-11-11 22:54:14 • speter
The Japanese version of InDesign has a composite font feature that can enable exactly this sort of thing. Some set of default glyphs in one font are replaced by glyphs from another font. I don't think it enables you to do them as alternates.

You could also use SING glyphlets to achieve the same thing, including your choice of having them as alternates or as default glyphs. SING is a newish Adobe technology for having standalone glyphs that add on to existing fonts. Again, it was developed primarily for the Japanese market. Currently it's available as a public beta for InDesign CS, but it would be reasonable to expect greater integration in the future.

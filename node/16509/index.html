Bitstream Font Navigator 5.0 that comes with Corel Draw 12. It's a maginificent, super-fast font manager. I have about 70,000 fonts in my database and Font Navigator takes 8 seconds to start. Extensis Suitcase took 5 minutes (!) to start with that database on the same machine. But when Linotype releases FontExplorer X for Windows, I'll migrate. Adam

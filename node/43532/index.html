2008-03-25 00:32:41 • Capnhud
This can happen if the outlines are converted to a TrueType font along the way. TrueType does not use the same kinds of curves that are used in PostScript fonts and many drawing programs (Bézier curves). TrueType uses quadratic curves. When these are converted back to Bézier curves, you end up with more nodes.

Also, some designers don't always use what might be perceived to be the "minimum nodes" way of working. There are cases where a curve won't flow smoothly enough without adding a couple more points. This is hard to see and explain since the difference is quite subtle, perhaps even in-perceivable to a lay-person. Type designers are a perfectionistic bunch though so a minute change will seem imperative to them. ChrisL PS: the upper drawing has some points which seem illogical though.

Hello all,

Any ideas on these two? The Ghostbusters logo was created in 1984. The type seems very familiar but frustratingly I haven't been able to find it. The type is distinctive by the spur on the G. I can't tell if the type below ("collector's series") is the same family or different typeface altogether (maybe another Norvarese?). That too is familiar, but my eyes or my brain are too tired now.

Silverado from Red Rooster seems close (including the spur), but Silverado was created after 1984. RR says Silverado was based on Eldorado, but they had to change the name since linotype had a dissimilar type by the same name. ITC Symbol is also similar, as is Memo, but both don't have the spur on the G.

As for the type in used for autamata (another good CD, btw), I haven't a clue.


Starting from the bottom upwards, "Collector's Series" is set in
Friz Quadrata. It's not the same type as the "Ghostbusters" logo.

"automata" is very close to Mark's Proxima Sans, but no cigar.

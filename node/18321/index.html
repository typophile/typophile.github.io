Hi Kristina,

It’s hard to say without seeing your file, but here are some thoughts in order of probability:

  1. There is superscripting built in to InDesign, and there is ALSO an Opentype superscripting available IF it’s built into the font. Perhaps you are selecting “Opentype superscripting” from the special Opentype menu – but if it’s not built into your font, nothing will happen. Choose the other superscript option from the character palette

  2. Maybe the preference is tweaked – you can set the percentage that superscript reduces the font

  3. Style sheets in both documents with the same name but different definitions?

Let me know if you figure it out.

2009-04-06 12:32:29 • dberlow
It's a (slightly bizarre) feature, not a bug. When you draw using the primitives tools (rectangle and oval), or using the paint brush, there is a "paint mode" that affects what happens when you draw. It doesn't affect the line and curve drawing tools, but it will affect the paths you have already created with them as soon as you use one of the "paint" tools (the rectangle, oval, or brush tools). Here is the paint mode selector on the paint toolbar: [img:sites/default/files/old-images/paintmode_4611.jpg] The modes, from left to right, are: Empty color Auto color Black color White color These modes affect what happens when you create overlapping shapes with any of the "paint" tools. Unfortunately, they also affect any existing overlapping shapes that you have drawn with other tools as soon as you use one of the "paint" tools. To prevent this automatic merging from happening, be sure that the "Empty color" mode is selected. The behavior you're seeing is because one of the other three modes is selected.

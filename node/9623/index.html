I think you will have a tough time finding these, because the red lettering looks like a customization of Benguiat Frisky (the S and B are upside-down; and a new a and g have been used) that has been given baseline bounce. The 'Children's Dentistry' font is similar to some of the 'Child's Play' fonts from FontFont, but only similar. It's very possible, since this style is so easy to create, that this is an original do-it-yourself font, or one from the countless amateur sites with handprinted fonts. You can have a font like this made for under $10 from your own printing, so anyone could have done it.

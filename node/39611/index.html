>What are the typical rights that go along with something like this. Typically you’d pay a fee for the mods and then be able to use the font under the standard EULA terms that accompanied the un-modified font. >Can we distribute it conditionally to vendors? Negotiating such a right would be possible, but depending on the size of your operation and number of users/workstations that will need to have the font installed (on-site or off) it would probably be less expensive to “count” users and get enough licenses for everyone. >Can we name it? Naming can be tricky as renames can dilute the original font trademark. However appending the company name after the font name may be acceptable – for example “Neo Sans Intel”

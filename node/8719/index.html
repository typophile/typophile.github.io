Because of the way those letters connect, I am convinced that that is handwriting and not a font.

For something similar I would suggest checking Part 9 of the Script Font ID Guide (Everyday handwriting). You might see something there that you feel has the same quality you like about this sample. You then look up the vendor using the links to the Script Font Reference List, which has information about the fonts.

My own personal choice to simulate this look would be either Shimmer from Blue Vinyl, or Velvet Script from Typadelic. (See samples from Script Guide below).

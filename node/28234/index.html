Hello, If you have set the option "Genrate OpenType [kern] feature if it is undefined or outdated" on - the malicious feature code may be created during the generation of the font. Try the folowing: "MetricWindow/KerningAssistance/Update[kern]feature" then you have the code for the kern feature in the OpenType panel and you may get a helpfull error message when you try to compile.

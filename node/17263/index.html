Have you tried the tools already in the software? Quark 6 (and others) has a table creator and Illustrator (for a long time) has a graph creator built in. The Quark one has a ‘crosshatch’ icon on the tool menu below the picture box icon. Illustrator has a graph icon also in the tool menu. It is possible to control style and font for both once you have created a master/style sheets. Tim

Oneline© , according to a post on http://logopond.com/gallery/detail/135264 in 2011 is an unused proposal for an insurance company logo by a Polish graphic studio Fajne Chłopaki http://www.fajnechlopaki.com/ The word Online© is subtitled UBEZPIECZENIA which means “insurance” in Polish. Don’t know if the script is an original by the studio, or whether it is a Polish adaptation of an existing logo.

In any case it’s probably handlettered. Note the differences in the repeating letters.

The bouncing lowercase has some similarity to Mistral, but is not monoline http://www.myfonts.com/search/Mistral/fonts/

Monoline script fonts that have some similarity, but are too regular Shelby http://www.myfonts.com/fonts/laura-worthington/shelby/ Confetti http://www.myfonts.com/fonts/tipo-pepel/confetti-tp/ Veltro http://www.myfonts.com/fonts/profonts/veltro-pro/ Matinee Idol http://www.myfonts.com/fonts/comicraft/matinee-idol/ Sneaker Script http://www.myfonts.com/fonts/flat-it/sneaker-script/

Plus a monoline script font that is similarly free-style, but different Toothpaste http://www.myfonts.com/fonts/schizotype/toothpaste/

BONUS Not monoline but freestyle with some similarity Amplia http://www.myfonts.com/fonts/jonahfonts/amplia/ Don

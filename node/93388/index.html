http://www.myheinz.com/Home.aspx#/choose This is a Heinz promotal site from a couple of years ago. The font being used to create those personalized labels is named “TK Super Duper-BG2”. LHF Hensler is much bolder and has a number of differently shaped glyphs, but it has a similar feel. The TK (for Tomato Ketchup, no doubt) font appears to match Heinz labels, whether it’s Heinz' original bespoke font or a cleverly made copy. Either way, it doesn’t seem to be publicly available.

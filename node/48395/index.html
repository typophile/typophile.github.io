2008-08-14 08:00:07 • Renko
> subtle curves into the slabs These are called "brackets". All the typefaces you list are unbracketed slabs. The flat bottomed 'a' is pretty unusual so I don't think you'll find a match for that. Best to start with something close and modify it. The closest thing I can summon is Amasis.

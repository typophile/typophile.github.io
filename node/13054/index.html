Hello everyone, i need help on one issue concerning a layout made on a mac that must be edited on a PC. First of all I need to use one single font file (Opentype or Truetype) to build this layout. Then, the document should stay editable on PC. Do to so i've made a Opentype and a TrueType test of my font. The main problem concern the key map. For example, the "é" character on Mac goes away on PC. There are also problems with the kerning… My question is simple : how could I keep the integrity of my font file and use it both on Mac OSX and on Windows XP. Thanks. Guillaume PS : Hope my english is understandable…
As you're working on Mac OS X, I'd suggest just making a Windows flavor TTF, with no Mac-specific cmap - just include a regular Windows Unicode cmap. Run the resulting font through Microsoft's font validator. With respect to the kerning, you need to turn kerning on in Word on Windows - it's under format, font, character spacing. Finally I'd suggest that using InDesign on Windows and Mac might result in less reflow issues (possibly none) as the product shares it's code-base cross platform - Word doesn't. Cheers, Si

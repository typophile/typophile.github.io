@ SuperUltraFabulous: Good call on checking the true PDF cover.

TIP: If you open it in Acrobat Pro and use the ‘Touchup Text Tool’ and highlight the type and right-click and get ‘properties’ it will tell you the font, font-size, horz scaling, color, etc. [img:sites/default/files/old-images/tungsten_5752.jpg]

In this case:

THE = HobokenHigh-SemiSans NEW RULES = Tungsten-Semibold

FOR = HobokenHigh-SemiSans DIGITAL GENTLEMEN = Tungsten-Medium

AND OTHER = HobokenHigh-SemiSans HIGHLY EVOLVED HUMANS = Tungsten-Medium

HOW TO AVOID thru BRAD PITT = Tungsten-Medium

It looks like Regency Script. There is a digital version called Regency Script FLF by Casady & Greene [2004] and there is also an OPTI/Castcraft version: [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=OPTI-Castcraft&ffam=OPTIRespighi%20-%20Script&fid=6f5f1e87b1b63843e71c7593e0191a7d&fsize=60&text=Harold%20H.%20Leese%20(A%20Boulton%20%26%20Sons)&wrap=2|OPTI Respighi]]

PS: Be careful when using OPTI “orphan” fonts in a commercial project. You can read more about OPTI here: http://luc.devroye.org/fonts-27506.html

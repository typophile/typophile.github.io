Phidias Greek is a metal typeface designed by [[Hermann Zapf]] for [[D. Stempel AG]]. It is designed to be used together with [[Michelangelo]]. A digital version of the glyphs is included in the [[Linotype]] typeface Palatino nova Titling.

Please see the [[Palatino]] entry for more information.

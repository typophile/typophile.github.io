None of my postscript fonts had kerning when I moved to OSX. Here's a crappy and inconvenient solution that should allow you to view kerning: Don't install your fonts in the library/fonts folder. Don't use font management software either. Illustrator does not read metrics info when the fonts are stored here. Instead you must put them here: hardDrive/library/application support/adobe/fonts. Try that and I bet you'll have kerning again. Newer versions of illustrator may have fixed this, I dunno.

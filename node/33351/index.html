Probably not. From what you say, I suspect that the font you have is not Unicode-compliant. If that’s so, then you will have problems when using an Unicode-compliant font, like OpenType fonts tend to be. You can, however, make your text Unicode-savvy, by using this tool: http://www.2cyr.com/decode/?lang=en

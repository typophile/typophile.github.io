There is a “Use them” section at http://www.photofont.com/photofont/use/

You can currently use photofonts in Adobe Photoshop and Adobe Illustrator using the free Photofont Start plugin. Plugins for Adobe InDesign and QuarkXPress are currently close to be released, as is Photofont WebReady, an application that allows web designers to produce dynamic web page headlines using photofonts (using a mechanism similar to sIFR).

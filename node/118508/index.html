2014-08-07 11:01:22 • riccard0
I use LaTeX for my letters; the letter class http://en.wikibooks.org/wiki/LaTeX/Letters works fine. To get palatino, you put in the preamble \usepackage{palatino} or \usepackage{mathpazo} if you have mathematical formulas. The class also works fine with XeLaTeX, which allows you to use opentype and truetype fonts.

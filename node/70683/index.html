I think you're confusing Neuropol and Neuropol X. The regular Neuropol was discontinued about 6 years ago. There are several versions of the original Neuropol of varying quality still hanging around of free font sites. In 2004, the old Neuropol family was scrapped. I redrew Neuropol X from scratch. In 2008, version 2 was released which had better vertical metrics. In 2009, more accented characters were added to support a few more languages. http://www.typodermic.com/19.html For a display font, the character set is probably sufficient for a logo and headings.

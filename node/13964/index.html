[[Indices]] : [[Terminology]] : Hinting

Hinting, more accurately called instructing, is a method of specifying how [[digital font|digital fonts]] display at small sizes on low-resolution devices, usually for on-screen usage. This is accomplished by providing instructions in the font file that define which pixels are turned on when producing [[bitmap]] images.

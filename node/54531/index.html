In Fontlab how do I find out what the path numbers of each path are in a glyph with multiple paths?
If you are doing PS outlines, you hit CTRL + F7 for the hinting mode, and it will show you the numbers (hierarchy) of the paths by first nodes. I forget if you can change their order or not.

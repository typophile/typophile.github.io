From the Quark forums:

When you launch QuarkXPress, some conflicting components may prevent the application from loading properly. The XPress Preferences file, fonts, XTensions, printer descriptions or other supporting files may be damaged or incompatible.

Solution: Rebuild a new set of Preferences in the local QuarkXPress folder. To do so:

For Mac OS 10.3.x - 10.5.x:

  1.   If the application crashes, Force Quit out of QuarkXPress.

  2.   On the Finder level, go to the Hard Drive > Applications > QuarkXPress folder.

  3.   Inside the QuarkXPress folder, create a new folder by going to File > New Folder

  4.   Label the new folder Preferences (use an upper case P - It is important that “Preferences” be spelled correctly).

  5.   Launch QuarkXPress from the QuarkXPress folder.

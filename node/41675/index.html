I guess you are using a mac computer? I would advise you to use fontexplorer. It's a free font management application. http://www.linotype.com/fontexplorerX Just download and copy to your applications directory. After that start up the program (just double click), then you can drag and drop the font files into the application window and that should do the trick for you. When you then start any application (illustrator, indesign) you will find the font in the font list of the application. good luck. dav2.

Hi Joel, 1. You have to unzip the zip file. That will bring you a folder with the font files (and, normally, some other files like the EULA etc.) I assume you are on Mac OSX? Then just doubleclick the zip file. 2. The font has to be activated, so that applications can use it. If you don’t have a lot of fonts, the easiest way is to move the files to your Fonts folder (Your-User-Name /Library/Fonts) mac post script is that right? Depends.

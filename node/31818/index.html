“this looks like an antiquated version of the “pound” sign. unicode ID # 2114. also known as the “number sign”. as in #." Ch, This symbol is actually called an Octothorp, otherwise known as the numeral sign. It has also been used as a symbol for the pound avoirdupois but this usage is now considered archaic. It’s name derives from its cartographic use as the symbol for a village (eight fields surrounding a square) as it literally means ‘eight fields’.

Open Type guide?

March 31, 2012 • inkexit

I'm looking for a good guide to Open Type and all of its features, especially the programming methods available. Preferable something a little less arcane than the standard "encyclopedia" that passes for a lot of programming manuals. Thanks all.
Both the Microsoft Typography and Adobe Opentype web pages contain official specifications as well as extensive listings of all currently defined features. Most of them come down to "this feature should be implemented as a Single Lookup, and that as a Contextual one" -- not much of programming required. Combining the few available commands can make things interesting; take, for example, Tal Leming's Fraction Fever 2. There are also a couple of threads in the Build section that discuss writing out arbitrary numbers in Roman form. Anything particular you are looking for?

OT feature code is not a programming language in the usual sense. There are no functions for system info, there is no random function, there are no functions period. I find it's easiest to think of it as a formalized kind of search and replace language, something like GREP or regular expressions.

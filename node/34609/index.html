You can’t make several different outlines into one outline font. Each outline font can only have one set of outlines per glyph. “Hinting” is an advanced technique that manipulates that one set of outlines to create different bitmap renderings. However, most of the current applications and operating systems do not use TrueType hinting the way they used to, so it is pretty much impossible to do what you’re trying to do.

Regards, Adam Twardoch Fontlab Ltd.

Ok let’s tell Sam straight:

You’ve done the wrong thing mate. You’ve drawn squared but vector font in Illustrator. Importing into FontLab won’t make it a bitmap font. It will still be vectors and it will look like those squares at bigger sizes as well.

Hinting is actually just a special technique/software algorithm of a font rendering engine, to make each font look as good and readable at small sizes as possible. You can somehow “help” this program that does hinting (to still preserve your style and readability of the glyphs), but it’s not in a way how you did it.

Dear Eggs,

“So what do i have to do to get the right size as it is into fontlab without scaling,”

Well, that’s possible, but “normally” you need to scale the type up in your case so the OS can scale them down later…if that makes sense. The real question is how do you scale them to get the same effect as fonts, as the effect they have as illustration.

If you don’t want to scale, go to FL/FontInfo/Metrics and Dimensions/ and set the em units to the number of pixels in your font, copy the contour into the glyph window and you have now set the em to receive the little letters you made at actual size. When you compose the font at that size in Illustrator, it should be identical.

The “more standard” process, is to set up FontLab to be an accurate representation of the screen. Assuming use of a 1000 unit em, this means that when the total pixels size of your font is 10, e.g. 7 pixel cap ht. one pixel above that and two below the baseline, the FontLab 1000 em needs to be divided by 10 to give you a 100 unit pixel.

So you go to FL/preferences/glyphwindow/dimensions/GRID STEP, and set that to 100 x and 100 y. Now take one of the characters you drew for a 10 pixel em and scale it until it fits the grid exactly. You should be able to figure this out once, and then all the glyphs will be scaled the same.

Now you have a font that will also give you the same fonts in illustrator by scaling them which will give you more “room: to design in FL.

“AND how do i get all these ’different’ fonts in one .ttf file?”

You don’t. TTFs are not so visionary as you. You may see my exercise on this topic at http://rogerblack.com/blog/screen_fonts_history…

Cheers,

David Berlow

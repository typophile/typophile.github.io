One solution I saw recently was to insert the soft return before the period. Like so…

blah blah blah website at www
.typophile.com blah blah blah

It seems odd to begin a line with a period, but it prevents the reader from thinking that the sentence ends with "www."

I usually work around the problem in the magazines I design:
I remove all the URLs from the main text blocks and replace
them with a visual reference (symbol, word in contrasting colour,
reference number…). Then I put all these URLs with a short line
of explanatory text in a separate text block that can run over
two or more columns.

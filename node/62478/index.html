2009-09-28 23:11:07 • afonseca1974

Use a sans serif font in a document. Save or export as PDF. Make sure you don’t embed the font in the PDF. Uninstall the font you used. When you open the PDF in Adobe Reader, you should see the Adobe Sans instead of the font you used. The weight and proportions will depend on what font you used. Adobe Reader will try to match it as much as it can using Adobe Sans. To match the sample you posted above, you will need to know what font it was originally set in.

I don’t think there is any way to select Adobe Sans directly. It wasn’t designed to work that way.

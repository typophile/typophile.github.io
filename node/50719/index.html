Announcing UAFDKOML

October 21, 2008 • benkiel

For those interested in issues involving the Adobe FDK and/or .fea file syntax, UAFDKOML (Unofficial Adobe Font Development Kit for OpenType Mailing List) is for you. Signup is here: http://groups.google.com/group/uafdkoml/. You’ll need to request an invitation (done not to exclude people, but to exclude spam), but that’s the only hurdle to joining.

Windows PC's pixel = ?# em units

May 8, 2005 • ebensorkin

What is the # of em units to match a PC’s pixel? ( Not a mac’s ).

The reason I ask is that I have font which uses 1000 UPM & has 100em unit pixels ( vector not bitmap ).

This font looks nice & crisp at 10 pt - on a mac.

But it does not look crisp at any size on the PC. It has been true type hinted ( which helped alot ) but it seems quite clear - it just isn’t fitting the grid.

Any ideas?

The default resolution on Windows systems is 96dpi, but many people tweak it. Otherwise, 7.5 points should in theory render your font crisply.

In the not-too-distant future, operating systems will automatically query the resolution off the display using Extended Display Identification Data (EDID), so a “pt” will, once again, be 1/72" or thereabouts.

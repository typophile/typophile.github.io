Hi, looking for some help identifying lettering for an old hotel. Any help would be greatly appreciated. Thanks!

Architectural type of this nature and age didn’t always correspond to type used for printing. It was most often hand-lettered and custom manufactured.

You might find something similar among the offerings of the Greater Albion Typefounders, which specializes in revivals and homages to type of this era: http://www.myfonts.com/foundry/Greater_Albion_Typefounders/#item_32

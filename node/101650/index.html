#1 looks like [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=Apple&ffam=SI%20New%20-%20Regular&fid=eaa52210817d4955de38753fd31bb64e&fsize=60&text=Obenwelt%20Der%20Silberne%20Faden&wrap=2|SI New]] Apple font manually condensed and modified. More specifically the copyright of SI New font says: “1990-91 Apple Computer Inc. 1990-91 Type Solutions Inc. 1990-91 The Font Bureau Inc.”

#2 must be [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=Letraset&ffam=Scriptek%20-%20Regular&fid=2c358e446161db38ac7c49006ffdd7be&fsize=60&text=Stellt%20euch%20einen%20Schrank&wrap=2|Scriptek]] by David Quay (Letraset, 1992) also available from ITC

Combining Avant Garde with Chaparral.

January 3, 2012 • skeetone

Hello people,

First of all I wish you all to be very happy this year!

Now to my question. What do you think of combining ITC Avant Garde Gothic Pro for titles with Chaparral pro (typekit) for body copy?

If you think it wouldn’t work, any tips would be very much appreciated!

Thanks!

I think if the difference in size is big enough, you can combine almost anything. But I don’t think the aesthetic of these typefaces work well together. I think Chaparral could be nice, it is a friendly and warm face, but avant garde is cold and stylized, and it is too geometric (in a bad way). If you like a geometric face, try Futura, Avenir or Neuzeit.

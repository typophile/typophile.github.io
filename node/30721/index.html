2007-01-19 04:16:13 • typotheticals

If you have a lot of money and are doing full-time font development, particularly in a team setting, there’s also DTL FontMaster.

But for most other folks doing serious work, if you are only buying one tool, FontLab is the only option worth considering.

Fontographer simply lacks key features that we need for serious work (such as OpenType support, decent Unicode support, native TrueType editing, adequate hinting for both kinds of outlines, and a zillion aspects of control over what goes into the generated font). It does have a drawing interface that a lot of people like, however.

Cheers,

T

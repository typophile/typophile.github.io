2007-01-18 17:08:02 • AndrewSipe

The only TrueType fonts required by Windows XP are:

  • Marlett (hidden by default)
  • Tahoma
  • Tahoma Bold
  • Microsoft Sans Serif
  • Franklin Gothic Medium
  • Franklin Gothic Medium Italic

If you remove any of the above fonts, Windows XP will automatically restore them within a few seconds.

In addition to those, I recommend that you keep at least the following fonts installed:

  • Arial (all four fonts: Regular, Bold, Italic, and Bold Italic)
  • Times New Roman (all four)
  • Courier New (all four)
  • Symbol
  • Wingdings

For Web browsing, you’ll probably want to keep the rest of the families included in the Microsoft Core Fonts for the Web:

  • Andale Mono
  • Comic Sans MS
  • Georgia
  • Impact
  • Trebuchet MS
  • Verdana
  • Webdings

That covers Windows XP. Pretty soon, though, we’ll be seeing a lot of the new fonts that come with Windows Vista and Office 2007. Of those, the most crucial font families are Segoe UI (the new default interface typeface in Windows Vista) and Calibri (the new default typeface in Word, Excel, and Outlook).

Which to use Fleischman BT or DTL Fleischmann?

Identifont will give you a side-by-side visual comparison of the basic glyph set, and calls out major differences: http://www.identifont.com/differences?first=Fleischman+BT&second=DTL+fleischmann&q=Go

Depending on whether you are using it for running text or display text, one might be more appropriate than the other.

2011-02-05 10:49:40 • DTY

… and the non-slanted ones regular (because they are a symbol that would normally be italic, within text that is as a whole italicized).

Which is really odd. It’s true that, in terms of typographic operators, italic² = regular.

But I’ve never seen this convention applied to formulae. Variables and function names are usually set in italics no matter the context (the s is, for example). This matters, because some functions and variables are distinguished from others by setting them upright — in quantum mechanics, for example, Schrödinger’s wave function is often written as Ψ in its time-dependent, and ψ in its time-independent form (at least in the textbooks I’ve come across).

As such, I would suggest that you keep all φ’s in italic here: both along the axes and in the caption.

2011-02-06 17:06:32 • SebastianK

Formulas should be the same, whether in an italic context (theorems are normally set in italics) or an upright context (definitions are normally set upright). If a letter phi represents an angle, then it should look absolutely the same inside a definition or inside a theorem. In the example above, not only the phi was changed, but also the s. You just don’t do that.

[img:sites/default/files/old-images/formulas_4245.jpg]

The conventions for the formulas are not universal. French mathematics uses upright roman capitals and upright greek letters (independently of the case). In the US, latin capitals and small greek letters are in italics, but greek capitals are normally upright. Journals may have their preferences for the font to be used for vectors, tensors, etc. Once you are given the conventions, you stick to them. If a symbol is to be written with an upright bold sans, then it needs to be an upright bold sans everywhere in the text, independently of the context.

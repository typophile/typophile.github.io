There is a match, without editing, to Lining Gothic 545 [img:sites/default/files/old-images/LiningGothic545_5766.jpg] created originally by American Typefounders or to one of the almost identical clones produced by most major foundries. AFAIK there is no digital revival. However this image provides a full character set and can be used to select substitutes. Don

Windows version expects InDesign files to have a ".indd" suffix. If this is missing, the OS won't know what app to launch when you double-click the file, and the file may not show up by default in a "file open" dialog. But you can still open the file, either by doing "show all files" in the file open dialog, or even just drag and drop the file into the PC version's window. Regards, T

Remove the fonts from fonts folder, put on trash all OT fonts created, then put the X font cache on trash: HD/System/Library/Caches/fontTablesAnnex restart the OS, create another OT, with different date and version number. If its still doesn't working… hum, well, don't know, the font is not good. :-) Good luck!

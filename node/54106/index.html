Record Label Logo

January 28, 2009 • Nicholas Pizzolato

A friend of mine is starting up a record label and I’ve been commissioned by him to do their logo. The label is called Shack Attack Records. I’m not a huge fan of the name itself, so I thought focusing on S + A would make a stronger impression. Anways, here’s some initial logos I’ve got so far. All type is Universe.

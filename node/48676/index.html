Have you tried just double clicking the empty slots (if you have both key letter and the diacritic mark)?

When doing this the glyphs gets collected from a file called “alias.dat” where instructions are written which glyph should have which accent etc. This alias.dat can also be modified to make your font production even faster.

By default (when I double klick or use the ”Create glyphs”, or ”Create glyphs if empty” command) all my components are automatically centered.

Btw, your command A+|acute=Aacute makes the component accent to placed to the left. If you just type A+acute=Aacute, everything is centered.

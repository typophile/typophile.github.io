Pieter, here's what I would do... Create classes for your numerals like this: @class_0 = [zero zero.OP]; @class_1 = [one one.OP]; @class_2 = [two two.OP]; @class_3 = [three three.OP]; @class_4 = [four four.OP]; @class_5 = [five five.OP]; @class_6 = [six six.OP]; @class_7 = [seven seven.OP]; @class_8 = [eight eight.OP]; @class_9 = [nine nine.OP];

I am afraid the No.1 choice for setting mathematics is still LaTeX in some of its reincarnations. Check, e.g., www.miktex.org. Not too friendly for a newcomer, but irreplacable for practitioners who need to set formulas on a regular basis. As a computer scientist I can report that more than 50% of researchers I have ever met (worldwide) use some form of LaTeX. The second largest user group seems to use - oops - MS Word with Equation Editor or some software based on that (or extending it). If you want to see an example of REALLY BAD output, give MS Word/Equation Editor a try.

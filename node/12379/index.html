[[Indices|Indices]] : [[How-To|"How To" Section]] : Scripting with Python Before you get started making macro programs for [[FontLab]], you must download and install [[Python]], the programming language that is used by FontLab to write macros. You can download Python from www.python.org. First you should begin by reading the chapter entitled "Macro Programming" in the FontLab Users' Manual, which can be downloaded from the following locations: Windows: US : Europe (.zip) Mac: US : Europe : Asia (.zip) Once you have done that, some great resources to start with: The Unofficial FontLab/Python API Reference The Letterror FontLab Scripting Mailing List Archives Discussion: Small release of several python scripts Book Recommendations: Python for FontLab Can You Program in Python...?

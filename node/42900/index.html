OpenType Stylistic Sets in InDesign CS3

March 6, 2008 • afilbin

I work for a magazine that prints a lot of programming code blocks in which we use only straight quotes. The problem is, InDesign is automatically changing all of our straight quotes into curly quotes because we need curly quotes in our general body text. I've discovered the Stylistic Sets option for Glyphs and have made a new set that has the curly quotes removed, but I don't know how to get our code Paragraph Style to use the Stylistic Set I created instead. Am I even attempting to do the right thing here? How would you go about getting a Paragraph Style using Letter Gothic to use only straight quotes? Thanks in advance for any help or suggestions! --af
Well, in CS2 you make a new paragraph style and under the Open Type features section there is a menu where you can select Stylistic Sets, if any have been created. David

Just set the language of the paragraph style for your codes to "none" and InDesign won't touch your quote signs. Ralf
2008-03-06 21:40:10 • Typography.Guru
To prevent use of typographic quotes in any newly created documents: 1. Open InDesign but do not open any documents. 2. Go to Preferences, and turn off 'Use typographers quotes' option. This will stop InDesign from converting straight quotes to curly quotes.

2010-05-28 18:09:31 • mili

I’m sorry for the difficulties! There are two problems going on with the Adobe Type Store.

  1. The North American Store and the other Stores use different architecture, so the Hypatia Sans release isn’t officially on the other Stores yet. That’s in process, but I’m not sure how long it will take. (In the longer term, the difference in architectures will also be fixed, but that’s a ways off.)

  2. Something has gone wrong with the keyword search function. The web team is investigating. Until it’s fixed, the best way to get to the Hypatia Sans page is to select the letter H from the alphabetic search tool (right below the keyword search tool). The Hypatia Sans packages are the last two listings on the H index page.

I hope this is some help.

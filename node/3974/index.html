2004-03-31 07:26:30 • hrant
Illustrator CS has the same optical kerning engine as InDesign, as well as the same spacing and justification capabilities. As long as you have the latest Illustrator, you should be able to duplicate the text layout pretty well.

As for Kepler, it's probably Slimbach's most underrated/overlooked work. But I don't think I'd invest in the MM version at this point in time (Adobe has stopped selling its MMs, and is encouraging its licensees to do the same). Get the OpenType version.

2004-03-31 07:56:28 • Thomas Phinney
Oh, resellers still sell MM fonts?! Wow, that's the best news so far this year!

I think laymen might indeed need to avoid MM fonts, but somebody like Hector who can create custom instances from the MM at will should definitely get the MM version, especially if he can easily add any OT features with the new FontLab. Not to mention that InDesign can automatically produce optically-correct typesetting with an MM font (that has an optical axis) but not with an OT font (yet). And even when InDesign support's OT's optical masters, it will be a discrete affair (so to speak), instead of the fully continuous scaling in an MM.

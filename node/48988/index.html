2008-09-03 19:02:14 • Goran Soderstrom
The theory is very simple: You cannot have two fonts with the same font family name and the same settings for “bold” and “italic” (the FontLab checkboxes.) Otherwise just one of these fonts will be available. So if you create a font family with up to 4 styles, just make sure you use the checkboxes. For more than 4 styles you need sub-families as described in the article.

Looking for matching free font

Your sample is typeset in Thirsty Script Rough—it’s a commercial typeface; there is no matching “free” version: http://www.myfonts.com/fonts/yellow-design/thirsty-rough/

There is a demo version of Thirsty Script Extrabold available from dafont.com, but it’s for personal use only (you can’t use it for commercial projects): http://www.dafont.com/thirsty-script.font?text=LSC

Similar, free (again, for personal use only) typefaces: http://www.whatfontis.com/Blessed-Day.font?text=Lavender%20Sleep%20Cream http://www.whatfontis.com/Niconne.font?text=Lavender%20Sleep%20Cream

Some free Google web fonts, one of which may work for you: http://www.google.com/fonts/specimen/Dancing+Script http://www.google.com/fonts/specimen/Lobster http://www.google.com/fonts/specimen/Playball http://www.google.com/fonts/specimen/Cookie http://www.google.com/fonts/specimen/Great+Vibes http://www.google.com/fonts/specimen/Rochester http://www.google.com/fonts/specimen/Lily+Script+One http://www.google.com/fonts/specimen/Grand+Hotel http://www.google.com/fonts/specimen/Satisfy http://www.google.com/fonts/specimen/Oleo+Script+Swash+Caps http://www.google.com/fonts/specimen/Sofia

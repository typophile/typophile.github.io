It must be DSBroadBrush by Dubina Nikolay [D-Studio, 1999] More on D-Studio fonts: http://luc.devroye.org/fonts-26598.html

There is also a font which matches (at least ) the latin subset which is named ft18 by FT Content Provider. More on FT Content Provider: http://luc.devroye.org/fonts-42785.html

Chances are that DSBroadBrush is the original and ft18 is a non legitimate clone, so please take care on using/licensing.

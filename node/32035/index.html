If it is an OT font and you are working in FontLab, go to the features panel and “build kerns.” Compile, and then write off the feature file, which you can open with a text editor. In that file, the “kern” feature will show both your plain pairs kerning and your class kerning. The classes as defined will be at the top of the file.
2007-03-07 17:51:35 • charles ellertson

The Classes panel allows you to save its contents, so you can apply it to another font. (Menu list at top left of the panel.)

If you make a class for “O_right” and include O', C, G and Q, as well as the accented versions of those letters, you only need to manually kern O (providing the other characters are similar enough in shape). And so on for all the other character kinds that require kerning.

Using this principle, you only need to manually set a couple of hundred kern pairs for each weight of a font, and the rest of the kerns are handled by the class-based system.

The rest of the information – the lists of glyph classes and their contents – is transferred in FontLab’s “flc” files, by using the menu at the top left of the classes panel.

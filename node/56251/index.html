How can I in Indesign CS3 create the multicolored highlights like in the 8VO poster below?

If i try to bring a red highlight next to black highlight the red automatically goes black.

One way I found around this problem was to give the red and black highlights there own boxes but this can be very inaccurate and text is never consistently spaced and lined up.

Thanks a lot

You could try using Paragraph and Character Styles for the type, and use Underlines for the highlights.

One paragraph style for the overall font style. One Character Style for each kind of highlight.

To create the highlight (within the Character Style), I’d use

New Character Style Underline Options Underline (On), Weight (24 pt or similar), Offset (adjust to taste), Select Color.

Then duplicate the Character Style and simply change the color of the Underline within the Character Style pop-up, but make sure each successive Character Style is based on your original, since if you’d like to change the width or offset of the highlights globally, you can do it by changing the original Character Style.

Good luck.

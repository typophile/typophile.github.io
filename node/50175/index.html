2008-10-05 19:12:20 • bemerx25
I can't think of any script fonts with such long connectors between the letters. You might be able to adapt Letraset's [[http://www.myfonts.com/fonts/letraset/rapier/|Rapier]] which is a late '80s design. Also look through all these scripts by: [[http://www.myfonts.com/person/leuschke/rob/|Rob Leuschke]], [[http://www.myfonts.com/person/borges-de-oliveira/charles/|Charles Borges de Oliveira]] [[http://www.myfonts.com/foundry/sudtipos/|Sudtipos]] Your example looks as if it was done in some kind of crayon or pastel. If you want this exact look it will be even harder to match with a digital font. You might be better off getting a few different script fonts and use them as a starting point to draw your own lettering.

In some of my Guides I have made an effort to give the design dates of the fonts (typefaces) represented in the Guides. This is most complete in the Serif and Script Guides.

There is a text file from the spreadsheet used to create the Serif Guide (link at the bottom of the page) which you can re-constitute back into spreadsheet form and scan down the Design date column to find designs from any era.

The Script Guide has the Script Font Reference List, which has a Google search for the bowfinprintworks.com pages. If you enter a date you will see results that include that date in the font entries, for any of the Guides. It could be a bit tedious, but it would work.

The Sans Guide is in the process of adding design data, but it is only partial at this point.

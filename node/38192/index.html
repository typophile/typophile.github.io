2007-10-18 13:13:19 • laurent

Don’t use stochastic raster, it will cause more problems than solutions. Today the normal amplitude modulated rasters are so fine that frequency modulated (stochastic) raster is not really relevant any more.

If you use a raster value in body copy then use only the black plate in your case. That way the print will look cleaner because there can’t be any out-of-register that will make it look fuzzy. Just set K=69% for the best result. Have the platemaker (assuming the printer goes CTP) rasterize in 200 lines pr. inch.

2007-10-24 22:57:07 • buddhaboy
This is Figgins' “Phantom” from the 1830s. Various other shaded and ruled faces in the past. And there’s that LettError bankish face that has size-specific pseudo-engraved ruled tone fill. But I think Figgins had the right idea, that the rules can be structural, not just applied tone. Some of those op art faces, eg Pump, and Mexico Olympics, have ruled structure that gives them a tonal value. But those are all display faces. However, I don’t see why a text type couldn’t be developed, for this high-res era, that has some built-in tonal quality. For that matter, it may be possible to use an outlined Thin weight at text size, but it would require some experimentation to get the effect just so.

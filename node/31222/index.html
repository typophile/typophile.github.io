What is the font hiding feature? What does it do?
2007-02-08 15:37:35 • ebensorkin
Font hiding was just a relatively straight-forward means to turn fonts on and off in a user's local .fonts directory. Clicking the hide button would change font.ttf to .font.ttf (and vice versa) so that the fontconfig system would no longer list it in the available fonts. The only font management feature that I've heard of in the Linux environment. Nothing as feature-full as a font-management program on a Mac, but combined with creating folders with special-purpose fonts, it made my life easier. Below is a message (in response to the wish-list item I created) concerning font handling in the soon-to-be-released KDE 4: "KDE4's installer features font hiding, and font grouping." I'll be very interested to see how extensive these features are. Scott Thatcher

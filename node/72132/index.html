You are going to need an as-neutral-as-possible serif to hide beneath both Mrs. Eaves, which has a lot of personality, and Meta which also has a lot of personality. Or you need a serif with personality that doesn’t detract too much from the those first two. I say “too much” because you are going to have some kind of tension that doesn’t work in your favor with 3 typefaces that are all so distinct.

I played around with Mrs. Eaves and Meta on a page, and without spending more than a few minutes, found that New Baskerville and New Century Schoolbook create a third leg to create a sufficiently balanced, albeit awkward, three-legged stool.

I had all four of those typefaces set as samples for a recipe book on font combinations I’m working on, so it was easy to try them out in a broader context.

I like the New Century Schoolbook combination best as it has a sufficiently dark enough color with neutral tracking to hold it’s own against both other faces which are heavy.

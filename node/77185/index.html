Yep, that's the one. Thanks, Sii. Reading Monotype EULA - 12. You may electronically distribute Font Software embedded in a "Personal or Internal Business Use" document (that is, a document other than a "Commercial Product" as defined herein) only when the Font Software embedded in such document (i) is in a static graphic image (for example, a "gif") or an embedded electronic document, and (ii) is distributed in a secure format that permits only the viewing and printing (and not the editing, altering, enhancing, or modifying) of such static graphic image or embedded document. Technically the HTML page with @font-face embedding falls under (ii) as it does not really permit "editing, altering, enhancing, or modifying". Sure one can save the page and then edit it as a text file, but exactly the same applies to PDFs and Flash applets, the use of both of which appear to be allowed. The Webtype selection seems odd - there's light and medium, but no regular weights. It is also a service. The same goes for Monotype's, though they do have a better selection including Trade Gothic Next and News Gothic in all weights and styles.

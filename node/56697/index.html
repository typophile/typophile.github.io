I’ve used it extensively for international clients, remote designers, and as a course convenor.

I’ve found it to be great for pulling together groups because it is very easy to use and orientation/learning happens through actual use, which helps speed up the start up stage.

Every group I’ve shared it with ends up using it in their own way. I usually make some suggestions at the start and then it beds in as the project rolls on.

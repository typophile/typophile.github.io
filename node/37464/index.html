InDesign question regarding "quad tabs"

September 24, 2007 • janinagray

Does anyone know how to set what would be the equivalent of a “quad tab” in InDesign? Using an example of 3 words: It would be a word (flush left) with a shift-tab to space to the second word, and another shift-tab to the third word which would appear flush right. The second word would be centered between the first and third word with equal space before and after. Can’t find anything relating to this in the Help Center, but I don’t know what it would be called in InDesign. It certainly isn’t called Quad Tabs!

The only way I know is to set your type in on line with one or more space between each other, force to justified and you’ll have you 3 words equal spaced. Hoping is what you’re looking for.

André

2007-09-25 00:02:29 • sim

set your type so there is a tab at the beginning of each line and a tab between each item. Then set your first tab stop to be right aligned, the second to be center aligned and the third to be left aligned.

Even though you make 4 tabs you only set 3 tab stops. The first one is just to move the text over.

Kerning: Restore Exceptions?

January 4, 2008 • Goran Soderstrom

Is there a way to restore a kerning exception so it again follows the key glyph in the class in FontLab? Can't find it... and I need it bad. I can only Reset the kerning. Thanks in advance. Cheers!
A single value? Open the kerning list in the Kerning Panel (second buttom from the right in the panel's toolbar). Navigate to the exception pair. Alternatively, type the pair into the preview part of the panel and click on the right glyph of the pair; this will highlight the pair in the kerning list. Click on this kerning pair's line in the kerning list; do this even if it is highlighted already! In OSX, hit the 'back' key. In Windows it might be the 'delete' key, I don't know.

It's here, "Remove Exception Kerning". It is important that you open the script in a text editor first and assign your own identifier tags which identify your classes as leftside, rightside (and exception). Else the script won't work. Also see the notes in the text file. Not tested much, so better try with a copy of the font first.  :) Kerning pairs of which one glyph is not covered in any class at all are not considered exception kerning pairs!

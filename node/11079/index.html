March 8, 2002 • andreas

This font I have started as display/logo font for a local American football team. Its inspired by "Aachen", "Princeton" and "City BQ".

Now I totally redesign the font and add many features like:

Ligatures
Alternates
Small Caps
Old Style Numerals

The hardest thing is to kern this font. So Opentype is the only format to handle this massive kerning tables. InDesigns "autokerning" engine - cant handle these faces in a good way - so I have to do all by hand.

Is is the first face "ThunderBold" of a family.

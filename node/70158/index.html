That looks like Helvetica Neue, and it sounds like an all-too frequent occurring problem on Macs.

(From hear-say:) The problem is caused by the Mac OS; the Helvetica Neue provided by the system is wonky, and causes problems when you are using it in a program such as InDesign. The solution is to install a new, “real font” Helvetica Neue.

Do some googling – it’s been seen before.

I’d think the problem isn’t just that the Helvetica Neue provided by the system is wonky, but you have a different version of the font than your colleagues. It’s also possible you have multiple versions.

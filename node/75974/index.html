Elbonium via Titillium

November 4, 2010 • chank

Here’s a group of fonts I’m working on. They’re all based on Titillium, which is distributed under the SIL Open Font License, version 1.1.

Slanted Titillium to get the Titillioblique fonts. Added some more interesting characters to make a family called Titillitalic. Added even more interesting, bendy characters to make it into something called Elbonium-Italic. Then slanted that back upright to make the Regular Ebonium fonts.

Hope these pics show up okay…

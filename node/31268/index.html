2007-02-17 19:46:12 • vinceconnare
XP works fine on bootcamp/MacBook Pro. But... Vista is not backwards compatible and probably won't work on anything but new. (very Apple System 7 like) - Apple's Sigma Tel audio drivers don't work in Vista (Sigma Tel doesn't support drivers or consumers) - Apple's Bluetooth drivers don't work in Vista - Apple's Startup disk in Control Panel doesn't work in Vista (you use the alt key on restart to select boot device). - Netgear's WG511v1.1 wireless network card don't work in Vista - Microsoft's Visual Studio .net 2003 doesn't install on Vista and isn't supported. - the SSH Client 3.2.9 non-commercial from http://www.ssh.com/ doesn't work - Ipswich WS_FTP you need the commercial version with SSH support and the Vista upgrade for it to work. - FontLab 4.6 may not install on Vista (I couldn't get it to install but 5.0.2 is fine) - MS Font Validator works fine as long as you have .net 1.1 framework installed. that's the list so far....

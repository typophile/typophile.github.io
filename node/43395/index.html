You can do this in the Illustrator as well using the same filters(Effect/Pixelate/Color Halftone) but here am showing how to do it in the Photoshop. Create a big black dot on a transparent layer with a white background [img:sites/default/files/old-images/1_3732.jpg] apply gaussian blur to the dot layer [img:sites/default/files/old-images/2_4943.jpg] Now merge all the layers(Ctrl+Shift+E) and apply halftone effect with following details. (Filter/Pixelate/Color Halftone) [img:sites/default/files/old-images/3_4023.jpg] Then you can take this image to Illustrator and apply live trace filter to get a verctor output.

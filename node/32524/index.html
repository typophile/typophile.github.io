There is really nothing like Missionary, but it’s true that it demands to be used very large. You might like some of the ornamented caps like Gill Floriated, Scala Jewel, or Wiescher’s Bodoni Classic. And for unabashed embellishment, there are few more delightfully frilly than Treasury.

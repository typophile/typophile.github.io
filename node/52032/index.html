There is no image posted for what you would like to match. If you want to try using this version of [[http://www.bowfinprintworks.com/SansGuide/sanssearch.php|the Sans Serif Font ID Guide]], you can try selecting the letterforms that you want to match and see which sans fonts in the database might be structurally similar.

Franklin Gothic has a combination of my seven key letters (a, e, g, G, M, R and y) that at least eight other families also use. However, you may not care about whether the G has a chin, or the a has a tail, so you can broaden the range of answers by selecting only the letters that matter to you.

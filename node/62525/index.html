Indesign list-indents on the right side

September 29, 2009 • PageMadchen

I’m using a preformatted template in Indesign.

When it comes to the lists, they indent on the right side and don’t line up with the rest of the left justified content (loosing precious space).

When I right click (on a PC) and choose “clear all overrides”, the problem is resolved, however the text is all befuddled and must be reformatted (font, size, and color)

Any suggestions as to what is causing this problem? I would like to deal with this issue without reformatting the entire document.

Thanks for any suggestions.

The settings for “Indents and Spacing” can be changed in the Paragraph panel. Your description is a bit unclear, but I think it would suffice to select the indenting paragraphs and set their left/right/first line/last line values to zero. Do this for each occurrence.

Normally I’d recommend changing this in the paragraph style for your lists, but judging by your ‘clear overrides’ problem, these haven’t been used. If you (or original file creator) had used styles, you’d only had to change it once.

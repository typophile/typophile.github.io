2009-04-25 04:44:23 • phrostbyte64

I don’t design type, but I don’t think you should make smaller non-letters.

Full caps are designed to be used within lines of miniscule letters; they’re not really designed to be sued next to each other. Small caps are designed to match miniscule letters in weight and size so that you can set full words in caps.

The rest of the typeface is designed to match lines of mostly miniscule letters, and small caps are capitals that look more like miniscule letters, so small caps should work better with the normal versions than the adjusted versions that you propose.

2009-01-21 12:29:31 • Typography.Guru
MB, Photofont is more of a "platform" than "program", developed by Fontlab Ltd. There is the photofont format, which can be fonts that consist of arbitrarily high resolution, and that support alpha transparency so you can make them work over arbitrary backgrounds. Then, there currently is Photofont Start (for Photoshop and Illustrator) as well as Photofont WebReady (for web pages), in which you can use the photofonts you made. Within the next few weeks, we will release Photofont ID (for InDesign) and Photofont QX (for QuarkXPress). We have no current plans to support Microsoft Word, though. Finally, there is BitFonter, in which you can create your photofonts, by either drawing directly or converting existing photographs into photofonts. You can find more information at: http://www.photofont.com/ Regards, Adam Twardoch Fontlab Ltd.

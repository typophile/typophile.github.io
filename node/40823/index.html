I don't have an answer for you, but I'm glad I'm not the only one who puts off the content of the syllabus in order to choose a new font I've always wanted to try out. I'm almost never happy with the pairings I get between serif and sans serif, however. I was happy once with a presentation template that used Adobe Garamond for titles and Cronos for body text. Perhaps [[http://www.myfonts.com/search?search%5Btext%5D=cronos|Cronos]] is a typeface to check out. I also saw a nice set of handouts that used Garamond for body text and Univers Condensed for headings. In that case, the contrast seemed to work out well. ST

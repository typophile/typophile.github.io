If I were making fonts commercially, or money were no object, I would at present use FontLab, as John suggested. As it is, I can't justify buying FontLab, and prefer not to use pirated software, so I use FontForge. It's a slightly unpolished work in progress, but certainly usable and reasonably capable. It is a generic unix program with an X11 interface, and although a native OS X interface will probably appear eventually, it won't be very soon. As such, you can't (for instance) simply drag-and-drop from Illustrator, but it will read EPS files, among others, and does read and write Mac font formats. It still suffers from the occasional crash (though it almost invariably recovers without losing any work) and other bugs (though the author responds very quickly to bug reports). There is a scripting system, but it is not nearly as good as FontLab's.

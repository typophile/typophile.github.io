As for Autoplay, as of OS X it’s a no-go. On a PC, it’s entirely dependent on the system you are using. You’ll have to enable it on that system, and create a file in notepad called autorun.inf with these two lines (substituting the name of your PDF file - no space characters allowed):

[autorun] shellexecute=“yourfile.pdf”

Save the autorun.inf file to the root level of your CD.

Keep in mind that many users do not like documents/applications to take over their entire monitor by surprise. It’s not expected behavior (except maybe for games). I’ve known folks who get very annoyed when a program does that unexpectedly.

Asking the user’s permission before going full screen is not necessarily a bad thing.

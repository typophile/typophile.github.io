Do I have any control over font selection, size and color?
2007-01-29 20:54:43 • AndrewSipe

Absolutely. Once you create the text box (Tools, Advanced Editing, Text Field Box), you can double-click on it, select Appearance, and it lets you define font, size, colour, etc.

(Of course, it presumes that whatever you define is also on the computer that is opening the pdf: you can’t select Snell Roundhand if it isn’t installed.)

2003-11-30 04:24:46 • kakaze
You need to set the preferences when no documents are open. If a document is open, the new preferences will be applied only to that document. If you set preferences without any documents open, the new preferences will be applied to any new documents.

Document preferences are saved with a document. So existing documents will retain the preferences they had, regardless of whether you change the global preferences when no documents are open. The global preferences affect new documents.

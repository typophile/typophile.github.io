How to ease wrong exceptions in a class kerning

September 8, 2009 • Ramiro Espinoza

Hi there, I have to improve the kerning in a font. The question is there are many wrong exceptions in the class based kerning. I need to erase all kerning values that are not equal to the leading pairs. Is there a way to do that? Thanks in advance. Ramiro.
Ramiro How many is 'many'? If you have a couple of hundred bad exceptions it is possible to just delete these from the metrics window kern listing. Make sure you have the kerning classes you require correctly added to the Classes Pallet. The kern values for your exceptions should now turn red in the kern listing. Now click the top of the kern listing on the header that says 'Value'. This will sort the listing and force all the red exceptions to the end of the listing. Go to the end of the listing and start deleting them. It is a bit infuriating that you can only select one entry at a time to delete which means if you have more than just a couple of hundred - it will take a long time to delete them all. Hope this helps Malcolm

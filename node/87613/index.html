The font has an Anglo quality, rather than Germanic: it is Bitstream's version of London Text. So I'm afraid the allusion "Blackletter = German" doesn't work for the typographically informed. I immediately thought that as an anglo style, it represented Keynes, not Weimar, which contradicts the article's content. Besides, for a graphic designer, "Weimar" suggest Bauhaus, not Völkisch. A modernized (sans) blackletter (e.g. Tannenberg) would have worked better, but I suspect that would have required a font purchase, and the Bitstream font was on hand, bundled or in a folio.

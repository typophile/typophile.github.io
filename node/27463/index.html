Fontlab is what I use, but they’ve recently picked up Fontographer after an 8-year slump – it works great under OSX (both G5 and Intel versions, though it’s not Universal yet).

I’d say that you should make your font in “Windows TrueType” (.ttf) format, which works on Windows and Mac OS X. All programs (FontCreator, TypeTool, Fontographer, FontLab Studio) allow you to do that. Since you say your funds are limited, the shootout indeed will happen between TypeTool and FontCreator. As I work for Fontlab Ltd., I’d argue that TypeTool is a bit more easier to work with, although FontCreator has some nice features that TypeTool currently lacks. I can mention, however, that we’re working on TypeTool 3, which I believe will offer unbeatable features for the price. Of course, if you buy TypeTool 2 now, you’ll be able to upgrade to TypeTool 3.

Both TypeTool and FontCreator offer demo versions, so I would just go ahead and try them out!

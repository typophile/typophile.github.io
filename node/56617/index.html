Printed samples of the florin sign?

April 4, 2009 • Andreas Stötzner

Germany’s DIN is currently working on a proposal to disunify the monetary florin sign from the afrolatin ccharacter 0192. To proof this issue we’d like to present some sample images of the florin’s usage in print. Does anyone have something at hand?

This is indeed a problematic unification, bit disunification can be even more problematic, as we discovered with the Romanian diacritics. In which direction would you disunify? What would be the identity of the existing codepoint? – bearing in mind that it is part of ISO Latin 1 as the florin, and widely implemented as such in fonts, but has normative case folding to the uppercase F-hook?

I believe a significant identity shift is already beginning to take place vis à vis this character. As more font support extended Latin character sets including support for African languages, the obvious implementation of this character is as f-hook, rather than as the florin symbol. We’ve already started changing the implementation in fonts for MS as we extend the character set to support African language. And of course the demise of the Dutch currency in favour of the euro makes this change easier.

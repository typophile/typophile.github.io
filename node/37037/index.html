i tried the feature “substitue from ” , putting all my A-variants in a glyphclass it works. BUT IN INDESIGN IT ALWAYS JUST TAKES THE VARIANT THAT IS FIRST IN THE LIST OF THE CLASS. this is how the the "sub from" syntax works. there is no way to get "real" randomness in OpenType (unless you somehow build your own plugin for the application, that does what you desire) but you can acheive some level of pseudo randomness. there are several threads here on typophile discussing how to achieve this effect, i'll let you wade through them yourself: http://www.google.com/search?q=random+site%3Atypophile.com if you want someone to implement this for you, you might contact [[tal leming]]

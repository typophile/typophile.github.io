A narrow x-height region (which is basically what "condensed" means) is only one way to save space. In fact the more paragraph breaks you have the more horizontal economy ends up going to waste. In contrast, a font with short extenders (read: wide and tall x-height) can be set at a smaller point size (without looking too small) and with tighter leading, resulting in vertical economy (irrespective of paragraph break frequency). hhp

Mandarin and Cantonese are dialects and do not refer to the written language. For written Chinese there are two forms: traditional and simplified. Simplified Chinese is used only in the PRC. Korean has fewer characters than there are letters in the English alphabet, so file size wouldn't be an issue there regarding web fonts. This site may be useful (turn Google translator on): http://www.justfont.com

You’re right, there are rarely hard rules when it comes to making information easy to read, but there are basic rules about which part of a phone number is ‘area code’ and which part is ‘subscriber number’. A London number has a three-digit area code and an eight-digit subscriber number. Most towns have a four or five-digit area code and a six or seven-digit subscriber number. Some small towns have a six-digit area code and only a four-digit subscriber number.

Nothing like keeping it simple …

As long as you keep your UK mobile phone ‘area code’ together (07816 or 7816 for international), it wouldn’t really matter how you arrange the subscriber number; xx xx xx would suit some numbers more than the more usual xxx xxx.

From the Ofcom Telephone Numbering Plan (via that Wikipedia link above): “It is very common to see the + notation being misused — +44 (0)xxx xxx xxxx is not an uncommon occurrence even in official documents. For international callers the number between the brackets is dropped. For callers within the United Kingdom the +44 is dropped and the number between the brackets used. Calling +44 0xxx xxx xxxx will not work from most operators, and the +44 (0)xxx notation is incorrect and should not be used. The correct usage of the + notation in this example of the three-digit geographic code is +44 xxx xxx xxxx.”

As more people start to use mobile phones as their primary telephone, it makes sense to use the correct international number – no (0) – so that you can dial it from anywhere in the world without having to add or remove digits. If you dial +447816111111 on a mobile phone it will work in the UK as well as globally.

I noticed that some pages on the Ofcom site – including the contact page of the Welsh language version – use the numbering scheme: (+44) (0)20 7981 3033 so it’s clear that these are only guidelines and even the regulators can’t stick to them.

I don’t think it really matters if you prefer to use the (0) but I was glad to read that it is unnecessary as I always found it cluttered and a little ugly.

More information here: http://paulm.com/how_to/write_a_phone_number.html

One alternate way to write this would be +44 (7816) 111111. While not so useful for mobile numbers, it becomes useful when using geographical (+441XXX_, +442X_) numbers in some applications like Microsoft Outlook.

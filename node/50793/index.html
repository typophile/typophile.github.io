Typetool is about as simple as it gets. However…

Two years ago, when I was a design student trying to figure this stuff out, I thought starting with Typetool would be a good idea. I quickly decided that it was actually a nightmare. Because I’d already been using Illustrator for years, I found something missing every time I turned around in Typetool and ended up doing a lot of work in Illustrator and then bringing the vectors into Typetool. Eventually I just caved in and bought Fontlab. If you decide to go with Typetool, run through all of the exercises yourself first to be sure that it’s not too simple.

Chuck,

TypeTool 3 is built on the core of FontLab Studio 5, so you can design professional-quality retail-ready fonts that will have the same industry strength as fonts built by major font foundries. Also the user interface concepts are closely related to those of FontLab Studio, but simpler.

Note that if your academic institution purchases 8 or more TypeTool licenses, the cost will be about US$35 per workstation. See http://www.fontlab.com/buy/academic/ for more information.

http://www.fontconstructor.com/ and http://fontstruct.fontshop.com/ are completely free but follow a different model. Rather than drawing lettershapes, you build them from blocks and elements. Also, the output quality of those tools is “draft”, so while the fonts made with those products may work in most operating systems, there is no guarantee that they will perform well.

Finally, http://fontforge.sourceforge.net/ is free and opensource but it’s rather complex and the learning process is not trivial.

Regards, Adam

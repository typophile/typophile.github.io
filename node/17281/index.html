HELLO AEROSYN-LEX MESTROVIC.

Extensis Suitcase does work with all fonts supported under Mac OS X. Your problem most likely is that you are transfering the font files in a way that strips off the old Mac “resource fork” data. Type 1 Postscript and traditional TrueType fonts for Mac OS have hidden data and cannot be sent as email attachements, ftp or on DOS formatted volumes with out being made into a .zip or .sit archive.

If you have Mac OS 10.3 or 10.4 find the folder containing the fonts. Control-click on the folder and choose Create Archive. This will make a .zip file of the fonts. You can safely transfer this file to another system and then unzip it.

So called “Windows” TrueType and OpenType files have a .ttf or .otf extension. These can be moved from system to system without being zipped.

The FFIL is often referred to a “font suitcase”. This could either be a traditional Mac TrueType font or a suitcase associated with a Mac Type 1 Postscript outline. The LWFN is a Postscript Type 1 Outline, sometimes referred to as a “printer font”. Some of this is confusing because these formats have been in use since the 1980s!

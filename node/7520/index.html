2004-11-07 03:45:52 • adriano
Believe me, we've looked into alternate board software. PHPBB is
enticing because it's free (opensource) and because PHP is so
powerful (customizable). However, one of our main problems
with considering a move to another board is that we're sitting on
3.5 years of archived (searchable) content.

So, any discussion about moving (upgrading) to new forum
software has to include strategies about porting or archiving
the existing content. PHPBB looks most promising - again
because it's customizable. And of course, the ability to post
images is a must that we have to build in.

The CSS is another matter entirely... The current Discus software
and CSS go together like hammers and screws. Discus is built
with proprietary tags that make any CSS usage like covering a
bad hair day with a band-aid. Suffice it to say, with an updated
board will come a revamped CSS.

And, yeah, Lucida Sans is really nice and Trebuchet is nice in
small doses. Have you Mac OS users noticed that you can
spec Hoefler Text and Myriad with your HTML (for Mac users of
course)?

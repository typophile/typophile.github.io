You’re right about the problems with setting up new fonts in th TeX tree (I did it with fontinst with the help of this manual and some 2 weeks of regular trial and error fiddling around). I am currently using XeTeX for typesetting multilingual (and/or rather multi-writing-system) stuff as it allows you to directly input unicode. Using several fonts in XeTeX is quite easy, you just have to declare them using their PostScript name as in: \usepackage{fontspec} \defaultfontfeatures{Mapping=tex-text} \setromanfont[Numbers=OldStyle, Ligatures=Common]{GaramondBQ-Regular} \setsansfont{Gill Sans} \setmonofont[Scale=0.94]{American Typewriter Condensed} \renewcommand{\sc}[1]{\fontspec{GaramondBQ-SmallCaps}#1\normalfont} I must admit that I do not have experience with accents in math mode (just using $\times$ occasionally, that is more or less my math mode experience). Another possibility would be to use something like LaTeXit, LaTeX Equation Editor or PDF Equation to create your math, and then import it to whatever solution you’ll come up with. (I don’t know anything about your Indesign question…).

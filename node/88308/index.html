Typesetting a Dictionary Using LaTeX

December 14, 2011 • Sylph

Does anybody know a good template or can create one for LaTeX for a dictionary – such a template which divides a page in two (or three) columns, with a dividing line between them, where the entries would be divided by a specified spacing (spacing after paragraphs) and the first line indented or where the entries would feature hanging paragraphs? Any code suggestions?
Typophile James Crippen wrote a Tlingit dictionary with XeLaTeX. His sources are available at http://www.drangle.com/~james/dictionary/. To get the vertical separating line, you only need to add one line of code using the command \setcolsepandrule.

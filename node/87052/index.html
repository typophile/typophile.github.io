2011-11-04 09:50:57 • Andreas Stötzner

It’s quite a few years since I spent this much time going through the distributors' online type catalogues. It strikes me how bad most of them works, in different ways. For instance, in at least some of the Linotype pdf specimens, the only uppercase letters you can see is H and O. Generally, for many typefaces there are no pdf specimens at all. When there are any, they are often not very good. The screen representations of the fonts are in some cases really bad, so that when you use the test drive, the result is so bad that you’ll probably consider some other font. And so on.

Working a lot with web, usability, ux and such myself, I find this quite amazing.

Opinion requested re: small caps use in PhD dissertation

November 8, 2014 • kev8d

I’m finishing up a doctoral dissertation at a university that doesn’t require students to use specific fonts or type settings (hurray!).

The font I’m using (Equity, licensed) looks great when printed and on screen, however it includes small caps but not small caps italics.

I wanted to set my headings in small caps, but I have a problem: Some of my headings include the names of books, which are supposed to be italicized. Do you think it would be best to either: not italicize the book titles when included in headings or not use small caps (so book titles in headings can be italicized) ?

Here’s an example of one of my headings (using all caps): THE INTERPRETATION OF BOOK TITLE

2014-11-08 05:14:25 • eliason
The italization of the book titles might have priority over font choice, because this is a part of the overall typographic system for the text. These are your options: • set the titles in normal Italics instead • abandon small capitals for the headings • order italic small capitals font from the font producer.
2014-11-08 07:47:36 • Andreas Stötzner
If you’re using all small caps for the headings, you could consider using all caps instead. I usually tend to avoid using italic inside roman headings. In a case like this I would probably set the book titles between brackets.

2014-11-10 00:10:42 • JamesM

Either go with a font that has italic small caps (I’m surprised Equity doesn’t), or use u&lc for the headings, or fake it, as Martin described.

Regardless of whether the font has italic small caps, I think using small caps for headings of significant length, or headings that contain titles, is visually questionable. I would save the small caps for elsewhere and do headings u&lc.

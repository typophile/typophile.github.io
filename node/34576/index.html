2007-06-19 20:05:53 • Zara Evens

We’ve had poor server performance as part of our “free” sponsored mediaTemple hosting service. Unfortunately, we’ve banged our collective head on the ceiling of server performance and storage space virtually since the beginning of our partnership arrangement. Requests to improve this have not yielded us an acceptable response, so we’re in process of moving over to a paid hosting package to give us better server muscle and lots of server space for some cool new features we’d like to roll out.

Please note that Wednesday’s maintenance upgrade will focus on Typophile’s codebase, not on server platform, so this problem may persist for another few weeks. We find it incredibly frustrating too (remember, we’re users too!) so please bear with us.

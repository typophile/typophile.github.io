The K in Macbeth is not the same style. I think this is a version of a typeface called Rubens Wood (see Solotype Catalog p. 39), or Rubens (p.9). My notes show a QBF font called [[http://www.qbf.de/dceng.htm|Nachtwache]] is a version of this. I think there are others with spelling variations on Rubens, like Rueben, or such. Mike F. probably has a list of them. We may have explored this topic before.

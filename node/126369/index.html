California's Indian Tepee

June 23, 2017 • OzCoop

I recently purchased this model on eBay, and I was hoping to identify the typeface that “Indian Tepee” is set it. The seller estimates it is from between the 1950’s and the 1970’s. I suspect that it is a lost typeface, and I was hoping to find out exactly what it was even if it has never been digitized. My best guess is a bold condensed version of Lydian or a close alternative.
Not lost, but you are exactly right—a weight that hasn’t been digitized of Lydian (Bold Condensed): https://fontsinuse.com/typefaces/6/lydian

Taking illustrator file into indesign

October 18, 2007 • vigeoh

I want to take a illustrator file into indesign. But when i do the text isnt a text box and each line is in its own box. Is it possible for each box of text to stay of box of text when taking a illustrator file into indesign? Thanks alot Tom
Hi Tom If you highlight the text with the type tool in Illustrator and then paste it into InDesign using the type tool it will remain as text. You will loose all text formatting applied in Illustrator, however if you set up a character style in InDesign with the formatting that you want and select it before you paste, the pasted text will appear in that style. Alternatively paste the text and then apply the style. Hope this helps

The reason that it works well between AI & PS, but not ID is that Illustrator & Photoshop share the same underlying text engine. InDesign has its own, which was tuned for much longer documents and more complex text flows.

I see three unIDd main fonts on this rather period-typical looking map on an atypical topic from NLT 1861 (via StrangeMaps blog, article at http://strangemaps.wordpress.com/2009/01/18/353-kanawha-and-a-landlocked-virginia/).

Can anyone kindly ID them:

  1. Existing state names (OHIO)
  2. Proposed new state names and bodies of water (KANAWHA)
  3. Non-capital cities (Pittsburgh)

Rivers, capital cities, and physical features appear to be obliques of the above three. There are a couple of other minor entries for scale, caption, and copyright that I’m less interested in.

Georgia was designed by Matthew Carter to be a web and screen font, and it is one of the ‘standard’ web serif fonts, along with Times. When those are not used you probably see Adobe Serif default font. At least this has been how it seems to work for years. Maybe there are newer ways to control fonts by embedding them, but I think this is not used much yet.

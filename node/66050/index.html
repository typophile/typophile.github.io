2010-01-08 12:17:49 • johnbutler
The current version of TLF does not support Contextual Alternates. The OpenType functionality currently supported can be leveraged using the classes in the [[http://livedocs.adobe.com/flex/3/langref/flash/text/engine/package-detail.html|flash.text.engine]] package. If you haven't done so yet, try the [[http://labs.adobe.com/technologies/textlayout/demos/|Text Layout Demo App]]. The TLF team has recently made available its [[http://blogs.adobe.com/tlf/2009/12/new-and-updated-tlf-samples.html|source code on their blog]]. The [[http://forums.adobe.com/community/opensource/tlf|Text Layout Framework]] forum is available for specific questions.

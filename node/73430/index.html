The differences you got are from screen tests with Acrobat/Reader or from prints?

I also got generally better on screen results with non-hinted versions. But non-hinted complex glyphs, like ampersand or section mark, sometimes render horribly –while the hinted version keeps a reasonable (although suboptimal) result.

With print tests, hinted fonts are definitively better, especially at sizes smaller than 14pt. PS hinting excels for printing but achieves limited results on screen. If your font is designed mainly for screen usage, TT hinting is a better choice.

I compared your PDF files on screen (Windows Vista, Acrobat 9). At 50%, the non-hinted reads better. At 100% the only noticeable difference is a pixel up/down in H bar, with very subtle changes in curves –but both reads well. At 150% they are rigorously equal, except for g –but, again, both are good.

Proper alignment zones (this is, zones which coincides with main stems/curves of your glyphs) helps to get better on screen result. But I don’t know if this feature is available in TypeTool (I use FontLab).

2010-08-22 16:19:44 • Michel Boyer

Both PS and TT hinting acts on screen and on printing. PS is simpler and focused on printing. TT is far more complex and targets screen results.

OpenType format could be generated in two “flavors”, OpenType PS (also called OT CFF) or OpenType TT. Both are mainly equal, the hinting system is their main difference.

You must use the hinting system according to the kind of OT file you plan to generate. Please note you also must set you UPM size differently to one or other format (PS default is 1000 UPM while TT default is 2048).

The vast majority of OpenType fonts are OT PS. One of the few exceptions is Vesper Pro.

Suggestion: does not forget about hinting. Set hints manually as you’re already doing and rely on print tests more than screen ones. Keep going with OT PS and put your focus on producing a good and consistent design. In the future, you can use this PS font to produce an OT TT for screen/web use if this is necessary.

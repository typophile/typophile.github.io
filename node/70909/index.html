I don't use other people's fonts these days. I don't like lists. Recent fonts I've liked,off the top of my head: Memoriam, Narziss, and piqued by the upcoming Dala Flodo. I think these interest me because I have occasionally thought of making types along these lines, and because their drawing is so virtuoso -- that's one of things I respect most in other type designers' work. The line of beauty, to use Hogarth's term, is infinitely subtle, which is why I reject the idea that "it's all been done before".

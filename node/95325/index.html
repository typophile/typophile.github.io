Testing Typefaces in Web Pages

August 4, 2012 • AlexanderKatt

There is a quick way to test your typefaces (especially those intended for web) and see how would they look in different contexts - the “Change Font Family Style” Chrome Extension.

  1. Install the Google Chrome Web Browser (if you do not already have it) and the extension.
  2. Install your typeface.
  3. Open the extension’s options and type the name of your typeface in the only field.

All pages that you open will be rendered with your typeface, instead of the original ones.

If you think some of them look better, post a pic.

A correction: Lining and Tabular figures are not the same thing. [[Lining Figures]] actually means the numbers line up horizontally. You could call them “all caps” or “titling” figures. Lining figs can be proportionally spaced. TF stands for [[Tabular Figures]] which means they all have same width for tabluar setting.

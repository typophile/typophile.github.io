2008-09-14 14:01:34 • mikeday

Latin cursive fonts almost exclusively use “calt” and “kern”. I have not seen a single Latin font that would use “curs” as the none of the Adobe OpenType Layout engines does not support it (AFAIK). To my knowledge, “curs” is only supported in Microsoft Uniscribe.

BTW, I found it quite surprising and amazing that you decided to write your own OpenType Layout engine rather than relying on an existing opensource solution such as ICU Layout.

2008-09-14 14:11:28 • twardoch

Any nasta’liq style Arabic font, the typical style for Urdu, is likely to utilise cursive attachment positioning. As Miguel, notes, the MS Arabic Typesetting font (naskh style), supports this feature.

Adobe’s lack of support for cursive attachment positioning, even in the Middle East versions, has been a strong discouragement to more widespread use of the feature. I have plans for Arabic fonts which rely on this feature, but which I would not bother to implement until I know they will work in InDesign ME.

If you are looking for additional fonts with this cursive attachment lookups for test purposes, I recommend joining the VOLT user community group on MSN. There are a number of people developing nasta’liq fonts, who would probably be glad to assist you.

Rule/Precedent for dropcaps with single-line paragraph?

July 28, 2009 • ripperbard

I'm looking for advice on how to handle dropcaps in an instance where a single-line paragraph kicks off a chapter. InDesign gives the new paragraph an extra indent on the dropcap's baseline, which I actually thought was a pretty fair fix, but the folks I'm doing it for aren't entirely sure about it. One suggestion was to basically make the dropcap a raised one in this instance, but I really don't like that shift in continuity. Was wondering if there's any preferred rule on this or any precedent from pre-desktop publishing days anyone knows off hand? Thanks and best, Aaron
The rule is to design the work to fit the text. Not the other way around, as seems to be happening here. If there are enough instances of this in your text then you are going down the wrong design road. Hell, if I had to design a book with even one such instance I'd ditch the drop cap and come up with a better idea. powers

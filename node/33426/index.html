2007-05-03 00:58:26 • cuttlefish

(Warning, I’m assuming you’re generating OpenType CFF fonts, only.)

Which AFDKO tools have you used? I’d say you should at least use CompareFamily, but it won’t hurt if you ran your fonts through CheckOutlines, and do some proofs of the OT tables and GPOS/GSUB features with spot. The AFDKO includes several tools if you want to be more thorough (e.g. you can test the fonts' rasterization with waterfallplot, and do some other proofs with tx).

Here are a few useful commands: comparefamily -d /Folder_TestFont -rn -rm -rp > CF_output.txt checkoutlines font.otf > CO_output.txt spot -P liga test.otf > liga.ps waterfallplot font.otf tx -pdf -1 test.otf > glyphset.pdf

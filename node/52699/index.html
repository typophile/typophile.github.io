Colourseparation done by hand. I did it as a grad project, ages ago.

Joost Swarte did a nice exposé about it, circa 1978. Only in Ducth, and extremely rare now.

Three main techniques:

  1. Using preprinted transparent screens that are applied (pasted on) to overlays (C, M and Y) on top of lineart print (=K). Takes great craftmanship.
  2. Using chemically prepared illustrationboard on which line art is bluelined and a limited range of screenvalues can be highlighted with specific fluid(s), as well as full value with india ink. Limited usabilitu, used for cheap comics.
  3. Using fixed screens to superimpose on repro film, using rubilith masks. Very elaborate, used for high-value stuff.

re: 1913 cigarette card.

That was probably drawn on coquille board with crayons or oil pastel of some kind, possibly augmented with some watercolor tinting.

Coquille board is a kind of drawing board that comes in a variety of roughened surfaces that break up crayon or brush strokes into texture more suitable for line reproduction. ([[http://www.top500.de/lexikon/coquille_board.htm|1]])

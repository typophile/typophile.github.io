The second one is getting a little bit too dark. I guess it's a toss-up between width, weight, and the extra MM effort you mention. I'd just go with your first setting, if that's all you'll use the smallcaps for. But if you're going to use a lot of smallcaps in text, do go for the wider style, assuming you can match the weight better - since you have the Pro, you could try using a larger optical cut of the font, which will be slightly lighter.

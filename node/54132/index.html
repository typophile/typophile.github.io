2009-02-02 07:43:58 • rushmore
I assume you’re using Word (I myself am using Nisus, where this problem does not appear). It seems that GFS Porson does not contain a separate acute accent, which is not actually needed when typing Greek. When you press the dead key for the acute accent (semi-colon), Word changes the font to one which contains an acute accent, namely Times New Roman, although we actually don’t want a separate acute accent. The only solution I could think of (there may be people here who know of a better solution) was to add an acute accent (and a diaeresis, which causes the same problem) to the font. I just sent a new version to you by email. I hope it works. Best wishes, Thomas

The STIX font project

July 29, 2009 • Andreas Stötzner

I’d like to know a bit more of the STIX font project (http://www.stixfonts.org/) but I failed to find the project’s website any helpful in this. If I’m not completely misguided: there’s not a single font sample to view, no character or glyph charts to look at, no subpage for registration, no download site to find. Instead, the sub-menus are apparently ill-structured and the “glyph tables” come out as clumsy tapeworm listings, which I find rather discouraging. Now I wonder if they want letting anyone know something about it at all. – Any hot hints for the curiuos?

I can only second your observations. It does appear that the beta versions of the STIX fonts have been available for download:

I’ve periodically tried to keep up with their development, but it’s seemed very slow from the outside.

Stuck trying to choose between two type(-related) options

April 22, 2009 • apankrat

This is for a software product website. The typeface in question is used for the navigation menu, taglines and section headers. The main body font is Arial. After shuffling through a lot of types I have converged to [[http://new.myfonts.com/fonts/fontfont/ff-din/medium/|FF Din Medium]]. It has clean and precise geometrical feel to it, which is exactly what's needed (the "real-time or .." header): [img:sites/default/files/old-images/din-1_5154.png]  I was very happy with the result, but then I woke up the next morning and the lowercase L felt wrong. Too playful and distracting, especially in a longer text. So now I made two alternative versions, which I think fix this problem, and I am now stuck deciding between them :) First is a pseudo small caps. 16pt R in medium followed by the rest in 12pt bold: [img:sites/default/files/old-images/din-3_3900.png]  Second one simply has the lowercase L straightened up: [img:sites/default/files/old-images/din-2_4956.png]  So .. which one ?

Newspaper nameplate? These were usually hand lettered. Some have been converted into digital fonts for private use. such as the New York Times font based on its historic letterhead. Foundries did produce some common styles for use by newspapers, but I don’t recall seeing this particular blackletter. There are a huge number of blackletter fonts only a few of which are digitized.

I agree with the description by Dino dos Santos of Voz Gotica as an exquisite blackletter font, very calligraphic yet modular. He is a very versatile and productive font designer, but since producing Voz Gotica five years ago has not released – as far as I know – any more fonts in this genre, except for Hades, a “dark contemporary blackletter” and Albo, a 16th century derived lowercase, enveloped in a flood of calligraphic swirls. I wonder if there are any more capital letters available for viewing other than the ones now on the DST website. Don

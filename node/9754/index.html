Once again, it looks like it is a font (although the sample is too small to be sure), because the m's look identical.

However, as with all hand-printed fonts, I think there is a large chance it is a custom font, because they are so easy and cheap to make. Virtually anyone who can print with a pen can have one made, for less than $10. (This ought to be part of the FAQ).

If this is a hand-printed font I have collected for the Script Font ID Guide, it would be in Part 11. If you do not see the exact match, you may note some that are close enough if you would prefer to have a font, rather than make the lettering by hand.

I will look, and get back if I find it, but you might also want to look, in case it is not among the approx. 500 fonts in this part of the Guide.

  body: "Hi Steffen,\r\nyour sample is <a href=\"http://www.fontshop.com/fonts/singles/itc/old_english_std_plain/?&sample_text=Ungl%C3%BCcklicher%20bayerischer%20Kronprinz&sample_size=32\">Old
    English</a> by ITC (or <a href=\"http://new.myfonts.com/fonts/letraset/old-english/?testdrive=dock%3Dfalse%26size%3D28%26w%3D720%26src%3Dcustom%26text%3DUngl%25C3%25BCcklicher%2520bayerischer%2520Kronprinz%2520%26fg%3D000000%26bg%3Dffffff%26goodies%3Dfit%26seed%3D28\">Letraset</a>
    respectively \u2013 these versions seem to be more or less identical.\r\n\r\nOld
    English is based on a [[blackletter]] by [[William Caslon]] (England, ~ 1760).
    Other foundries had their own versions of this style, like the Manuskript Gotisch
    that Georg has shown above. At the beginning of the 20th century, [[Morris Fuller
    Benton]] designed a version for ATF.\r\n\r\nMore information about the history
    of Old English typefaces can be found in this <a href=\"http://www.linotype.com/2195/oldenglishfonts.html\">Linotype
    Font Feature</a>.\r\n\r\nToday, there is a number of digital \u2018Old English\u2019
    fonts:\r\n<a href=\"http://www.fontshop.com/fonts/singles/monotype/monotype_old_english_text_std_plain/\">Monotype\u2019s</a>
    (and <a href=\"http://new.myfonts.com/fonts/itfmecanorma/old-english/?testdrive=dock%3Dfalse%26size%3D28%26w%3D720%26src%3Dcustom%26text%3DUngl%25C3%25BCcklicher%2520bayerischer%2520Kronprinz%2520%26fg%3D000000%26bg%3Dffffff%26goodies%3Dfit%26seed%3D32\">Mecanorma\u2019s</a>)
    is pretty wide and loose. \r\nThe <a href=\"http://www.fontshop.com/fonts/singles/urw/old_english/?&sample_text=Ungl%C3%BCcklicher%20bayerischer%20Kronprinz&sample_size=32\">URW
    version</a> is lighter, with the \u2018i\u2019 dots shifted a bit to the right.
    It has the long s (&#383;) as default.\r\n<a href=\"http://www.fontshop.com/fonts/singles/bitstream/cloister_black_regular_ot/?&sample_text=Ungl%C3%BCcklicher%20bayerischer%20Kronprinz&sample_size=32\">Bitstream\u2019s
    version</a> is called \u2018Cloister Black\u2019 and can be easily distinguished
    by its sporns on \u2018b\u2019, \u2018h\u2019, \u2018k\u2019.\r\n\r\nFurthermore,
    there are numerous \u2018spin-offs\u2019, e.g. <a href=\"http://www.typedesign.com/fonts/aveblk.html\">Avebury
    Black</a>, an ultra-black and wide display version by [[Jim Parkinson]].\r\nF\r\n"

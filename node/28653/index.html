Up to 1974 there was no exactly defined way of classifying fonts. So lots of more or less phantasy names were used, like Egyptienne in Germany e.g.

1974 DIN 16 518 made it easier. All the fonts like Memphis, Rockwell, Beton, Schadow became “Gruppe V - Serifenbetonte Linear-Antiqua”.

This makes it easier.

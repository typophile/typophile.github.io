Hi :O) Can anyone tell me what the position is with regard to modifying a free font for use in a visual id/logo please? :O) I've tried to find contact details for it's creator to no avail and the licence info. on the free font site states that modification for repackaging or resale is not permitted but I want to be 100% sure what I've done is ok before it's published? :O) Any help or advice very much appreciated? :O)
They're referring to if you were to take their font and change it slightly then resell it as if it were your own. If you're just using the font to create some words and you're tweaking letters that's fine. In fact that's pretty common when it comes to creating a logo.

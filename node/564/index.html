Hello there,
Yes it is true that the FF Meta + has been rereleased as Meta.
The Meta + came in the following weights:
Bold Roman, Bold Italic, Bold Caps, Bold Caps Italic
Book Roman, Book Italic, Book Caps, Book Caps Italic
Black Roman, Black Italic
Medium Roman, Medium Italic, Medium Caps, Medium Caps Italic
Normal Roman, Normal Italic, Normal Caps, Normal Caps Italic

The new Meta has lining figures and expert sets added to each weight.

The metrics of the font have also been changed.
Upon request FontShop provides customers who need to match the Meta+ for older documents with the old data along with the new version of the fonts. (1-888 FF Fonts)

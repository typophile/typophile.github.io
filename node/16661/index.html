Both the "Futura" and "Avante Garde" are Century Gothic. (It's a bit ironic that you ID'd them as you did--Century Gothic is based on Twentieth Century, a Futura look-alike, redrawn to fit the same character widths as Avant Garde for PostScript clones.) The serif face is ITC Bodoni Seventy Two.

Mark, do you know more of the history of Century Gothic? Not a lot, except that it was part of the set of fonts developed by Monotype in the late eighties as work-alikes for Adobe's core 35 fonts that were built into PostScript printers.

For a journal i have to design for september, i have choosen Storm’s Lexon for body text. I wanted a solid face with a lot of personnality, and i think it works well. But Lexon is tricky to marry ; its bold being a sans (and headline) doesn’t make choices easy. I want to use the headline version for the main titling, but i would need some others* — one for alternate titling, and some for specific titles, etc. And the more tricky thing is to find a good little texts sans, which would have ideally to be part of the alternate titling familly. I have quite the desire to find another serif as a compagnion, but matching two text serif looks hard to me. I have also thought about scripts which would have suitable versions for short texts, and i’m virtually open to any idea, all the more as i haven’t to make that choice now.

*ideally, if i had had some money, i would have loved to make Mr Storm expand the family to cover all i want, but…

The lettering in the bottom slide is similar to [[http://new.myfonts.com/fonts/berthold/flange-bq/|Flange]] from Berthold. Another similar font for that one is Bob Alonso's [[http://new.myfonts.com/fonts/bagraphics/crescent/|Crescent]]. His font "That's Amore" is a heavier version of the same design. With Crescent, the 'y' would be similar if you reversed the tail. Overall, for most letters, I think Flange would be a good approximation. - Mike Yanega

Playful...yet professional typeface

February 11, 2008 • julia.warren

I am designing a catalog for a company that is touting Disney licensing. Because it is Disney, I want to use a slightly more playful font for headers, etc. I was using Swiss 721 Rounded, but got some complaints about it being too “bubbly.” Any other suggestions? Thanks.

Try Vista Sans alternates. Clean, professional, and fun. I have found Vista Sans to be an incredibly versatile face.

Enjoy!

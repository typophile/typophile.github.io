Hi all,

Is it possible to save the metrics from a font so that the side bearing values are saved.

I have changed the width values of some gylphs in my font but wanted to keep the side bearing as before.

I did the alterations in a duplicate vfb file and then imported the metrics form the original vfb file, but the metrics file only seem to import the left sidebearing and the width of the glyph, therefore the right side bearing is always incorrect.

Is this the case and is there any way of exporting and imprting the metrics so it uses the sidebearing values.

Rr

The quickest (and easiest) way to do this is to copy the glyphs from the font you want to take the sidebearings from.

Then mark the glyphs on the new font that you want to change and do “Paste Special” on them - and choose only right and left sidebearing in the following window that pops up when you do Paste special.

Paste special is my favourite commando ;-)

You could also save the sidebearings, but this is quicker.

Be careful if you paste the sidebaerings on compsites however, there is a bug sometimes making the composites destroyed, better to adjust the composites manually or by clicking the little arrow in the metrics window (copy to composites) OR use classes to set the composites to the same value as the component glyph.

In the image shown in my previous post, you see that there is written AFM (two times)? The first one opens/imports metrics files, and the second one (looking like a disc) saves AFM-files, which is Metric information. Could be kerning and sidebearings. If you sweep over the buttons you will also se what all buttons means in FontLab.

Now, when you copy metrics, you can just mark the glyphs you want to copy metrics from. Lets say you mark A-Z, all 26 letters. Then do normal copy (as if you were copying them).

Then, shift to the other font. Mark A-Z here to, and then do “Paste special” instead of just “Paste”. You’ll find the command up in the Edit menu. Paste Special, lets you paste what ever information you want onto the marked glyphs.

When you do Paste Special, you’ll see a following window:

If you click on these two I’ve checked here – the only thing you’ll paste is the sideabearings.

Hope you get it working, Rachel.

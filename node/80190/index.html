How to get this effect in Illustrator or Corel?

March 19, 2011 • artslons

Below is the image of text. If you can see it is not a solid color. I want to know how can I get this effet in Illustrator or CorelDraw to it print like this?

How many times have you posted this question? I’ve seen it in at least two other forum sections.

In Illustrator, make a large area of your screentone pattern. Put it under your text. Select both the text and pattern and “Make Clipping Mask”. (You might need to have converted the text to outlines first.)

You can do it with the screentone pattern above the text but it seems you need to make it into a Compound Path first.

I use a fairly old version of Illustrator, from CS1, so there might be other ways in newer versions.

It’s basically a halftone bitmap image masked within text. To create a bitmap, open an image in Photoshop that has varying levels of gray, if it’s cmyk or rgb convert it to grayscale first.

Here’s how to make a halftone bitmap image.

Image>Mode>Bitmap…

Method, Use: Halftone Screen…

Adjust the Frequency, Angle and Shape to achieve the results you want. Save as a bitmap .tif or convert back to grayscale .tif then proceed to masking your type in whatever program you prefer.

What a coinkidink: Just this morning, I was noodling a project that will require a good technique for achieving this effect.

I see two solutions proposed here and in the link above, and, unfortunately for me, neither seems really elegant. One can either create a halftone in Photoshop (which leaves us with a raster image) and optionally Live Trace it in Illustrator (which leaves us with irregular shapes), or one can buy an expensive plug-in.

I don’t mean to belittle the talent that goes into developing that plug-in (it looks really impressive), but the prospect of buying it for a single project isn’t appealing to me.

It’s unexpected and disappointing that while Photoshop has had this functionality for a long time, Illustrator still doesn’t seem to. It seems like an effect that is inherently more appropriate for vector images.

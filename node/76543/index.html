Alternatives for Meta

November 21, 2010 • mattiman

Looking for a new typeface to use for my webdesign company I found FF Meta to be potentially the right one. It's modern, clean, no-nonsense, but also a bit "creative", has a bit more personality. I use Frutiger now, but I'm kind of bored with it. I like clean typefaces like Helvetica and Frutiger, but it's just a bit too plain. However, as with many of the "classics", Meta is everywhere! On trucks passing by, on the ad display next to the road, on the local butcher's window, etc. So looking for alternatives. One I still like is Whitney. However, it's quite expensive and its license seems a bit too restrictive. If I find a typeface I can use on my website (@font-face or typekit) that would be a pre. Otherwise, I would have to create images for the main headlines on the site and use that. Potential alternatives I have found so far - ITC Officina Sans. Also by Spiekermann. Modern and versatile, with just some little details that make it less boring. - FF Milo. At first sight quite plain but also has a few nice details. - Seravek. If I'm correct I can use Milo and Seravek in some way with font embedding for the website. Any other interesting ones? If I would have to describe the ideal typeface, it would be: clean, modern, no-nonsense, friendly
[[http://new.myfonts.com/fonts/shinn/sensibility/|Sensibility]] by Nick Shinn

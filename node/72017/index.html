You can find an abridged version of Bringhurst’s table in the LaTeX memoir class documentation

http://mirror.ctan.org/macros/latex/contrib/memoir/memman.pdf

(see pages 13-14). The LaTeX source, memman.tex is to be found here:

http://mirror.ctan.org/macros/latex/contrib/memoir/doc-src/memman.tex

Just search for “characters per line” in the file and you get the table in LaTeX format.

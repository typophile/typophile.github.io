2009-11-05 23:18:36 • nina

I think one of the main reasons to get a Mac if you’re a designer is so you are more marketable when looking for jobs. I know it’s not not necessarily the same elsewhere, but in the US, Macs are the standard for design. I’ve never been in an ad agency or in-house design department that used anything else. Sure, the programs are going to operate more or less the same, but really knowing the computer, especially in in-house design jobs where you’re at places with IT departments that only know Windows, is a HUGE plus.

If you’re just planning on working on your own, I suppose it doesn’t matter much, but if you’re looking for a job, or if you want to more easily collaborate with most other designers, the advantages become pretty clear.

While one can run OSX on a Windows PC, it’s not officially supported (or even allowed).

As such, if you’re a web developer, it’s nice to have a Mac for that reason alone. You can easily test a wide range of browsers fairly easily.

That said, there aren’t huge issues in the OSX browsers for the most part. You can install Safari on Windows easy enough and Firefox is pretty much the same.

Obviously, if you want to get into iPhone dev, having a Mac is a prerequisite.

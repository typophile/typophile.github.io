2005-06-29 16:27:30 • aluminum

Snapz Pro can do it. That’s what I’ve been using lately.

The ones in my Typecasting series were done over several years. The earliest ones were done with a VCR hooked up to an AV-input-equiped Mac and the Apple Video Player (pre OS X) with which you could use command-C (Copy) to grab a frame. Most of the later ones were done with VLC. It works okay, but navigation was kind of tricky (maybe it’s better now–I haven’t tried the latest version recently). Snapz Pro lets you get frames from the Apple DVD Player, which I prefer because performance and navigation are better than VLC (assuming it hasn’t gotten better since the last time I used it). Even so, the video card may prevent it in some models. (I think it has to support Quartz Extreme or something.)

I don't think that the lettering on the one dollar bill is type. I think that it is all hand-lettered, that is, invented by the engravers. Maybe they looked at some printed specimens to get the letter proportions right, but I don't think that it is based off of any font. Federal is the only thing available that matches the effect. But a lot of fonts have the same letter-style (without the 3D or hatched effects). Would one of these be ok? __ www.typeoff.de

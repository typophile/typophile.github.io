The Index mode shows you the order in which the glyphs are organized in the font. The number displayed above the glyph is the glyph ID, a.k.a. GID.

(For example, InDesign’s Glyph Palette displays each font repertoire in the same order as seen on FontLab’s Index mode.)

Once you have your encoding file, you can do Glyph –> Sort Glyphs –> By Encoding, and the glyphs in your font will be neatly arranged :)

Font for electronics catalog redesign

July 8, 2008 • summergirl

I'm redesigning an electronics catalog geared towards 18-45 year old males. Needs to have a do-it-yourself (install) vibe as well as appear classic and reputable (company has been around for 30+ years). Any font suggestions?
How about Letter Gothic Text? It's a proportional version of a monospace style. So the idea is it has a kind of technical "code" vibe, but at the same time it sets very smooth and compact. For more personality in display, you could match it with one or more of my monospace-unicase Panoptica styles. Pixel might be good.

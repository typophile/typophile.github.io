It’s very hard to give «blanket» advice without knowing more precisely what you (or the client) have in mind. For example something like a word where every letter is from a different font can be made to work I think, it’s just very hard to pull off without looking ransom-note-ish (or just plain bad). It might be an interesting alternative to look at font systems that combine multiple stylistic variants, like FF Polymorph, or the conceptual work that Letterror did for the Twin Cities (the latter, I think, is not available as a font? But can maybe give you some ideas).
2009-12-15 20:15:26 • nina

Ask them how exactly diverse typefaces relate to diverse ethnicities. I have seen ads where different languages are used to show diversity, but to try to do the same thing in a logo would be a mistake, I fear. For one thing, really good logos don’t use any type, and those that do only use a few letters.

If you have so many words in a logo that you can start using that many different faces, then you really aren’t building a logo, but an advertisement. And if they are thinking of using different faces in a single word … (shudder).

Viewing glyph outlines in FL Metrics WIndow

October 14, 2009 • .00

I demonstrated this in my type design class last night. All of the students were surprised that this feature existed, so I thought maybe others were unaware of its existence.

While viewing a glyph string in the metrics window, on a Mac, shift command P will unfill the glyph shapes and render them in their outline form. Very helpful if you are spacing a connecting script. Shift command N will give reveal the control nodes as well.

Just thought this might be of interest. If it is old news my apologies for wasting your time.

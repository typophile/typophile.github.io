Ha! Some kind of consistent match for the colour schemes, typefaces and arrangements might lend some continuity there between the jacket and case.

I’ve never felt it necessary to stack the author name and/or parts of a title on the spine; because I think the convention is that they should share a common horizontal axis, if not a baseline.

Although I quite like that turquoise/black colourway and the (Bodoni? Didot?) capitals on the jacket, the case just looks like an unconsidered last-minute addition.

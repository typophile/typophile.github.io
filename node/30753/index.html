“Designing an HTML page has its sole benefits of good SEO and SEM abilities. "

SEM?

There’s many reasons why HTML has benefits:

  • device independant
  • when done right, completely accessible to a wide range of people
  • can be a good way to single-source content
  • computer readable (SEO)
  • easier to share

The easiest way to deal with custom faces is to just make them images. Obviously, this should only be done for display face uses (headlines, titles, etc.) but works just fine when accompanied with the appropriate alt attributes.

You can also use sIFR…which this site uses…to create the headline type as flash. Alas, it has it’s problems. (For instance, for whatever reason, none of the thread TITLES in Flash show up for me on this site when using Firefox).

If you are talking about InDesign, in CS2 we didn't utilize the language & script tags when applying OpenType features, but in CS3 we did. I suspect that the ligatures may be marked as only running in a particular language or script which isn't matching the actual text. In CS2 we used the equivalent of "default" language & script which you can get to in CS3 by applying [No Language] to the text.

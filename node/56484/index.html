After the release of DTL FontMaster 2.7.0 a couple of months ago, a Light version is now available also from the DTL FontTools web site.

Release notes can be found here. FM 2.7.0 supports the AFDKO 2.0 (already an update is in production that will support the latest AFDKO version 2.5, like DTL OTMaster does).

The kit for running the Windows version of FM in a ‘Wine bottle’ under CrossOver for OS X (actually X11) or Linux has been updated also. At the Dutch Type Library the ‘Wine bottles’ have proven to form very fast and stable environments for running FM.

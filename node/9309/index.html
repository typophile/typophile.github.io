is there a simillar font available. TXS, Sandra
Since 'similar' is best defined by the person asking, you might see something that you consider similar enough by looking through Part 7 (Casual Flowing Script) in the Script Font ID Guide. There are 11 pages of fonts that resemble this style to one degree or another. It's also possible that Part 9 (Everyday Handwriting) might have some more candidates, but I think I would look first in Part 7.

If you want a few suggestions for 'similar' to my eyes, Bickley Script (Part 7 pg 1), Rage Italic (pg 3), or Shimmer (pg 11). In Part 9, you might consider Handsome (pg 2), Stanford Script (pg 4), or Velvet Script (Pg. 7).

Use the Script Font Reference List links on the bottom of the pages to find the designer and supplier of any you are interested in.

I must correct Si, at least as far as MyFonts is concerned. We have always measured our main "font count" in terms of Styles (aka Weights), not Versions.

Multiple versions in OT, PS, TT available for the Mac and Windows platforms count as Versions, not Styles. We also collapse various linguistic codepages (e.g. Cyrillic, Greek, CE, Turkish Helvetica) into the same Style.

Current count of fonts live at MyFonts:

* 11,440 families
* 39,231 styles
* 182,730 versions

We also hold onto old versions and versions deactivated for other reasons, just in case customers need access to fonts no longer on sale. These are normally available from one's "My Account" download area.

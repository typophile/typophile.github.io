Central Lettering Record?

August 27, 2010 • hashiama

I was reading through the Texts on Type book and came to Jonthan Hoefler’s essay: On Classifying Type which talks about an ambitious project to archive lettering called the ‘Central Lettering Record’. This essay was written in 1977 though; I was wondering if anyone has a clue about whether or not it is still available and if so where?

Searches on the internet turns up Phil Baines' biography more than about the actual CD itself…

The Central Lettering Record is a project now run by Phil Baines and Catherine Dixon at Central Saint Martins in London. It started in 1963 and as far as I know it is a photographic collection of lettering.

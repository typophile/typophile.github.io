Apple's absolute first logo

October 28, 2004 • anonymous

Before Rob Janoff designed Apple's famous logo, they had a, to put it mildly, _less_ professional logo...

It looked like this:

http://www.mactechnews.de/user_images/forum/__AppleFirstLogo.jpg

It was hand-drawn, so I'm not sure there is an established typeface that looks exactly like the one used in he banner, but I would still appreciate if you could attempt to classify it.

It looks similar to some versions of Caslon. Many of the characters are pretty close to Big Caslon included with Mac OS X. On the other hand, the "R" is very different, the O is drawn with an almost constant stroke width, while in Big Caslon it is elongated. The difference in weight of the two strokes in the "A" is greater in Apple's logo. I browsed some Caslon-related typefaces on [5] (http://www.myfonts.com/fonts/adobe/caslon/), but couldn't find a better match, though. The Caslon at the very end of this page (http://graphicdesign.sfcc.spokane.cc.wa.us/tutorials/process/type_basics/type_families.htm#oldstyle) looks like an excellent match too, but the page doesn't specify exactly which font was used.

Is there some other family that's a closer match? I'm researching this so that I can add it to my article on Apple's typography on Wikipedia: http://en.wikipedia.org/wiki/Apple_typography
Bump. No thoughts at all on this?

I would say it is more or less in the Caslon vein, but idiosyncratically executed by the artist. The R is the most unusual departure. I can only think of a few typefaces with an R like this (ITC Bodoni, for instance). The C is also a bit exaggerated.

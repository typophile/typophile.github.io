This looks like one of the brush sign scripts which URW has digitized, based on old photo-lettering typefaces. In the Solotype Catalog there is a whole spectrum of these scripts, with all ranges of stroke thickness, slant and lettering styles between printed letters and connected script.

There was a (bad) collection of Expresiv Brush Script fonts sold by Tiger Direct in the 90’s that had styles like this too.

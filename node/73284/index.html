OK, it took me 14 months to figure this out:

If the top of your Word 2010 document window says “Compatibility Mode” — as it will if you open a document created in an earlier version of Word — Word 2010 will disable the OpenType features (to preserve layout compatibility with previous versions).

To fix this, go to File > Info > Convert and upgrade the document to Word 2010 format. Then the OpenType features will activate.

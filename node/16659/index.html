I downloaded the free Meta Pro font and opened it in FontLab. The st ligature is indeed included in the liga feature (the standard ligatures). This seems like a mistake to me.

They also included the ij which is (I think) only used in Dutch. Not that it matters since it is indistinguishable from setting an i followed by a j. I think this is normally supposed to be handled like the German double s, not by a ligature feature.

February 26, 2010 • yanone

Dear all,

I tried a mark positioning feature for an arabic font with the latest FDK, but didn’t succeed. Feature looks like this (anchor positions are made up for now), as suggested in the FDK syntax definition:

feature mark {

markClass [uni0651 uni064B] <anchor 250 20> @TOP_MARKS;
position mark [uni0631] <anchor 400 200> mark @TOP_MARKS;

} mark;

But the marks don’t position (InDesign CS3 ME 5.0.4) Edit: the base and mark glyphs now also defined in GDEF/GlyphClassDef, but to no avail.

Any ideas? Thank you in advance.

Note to self:

Check InDesign ME tool window “Diacritic Positioning” for value “OpenType” before asking stupid questions on public forums. Problem solved.

2005-11-21 21:04:12 • Toby

Why isn´t there a plain stroke tool in Fontlab?

Strokes as such are not supported in any widely used font formats. The old Adobe Type 3 format allowed it (plus anything else that PostScript can do), but that format has been obsolete for over ten years. Fontographer can produce Type 3 fonts (including strokes), but you need an ancient computer and OS to use them.

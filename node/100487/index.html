All script fonts overlap. It's not a problem. But... overlapping is no problem if all the contours on all the glyphs are in the right order. If they are in the wrong order, you may get "inverse" overlapping. Always use "Contour -> Paths -> Set PS direction" before generating the OT fonts (or set TT direction if you are generating TTF files).

Classic video game "Spectre" title font

April 2, 2010 • cfryar

Dear Typophiles,

Can you identify the attached font? It was used as the title in the main menu of the first network enabled, multiplayer 3D cyber space video game entitled “SPECTRE”. This game was developed by Peninsula Gameworks for the Macintosh in 1990 and published by Velocity in 1991.

[img:sites/default/files/old-images/SPECTRE logo (from original game main menu)_5104.png]

We are developing an updated version for the iPhone and cannot locate the original font resources in our archives. None of us remember the font name (it’s been 20 years!).

Also, it is not necessary that we find a version with the chrome type of fill, as solid black would suffice.

Thank you for your time and interest. Best regards, Craig Fryar, President Peninsula Gameworks

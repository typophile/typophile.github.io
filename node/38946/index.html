2007-11-09 16:12:16 • William Berkson

If you wanted to simulate the appearance of un-kerned metal-era type. Or capture the flavour of unkerned shareware fonts.

Or if you have a connected script font in InD, set to “optical kerning”. Actually, there’s no way to “de-kern” a font in InDesign. But you can do it in Quark, I believe, by using the kern edit table.

2007-11-09 16:39:21 • Nick Shinn

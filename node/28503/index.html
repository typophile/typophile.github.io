Not sure if this has been said, but with the copy: I read "20 million" and then "Sperm, Only one egg... Like the odds?" Breaking it between the two is awkward and confusing -- I don't know how to read the sentence. I think "sperm" needs to be kept with 20 million for the message to get through first read. Also the capitalization indicates that "Sperm" is starting a new sentence or thought, as does Only, except it comes after a comma... and then the reader gets confused. Try "20 million sperm. (Only one egg. Like the odds?)" I think having the word sperm up there nice and big will grab way more attention. K.
2006-10-04 17:40:22 • Kristina Drake
Headline wise, it's best to start with a punchline. Try this: 20 million sperm. 1 egg. Like the odds? On three separate lines. Use a nice condensed sans (that's still friendly enough, of course) and the retrofit each line, so "1 egg" would have a bigger size than "20 million sperms". The nice thing about this method is that it'll really highlight your main point. Visually, I like number 6 if you would further highlight the condom. I can't see the suggested "tip" very well.

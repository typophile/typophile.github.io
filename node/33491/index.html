Websites That Use Type Well

May 3, 2007 • Peter G.

Web typography is surely not as sophisticated as it's printed cousin (or should it be mother?), but nevertheless, some web designers have worked hard to do the best they can with what they have. So I'm curious what websites the type community thinks make the best use of type on the web. I'll start. A List Apart takes full advantage of the best of the web typefaces. Sure they're just Georgia and Verdana, but when used well as they are here, they add real sophistication to a design. Khoi Vinh’s Subtraction proves that type can look great on the web if used properly. Here a single font (Arial) is leveraged to its full potential with the aid of a strong underlying grid layout. What sites inspire you to use type better on the web?

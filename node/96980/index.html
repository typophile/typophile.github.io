Hi, I`m looking for the font used in the CELINE logo : http://www.celine.com/ . I only found out that: “The logo itself consists of the name in a digital font, inspired by a mid-20th-century Italian one found in an old type specimen book. “. Looks like the whole CELINE website uses that font. Any ideas what it may be?

It looks like a custom typeface called “celinefont”. You could be interested in something like Novecento: http://www.myfonts.com/fonts/synthview/novecento/

Or some version of Semplicità: http://www.myfonts.com/fonts/canadatype/semplicita-pro/ http://www.myfonts.com/fonts/durotype/simplo/

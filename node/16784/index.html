TrueType fonts cannot have characters in those slots. PostScript fonts can, but access is very limited in older Mac systems, and not supported in later systems. In Mac PostScript fonts, for instance, low ASCII slots are sometimes used by font developers to store Windows-only characters for convenience sake. But there is normally no way to access them on the Mac, and definitely no way to type them.

I should clarify that, even if you are able to put characters in the lower ASCII range (as in a PostScript font), you may not be able to access them. The lower ASCII range is not meant as a place to put “normal” characters. This is where special codes like the carriage return, line feed, delete, and tab are found. Using them for normal characters will probably not work, even if you are able to make such a font.

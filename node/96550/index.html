Gothic 821 Condensed, Bitstream’s version of Block Berthold Condensed, designed by Heinz Hoffmann in 1908. There’s only one Bitstream font; Berthold has an 8 font family with an oblique but no true italic. See Dieter Steffmann's similar [[http://www.dafont.com/rolling-no-one.font|Rolling No. 1/Rolling No One.]] (free) Also at http://moorstation.org/typoasis/designers/steffmann/samples/r/rolling.htm

Basic Alphabet teaching for children?

October 20, 2008 • kegler

There are different letter drawing teaching techniques for children such as the Zaner-Bloser and D’Nealian methods. Some character formations are unique to different languages and countries such as the 7 with or without a slash or a q that has a curled tail vs. straight. Are there any good resources for some of these basic learning alphabets for young children in languages such as Arabic and Hebrew and even Ukrainian & Slovenian where the local preference in teaching character formation in learning to write is explained?
i would grab some primers, either for child learners or for students for whom these might be second languages. I have some great textbooks for Russian, but I’m sure one could find these in a decent sized library for the languages for which you are seeking. i tried to look at primers to find regional preferences for languages that use Devanagari script and found these to be useful resources when i could find them.

Additionally, it may be hard to find certain models for letter drawing techniques. When i was learning Russian, we only ever did script writing excercises and were never formally taught to write in block letters, i’m guessing this would be similar to the experience of most young students learning to write in countries which use the cyrillic alphabet. Anything I might say about Arabic would be mostly guess work, but I would think that teaching of the relation between initial/medial/final forms would be an important aspect in teaching the basics of the script to new learners.

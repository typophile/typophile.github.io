Hello Typohilers, I'm hoping someone can help. I have been looking for info on how to change the font menu display in Word 2007. The default setting is for the actual style of the font as well as the font name to be displayed. This is quite advantageous for most users, but we want to turn that feature off and revert back to the old display where the font menu just displays the font name in a standard font for all fonts listed. Having searched high and low, we can't seem to find an answer. We work a lot with barcode fonts, and it is proving problematic just seeing the barcode characters in the font menu rather than the font name. Any help is appreciated? Many thanks Neil
On most versions of Word you can find the menu item to shut it off in Tools:Customize then you should see a checkbox for 'WYSIWYG Font Menus' uncheck this. You maybe have to restart the application. I only checked this in Word:Mac I have Office2010beta running on Windows.

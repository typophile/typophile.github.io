Many Criterion DVD covers were done by [[http://www.kellerhouse.com|Kellerhouse Inc.]]. May be it’s one of their many artworks using some bespoke typeface as they’re well-known for. Looks like they artificially emboldened by adding an outline stroke to some Signum-like (thanks Don) font and reworked counters. BTW, following Don’s previous modification guidelines, some more alternatives even if Trump Gothic is a nice find: [[http://www.myfonts.com/fonts/mendoza-vergara/proteina|Proteina]], [[http://www.myfonts.com/fonts/grouptype/raleigh-gothic|Raleigh Gothic GT]], [[http://www.myfonts.com/fonts/dizajndesign/komu|Komu]] A, [[http://www.myfonts.com/fonts/gautier/le-gendarmerie|Le Gendarmerie 205]], [[http://www.myfonts.com/fonts/storm/header|Header STF]], [[http://www.myfonts.com/fonts/typetrust/heroic-condensed|Heroic Condensed]], [[http://www.myfonts.com/fonts/jnlevine/movieland|Movieland JNL]], [[http://www.myfonts.com/fonts/bagraphics/title-gothic-light|Title Gothic]]

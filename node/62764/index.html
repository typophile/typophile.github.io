Tastaturaufkleber - German fonts

October 6, 2009 • tastaturaufkleber

Hello World!

this blog has started! It meant to be a blog for multilingual writers. Tastaturaufkleber is a German word that people use to describe stickers that are used for multilingual tipping. It’s all about the smallest efforts for maximal results. I want to share my experience and also see which experience other people already have with that type of a solution. I have also started a blog http://www.aufklebertastatur.de to see how people handle this problem of bilingual computer use. I’m also wondering if there are any different fonts on keyboard stickers. Does somebody know this? I’ll be pleased to read your comments!

Hannah

January 5, 2009 • IAmMintCondition

[img:sites/default/files/old-images/Hannah_frontpage_5898.jpg]

Just released Hannah this week. It is a hand drawn font comprised of three versions that work together as one, producing not variation and contrast between weights but between widths. The ability to mix and match along with some contextual alternates in OpenType allows for many interesting combinations for display. Hannah is available [[http://www.youworkforthem.com/product.php?sku=T0235|here.]]

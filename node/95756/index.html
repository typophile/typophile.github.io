I tried some of the free image uploading identifying sites and they come up with gibberish. This font was used by USCIS to print A-Numbers on I-94 forms; http://international.caltech.edu/photoimages/i94card.jpg [img:sites/default/files/old-images/font_6274.png] Thanks in advance.
This is essentially a credit card imprint - the same digits as are embossed on many credit cards. Here is an emulation (quite a bit bolder) - see the LARGE Numbers in [[http://www.myfonts.com/fonts/wundes/caard/|this font, Caard]]. There are other credit card imprint fonts used; some are available. BTW - WhatTheFont found this with no problem. You have to give it an image without extraneous borders, which the .png file had. Had to edit the image and 'flatten' it. - Herb

Chakma script - the first Unicode font available

October 31, 2012 • JanekZ

“Version 6.1 of the Unicode Standard continues the Unicode Consortium’s long-term commitment to support the full diversity of languages around the world.”* Among others Chakma script have been added (11100..1114F). The standard is not enough—people need the font to write. Some days ago the RibengUni font has been released, see: http://uni.hilledu.com/download-ribenguni/ , designed by Bivuti Chakma (Suz Moriz) & Jyoti Chakma [Bangladesh] with assistance of Jan Żurawski [Poland]. The font is released under SIL Open Font License. Latin part is adopted from the other OFLicensed font released recently. You can see the font in the wild http://www.facebook.com/groups/hillbd/

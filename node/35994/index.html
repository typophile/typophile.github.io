Gg - Fonts with a cool G/g letterform

August 10, 2007 • Artform

I'm going to open a can-o-worms here. I think the letter 'g' is one of the few characters that really stand out in a type face. There are a multitude of ways it can be crafted. With so many fonts out there, I look at a few letters -- the lowercase g in particular. This is how I propose the thread will work. I'm not here to militarize things... just some friend standardization. 1. Go to a font site (I prefer http://www.fontshop.com/ because you can link to samples) 2. Find your font 3. Post a link to it (if possible to the sample) along with the font name. For example: Natalie Script OT http://www.fontshop.com/fonts/downloads/paratype/natali_script_ot/?sample_text=Gg&sample_size=36 You can also post an image. Let's see your G-sus.

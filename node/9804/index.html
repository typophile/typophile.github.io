I believe this is a customization of Supermarket, possibly Supermonomarket, which is an inhouse Group94 customization of Supermarket by Sven Stuber, which can be found at Superlooper. The customization is not, as far as I know, publically available. Group 94 also had some larger pixel fonts that were custom made (perhaps stansharp, among others), and there were rumors that some of these may eventually be made public, but I don't know what happened to those rumors. For more information, consult these Typophile threads here and here.

The names of the dishes are definitely not made with an IBM Selectric Composer in Univers. This is done with the mechanical four-unit Varityper. (An IBM Executive typewriter with the Mid-Century face, based on Futura, which might be considered to be close, is also not a possibility; this is a Varityper.)

If you want a truly minimal /System/Library/Fonts folder and do not require CJK support, you can remove everything but the following.

Helvetica.dfont Keyboard.dfont LastResort.dfont LucidaGrande.dfont

However, you should probably also leave the following alone as many applications won’t work properly without them:

Geneva.dfont HelveLTMM Helvetica LT MM HelveticaNeue.dfont Monaco.dfont Symbol.dfont Times LT MM Times.dfont TimesLTMM

Note that if this list is specific to Leopard. Other fonts may also be required under other versions of MacOS. Since the above list includes no CJK fonts, you may run into serious problems if you ever switch input methods.

Anything which is not stored in /System/Library/Fonts is pretty much guaranteed to be safely removable.

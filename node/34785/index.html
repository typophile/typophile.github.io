Re-naming updated typefaces

June 27, 2007 • Ricardo Cordoba

I apologize in advance if this topic has been raised before here... I am using an ancient operating system and can't find Typophile's "Search" feature. I am curious about something I just noticed on MyFonts: two "new" typefaces that are actually updates of pre-existing ones, but with new names... Loose Caboose NF is actually Seymour Chwast's Artone, while Flap Jacks NF is an update of Black Jack, by Vincent Pacella... While both designers are credited by Nick Curtis, I wonder if it isn't confusing to rename these typefaces. Any thoughts?

2007-07-08 18:18:16 • Ricardo Cordoba
If you're doing a revival I think it's sensible to avoid using a name that someone may be able to claim rights over, even if there is no formal trademark. However untrademarkable names like Hellenic (descriptive) and Garamond (historic) are considered fair game.

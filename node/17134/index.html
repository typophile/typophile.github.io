How does Indesign count the number of characters?

2006-01-08 00:37:49 • Sebastian Nagel
By the backing string, i.e. the stored character codes that the type represents. So if you type ffi and it renders as a ligature, InDesign still knows that there are three characters in the backing string. The whole point of caret placement is to edit text, so you want the positioning to relate to the characters underlying a ligature, so that you know if you position the caret between the two fs of the ffi ligature and hit delete, you will remove the second f leaving fi.

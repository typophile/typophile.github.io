2003-07-16 01:17:14 • cheshiredave
Tiffany, where on earth did you find that?

By the way, if anyone is designing an uppercase Eng, be aware that there are at least two different forms. The one Tiffany has is an example of the European Eng, which is used to write Northern Saami. There is another form which looks more like an oversize lowercase n with a curved right side and a hook at the baseline. This second form is used for many African languages. There is a third form which is similar to the latter, but with a descending hook. This seems to be a preferred glyph variant for some African languages.

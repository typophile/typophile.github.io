2011-07-12 20:48:44 • oldnick
There is a real problem with some URLs when there is no way to parse the words. To get someone to type in a string of characters from memory is actually very hard unless they understand the words that it spells. Initial caps is an excellent strategy to overcome this problem. There are other ways, such as color or boldness. In the end it is not a matter of right or wrong, but a matter of achieving a goal such as getting people to actually visit your website.
2011-07-12 22:36:36 • Luma Vine
Note that the base domain name (supergreat.com) is not case sensitive, but the path (anything after the slash: supergreat.com/greatstuff) is, so incorrect capitalization will make the path invalid.

2004-07-28 09:57:04 • steve_p
Steve, I like tabbed browsing because I can group my browsing in a subject-oriented manner, rather than in an unsorted mess.

If someone on typophile links something, I can open it in a tab attached to the originating window, then Cmnd+Tab over to my email window which might have a tab for Hotmail and/or Gmail along with tabs for any links I got in emails. It's nice for research too, since I can run 4 Google searches in 4 windows and open links related to each in tabs under that originating Google search. I find IE to be laborious in contrast.

This hits on a fundamental difference in the 2 big OSes (mac and windows) as well: logical/hierarchical segmentation. While tabbed browsing is possible in both OSes, in Windows every document gets its own Taskbar item (except in apps like Photoshop), but in Mac OS every app gets its own Dock item, then you can move between documents in the app you choose. There are benefits to both, but I prefer the Mac method of hierarchical segmentation (this isn't an official term, I'm just using it for convenience). Also, OS X allows document windows from different apps to intermingle, so you can pull your Illustrator work to Photoshop without having to move every window out of the way, so it's not too strict about the segmenting (OS 9 does not have this ability).

I'm not a PC-hater, and I don't use every single widget and feature of my browsers, but even for me IE is behind the curve in a few siginificant ways. And, if I understand correctly, MS is ditching IE anyway (probably so they can just integrate browsing directly into the OS).

It looks indeed very close to New Century Schoobook but I think it’s not. It’s a more “round” font called “Computer Modern” which created by Donald Knuth (using his Metafont, stroke-oriented font editor) to be used in Tex (another creation of D.Knuth). Some versions of it have been converted later to normal TTF/OTF font formats: http://en.wikipedia.org/wiki/Computer_Modern

Mr. D.Knuth Mr. started to create Tex/Metafont in 1977 when he was pissed off that there was no appropriate method/program to print a mathematical paper containing complex equations and scalable mathematical symbols. The fact the sample is a research report and that the text has been seen before in mathematical proofs, is an additional evidence it’s actually Computer Modern.

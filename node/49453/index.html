Robofab - Remove Kerning Pairs ?

September 15, 2008 • RachelR

Is it possible to remove kerning pairs from a font with Robofab based on a condition. I want to remove all the kerning apart from those that are applied to the smallcaps. I was thinking something like : from robofab.world import CurrentFont f = CurrentFont() kerning = f.kerning if(kerning != .small) remove I don't know how to write conditional statements for Robofab so I know this syntax is completely wrong
Maybe you could try the eliminate() method mentioned at http://robofab.com/objects/kerning.html: from robofab.world import CurrentFont f = CurrentFont() # create a list of smallcap glyphnames which you # identify by the ".small" suffix: smallcaps = [g.name for g in f if ".small" in g.name] # eliminate kerning involving a smallcap: f.kerning.eliminate( leftGlyphsToEliminate=smallcaps, \                      rightGlyphsToEliminate=smallcaps ) f.update()

altering typefaces?

July 20, 2005 • thierry blancpain

when i thought of Helvetica Flair yesterday, i asked myself if things like that are even legal - taking a letteshape, altering it (maybe better talk about “extend it”) and using it again. please be aware, im not talking about selling it - just using it for personal or business matters.

by the way, most of you will know it, but on typograpica, you can find a nice interview with phil martin.

it all depends on the foundry’s [[EULA]]. Some allow for font alterations, some don’t. Be sure to read the documentation that comes with your font license.

That said, some feel that once you license a typeface you should be able to alter it however you please, but I won’t go there… you can find that discussion elsewhere on typophile.

My observation, is that when you come across a typeface, whether already in print or digital, for a project ie., poster, logo (with rights procured) postcard, one might be bound to add or subtract for purpose to add a meassage, this is in the realm of acceptable. If your purpose is to redesign and distribute as altered face, this is unacceptable. Addition or subtraction can add a great deal in the visual sence. A designers comfortability with type is essential. I believe that stretching, scaling or obsuring a typeface’s general being is morally unacceptable more than creating something new. By redrawing or deconstructing a letterform is to use a typeform naturally and extend the meaning or purpose for which it was initially designed.

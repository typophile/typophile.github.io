2012-09-12 11:12:04 • daverowland

I can’t be sure that missing the ‘MSVCR100.dll’ is the only problem, but it is a sufficient problem. The issue is that the FDK is compiled with an older version of the Microsoft compiler tools, in order to be compatible with Windows XP. Windows 7 no longer includes this library. I thought I had solved this, by compiling this library into the autohint program, but this is obviously not the case. You can work around this by downloading the file, and copying it to: C:/Windows/system32/

You can just google “MSVCR100.dll download” but it can certainly be downloaded from: http://www.microsoft.com/en-us/download/details.aspx?id=5555 Please download and install this file, and then let me know if you still have a problem

Postscript outlines, TrueType hints?

July 15, 2009 • blank

Am I correct in believing that CFF OpenType fonts can only contain Postscript hints? Or does the OpenType spec allow TrueType hints in a font with Postscript outlines?
Your initial belief is correct. The TT and PS hints are closely linked to the outline formats, so there is no way to apply TT hints to a PS outline.

The one solution I’ve found to this issue is to convert the font to individual .pfb files with Fondu and then convert these files to a more useful format. I have used FontForge for this. Based on the Emigre FAQ, I think this kind of conversion is allowed:

The Emigre Fonts license does allow the user to convert the fonts to other formats as long as the user has purchased the appropriate user license and the converted fonts continue to be used in accordance with this license. In addition, the converted fonts must retain the original Emigre copyright notice.

After conversion, the spaces appear as expected.

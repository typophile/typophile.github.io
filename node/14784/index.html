David,

Apparently, the way it works is that when you choose insertion point in the text or select some text, the Glyph palette (or job case, if you wish) switches to whatever font is before the insertion point or in the selected text. (If the selected text includes more than one font, it switches to no font at all.)

So, the trick is, choose the location first, then you can switch to a different font in the Glyph palette and when you double-click on a character, it will insert the correct character from the font you chose, even if it is a different font from the surrounding text.

Does that make sense?

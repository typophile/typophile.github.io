If the typographic family contains four styles only, from which one is displayed in the font menu of word, there are no doubts, that the displayed style is style linked to bold. But if the typographic family contains more weights, the user has to think about, which weight is style linked to which weight. You have not to expect technical problems, but it is uncomfortable in my opinion. In many fonts medium is style linked to regular, but not to light, because the contrast between regular and bold would be too big in body text. I prefer, to clearly know, which weight I select. I want to be able to pick up the weights from the drop down menu, but not with the help of the bold-switcher. There is no duty, that your styling group contains exactly four styles (regular, bold, italic and bold italic). Four styles are the maximum only. You can build styling groups, that only consist of the upright and the italic style of the same weight.

Need suggestions for a compnaion face to Neutraface

August 5, 2009 • takach

Does anyone have suggestions for a text weight companion typeface for Neutraface, slab preferably, maybe a serif though? I know there is a Neutraface Slab, but I didn’t want to do something that obvious. Or should I just suck it up and finally purchase that typeface since Neutraface is so peculiar? Thanks for the help.
If you are going to use another geometric I think you are better off going with Neutraface Slab because it is idiosyncratic and any other geometric would not match. However if you are just looking for something to go with Neutraface then you can really mix it up. For instance I’ve used Emigre’s Solex with Neutraface. (Keep in mind that I was using Neutraface and Solex in contrasting ways. Solex was never used at the same size or the same way. They served different purposes.)

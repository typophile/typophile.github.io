Hi Ray,

You will have to make your own encoding (or borrow one) with these 4 glyphs in the beginning. (there is a nice little software out there to edit the encoding file’s numeration -take a look to an encoding file and you will understand what I mean)

Here is: http://typophile.com/node/46625

You can edit and rename an existing encoding and once it is placed in the encodings folder you’ll be able to select it in the pull down menu (after re-starting FL).

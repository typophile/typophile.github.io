Hi Daniel, this is Viva. Viva is the work of California designer [[Carol Twombly]]. It is a [[multiple master]] typeface with weight and width axes that allow variations of Viva to be generated by the user. Viva’s weight axis can be thought of as a shadow axis: as the weight increases, the design casts a longer shadow in a weight range from light to bold. The width axis ranges from condensed to extra-extended. — linotype.com

I didn't see this until WLD posted. You need to restart under System 9, then install BorderMaker. You should then be able to restart under OS X and run BM using Classic mode. If you haven't done that before, go to System Preferences and click on Start Up Disk and select OS 9, then restart and install BorderMaker. To switch back to OS X while using OS 9, go to Control Panels. From there you can select OS X from the Start Up Disk preferences. The problem is that BorderMaker is looking for system preferences and extensions that weren't install. And you can't install them while running in Classic mode. At least that's been my experience. I hope that works for you.

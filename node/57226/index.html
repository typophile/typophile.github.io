I had a similar problem after installing OpenOffice.org 3.0 on my new iMac. Even though the DejaVu fonts appeared in my OpenOffice font list, I could not see them in other applications, and I could not find them with Spotlight. Because I was new to the Mac environment, I did not realize that a font included within an application package can be used inside that application while remaining invisible to the rest of the system. If the suggestions above haven't helped, try searching Photoshop's application package. Go to your Applications folder, right click Photoshop, select "Show Package Contents", and use Spotlight to search for a file name including Bickham within the Photoshop package. Gerry K

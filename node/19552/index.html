A bit more than that. Just have a look into any AFM. (a)  In the header there are things like font names, italic angle, underline position and thickness, vertical dimensions, bounding box, &c. [All that you would determine in the FontInfo dialog.] (b)  Then, name and metrics for every glyph. Metrics are given as total width of glyph (including side bearings), and then: glyph's outer left point, glyph's outer bottom point, glyph's outer right point, glyph's outer top point -- measured from character origin. [An effect of this: When copying metrics from font to font via AFM in Fontographer or FontLab, if both fonts' glyphs had different total widths, then the right side bearing was messed up. Because FOG and FL were just copying each glyph's total width and left side bearing.] (c)  Finally, kerning information. So an AFM contains almost a complete font -- except glyphs' outline descriptions and hints.

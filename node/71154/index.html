2010-06-08 22:06:29 • dezcom

Use this

feature locl { script latn; language TRK; lookup ITRK { sub i by i.TRK; } ITRK; 
	language AZE;
		lookup ITRK;

	language CRT;
		lookup ITRK;

} locl;

With this

feature smcp { sub @latin_lc by @latin_sc; sub dotlessi by I.smcp; } smcp;

And make the glyphs i.TRK for lowercase, Idotaccent.smcp for small-caps, and Idotaccent for upper-case.

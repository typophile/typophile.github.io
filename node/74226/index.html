2010-09-15 14:25:49 • Ramiro Espinoza

A good start for individual situations is the following:

feature calt { sub n b' by b.alt5; sub m b' by b.alt5; sub space @lc' by @lc.alt; } calt;

If opentype will find an instance of nb or mb it will change the b (since it has been accented with the apostrophe) with b.alt5. You can also work with groups by adding the prefix @. If a space is followed by one or more letters in the group lc, it will change the first letter to the other group.

It’s really easy once you get the hang of it.

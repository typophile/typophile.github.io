Although OFL does not allow the general redistribution of a derivative for money
It allows the redistribution of a derivative for money, but not by itself. For example, you can embed the font into an iPad app and sell the app, but not just the font by itself.
it does allow somebody to get paid to create a proprietary derivation for a client
It allows anybody to get paid to create a derivative for a client, but it must remain OFL, not proprietary. Also, if the font uses RFN, you may need to a) Rename it, or b) Ask permission to continue using the same name.

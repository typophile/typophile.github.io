2008-07-30 10:31:06 • kiko

Make sure you’ll do the following:

  • Download the latest version from robofab.org (Robofab + FontTools + DialogKit)
  • After unzipping, place the RoboFab folder somewhere on your harddrive. I’ve put mine in my applications folder
  • Run install_zip.py
  • Copy only the folder named ‘Scripts’ to your FontLab macro folder

If you move the folder with the Robofab data to another location you need to run the install script again.

Hope this helps.

  • To install Numpy for Python2.4, I used and old Numpy1.2.1 release.

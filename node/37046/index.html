It's a mixture of script styles, with an s that is normally not connected and and r-style that usually is connected. 'Radio' that Stephen suggested uses the more conventional connected script 's'. This logo was probably designed to be script-like, but a bit more legible, as you suggested. I doubt if it based on any font, except in a vague 'generic' sort of way. In my opnion it works from a distance, but doesn't look so good on close examination, like the 'Franken-font' it is. - Mike Yanega

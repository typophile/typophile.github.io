2011-10-18 14:58:19 • Morpheus10

If the glyphs are exactly the same, remove/delete one or reassign as the Ohm (electrical resistance) sign (code U+2126), if the font doesn’t have one already.

If they differ, consider which one is a better Omega, then proceed as above.

Aside: The Mho (electrical conductance) sign has the code U+2127 and is an upside-down Ohm sign.

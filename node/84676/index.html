Hi Everyone,

I’m working in a new typeface that is taking all my waking hours away!

Inspiration comes from nixin tubes, [[http://scherle.com/wp-content/uploads/2009/03/nixie_blue.jpg]] The basic shapes are very simple, so I wonder if there’s already another typeface like it around?

Is it a go? Critics and ideas are always welcome =)

Well, the field of geometric typefaces like this is quite ‘grazed’.

‘The Northern Block’ offers several similar typefaces (of rather poor quality as far as I can tell from the samples): http://new.myfonts.com/foundry/The_Northern_Block/

A professional take on this ‘naïve’ style is Helsinki by Ludwig Übele: http://new.myfonts.com/fonts/ludwiguebele/helsinki/

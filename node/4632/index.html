Twila,
That 2bead.com web page's HTML specifies only "sans-serif". If you're not using Windows, then disregard what follows …

In Windows, at least, the typeface used when the operating system or a program calls for sans serif is, I believe, set by the operating system to default to the ugly MS Sans Serif bitmap font. It sounds as if maybe yours is missing and so some other, possibly random, font is used. See if you have MS Sans Serif in your Fonts folder. It will likely have a different icon than the TrueType fonts. Note that you should also have a MS Serif bitmap font in that fonts folder for when a call for simply serif is made.

The font you set as the default in Internet Explorer only applies when a website doesn't specify any font at all.

So it is possible to import kerning into a font without effecting the pairs that are already there - like add pairs that don’t already exist and don’t change the kerning that is already present

You have not stated what font editor you’re using, so it’s really hard to give specific advice. If you happen to use FontLab Studio 5, FontLab 4.x or TypeTool 3, the advice below should give you what you need. If you use Fontographer, DTL FontMaster, FontCreator, FontForge or one of many other font editors out there, you’ll need to look for alternative advice.

So, assuming you use FontLab Studio, FontLab or TypeTool, and if you have some kerning in the font B, and you want to add the kerning from the font A to the font B but leave the existing kerning that was in the font B intact, you need to:

  1. Go to the Metrics Window of the font A.
  2. Choose Tools / Save Metrics and save the metrics as “A.afm” in a working location (e.g. the Desktop).
  3. Go to the Metrics Window of the font B.
  4. Choose Tools / Save Metrics and save the metrics as “B.afm” in the same location as above.

After these steps, you have the full kerning data of both fonts saved in two AFM files.

  1. While still in the Metrics Window of the font B, choose Tools / Open Metrics, click on “Continue” and choose “A.afm”.
  2. In the “Import Metrics File” dialog, set “What do you want to do with metrics data” to “Do nothing” and set “What do you want to do with kerning data” to “Add imported kerning data to the current font”. Click on “OK”.

Note that after this step, the kerning pairs that existed in both A and B but had different values have been replaced with the values that came from A. So values from B have been only kept for pairs that did not exist at all in A.

If you want to retain the B values for the pairs that existed in all fonts, you need to complete the two final steps now:

  1. While still in the Metrics Window of the font B, choose Tools / Open Metrics, click on “Continue” and choose “B.afm”.
  2. In the “Import Metrics File” dialog, set “What do you want to do with metrics data” to “Do nothing” and set “What do you want to do with kerning data” to “Add imported kerning data to the current font”. Click on “OK”.

You’re done.

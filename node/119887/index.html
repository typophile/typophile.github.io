2014-09-25 20:38:35 • riccard0

Yes, some proportionally-spaced faces for electrical typewriters had a very limited set of spacing options:

IBM Executive (the most common): 2, 3, 4 and 5 units

Vari-Typer: 2, 3, and 4 units

Underwood had a proportional typewriter as well, and one of the typefaces for it, called Raphael, had sizes of 1, 2, or 3 units - just what the OP suggested.

As well, with computers, some systems displayed mixed Latin (and Greek and Cyrillic in some case) alongside Chinese characters, using one byte to code the former and two bytes for the Chinese characters, in a traditional character generating video card - and so the Chinese characters were double-width on the screen, matching the double size of the code representing them in the video card memory.

In addition, some Monotype matrix-case arrangements for the Hebrew language had only a limited number of widths, to facilitate applying vowel points to the letters.

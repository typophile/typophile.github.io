How did you send the fonts to the printer? If you send a font suitcase over the internet without enclosing it in a StuffIt file, there is a good chance that the file will not survive the trip. (A .zip file created using the "Compress" command in the OS X Finder will work, too, so long as it is opened by OS X Finder on the receiving end.)

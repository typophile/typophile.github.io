Actually, InDesign supports the 'hist' feature. However, it can only by accessed through scripting, not through the UI. The property is called 'otfHistorical'. The last page of the OpenType User Guide for Adobe Fonts (PDF file) shows you a list of which features are supported by Adobe's applications.

2007-02-26 15:14:47 • Choz Cunningham
Oh, liga and clig (for standard and contextual standard ligatures) for ligatures, and dlig (for special ligatures), should be your first choices. If an application supports OT features, then these two are most likely supported. For additional things, maybe one of the Stylistic Set features might do. But these were not intended for ligatures and are available in InDesign CS2 only. (And in some recent Apple applications, but here, one can only select one Stylistic Set feature at the same time. Which is why I mention it only in parentheses.)

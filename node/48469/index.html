Enjoying P22 Type Catalogue

August 16, 2008 • dezcom

I recently purchased P22's new type specimen book and have been enjoying "reading" it! Yes READING! The chosen text for each sample is both informative and enjoyable. The visuals are fun as you might expect from Mr Kegler, et al but the writing brings back fond memories of the typebooks of old. The book is also very helpful to those not familiar with the P22 web site and makes shopping for type much easier than trying to navigate the web site unaided. http://www.p22.com/products/specimenbook.html ChrisL

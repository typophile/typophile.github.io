2004-01-29 15:54:51 • nike
Make no mistake, it's the same type of script used on an old Swingline Stapler or Philco TV :D

This falls in a category I refer to as a Streamliner Scripts. Yes, Leslie has a nice one called Streamline which is similar, I have a few at the Diner, but I noticed the Sinatra script when the re-release of the Sinatra collection on CD came out a few years back.

I do intend at some point to develop a font inspired by this lettering example for the Diner, but I just haven't gotten to it and it's not planned into the next font set release.

To be honest, I would recommend starting with Dragonwick and recreate your own cap letter to get this effect.

Stuart :D

Is there any way to batch process removing or at least identifying the True Type fonts so my library is completely PostScript? Any ideas welcomed, thanx
It is possible to do this with Font Doctor. Sadly I'm not sure if my instructions will help or not as I'm not sure what you mean by FFIL and LWFN ... sounds like PC font naming. Here goes anyway.



Before you start the process you need to set up your preferences. This can be done by clicking on "diagnose options".



At the bottom left of the "Diagnosis Options" window you will see that you can diagnose mixed font types. You can also have it remove the others to the trash. Once finished, if you want to keep those trashed fonts, simply put them in a folder named so you know what is in there.

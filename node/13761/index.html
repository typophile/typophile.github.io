[[Indices]] : [[Terminology]] : Em

Pronounced like it is spelled, an em is a measurement equal to the current point size. An [[Em Dash|em dash]] is a dash the width of the given em. Because “em” and “en” sound so much alike, the em is sometimes referred to as the “mutton.”

See the related [[En|en]].

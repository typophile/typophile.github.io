For a font with the straight portions of the inline check Core Circus 2d In https://www.myfonts.com/fonts/s-core/core-circus/ Use as is or extend the inline to make the curved portions of the Kreisler K logo. The weight of Core Circus is a little lighter than the Kreisler K logo. So if you are editing you could make the K bolder. Or you could do all of the inline from scratch, using a base like ITC Avant Garde Gothic Bold, as suggested by Fivos. Don

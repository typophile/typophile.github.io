2009-06-24 08:50:07 • satya

When come to school assignment for redesigning an existing logo, one of the rules I apply is never choose a famous brand, unless it has an obvious flaw.

Why choose a successful band logo (which likely to be designed by a pop-star designer with years of impressive portfolio and client has spent millions on the project ) then make the student change it? Most of the time it would make them looks stupid in comparison and such assignment is not very constructive. (unless it’s an assignment designed with specific purpose in mind like to challenge the status quo and see how far you can push it…)

If it’s not broken,why fix it?

The lecturer is the one to be responsible here. Please tell the students in the first place what’s wrong with the current Nike logo (if you can identify one) and what is there to improve? If not how would you judge whether the newly modified version by the student is an improvement?

Design is about solving problems. How would the student learn this process if they can’t identify the problems?

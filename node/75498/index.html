Comic Sans is not a bad typeface. It is suitable for use as an informal script typeface for materials that can use a bright and cheerful touch, especially if they're associated with children. But because it's a standard face that comes with Windows, it's often used, because nothing else is at hand that comes close, for purposes to which it isn't suitable. For example, it might be used to simulate hand lettering - such as in word balloons in a comic. Or it might be used where Papyrus, or Hobo, or even Dom Casual might be called for. If people use Georgia when they really wanted to use Caledonia or Baskerville or Garamond, little harm is done - all these faces are nice standard text faces, suitable for book use, which don't happen to be Times Roman. But when it comes to display faces, there are many different display faces which have different purposes. You wouldn't think that Bifur, Peignot Bold, and Avant Garde were interchangeable. Comic Sans, because it's handy, gets used beyond its range of applicability. And often by people who don't know or don't care about typography - and in bright, clashing colors - so good results are not to be expected.

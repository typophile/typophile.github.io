2011-05-25 01:52:44 • hrant
Go here! http://www.wells.edu/pdfs/Book_Arts_Summer_2011.pdf The Wells College summer program has a variety of classes related to typography, type design, typefounding, and book arts. I've signed up for a Monotype casting class at the Bixler type foundry, but I'm only the second person and they need at least four to run the class. If you sign up, more chance it'll happen! I went to Wells a few years ago and had a great experience. Some big names teach and attend the workshops; Julian Waters, for instance, is teaching this year. I'm a teenager too, by the way. I've found that if you're knowledgeable, respectful, and enthusiastic, you're usually welcomed at any event.

2007-04-25 14:39:16 • guifa

Tools > Kerning Assistance >Expand

This will convert your class kerns into “hard” kerns stored in a separate font table which can be accessed by previewers.

You can also inspect and edit these hard kerns in the list on the right side of the Metrics window.

FL provides various criteria by which you may limit the number of kern pairs generated by the Expand process, allowing you to limit ballooning.

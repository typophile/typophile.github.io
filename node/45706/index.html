I recently purchased Merlo. http://typophile.com/node/44367 The version from Vllg is actually better than the version Mario actually sells. It has more ligatures, dingbats, and several other important features like fractions. There also differences in the vertical bearings and kerning: vllg: [img:sites/default/files/old-images/vllg_5649.png] felliciano: [img:sites/default/files/old-images/felliciano_3949.png] However the version that he sells has alternates in the italics that I really love: [img:sites/default/files/old-images/merloalts_5447.png] Also, Mario tells me there is an opentype version on the way soon. Hopefully all the features will be united. Until then, buy the Vllg version then request from Mario the version he has. I really love using it. Next on my list from him is Rongel. I really love his work. All my best, Mike Diaz :-)

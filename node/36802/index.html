calligraphy for left-handed

September 5, 2007 • martina

Hi, I have a question..did anyone try to learn calligraphy using left hand?

It it possible. You just need to (re-)learn writing with the paper in a different angle. See this movie: http://youtube.com/watch?v=ga3u9XvCLJc

When I was in a calligraphy workshop I simply made all the strokes from the end to the beginning, for example the i from the bottom to the top.

Hi Isa,

this might be a good starting point: The Elements of Typographic Style Applied to the Web. A List Apart has a number of interesting articles on web typography, too.

The most important thing to know is: everything is relative. Unlike with print, you don’t know nothing; you can’t be 100% sure about the font(s), the size, the available space, the colours … Still, it can be fun, it’s just a different approach. I’d say most of web design is typography.

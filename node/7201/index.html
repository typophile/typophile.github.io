African Text Font

October 18, 2004 • speter

I have a linguistic book project coming up that involves an African language written in the Latin alphabet. I'd love to be able to set the text in an African-designed text font, but all I've been able to locate so far are decorative fonts. (Setting a dictionary in an alphabet based on birds probably won't please too many users!) Any pointers?
I don't know of any text fonts made in Africa*. Except maybe if you include Egypt, but even then I personally don't know one. One option might be to go with something Joshua Darden has designed - although they don't necessarily look African. And really, anything that's strongly African -or of any other landmass, or even any group of people- wouldn't be a good text face. Text is all about subtlety, which does however leave room for some expression. For example if you believe that Africa has preserved its respect for Nature whereas much of the rest of the world has lost it, you might use one of the "fauve" fonts from Eastern Europe or South America. * Look to Saki Mafundikwa for display work. Lastly, there's Gentium by Gaultney. Victor is not African, and there's nothing inherently African about Gentium really (unless you think chirography is fauve - not a implausible idea) but it does happen to be one of the very few typefaces designed to serve African languages, and serve them well. hhp

I understand that many Latinized African languages require combining mark combinations that are not precomposed in Unicode. So another limiting factor (if you want to use an OpenType Unicode encoded font rather than a hack-encoded font) would be support for combining marks - if so 'Doulos SIL' might be worth looking at.

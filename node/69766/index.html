If you aren’t afraid of (or familiar with) Python, take a look at NodeBox. It’s open source and, AFAIK, currently available for OS X and Linux (though the new Beta seems to have a Windows port too).

It takes some time to get familiar with, but there’s already a huge host of NodeBox visualization libraries around.

Edit: Sorry, didn’t see you don’t like programming :) … I’m afraid for maximal control (read: good results) you’ll want some coding skills though. It’s not hard to learn, and there are great tutorials for both Processing and NodeBox. Worth your time. Promise.

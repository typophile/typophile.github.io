I'd disagree. You can use the names for fonts (Drysdale, etc.) provided no existing trademark exists in a similar trade channel. However, you can't use "Beverly Hillbillies" Font Pak or something because BH is a registered trademark in various classes and they'd get you for diluting the mark. bj

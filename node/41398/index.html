Just to be grumpy, it isn’t the PC, it’s Windows. We used the PC for years and set full ligatures – the five f-ligs, plus whatever else was needed. But that was using TeX in a DOS box within Windows.

I believe some of the expert sets included fi and fl as well as the double-f’s. As with Mac, their names in such expert sets are a lie. If you are using one of those Type 1 fonts plus “expert” set, you could set the ligatures with a font change + new character assignment.

I constantly move between applications (in various release versions), but just for vector images (not including bitmaps). Probably the most reliable way is: Freehand to InDesign: Copy using "copy special", with the option "EPS" format ticked. Paste in an Illustrator document / Copy Paste into an InDesign document. InDesign to Freehand: Create a PDF file. Open the PDF file in Illustrator / Copy Paste into Freehand. Please note these important things: the copy/paste settings of the applications should be correctly set (in Illustrator you should choose to preserve paths, I don't recall the option precisely…) Note also copyng and pasting text blocks results in an inconsistent behavior. In general, text objects can't be properly copy-pasted from an application to the other. Also, using the "EPS" format with "copy special" allows to copy vector data from Freehand into FontLab without problems.

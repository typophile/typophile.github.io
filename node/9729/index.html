These are not legitimate forms. Cute though.

There is only one instance in which the 'commaaccent' normally appears above a letter, and that is the lowercase g – where there is no room for the mark below the letter) –, but in that case the accent is rotated 180 degrees. When we were making the Zapfino font for Apple, the first version to include Eastern European and Baltic character support, I adapted this practice to the uppercase G forms, which have a descending swash element that prevents placement of the accent below the letter.

You can design directly in any of the above mentioned programs. TypeTool is slicker than FontForge and will ease you into FontLab, should you desire to go that route. FontLab does have a trial version, so download that and FontForge and see which you like better and which fits with your budget better.

TypeTool is a simplified version of FontLab Studio, so whatever you learn in TypeTool, you’ll be able to apply in FontLab Studio later. Also, the file format (.vfb) is fully compatible.

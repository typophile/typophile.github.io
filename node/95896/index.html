Only to start the discussion as I wasn’t able to find any similar, even loosely. I like [[http://www.myfonts.com/fonts/fountain/nordic-narrow-pro/|Nordic Narrow]] for the first one and [[http://www.ffonts.net/Romeo-Giulietta.font|Romeo Giulietta]] for the second one (even if outlines would need to be cleaned up). For the second one, there’d probably a way by considering it as a mix of fonts. Even if it sounds odd, I would probably start with [[http://www.myfonts.com/fonts/itc/bauhaus/|Bauhaus]] (mainly because of the /S), redrawing almost everything starting from this base. I like also [[http://www.myfonts.com/foundry/TipografiaRamis/|Tipografia Ramis]] fonts (mixing them).

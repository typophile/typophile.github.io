Type on a civil war era Marine recruiting poster

September 7, 2014 • bmagee01

“Prize Money” appears to be Antique Condensed (p. 91, Boston Type Foundry, 1860).

“Able-Bodied, Sober, Intelligent Men” appears to be Gothic No. 2 (p. 89, Boston Type Foundry, 1860).

“Wanted for the U.S. Marine Corps” appears to be Great Primer Condensed or Extra Condensed, (pp 71, 93, Boston Type Foundry, 1860).

“Orderly Sergeant,” “D.M. Cohen…” et. al. appear to be Clarendon or Antique (p. 87, p. 91, Boston Type Foundry, 1860).

Not sure what the text face is.

All sourced from: http://www.archive.org/stream/condensedspecime00bost#page/n49/mode/2up

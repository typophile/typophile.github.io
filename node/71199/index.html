Python search & replace script (for commenting TTX-dump)

June 9, 2010 • Arno Enslin

Does anybody know a Python search & replace script, with the help of which I can comment the output of TTX, the name table for example? It should understand regular expressions and keywords for new lines.

To comment out the name table, you need only replace <name> with <!– <name> and </name> with </name> –>. If fin is the input file and fout the output, then intext=fin.read() fout.write("%s" % intext.replace('<name>', ‘<!– <name>').replace('</name>’, ‘</name> –>')) does the job.

If you have many substitutions and you want to perform them in one pass, then regular expressions may be useful indeed. This link discusses how that can be done.

Michel

Added: you need from string import replace.

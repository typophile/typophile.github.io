January 7, 2008 • dberlow

Using FL 5.0.1, if you don’t mind, and make sure you’ve got your font safe. On a font with a full set of accented latin glyphs, and the circumflex in the glyph window: From the outline layer, (ahem) move the origin or left side bearing, (I know no one is likely to do this intentionally, but pretend you’ve done it by accident). Undo this move. Do your accented characters using circumflex still show bad positioning?

Thanks if you try & let me know.

Cheers!

FLS 5.0.4 Mac: Moving an accent’s left sidebearing does no harm accented glyphs. It’s the undo which does.  :)

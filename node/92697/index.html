This has nothing to do with typography, but … You can change Windows' system icons (this is for XP - later versions should be similar) Right-click on an empty area of your desktop. Select properties. Click the Desktop tab Click the Customize Desktop button. Select the icon you want to change and click on the ‘change icon’ button.

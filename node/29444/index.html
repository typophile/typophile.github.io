I like its character, too. But each letter has a profoundly northeast direction… each letter’s motion goes toward the upper right corner. I thinnk that you’ll have to work on this. Text is a straight, horizontal line (in most cases ;-) ). The letters would work together in a line of text better if each letter would push the eye forward across the line of text, instead of somewhere on top of where the next letter should be. Does this make sense?

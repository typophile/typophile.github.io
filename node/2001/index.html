Those slots you refer to are part of the MacRoman character set. You can access them from the Mac keyboard using with the following keys combinations (U.S. English keyboard):

tilde = shift-option-n
macron = shift-option-comma
breve = shift-option-period
dotaccent = option-h
ring = option-k
cedilla = shift-option-z
hungarianumlaut = shift-option-g
ogonek = shift-option-x
caron = shift-option-t

Other accents can be accessed using the "dead" keys that are normally used to type standard accented characters. But, instead of typing the dead key followed by a letter, you type the dead key twice.

By using negative kerning or other tricks, you can make custom accented characters in standard graphics or page layout programs.

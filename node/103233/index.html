2013-05-24 00:27:44 • eliason
Many casual scripts look awful, but have their own appeal for some purposes. Think of – using the pre-digital names– Choc, Mistral, Signal, Reporter (Gong), Pepita etc. As for the ways that the letters connect in the image this would require OT features that allow the choice of “top” or “middle” joins between charcters. Consider the following using Nick Shinn’s Handsome Pro http://www.myfonts.com/fonts/shinn/handsome-pro/ The letterforms are more rounded than in the image, but you can see how the connections work. Some effects, such as the overlapped th would require special ligatures. Anyway, this design is probably hand lettering or Photoshop. It does not appeal particularly to me as being worth fonting. Don

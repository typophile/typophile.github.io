Summarizing: Compared to Romanized , Jawi Malay is easier-on-the-eye @ smoother-on-the-hand.

  1. Romanized has 52 letterforms (26x2: Capital @ Small). Dropping Dots, Jawi has ONLY (19) dynamic forms as elaborated here: http://typophile.com/files/QB3.3.pdf

  2. Malay is a phonetic language. It is better written (and crossworded) in Jawi. http://typophile.com/files/Silang%20Kata%20Kelopak_0.pdf

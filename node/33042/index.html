April 13, 2007 • rosem

Hey everyone, I'm looking to get some feedback on a symbol/logo I created for a school project. We were given an insect that would have it's own exhibit at the Smithsonian and need to develop an identity for the insect we receive (the Moth in my case) - along with a piece that include the identity. I tried to simplify the Moth into just a few lines, and then went with some type that match the symbol (which happens to be Neutra Display in this case). The name of the exhibit is "Hidden Beauty"... Just looking for some feedback thus far. Thanks, Mike

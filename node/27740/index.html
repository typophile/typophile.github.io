Hi, first-timer here, so bear with me. I attended a lecture by Matthew Carter some time ago where he discussed his method for developing the font Fenway for Sports Illustrated. To satisfy SI's needs, he had to create a font that seemed roomier than Times New Roman, but did not take up any more space. In order to implement this, he used a character chart/map that included a PROPORTIONAL representation of how often each character is used in running text (15 A's, 6 B's, 8 C's, etc.) This allowed him to more accurately determine whether the font was going to take up the right amount of room than typing ABCDEFGHIJKLMNOPQRSTUVWXYZ over and over, as Q, X, and Z are rarely used in running text. At our magazine, we have been using Gill for years, but want to find another face that looks similar without cutting down our word counts. I've been Googling for an hour and don't seem to be searching for the right thing. Is anyone familiar with where I can find such a character list? Matthew
There used to be such a thing as character count charts, back in the grisly old days of copyfitting. These gave you an average width per character per font per point size, for the most common text fonts. I don't know if one is available online but will take a look.

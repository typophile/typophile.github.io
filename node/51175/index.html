If you remove all other Helvetica fonts and install only these 4 font files in Windows: HelveticaNeueLTStd-BdIt.otf HelveticaNeueLTStd-It.otf HelveticaNeueLTStd-Roman.otf HelveticaNeueLTStd-Bd.otf you will get what you need in MS Word. "75 Bold" is the file HelveticaNeueLTStd-Bd.otf The font name shown varies depending on the MS word version.

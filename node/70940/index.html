This looks like a ‘FrankenFont Monster". Whatever you use to re-create this thing, it will take some work, but my suggestion is to start with a somewhat similar narrow sans, such as Diosa Rubia from Nick Curtis. Then find (or draw) some rounded serifs to start grafting them on the sans letters.Some parts of the sans letters will need to be amputated first, like the top arm of the Z. The A is the only letter that is not approximately uniform stroke weight, but you could make that from the left half of the V (as the right leg) and then add a thinner left leg. The C might need a graft from a narrow serif C.

Frankly, it’s a mess, but if you gotta, you gotta, I guess.

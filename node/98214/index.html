Hi,

I am trying to ID the typeface used for the cover of Richard Price’s book CLOCKERS. (The version with red type running vertically on a black background). Any help would be great.

Thanks in advance

All text looks like [[http://www.myfonts.com/fonts/nicksfonts/bindlestiff-nf/|Bindlestiff NF]] by Nick Curtis (Nick’s Fonts, 2011) a revival of [[http://www.flickr.com/photos/myfonts/7316615096/|Binder Style]] designed by Joseph Binder and released by D. Stempel AG in 1959. The |C| looks a bit modified.

I found it using Find my Font - http://www.findmyfont.com

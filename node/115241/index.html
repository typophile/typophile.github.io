Don't know of a formal classification. Rounded Serif/Bulbous Serif works for me. Here is a fairly comprehensive collection of fronts in this category that are known to be legitimate: Brag by Schizotype http://www.myfonts.com/fonts/schizotype/brag-pro/ Cooper Gator Canada Type http://www.myfonts.com/fonts/canadatype/gator/ Goudy Heavyface http://www.myfonts.com/search/goudy+heavyface/fonts/ Henriette http://www.myfonts.com/fonts/typejockeys/henriette/ Check out the unique terminal on the _r_. Now that is rounded! [Black is currently free] Post Oldstyle as ITC Gorilla http://www.myfonts.com/fonts/bitstream/itc-gorilla/ and in a smoothed version as Jamp Post JNL http://www.myfonts.com/fonts/jnlevine/lamp-post/ ITC Souvenir http://www.myfonts.com/fonts/itc/souvenir/ Robur Canada Type http://www.myfonts.com/fonts/canadatype/robur/ Stratford EF http://www.myfonts.com/fonts/ef/stratford/ Verona http://www.myfonts.com/fonts/ef-typeshop/verona/ Westin Black http://www.myfonts.com/fonts/millertype/westin-black/ Windsor http://www.myfonts.com/search/windsor/fonts/ Enjoy!

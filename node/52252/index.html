Oh yeah, and for cache problems, you can use a free app called Font Nuke: http://www.versiontracker.com/dyn/moreinfo/macosx/29757

Or if you use Linotype’s free Font Explorer (font management app, better than Fontbook, looks like iTunes) it has a menu command for clearing font caches. http://www.linotype.com/fontexplorerX

Either way, a reboot is necessary to clear the font caches.

Hmm - if all you were doing is moving the K slider around it shouldn’t create swatches. Mysterious.

So if I understand you correctly you are trying to output grayscale?

If so Select All and then Edit-Edit Colors-Convert to Grayscale.

Another way to check on what swatches you are using is from the Swatch palette flyout menu. Choose Select All Unused and then Delete Swatch.

There should only be a few left, like Registration, Black, White, and perhaps a few other stock ones but all those swatches you listed in the original post should be gone.

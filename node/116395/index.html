As Nick Curtis notes, Mighty Ditey NF is based on a typeface by Richard Nebiolo called Aphrodite for Photo-Lettering Inc. in the early 1970s. Presumably Reisling used the same source. For an interesting view on the license issue for Reisling see http://www.fontsquirrel.com/fonts/Riesling?q=riesling Don

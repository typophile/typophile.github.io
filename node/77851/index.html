@Eliason, Bendy, frode, frank, Stephen: Thanks! The b, n and o where fixed, connections smoothed, and a few ligatures added. The original swash M was left as an alternate only, since it was too "formal" for this "casual" script. It's called "Dancing Script", and version 1.01 was just released under the Open Font License. http://www.impallari.com/dancing/ The source FL files are available in the download .zip Anyone wanting to contribute improvements for the next release is welcome.

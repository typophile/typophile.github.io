While wood and metal type are still used for letterpress, 3D plates can be created digitally (you create a film negative and use it to expose a heat-sensitive polymer plate)--which means that as long as the type isn't too small or too thin, it can be letterpressed. Hard to be sure with the perspective of the original, but Alternate Gothic seems a likely candidate (for both lines of type): http://www.fontshop.com/fonts/family/alternate_gothic/

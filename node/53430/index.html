Generate Opentype .ttf font from vfb ?

January 13, 2009 • RachelR

I have a complete vfb file from which I'm generating Post Script flavored Opentype (otf) fonts. Can I use this vfb file to generate a True Type flavored Opentype font (ttf). Is there anything I need to change in the vfb to generate a proper ttf font. Do I need to change the direction of the paths or change the hinting ?
You will need to change quite a bit, however there is an Action Set that will do a lot of the work for you. You will need to check the TT hinting, to see if FL has screwed anything up. Also check the gray scale settings and the additional blank glyphs required for TT flavor.

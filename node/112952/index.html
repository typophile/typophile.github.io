If I clearly remember (already tried to identify it), this is no type. All these signages for the NYC subway were hand-drawn/hand-painted. A list of font that could be used to reproduce these letterings (mixing them sometimes to recreate original inconsistency): Driver Gothic, Agency, Aldrich, Solano Gothic, Refrigerator Deluxe, Strassenbahn, Raleigh Gothic, Korolev, Tempus Gothic, Nomad, Cougher, DIN Engschrift, Van Condensed, SNV, Cst Berlin and West, Saa, Sign Kit, Inventory, Signal, Highway, Route 66, Outer Loop Jeff Levine has some interesting digitizations of signage fonts.

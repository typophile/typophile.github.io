Nummer 5 has the best ‘colour’, by which I mean it fits best in the overall tone of the page. Number 4 would be okay if you would use a more display-ish version of the typeface (a thinner cut — the serifs are too heavy now and the thin/thick contrats too weak). If you could achieve this, that’s the way to go, imo. The italic/swash versions are too distracting.

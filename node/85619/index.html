Have you considered the case this is not an actual font? It can be easily constructed using 2 different typefaces:

(a) One for L & E (b) Another for O & V (using letter D for the O)

I say so, because if you remove the left part of each glyph, you can read the remaining as LDVE with L,E having a different weight and being a different font from D,V.

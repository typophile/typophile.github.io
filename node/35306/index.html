There’s a kind of uneven quality about the way these letters are formed that suggests to me that this is hand-drawn. The height and size of the l compared to the d is an example of what I mean. (I guess it’s also possible that it’s an amateur font design, but I’d be surprised if a commercial script would be designed to fit together so poorly.) The G is similar to Grocers Script/Philly Sport Script, but the looped l and d are not at all like those fonts.

My guess is that the lettering was modeled after existing type styles, but that it was done by hand.

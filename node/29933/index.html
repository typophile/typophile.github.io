I think the comments make it clear that this is an amateur font, made from someone’s writing, with no attempt to make the connections join. With hand printing, at least you get something that looks like printing on paper. With a script font made this way, you get nothing that even barely resembles real handwriting.

There are much better script handwriting fonts. See Part 9 of the Script Font ID Guide for some samples.

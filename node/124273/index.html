alright, alright - this one is not much of a charm for the eyes (in mere optical terms of course) but hours of searching on auction-sites didn’t bring up anything better. obviously WTF didn’t succeed in suggesting anything remotely close.

examples are from two different generic album-styles (one originally blue on silver/white the other white on black) - i do however think it is the same font.

i pin-pointed to some of the features that might set this font apart: G - missing horizontal bar É - shrunk size Q - tail cutting through a - fits more into serif font scheme W - wide angle 165 - e.g. 5 leaning to the right a bit (i couldn’t find a set of numbers in my fonts that has a consistent look)

(missing samples would be D J X Z)

thx

Reducing the height of capital letters to allow space for accents was a practice followed by some German and Scandinavian foundries. As for the tail on Q you might find something similar by using this link http://www.identifont.com/unusuals?13 As for the tail on a you might find something similar by using this link http://www.identifont.com/unusuals?20 Unfortunately, Identifont did not create a page tagging fonts with a G that has no bar. telling from the logo (not depicted here) i can say that this doesn’t come from any of the european columbia(s) but in fact from columbia phonograph u.s.

i am aware of the identifont tools. in this case unfortunately they were not too helpful for the obvious reasons. this is basically about FONT RECOGNITION - in the sense that somebody has actually come across this font before and remembers; strongest clue being the G

it has to be something no later than ‘38 when columbia u.s. was bought by CBS; and upon researching a bit i can give 1931 for the recording date of the gieseking; the album style (“royal blue”) is 1932-1935!

Based on your update which provides a case that this is indeed an American typeface from the pre-phototypesetting era, I might suggest that somebody go through Mac McGrew’s American Metal Typefaces of the Twentieth Century http://www.amazon.com/American-Typefaces-Twentieth-Century-Revised/dp/0938768395 and do a page by page visual comparison. Not a five minute task. However, if it yields a match, it would give you a complete character set for the font, which could be used to adapt a contemporary digital font. Don

The Brooklyn Dodgers logo is custom lettered, you won't
find anything like it as a digital font. However, I noticed the
T-shirt advertised on that same page uses Brush Script.

There are some scripts that have a swash like that, notably
Letterhead Fonts' Ballpark Script (The Font Formerly Known
As Pilsner Script) and Nick Curtis' free Teamspirit.

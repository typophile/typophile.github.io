Christopher,

The “number” there should be one of the Numeric keypad numbers – and with Num Lock On (or Off – I can never remember which one, but it makes a difference). You can use it with any combination of Shift, Control, Alt, and Command (if you have that key).

It’s the only key combo that ID allows to be used directly for Paragraph and Character styles (and object styles as well).

If you really, really want to use other keys, check the InDesign User to User forums (what Miss Tiff suggested), because you can set any hotkey for a script, and you can easily write a single line script that applies a single style (and that’s already Advanced Stuff).

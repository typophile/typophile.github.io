2007-01-30 22:27:45 • WeAreLoa

Create two rectangles (one black one white). Create a blend between the two. Set the blend to smooth color and to align to path. Create an art brush out of the blend. apply the brush to a circle. Increase the stroke weight until a point is created at the center. Let me know if any of this doesn’t make sense. I was able to recreate your example perfectly.

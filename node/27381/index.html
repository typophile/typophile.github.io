Only when I found this thread did I discover that it's not possible to kern input fields. However, this is the fix I'm using just now: Firstly, make sure: - You have an input TextField created - You have setup a TextFormat object with 'kerning' set to true and 'letterSpacing' set to, for example, -1. Then use the following code: myTextField.onChanged = function():Void { myTextField.setTextFormat( theTextFormat ) } I hope that helps. If I've missed something and it's actually not a fix, please let me know! Edit: I just realised this thread's more than a year old :D

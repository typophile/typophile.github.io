Illustrator CS newly shares some text-handling components with both InDesign and Photoshop, and the main text layout engine with Photoshop.

This represents a complete and total overhaul of basic text functionality in the application. I would be very surprised if it didn't handle kerning pairs from TrueType fonts. I'm a little surprised if Illustrator 9 didn't, but it's possible.

Actually, did you revise the font and add the kerning and replace the already active version? It may be that you managed to confuse Illustrator. Try quitting all Adobe apps, deactivate the font if you're using a font manager, and then delete and "adobefnt.lst" files from the same directory as where the font is being activated from. Then reactivate the font and relaunch Illustrator.

These typefaces have glyph sets that support Cyrillic: http://www.myfonts.com/fonts/paratype/peter-skoropis/ http://www.myfonts.com/fonts/nikolai-sirotkin/taumfel/ http://www.myfonts.com/fonts/paratype/federico/ http://www.myfonts.com/fonts/ihof/p22-dearest/ [Also supports Greek] http://www.myfonts.com/fonts/glc/1695-captain-flint/ http://www.myfonts.com/fonts/paratype/pushkin/

Only found one other handwriting typeface that could pass for 18th century that supports Greek: http://www.myfonts.com/fonts/ihof/p22-operina/ [Greek only supported in Operina Pro; also supports Cyrillic]

Other forum members will likely be able to suggest others….

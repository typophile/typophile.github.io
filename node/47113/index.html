Well, here is how I would do it:

— Create a new image in Photoshop with a white background and drag in the picture to which you want to apply the effect.

— Duplicate the layer containing your chosen picture and set the opacity of the bottom one to about 10%

— Create a layer of text that covers your subject matter, filling it with whatever text you please.

— Right-click (or control-click) on the text layer in the Layers Palette and select “Create Work Path”.

— Select the top picture layer (the one at 100% opacity) and then go to the menu bar and choose Layer>Vector Mask>Current Path

— Hide the text layer by clicking the little eye next to the “T” in the Layers Palette and you’re done! (Just click on the background layer to get rid of the outline of the vector mask)

Like someone said there can be many ways to develop such ASCII portraits. Depends on what kind of results you want.

Method one: http://asciiconvert.com/ http://www.ascii-art-generator.com/ http://www.altarsoft.com/ascii_art_maker.shtml http://www.network-science.de/ascii/

Method two: Use Illustrator’s spray tool and experiment.

Method three: I guess this is how the sample image you have posted was been produced.

Create two layers in the the Photoshop. The top layer should be the text layer and the bottom one should have the image you want to apply the effect. Make sure that your text is set either in white or Black. Now select the top(text) layer and keep playing with the various layers/transperency modes. Shortcut for this: “Shift” + “+”.

I made something like this once before.

http://designrelated.com/portfolio/darintenb/page/1/entry/8375

All I did was put all the text in a layer in Photoshop with the original photo in another layer. Right click on the layer with the text and click select pixels. Then invert the selection and change to the layer with the photo. Hit delete and all the pixels that aren’t the text will be erased leaving only the portions of the photo where there was text. Hide the original text layer and presto, you’ve got what I did.

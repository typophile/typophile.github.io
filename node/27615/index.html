More probably the 'Shear' function was used in Illustrator, which is a better application for playing around with lettering. It allows you to specify in degrees how much you would like to slant the lettering. If you hang out here long you will see all kinds of graphic manipulation of type, with one of the most popular being one we like to call 'Squooshing'. I think it was Tiffany who coined that one. Sometimes the graphic power of some software tools is irresistible, much to the detriment of the type being used, but often it's just in the eye of the beholder. (You can tell from this remark that I am not above graphic manipulation myself ;) ) - Mike Yanega

Adobe Illustrator was first produced in 1987 and currently it’s latest incarnation is CS4 (v14). It’s main function is for producing vector based graphics/designs. It’s native file format is .ai but also can output .eps and pdfs. Newer versions can export most mainstream bitmap file formats. The key benefit of vector drawings is that they can be scaled from a fraction of an inch in size up to a few feet across with no loss of quality.

for further information see http://www.adobe.com/products/illustrator/main.html

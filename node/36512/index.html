Hyphenation and spelling for Russian is not officially supported in InDesign CS, only in CS2 and CS3.

However, if you own Adobe Acrobat 6.0 or newer, you can copy all files from C:\Program Files\Common Files\Adobe\Linguistics\Providers\Proximity (Windows) or Library/Application Support/Adobe/Linguistics/Providers/Proximity (Mac OS X) into C:\Program Files\Adobe\InDesign\Plug-Ins\Dictionaries\Proximity (Windows) or Applications/Adobe InDesign CS/Plug-ins/Dictionaries/Proximity (Mac OS X).

This way, you’ll be able to use spelling and hyphenation in Russian, Polish and a few other languages in InDesign CS.

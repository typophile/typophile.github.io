2003-12-29 21:23:56 • bowfinpw
Here's a scan from the Jaspert book of Greco (or Bristol) from a Gans 1925 typeface. You can see the A and C are like the sample, and the lower case matches as well, in as many common characters as are shown. Unfortunately this sample doesn't show the entire character set, but I think it shows enough to make the case. Note that the stroke of the right leg of the A is simpler than the Agfa-Monotype 'Greco' from the link I gave previously. Apparently the newer Monotype/Linotype versions exaggerated that feature, or else developed a solid version of the Adornado face that is not the same as Greco Bold. (The Linotype version was called Vulcan, but looks almost identical.)
2003-12-29 22:16:17 • bowfinpw
It is Greco Bold, but the e's have been tilted slightly. I know this for a fact because I have worked with the original artwork files for the logo and happen to know the designer who did it. The artwork, incidentally, was created by scanning from a type specimen book, not with an electronic version of the font.

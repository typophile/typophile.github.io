There's a pretty good reference chart at http://www.typotheque.com/fonts/opentype_feature_support showing which apps support which OpenType features. It's slightly out of date in that it doesn't list Adobe CS4 apps, but it is still quite useful and reasonably accurate. TextEdit is not listed, but my own testing suggests that the 'mark' feature is not supported there, so it's probably not a great test app. Mellel (2.2+) does support 'mark' as well as 'mkmk' so may give better results, but actually Microsoft's NotePad app is the best place to test VOLT work, outside of VOLT. As to your specific issue: could you post a screen capture of the VOLT Script -> LangSys -> Feature -> Lookup tree (the leftmost pane of the main VOLT window)? Perhaps there's something awry there.

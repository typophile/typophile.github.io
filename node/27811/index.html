Hello:

This bug (incorrect artwork for ‘k’ in FreightText-BookItalic and FreightText-BoldItalic) was identified shortly after the release of Freight, and new versions were generated. Evidently, some distributors have implemented the update less swiftly than others.

I’ve responded to Mili directly on this, but if anyone else should encounter the same problem, I would like to hear about it. Thanks!

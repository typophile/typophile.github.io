Hi Tilman, I'm afraid that my specimen PDF is not faulty :-P The idea is that the user can combine two Alternate/Style setting to access the variants in this typeface. Alternates / Style 1: By choosing Stylistic Alternates (in Illustrator for example) or Stylistic Set 1 (in InDesign) you’ll get round punctuation marks and alternates of "I", "a", "b", "g", "i", and "l". Style 2: Only choosing Stylistic Set 2 will give you access to an alternate "a" as well as "y", upright italic designs if you will. Alternates / Style 1 + Style 2: Combining both styles will give you the same alternates as in Style 1 but with (humanist) upright italics for "a", "g" and "y".

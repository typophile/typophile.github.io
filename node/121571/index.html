Middle Eastern style hotstamp on poker chips

November 25, 2014 • mipevi

Hi, These are home poker chips that were designed about ten years ago, with the artwork stamped on them in foil. I've gotten permission to recreate the artwork for my personal use, but the owner no longer has any art files. So while this post is mainly for the top font "CASBAH CLUB", I'm also interested in what was used for the denominations (if different) and for "LAS VEGAS" at the bottom. I'll of course create new requests for them if necessary. Huge thanks for any help you can give me! Cheers, Mikko [img:sites/default/files/old-images/casbah club_6485.JPG]
A quick google run-through of "faux Arabic" and possible names like Sheik, Caravan, Aladdin, genie, magic lamp, Sultan, Shriner etc didn't produce any thing even close. If you only need the eight letters it's perhaps as easy to make them from scratch as to search for an existing font. But possibly someone else has found something. Don

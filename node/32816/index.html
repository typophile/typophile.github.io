Intercut experimental wood type system

April 4, 2007 • Nick Sherman

Before graduating from MassArt last May, I spent a semester (and then some) working on an experimental wood typeface design / production project. The project was recently selected to be featured in a Contemporary Letterpress exhibition, so I figured it would be a good time to put all my process together for others to see. You can read more than you probably ever would want to at nicksherman.com/design/Intercut.

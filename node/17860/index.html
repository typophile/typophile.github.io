I've done my diploma thesis on this topic, analysing two fonts in detail: RialtoDF and Syntax. Each font got a small presentation book (~50-60 pages), analysing formal aspects, influences like zeitgeist, technique, font designer and purpose of use. I chose these two fonts because they are quite different (sans serif / serif) but have lots of things in common (humanist ductus, ...). The theoretical part describes these things in a more common way.

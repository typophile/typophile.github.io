This is another good example of a pseudo-blackletter. It uses features of certain blackletter typefaces, but it simplifies and modifies letters for legibility, and manufacturability, and also for style. For example the 't' is not like the 't' in almost any blackletter type I can find, but all of us see it as a clear 't' and it keeps the baseline clean (and flat), which is also not like most blackletter types. This only resembles blackletter. - Mike Yanega

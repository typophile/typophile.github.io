body: "<strong>[[Indices]] : [[Terminology]] : End User License Agreement (EULA)</strong>\r\n\r\nA
  legal document that outlines the licensing terms for a particular software. These
  terms usually list the manners in which the software may or may not be used. In
  regard to fonts, there are several common topics that [[foundries]] must consider
  allowing or disallowing. Among these are: Number of Output devices covered by license,
  server installation, back-up copies, embedding, distribution to a service bureau,
  modifications, transfer of license, &c.\r\n\r\nFor a detailed understanding of the
  issues surrounding EULAs for fonts, read [[Tiffany Wardle]]'s article <a href=\"http://fontlab.wikidev.net/The_Case_for_a_User-Friendly_EULA\"><em>The
  Case for a User Friendly EULA</a></em>.\r\n[[EULA chart]]\r\n\r\n<strong>Discussion:</strong>\r\nFor

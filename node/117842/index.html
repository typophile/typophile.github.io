Augmented Reality font

July 9, 2014 • Coen

Hi,

I’ve designed a font in one of my classes at design school, the idea was to have the font feel as if it had been folded out of a sheet. And then I also made tehe font from a sheet, folded it and fotographed it so I had a Clean and a real version.

Now I am working with a fiend to make the letters in 3D and stick it in an Augmented Reality app so that you can see how the font is only realy visible from one perspective.

It’s still work in progress but I was wonderin how you would react, if we could get any tips, maybe where to drop this idea or which type of people might be interested in this so we have an idea wehere to take this project.

Thanks!

Coen.

2007-08-16 20:58:06 • Paul Cutler
I started out with just two typefaces (purchased ones, that is, not the freebies that came with the computer operating system): Garamond and Myriad. Then I added gradually: workhorse text fonts that I use all the time, project-specific fonts (usually display fonts) that are archived and might never be used again, and beautiful typefaces that I didn't really need but couldn't resist buying because they happened to be on sale you never know, they might be perfect for a project one day. I don't use a font manager. I manually add/remove fonts from the active font folder as necessary. Generally have only a couple of dozen fonts active, mostly text fonts (as my job is typesetting books). I have pruned the odd font that proved disappointing. Although I try to buy only from reputable foundries, I've been burned a couple of times with supposedly pro-quality fonts that were badly kerned or, as I've discovered now that I purhased Typetool, have outlines that resemble bad autotraces that were never cleaned up. Of course I steer clear of those folks now. I don't think I could ever have too many typefaces.

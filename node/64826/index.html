Is there some reason that it must be this exact hand-printing font? If not, there are literally hundreds, if not thousands, of fonts done by amateurs. You can see a large sample of them included in [[http://www.bowfinprintworks.com/Script11Pg1.html|Part 11 of my Script Font ID Guide]]. Five pretty similar fonts shown in the first 5 or 6 pages are Valley HW, Wolf HW, Jeff HW, Beatty HW and Basha HW. All of these come from the ClickArt Handwritten Fonts collection which is now sold by Broderbund as part of their newest ClickArt Font collection (should mention ‘300 handwritten fonts’).

You can also search Dafont.com for handwriting fonts and find many free ones.

(I did look through hundreds of samples before deciding this was a better way to answer this question.)

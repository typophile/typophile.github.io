I don’t know the name of the font, but the OSX application ArtText has a gear in its library of vector shapes. It’s in the Symbols category.

It’s easy to create a gear in a drawing program with oblongs, circles and path operations such as Illustrator’s Pathfinder filters.

Draw an oblong, duplicate, select all and combine to form a cross. Duplicate the cross, revolve by 45°, select all and combine. This creates an 8-pointed star. Draw a circle, place on top of the ‘star’ shape, select all and subtract.

2006-05-10 11:09:42 • SuperUltraFabulous

…and besides there is no Gill Serif to go with it.

Actually, there sort of is a “Gill Serif.” Joanna, which is designed by Eric Gill is often used as Gill Sans' serifed counterpart.

You might also find that Gill Sans/Joanna have much in common with Scala Sans/Scala. From what I understand, Martin Majoor closely studied Gill’s designs when creating the Scala family.

As for Gill Sans being overused, I don’t think anyone can justly fault you if you use it well.

2008-10-23 12:56:29 • Jos Buivenga

It also depends on which languages your font should support. E.g. Polish uses only Aa and Ee with ogonek. But then you also need to draw the other special characters for Polish.

Lithuanian uses Aa, Ee, Ii and Uu with ogonek, but likewise to support Lithuanian, you also would need c with acute, c with caron, e with dot, s with caron, u with macron and z with caron.

I don’t know which language uses the o with ogonek and the o with ogonek and macron.

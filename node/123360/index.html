Open the font in Type light (freeware): http://cr8software.net/typelight.html (Windows) http://cr8software.net/typex.html (Mac /Linux) Go to font options on the font menu. If the hinting box is ticked then it contains some form of hinting. To remove all hinting: untick the box and save the font.

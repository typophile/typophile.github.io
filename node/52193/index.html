If this is STILL on his grave, that would be a disgrace. This is a temporary marker, placed prior to the final marker. And it is done in a generic typeface used on (plastic) engraving machines (think pantograph and stencil-like masters).

A modern equivalent would be Interstate by Font Bureau.

. . . Bert Vanderveen BNO

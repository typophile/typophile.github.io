2010-05-23 16:44:51 • Florian Hardwig
For nice calligraphic Fraktur typefaces, check out those by Julius de Goede. More versatile, with alternates: [[František Štorm]]’s Monarchia. ‘snapshot20100520191132’ is a style that was popular in early 20th century German commercial lettering, see the works of Louis Oppenheim, Lucian Bernhard and others. ‘snapshot20100520192936’ is some sort of Bastarda. Very few options for that style in typeland, you could try St. Augustin Civilité. ‘snapshot20100520185851’ is a Faux Arabic. FF Falafel is a bit similar. See also Legende. ‘snapshot20100520193709’ obviously emulates Asian calligraphy, as Riccardo noted. Try searching for ‘Faux Chinese’.

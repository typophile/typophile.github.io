AFDKO Compare Family ?

April 15, 2008 • RachelR

I have downloaded and installed the Adobe AFDKO kit, everything works fine, but I can’t see the Compare Family option. Should this be there or have I done something wrong on the installation.

You will not find it in FLS. You need to use the Terminal.

First, you should place all fonts which you want to test in a folder. Then you open the Terminal and type    comparefamily -d [a space here] and drag/drop the font folder onto the Terminal so its path will be added to the command. Hit return. For more options, see the font production and font testing PDFs linked from the bottom of the wiki entry (especially p.8 of the font testing PDF).

Btw, with AFDKO you have installed TTX too.  ;-) Use it like this:    ttx [drag/drop font files here] and hit return. More details with    ttx -h and with    ttx -u

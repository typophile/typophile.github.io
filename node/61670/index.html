Since I designed Mommie you can see a lot of script typefaces with big contrast for big sizes. With Narziss I created a roman interpretation with the very expressive Swirls version. Glenda is again an italic script like Mommie. In the Regular the characters join and it looks like a neatly written sprencerian handwriting. But like Narziss it has got a Swirls version too. Every glyph and swirl is carefully designed to work in every connection.

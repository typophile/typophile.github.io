Hi, Is it a typecrime to use the same face on letterhead text as you used for logo? Meaning address, phone, website copy. Any recommendations on current fav book on typography? Thanks in advance.
I'd say that it rather depends on what type the logo is set in. If your logo is say ...yawn... Helvetica, you are probably ok to set the other text in it. Anything with a bit more character may lead you into criminality. In general though, it's probably best to find a complimentary face to set the other bits in. If it's the typeface that the actual letter writers will eventually set their words in you'll possibly achieve even greater harmony. Or, at the very least you won't have three typefaces on the one document. What typeface is the logo set in?

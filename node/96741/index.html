2012-09-28 23:21:01 • Andrew Osman

There is the main Type ID board, where you post ID requests: http://www.typophile.com/typeid

When an ID is solved, a moderator moves it to the Solved boards: http://www.typophile.com/typeid/solved

Sometimes, people accidentally post there requests on the Solved boards instead of the main Type ID board. It’s actually a very common mistake. Especially for new users. When someone is looking to identify a typeface for someone they go to the main ID board to review posts. They do not go to the Solved boards. If you post a request on the Solved board, it is unlikely that anyone will ever see it, and you won’t get any help.

Long/short, if you want someone to identify a typeface for you, don’t post on the Solved boards, post on the main Type ID board.

Does this clarify?

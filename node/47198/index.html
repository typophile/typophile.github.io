2008-07-08 22:59:26 • Stephen Rapp

Hi Goran,

Why don’t you try the following script:

from robofab.world import CurrentFont

f = CurrentFont() for g in f: g.note = "" g.update()

It should blank all notes. (note that you will have to have Robofab installed).

Cheers.

You may like the ‘More fonts like this’ feature at MyFonts.com. You can also examine the tags that were assigned to Handel Gothic. Note that you can combine several tags in a search.

Furthermore, there are lists that feature stylistically similar typefaces, like Mike Yanega’s Bauhaus-Style Guide or FontShop’s Star Trek FontList.

Another way that often is promising: search Typophile for terms like ‘similar to Handel’.

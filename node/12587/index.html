[[Indices]] : [[Tools]] : [[Type Design Tools]] : Visual TrueType (VTT)

Expert’s font tool from Microsoft for dealing with the low level software language of editing [[hinting]] information for [[TrueType]] fonts, and [[OpenType]] fonts using TrueType outlines (Quadratic Bezier) for the glyphs. The program itself is friendly enough, but [[TrueType hints]] are encoded in their own stack-based programming language – as complicated as the language used to describe font outlines themselves – and the GUI can go only so far to mask the complexity and power of the hints.

Originally designed by the architect of the TrueType font format, Sampo Kaasila. Released as ‘TypeMan’ for the Macintosh. Completely rewritten by Beat Stamm of Microsoft Typography and released for both Windows and the Macintosh.

Available from Microsoft Typography upon completion of an application. Both Windows and Mac OS are supported.

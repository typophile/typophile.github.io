Vince, have you rebooted your Mac since the initial installation of the update? Sometimes a second reboot will force all of the automatic system-level maintenance scripts to run and do a bunch of stuff that needs to happen on a regular basis. It really sounds like you need to run a system utility like Cocktail or OnyX to clean your application and system cache files and repair permissions. Something has caused some file access problems with a few of your applications. You shouldn't need to reinstall anything. If you are unable to download and install a new program like Cocktail, then boot your Mac off the original OS X CD that came with it and run the Disk Utility from the CD. It will allow you to repair permissions at least.

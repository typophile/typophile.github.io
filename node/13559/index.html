[[Indicies]] : [[Typefaces]] : Gotika

1932–

[[Imre Reiner]] designed this typeface for the [[Klingspor]] foundry in Germany.

[[Peter Bain]] and [[Paul Shaw]] have this to say about Gotika: It is both highly stylized and simplified. It could be characterized as an Art Deco interpretation of a [[textura]].

[[Paul Renner]] recommended using Gotika as a headline face for text set in French Didones. Wow.

There is currently no commercial digital revival of this typeface.

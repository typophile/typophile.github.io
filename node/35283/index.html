The TTF does not contain multiple sizes. The “1000 x 1000” is not a number of pixels, but of “em units” which do not have a fixed size. When you produce text at a certain size, it resizes the letters in the font to be the right size.

You could make a font where the Roman and Thai appear at different sizes, but you would need permission from the designer/publisher of the font, and all your users would need to download and install the font.

I can see one more problem: If you increase the size of some glyphs (letters), then you will have to increase the size of the em, which increases the line-spacing. So lines without any Thai letters will look very widely separated.

Otherwise, it should be pretty simple.

Once you have permission from the owner of the font (the designer or publisher) — alternatively use an open-source font, get a font-editing program. I would suggest FontForge (http://fontforge.sf.net) if you’re using Linux or Mac (some people here don’t like it, but this is an easy enough job, and I’m a FontForge fan anyway). Using FontForge on windows is a bit more difficult.

If you choose FontForge:

Start FontForge. Open the font you want to edit. Select the glyphs you want to resize. Choose Element->Transformations->Transform… Set the origin to “Glyph Origin”. In the drop-down box, change “Move” to “Scale Uniformly”, and adjust the number to, say, 200%. Click OK.

Choose Element->Font Info->General. Adjust the “Em Size” by the same factor as you scaled the glyphs (if you chose 200%, double the em size, etc.). Click OK.

Choose Generate->Generate Fonts, change things as you see fit, and click Save.

I hope that helps.

Being allowed to use the font (and even distribute it) does NOT necessarily mean you are allowed to edit it. When you got the font, you should have got a licence with it (if you didn’t, it’s probably pirated in the first place).

But there are open source Thai fonts. These allow you to use, distribute and modify freely (they may or may not require you to pass on these rights on to the people you give/sell the font to). The Thai Linux Working Group and NECTEC has made several fonts (apparently of good quality) under the GPL (open source; you have to distribute modifications under the same licence). You can get them at http://www.ctan.org/tex-archive/fonts/thai/thaifonts-scalable/nectec/ and http://www.ctan.org/tex-archive/fonts/thai/thaifonts-scalable/tlwg/ (these sites offer “.sfd” files, which can only be opened in FontForge).

Yes, the difficulties of Thai fonts affect type designers too, albeit in a different way. However, you should not be affected, as the font you choose should already have this done (if it is of decent quality).

It’s a quirk of Adobe fonts, a little bit of branding, no doubt. The OpenType feature spec allows that it’s up to the foundry publishing a font to decide whether a ligature in it should be on by default (liga), or discretionary (dlig).

http://www.microsoft.com/typography/otspec/features_ae.htm

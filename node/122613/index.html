http://www.printershowcase.com/ Read the specs, decide which lasers you want to see samples from, then order a Test Drive from them for $19.95. You provide the test files of your choice.

I did that when I bought my color laser and was really surprised at how bad text looks on some of the big-name brands you might think would be the best. I ended up buying a Ricoh because the quality on 12-point type was excellent and it has true Postscript 3. The Ricoh black-and-white lasers use a Postscript emulator. Doing a Test Drive saved me from making an expensive mistake.

What is a naked email address? Is this a problem? How do I fix this?

Christopher, I believe I tried to warn you about this in an older post of yours. By naked e-mail address, Ben means that you are using the e-mail address, as is, on a website (this one), where it can be picked up by a spambot. A way of avoiding this is by typing it indirectly. For example, like this:

typographer [at] gmail [dot] com

or like this:

typographer-at-gmail-dot-com

The idea behind this is that thinking humans will get it, but unthinking spambots won’t. :-)

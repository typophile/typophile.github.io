500 (from the Michigan Spartans University): see [[http://www.typophile.com/node/99339|here]] for a previous discussion on this topic (confirming Machine ITC). More alternatives: [[http://lineto.com/The+Fonts/Font+Categories/Headline+Fonts/Hoboken+High/Sans+Serif/|Hoboken Sans]], [[http://www.myfonts.com/fonts/ingrimayne/masheen/|Masheen]], [[http://www.myfonts.com/fonts/jnlevine/base-runner/|Base Runner]], [[http://www.myfonts.com/fonts/typodermic/octin-college/|Octin College]], [[http://www.myfonts.com/fonts/itc/superstar/|Superstar]], [[http://www.myfonts.com/fonts/k-type/nyc/|NYC]] UNDER ARMOUR: to me that looks to be based on [[http://www.myfonts.com/fonts/urw/eurostile/|Eurostile Extended]]/[[http://www.myfonts.com/fonts/urw/microgramma/|Microgramma]]. Alternatives: [[http://www.myfonts.com/fonts/linotype/eurostile-candy/|Eurostile Candy]], [[http://www.myfonts.com/fonts/naghachian/aban/|Alban]] (another rounded Eurostile), [[http://www.myfonts.com/fonts/typodermic/reversal/|Reversal]] (another Eurostile clone), [[http://www.myfonts.com/fonts/dsgnhaus/saturn/|Saturn]] (one more Eurostile), [[http://www.myfonts.com/fonts/fw-identikal/reaction/|Reaction]], [[http://www.myfonts.com/fonts/fw-identikal/phantom/|Phantom]], [[http://www.myfonts.com/fonts/fw-identikal/stak/|Stak]], [[http://www.myfonts.com/fonts/cheapprofonts/geometry-soft-pro/light-c/|Geometry Soft Pro Light C]], [[http://www.myfonts.com/fonts/canadatype/miedinger/|Miedinger]], [[http://www.myfonts.com/fonts/fw-identikal/phat/|Phat]]

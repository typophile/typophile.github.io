Poetica stylistic set 6 gets you an F that is close enough to easily modify, and swash gets you the h:

Pouty might work: http://www.myfonts.com/fonts/fontbureau/pouty/

Medici Script could be altered (F): http://www.myfonts.com/fonts/adobe/medici-script/

Same with Le Griffe: http://www.myfonts.com/fonts/linotype/le-griffe-2/

But if your client is insistent on Captain Quill, just smoother, then Nick’s suggestion is the only way to go. Use the smoothing tool on an outline version of the glyphs.

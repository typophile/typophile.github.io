You could go for Linotype’s Font Explorer, see the righthand side for download for PC. Tim, that PC-Download is [[FontExplorer]] not [[FontExplorer X]]. FontExplorer is just a tool to view and search for Linotype fonts. "FontExplorer X" is the a font management tool from Linotype, which will additionally let customers easily purchase and install fonts from Linotype and other foundries on their computers. Currently, it is only available as a beta-program for Mac OS X. But the full version for Mac, and a public beta version for Windows, will be available soon. So, unfortunately, PC users aren't able to manage their fonts with FontExplorer X just yet. Sorry for the confusion.

For a Windows font manager, can I suggest Corel's BitStream Font Navigator. It comes as part of CorelDRAW Graphics Suite 12. However, a fully-functional version is included in the trial download of CorelDRAW Grahics Suite 12. Simply download the trial version (you will need to sign up to Corel's website, if you have not already). When installing, use a custom installation - choosing only to install BitStream Font Navigator (it is in the Utilities section). It is free, easy to use, quick, and has pretty much all the necessary features.

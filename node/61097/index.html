typoaddict>Is there any program or procedure that provide me info on font originality?

We are working on a standard to provide you with a wealth of font information including that the original designer, or owner of the design, has permitted the font to be committed to the enclosed format, that the founder manufactured it with this permission and to whom the font was licensed. All of this would be signed, so you could know that it was the original and authentic version of this information, in the original font, or its approved derivative.

In addition, this standard would allow for the original designer to provide recommendations for the best uses of the font.

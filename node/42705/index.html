Sheesh. I’m sure there is an excel help group on the web, but I’d rather ask you guys first.

I was giving a list of names, addresses, and birth dates. They are organized by last name and I need them organized by birth date. Is this possible and is the way to do it understandable?

Assuming the birth dates are in a column by themselves, yes, it is quite easy.

Highlight all the rows you need to include (that would be the entire document, then, excluding headers and footers). Go to Data -> Sort, and choose the column letter (or column name, if it’s been given a proper header) that holds the Birth Dates from the drop-down menu under “Sort by”.

If you wish, you can then add a secondary parametre (and even a tertiary), by choosing one of the remaining columns—for instance Last Name, and then First Name—from the two “Then by” drop-down menus underneath. This will make sure that, in the instance there are more than one person born on the same day in your sheet, they will be sorted alphabetically according to their last name; and if their last names happen to be the same, too, they will be sorted alphabetically according to their first name.

This applies to the Mac OS X version (more specifically Office 2004). I’m not sure if the menus are called the same in the Windows version—and if you have the newest version of Office (Office X or whatever it’s called), then I’ll have to defer to others, since I can’t remember which of those horrendous pseudo-tabs Microsoft have decided to hide the sorting function under.

That might be tougher. I’m no Excel guru, but this might work:

Select the column that contains the birth dates. Right-click anywhere in the selection and choose “Format Cells”. In the “Number” tab, choose “Date” in the list at the left, and then choose the layout you want (if you want to sort by month, the “Mar-01” and “March-01” options would be your best bet) and click OK.

If you sort by this column again, it should sort everything according to month (and for multiple instances of the same month, according to year, since it’s still just sorting alphabetically). I’m not 100 per cent sure this will work, though, since the layout of the cell doesn’t change the underlying content. Also, it requires well-formed, numeric dates, so if the dates are written as “15 Mar 1976” or something along those lines, it will probably just turn all the cells into random gibberish numbers.

Add another column next to the one with the birthdates, and fill each cell with this:

=text(B2,“mm/dd”)

where B2 is the address of the nextdoor cell with the date in it. That will put the text of the birthdate month/date in the cell.

Then sort (per Oisin’s instructions) on that new column.

>If I (usually working on Mac) want to check what screen type (like on a website) looks like on Windows, is it fine/enough to run Windows in emulation Not at all stupid, I did the same sort of double take when I did this. I then took my schizoid mac to a second, windows-only, windows machine to confirm, granted through the lens of a different screen, that it is the OS causing the rendering differences. So, the answer is that what you're seeing looks like windows running on the same kind of screen as the Mac. And all the rendering differences that remain are due to "system architecture etc.", and also preferences, as I understand your question. So you could also look at your site with all kinds of preference changes to see what that does to your site(s). Cheers!

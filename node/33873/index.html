Global Type's database online

May 18, 2007 • Lars Kaehler

Hello! We would like to announce that Global Type's database is online now. Based on the typeface collection of Hans Reichardt, Frankfurt am Main, who has been collecting typefaces for over 40 years, you can search in 91,076 typefaces, 2348 type designers or punchcutters and 1047 foundries. For the first time there is an overview about the typefaces of the 20th century, almost complete by name. But – that is to say – we only have a few samples by now (2749 pictures), and they do all come from the German type specimen book “Das Handbuch der Schriftarten” (Seemann, 1926). We have scanned all the images, written down all the information in the left row, so that all the information of this book is available online now with some restrictions. Please don't misunderstand: We don't want to compete with the existing commercial solutions, and our database is intended more to give help for research questions. The existing content is protected by now, but later you will be able to log in and expand the collection. The input will be inspected and released afterwards. Have fun! Lars Kähler http://www.global-type.org/Searching.743.0.html

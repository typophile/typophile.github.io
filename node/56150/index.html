It's Caslon Antique, but none of the digital versions ( here ) matches exactly. I have a Barnhart Brothers & Spindler catalog from the 1920s that shows a 48 pt. cut (metal foundry type) that matches quite closely, although the spacing is not as tight. The digital versions available appear to be based on smaller point sizes of the original metal face. Each size was slightly different. [img:sites/default/files/old-images/caslonantique-bbs_5832.jpg]

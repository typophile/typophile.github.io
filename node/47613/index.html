Classic Typefaces Used in a Contemporary Context

July 22, 2008 • Miss Tiffany

It might not be a popular sentiment here on Typophile, but in today’s world of one billion fonts many designers are still opting to use tried and true typefaces. Whether in a traditional book setting or reinvented magazine spread, these typefaces will never go out of fashion. I’m doing research for a book about classic (and some new classic) typefaces used in contemporary design. So if you have you done work — or know of good work — using one of the fonts listed below then please contact me or post a link below.

I thought it looked familiar… it’s Georgia, squooshed. (Not the “K” logo thing – the rest.) In the KNICKERBOCKER GROUP those are fake small caps, which I was able to reproduce with Georgia Bold caps, tracked out by 270, horizontally-scaled to 85%. I’m done abusing type for tonight but I’m pretty sure the rest is Georgia too.

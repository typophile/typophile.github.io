2005-06-13 20:03:48 • Miss Tiffany

InDesign can only open Quark 3 and 4 files (CS1, I don’t know about 2 yet).

If you use Quark 5 you have to save down to 4. I haven’t used Quark in long time, but I think if you use version 6 you have to save to version 5, then open up Quark 5 to get to 4. They have special algorithms and such that don’t let InDesign or other apps read the files.

On a brighter note, the road ahead looks great for InDesign :)

2004-11-30 06:15:51 • dan_reynolds
Copying and pasting from Illustrator CS works fine with FontLab 4.6 and MacOSX. If it's not working, you need to check the preferences in Illustrator CS.

Select "Illustrator CS > Preferences > File Handling & Clipboard..." . Make sure "AICB (no transparency support)" and "Perserve Paths" are enabled. Now it should work as it did before for you. (Note: AICB is disabled by default when you first install Illustrator CS.)

The problem with Macromedia's Fontographer is thay it's so old it can only import .eps files that are level 1. Since all the latest versions of Illustrator only convert to level 2 or 3, you need a workaround.

Copy your path from Illustrator to Photoshop and export as Illustrator path. This Photoshop export is apparently Level 1 because I can import these .eps files without a problem.

Hopefully this works for you.

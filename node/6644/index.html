If you read my previews text, it have all the info you need. ;-) The Italic thing is ok, but don't use single quotes (out of the double quotes content) in a professional text. If you find it ugly, guillemets are quite good in German too and its common for your Swiss audience. But note the difference in the directions for the German usage of guillemets. ;-) If you have to deal with other languages too, you should prefer the guillemets. BTW. The Duden is a German only book. An english version like the Oxford-Duden German Dictionary should you serve better - but its only a thought.

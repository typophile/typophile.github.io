Stephen, I have the Microsoft True Type Mac collection that included it. The booklet gives fairly good details about the origin. The lower case was designed to complement the original typeface which was apparently only upper case. There are a number of adaptations of the Gradl design (shown in the Dover book by Ludwig Petzendorfer): Swaak Centennial (Mecanorma), Maigret (Panache); Reynolds Caps (Intecsas/Rakowski), Santini (Canada Type); Sarah Caps (Sam Wang); and Pigalle (Solotype Catalog) - Mike Yanega

Steve as you probably know, I use our software (Find my Font) to find the fonts. The letter selection function of FmF works like a flood-fill operation - actually: a flood-copy operation, which copies the letter you click on, and works the same on black or white color letters without any need to pre-process the original image, the only requirement is to have a different color boundary.

See in the image bellow how the application screen will look like when you select the last letter (all selected letters are transferred in the lower left panel with black color, and then you type the letters and hit the match button).

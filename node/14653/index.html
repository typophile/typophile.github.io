2005-08-17 18:55:32 • hrant
> Wrestling: I assume you mean the lucha libre stuff, not Greco-Roman. In which case you might consider Motter Corpus. I think Lithos would work in both cases. But, seriously if you're looking to include the font or a derivative in your game (say for UI use) you'll need to work out licensing rights with the font owner. I've not seen a standard off-the-shelf EULA that allows distribution of fonts or derivatives with games. If the font is going to be used to generate a static bitmap logo then almost any font will do - almost all EULAs allow you to make complete logos and redistribute them in bitmap form with products like games. More info on the type of game may help us guide you towards some good font choices. Are you looking for fonts with rivets? Cheers, Si
2005-08-17 19:07:58 • Si_Daniels
Spook, I think you'll be able to find some great typefaces that would suit your needs for an identity and/or interface, without resorting to the destruction of something else, (not to say that this method doesn't work for some people). There are a lot of questions to be answered first in order to give a proper recommendation. What kind of game is it?* I have looked at a lot of new game promos recently, and much of what I have seen in the logos don't exactly fit a distorted, or even a grunged out aesthetic. Many of the darker games are using [[Blackletter]], and I see a lot of clean, sans serifs being used for war-based games and fantasy RPGs. I guess I just hate to see game design fall into such a generic style. *sorry, I missed the Wrestling part earlier

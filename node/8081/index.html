Hrant, I use a font freeware font renamer for Windows that goes the other way around: it changes the filename of a font so it becomes identical with the PostScript font name inside of it. I can't stand font filenames such as am_b_.pfb or something and prefer them being named AmericanTypewriter-Bold.pfb and the sorts. There is a nice free app that does this sort of thing in batches, for subfolders, but I don't know its name now (I'm on my Mac now). As for changing the internal names for fonts, I myself use Python with FontLab and TTX. Not completely straightforward when it comes to Type 1 fonts, especially Mac Type 1 fonts, though. Best, Adam

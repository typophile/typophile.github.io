Creating a common suitcase

June 18, 2006 • Toby

How do I create a suitcase for a full range of styles, from thin to black, with a common screenfile for all of them? I can´t find a function for this in FL. Thought File/Create all would do the trick, but I was wrong…

You can create a suitcase with four styles (plain/italic/bold/bolditalic) in FontLab using File > Generate Suitcase….

If you want to combine more fonts than that in one suitcase, you need a Mac running OS 9 or earlier. You can simply drag one suitcase file onto another in the Finder to combine them, more or less as if they were folders. There may be utilities that can do this in OS X, but I can’t think of any off the top of my head.

Regardless of whether you combine fonts in a suitcase, you can associate a large number of fonts as a family by using the Family Name and PS Font Name fields in the first screen of the Font Info dialog in FontLab. Note: This is assuming a PostScript font for use on Macs.

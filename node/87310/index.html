The profit part was [[http://en.wikipedia.org/wiki/Underpants_gnome#Plot|joke]] :) any way I don’t sell fonts, so I’ve no idea.

There are generic [[http://www.adobe.com/devnet/opentype/afdko/topic_feature_file_syntax.html|feature file syntax]] documentation, but nothing Arabic specific. For understanding how Arabic is handled in OpenType, check the [[http://www.microsoft.com/typography/OpenType%20Dev/arabic/intro.mspx|Microsoft documentation]] (especially the shaping engine page), then you have to implement those features in feature files. Checking the layout code in existing fonts can be very helpful as well.

Hi guys, i’m new here. i’m a graphic artist/illustrator and i am always searching for font IDs and purchasing new fonts (faves of mine are from letterheadfonts and fontdiner).

I found you on a Google search while searching for a Windows Version or a very close substitute for the Apple font “New York” … is there such an animal?

New York is a peculiar specimen of early digital typography. It is one of the first to be developed specifically for the screen, so it has an unusually high x-height for an old style serif because it was designed as a bitmap first. There isn’t much like it, but, besides Times, it seems like Palatino could have been a model for Bigelow and Holmes' outlines.

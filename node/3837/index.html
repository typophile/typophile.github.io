Thomas, I doubt there is a font to match this hand drawn lettering, but these kinds of initials are often offered as clipart. The Dover books are a good source. Search for "dover initials" at amazon.com, or abebooks.com. Also, Linotype's Madame isn't so far off. Interestingly, the forms are so complex, it requires several keystrokes to create each letter.

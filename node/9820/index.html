Yves, you need Helvetica Linotype. This is a family of four OpenType fonts. The four weights available are just the ones you need (the "italics" are only obliques). Each of the fonts has Western Latin and Extended European (CE, Baltic, etc.), Greek, and Cyrillic characters, as well as Hebrew and Arabic. You can read the code page info in the description at the bottom of this page: http://www.linotype.com/48344/helveticalinotype-family.html You can also purchase the fonts there, too ;-) Helvetica Linotype was released in 2002

I'll just note that the design of the Cyrillic and Greek letters in Helvetica Linotype differs very considerably from the earlier Helvetica Cyrillic and Greek designs (the latter from the 1970s, not sure about the Cyrillic). When Linotype approach me about making a version of Helvetica for use as core fonts in software systems, I recommended redesigning the Cyrillic and Greek, and was very happy when Linotype okay'd the budget for this. It should also be noted that Helvetica Linotype was completely respaced (Latin too), because the older Helvetica fonts (and the Helvetica Pro fonts that are based on them) had inherited 18-unit spacing from earlier font formats, so had inconsistent sidebearings. This is important to note, because you won't get the same line-layout between the Helvetica Pro fonts and the new Helvetica Linotype. So if you have existing documents set in the old Helvetica fonts and don't want them to reflow, you should avoid using the new font for that text. So, for instance, you might find that you end up using the Helvetica Pro fonts for Latin text and Helvetica Linotype only for the Cyrillic and Greek.

Yves, I'm not sure if any of the other Helvetica versions can solve your problems in four single fonts. (The following is not an official statement) For instance, from what I can tell from the Linotype webshop, Neue Helvetica Pro fonts have Western Latin and CE character sets in them, but not Greek or Cyrillic. The OpenType fonts in the Helvetica family on the other hand are different. Std seems to have Cyrillic characters in the character set, but no CE. The surperbly hinted Com version has Latin Extended characters, which include CE glyphs but not Cyrillic ones. If text flow issues would be a problem, and you don't think that Helvetica Linotype would be the best solution, I would recommend sending an e-mail to Linotype customer service, at info@linotype.com. If you tell them exactly what your requirements are, they should be able to give you all of the definitive answers you would need. Additionally, you can call them up if e-mail isn't your thing. Info on that here. I should add that, whenever anyone has questions about Linotype fonts, e-mailing the company support address (info@linotype.com) is always the best way to get information.

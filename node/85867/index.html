Hi J I would suggest to use GREP rather than nested styles. In Cs5 you will find it in the paragraph styles - GREP.

This will give you the possibility to change the letter “P” to a different character style. Im just trying to get you the GREP code as well.

Edit: here a code which seems to work

\sP\s

I never heard about ATM and Win XP not getting along...
That said, you install PostScript fonts on Windows 2000 and XP the same way you install TT or OT fonts - go to "Fonts" in the Control Panel, or to the Windows\Fonts folder with Explorer, then go to File > Install New Font.

Windows XP and 2000 natively support PostScript fonts.

Internet Type Museum

January 30, 2007 • matthew_desmond

In recent weeks, I’ve been doing some research on older typographic subjects and a thought popped into my head. We should have a all-inclusive online type museum with high quality scans of classical materials as well as a wiki-like setup. Think of the possibilities. We could cross reference all of the newer type revivals with the old stuff. It would also make finding rare type specimens a cinch. This would not only be beneficial to type designers or graphics people, but also the whole world in general. Need a font from the 1400’s? look it up on the site. See how it was used. I think it could be brilliant.

Anyone want to help?

I’d love to get this going.

Oh, and it could include written language information also, not just type.

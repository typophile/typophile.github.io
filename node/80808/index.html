Every big reseller (MyFonts, FontShop, Linotype, Veer, etc.) often have some blurb about the fonts they sell. As for sale-agnostic sites, http://www.identifont.com/ has some info, but you can’t add yours, while http://typedia.com/ would be exactly what you’re looking for, but it hasn’t yet gained enough entries to be a reference.

Anyway, from Ascender’s site:

Lily™ UPC is a Thai font designed by Unity Progress and offered under license from Microsoft. The Lily UPC Family (4 fonts) includes the Thai code page 874 and Latin 1 character sets. You should be familiar with the use of Thai fonts and multilingual fonts before purchasing Lily UPC.

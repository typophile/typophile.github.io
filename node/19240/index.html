2006-04-13 20:25:15 • neutura

Fontlab’s (v4.5) alias.dat file inside the folder “Mapping” might not have the necessary entries.

Use T+commaaccent t+commaaccent S+commaaccent s+commaaccent instead of Tcommaaccent tcommaaccent Scommaaccent scommaaccent

Of course, you’ll need to have the necessary glyphs (T t S s commaaccent), before you use the command “Generate Glyphs…”

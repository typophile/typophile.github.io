So small. About all I can say is that is an italic and seems to have contrast matching a "modern" typeface such as Scotch Roman, which is a font often used in British scientific books. There is a very good Scotch Modern by Nick Shinn http://www.myfonts.com/fonts/shinn/scotch-modern/ If you find it too expensive for a school project there are others such as Scotch Roman MT http://www.myfonts.com/fonts/adobe/monotype-scotch-roman/ There is a wiki that is very informative about the Scotch Roman/Modern typefaces http://en.wikipedia.org/wiki/Scotch_Roman Don

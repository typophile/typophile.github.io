Indesign workflow question

December 29, 2009 • blank

Is it possible to place a document containing text into multiple Indesign files and then style that text differently in each of the multiple indesign files? What I want to do is maintain a single text document for text that I use to proof all of my different fonts without having to change that fonts in used in the original every time I need to print a proof. And if this isn’t possible in Indesign, is there some other option?

In InDesign, go to Preferences > Type > Links, and select “Create Links When Placing Text and Spreadsheet Files.”

After placing a text doc, it will appear in your Links palette, and will display the yellow “I’ve been modified” triangle when you make changes to the original text doc.

The challenge comes when you update the link. Any changes you’ve made to the file in InD (such as applying styles) are wiped out, which would defeat your purpose.

I’ve not tested this, but I believe the workaround would be to create and apply style sheets in your word processor, then modify those style sheet attributes in InD. (Assuming that you’ve set your text import preferences to bring along the word processor’s style sheets, of course.)

2005-07-03 18:22:06 • levonk

The feature is intended for use in vertical setting of East Asian scripts. It is not intended for raising or lowering glyphs in the context of horizontal text. The feature is not applied by any applications outside of vertical setting of East Asian text, e.g. in the Japanese version of InDesign.

If you want to contextually raise or lower glyphs in horizontal text, you should use the normal feature. I’m not sure if you can do this with FontLab, though, because of the limitation that you encoutered. You can do it with the MS VOLT tool.

If there is a font exactly like this, I don’t recall seeing it. The lettering is an adaptation of a classic blackletter style called textura. The font with the closest basic letter-forms is Fette Gotish https://www.myfonts.com/fonts/linotype/fette-gotisch/ See also fonts like Unger Fraktur Fett http://www.myfonts.com/fonts/rmu/unger-fraktur/ and, particularly for the s, Fette Fraktur http://www.myfonts.com/fonts/linotype/fette-fraktur/ Don

Hello, is there any chance to find a font like Bruegel's signature? …. thank you very much m.
There are a few pen scripts that Font Book puts in the Blackletter category that have some similar features: Carus by Jurgen Brinckmann; FF Madonna by Jurgen Brinckmann; Poggio Bookhand by Philip Bouwsma; and Pompeijana by Adrian Frutiger. I would say that these are somewhat 'like' Bruegel's signature.

See this wonderful site [link]. Mostly all information you need to produce Unicode keyboard layout for your language is there. Big thanks goes to Mr. Alan Wood… :-) I suggest using Google to find information about your language usage on computers before. May be it’s already done for OS X …

From this page: “Ukelele provides a graphical interface for creating or modifying Unicode keyboard layout files for Mac OS X 10.2 onwards. It allows you to start from scratch, or to modify an existing keyboard. Characters can be dragged from the Character Palette, or from other programs that can display the characters that you want to use.”

2004-03-15 15:13:46 • bowfinpw
Mike, it seems like you're overlooking one of the most obvious possibilities: Helvetica Inserat, which, incidentally is what CG Trimverate Inserat is a knock-off of.

SM, it's normal for the top arms of an X to be shorter than the bottom ones. It's not usually obvious, though. The reason this is done is to compensate for an optical illusion. If you draw them exactly the same length, the top arms will appear to be slightly longer. When they are drawn slightly shorter, they appear to be the same length. If you don't believe it, find an X in a font which appears to be perfectly symmetrical and turn it upside down.

2012-10-31 09:24:20 • manyaldesign
This inline style is not hard to reproduce using Illustrator or Corel Draw. Spotting a condensed unicase gothic, you’d just have to set main color to transparent and add one to stroke. Then, vectorize text and remove all finial strokes to recreate this exact style (“gas-pipe”). You can also have a look to some tags at MyFonts: [[http://www.myfonts.com/search/tag%3AInline|Inline]], [[http://www.myfonts.com/search/tag%3Alined|Lined]], [[http://www.myfonts.com/search/tag%3Aoutline/|Outline]], [[http://www.myfonts.com/search/inline+sans/|Inline Sans]], [[http://www.myfonts.com/search/inline+gothic+unicase|Inline Gothic Unicase]], [[http://www.myfonts.com/search/neon/|Neon]], … Some suggestions: [[http://www.myfonts.com/fonts/reserves/enamel/|Enamel Inline]] (not unicase), [[http://www.myfonts.com/fonts/typodermic/droid/|Droid]] (free, unicase, not inline but easy to reproduce), [[http://www.behance.net/gallery/Metropolis-1920/3244285|Metropolis]], [[http://www.myfonts.com/fonts/redrooster/los-alamos/|Los Alamos]] (unicase, inline), [[http://www.myfonts.com/fonts/adobe/industria/|Industria Inline]], [[http://www.myfonts.com/fonts/llw-studio/market-street-neon/|Market Street Neon]], [[http://www.myfonts.com/fonts/canadatype/press-gothic/|Press Gothic Biform]], [[http://www.myfonts.com/fonts/canadatype/wagner-grotesk/|Wagner Grotesk]], [[http://www.myfonts.com/fonts/jnlevine/inline-lettering/|Inline Lettering]], [[http://www.dafont.com/ds-narrow.font|DS Narrow]], [[http://www.dafont.com/buenos-aires-nf.font|Buenos Aires]], [[http://www.myfonts.com/fonts/fenotype/rock-it/|Rock-It]] (already roughened, just need to be “inlined”), [[http://www.fontshop.com/fonts/singles/fontfont/ff_softsoul_ot_light/|FF Softsoul Light]], [[http://www.myfonts.com/fonts/canadatype/gala/|Gala]], [[http://www.myfonts.com/fonts/redrooster/phoenix-pro/|Phoenix]]…

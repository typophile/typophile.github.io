The sample is pretty small. Also, do you need just the SOAP letters? Yi Baiti is a font created for a very specific non-latin script* so arguably the latin part of it is based on an already existing latin typeface. I would try something like Nimbus Sans Novus** or other Helvetica (Neue) version. * http://en.wikipedia.org/wiki/Yi_script ** http://new.myfonts.com/fonts/urw/nimbus-sans-novus/ By the way, nowadays Macs use Windows fonts just fine.

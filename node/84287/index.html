First check to see if the font you're using just might have the ǿ character (U+01FF). Then check to see if it has combining accents, which are zero-width characters with the glyph set to the left, so that it appears above the character preceding it. The combining acute is U+0301. You can copy it from this ǿ and see if it shows up correctly. In the absence of either of these, go ahead and kern them together, but be aware of what will happen if you ever need to copy the text and lose the formatting. For positioning, refer to the ó in the font you're using.

I would like to announce the release of a light version of my Windows font editor - Type 2.2.

Type light is a fully functional, freeware font editor, with a basic drawing toolbox. It can edit, create and convert OpenType TrueType (.ttf) and OpenType PostScript (.otf) fonts.

Available for download here: [[http://www.cr8software.net/typelight.html]]

Jean Baptiste,

most of what you ask is possible to do using Python scripting, but not through some simple calls of some one-line commands. For most of the filetypes you’re asking for, one needs to actually write some code – procedures that collect the data and aggregate them in some text form.

Also keep in mind that RoboFab stores a lot of font-related information in text files (XML-based UFO/Glif).

There are Python developers on this forum (including myself) who might be available for writing this kind of code for hire.

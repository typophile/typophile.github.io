In what country are you? If in the USA, you can obtain a Design Patent for a typeface design. Note that this is different from a Utility Patent. In other countries, similar protection is referred to as e.g. Registered Industrial Design.

See http://en.wikipedia.org/wiki/Design_patent for more information.

The easiest way to register a Design Patent is to employ a patent lawyer to file the paperwork, but you could presumably do it yourself if you find out from the US Patent Office what is required.

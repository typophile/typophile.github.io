2007-07-30 06:14:41 • Stephen Coles

With something like this, as you connect more lines, just like when blackletter joined stems (is it mun, num, mnu, mum, mmu, umn, etc), it can be hard to recognise letters are they should be.

On first glance I couldn’t immediately process the letters, and after thinking about it for a while (without reading your post) I figured it was either carikou, or carijoou (I’m sure those mean something in some language —wait, nevermind, actually there is a musical artist Len Carikou, though according to Google, Yahoo!, and MSN, I’m the first person on the internet to type in carijoou, wahoo)

Anyways, some alternate interpretations would be caruoou, carykou, couribou, courikou, etc. If you have to go for the connected letters look, make sure it’s unambiguous as to which letter it is. If you have to go for the long connected descenders, maybe try a slight break (like you have between the top of the c and the bulb of the a) so that we know to see two descenders, and not a connected u. I agree with some of the others who’ve already posted, the o on top of the b and u is really distracting, and hampers reading (it makes the b look like k).

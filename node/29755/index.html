First of all, GoLive is a dead-end product. Adobe will be dropping it and Dreamweaver will be taking over. You might just want to start with DW. Secondly, the best way to learn any of the programs is to start learnign HTML and CSS. Once you grasp the concepts of those, using the actual web editor becomes trivial. So, that said, don't bother picking up any books regarding DW or GL themselves. If you want a book, pick up one on HTML/CSS. If you are a complete novice, www.w3schools.com is a good place to start.

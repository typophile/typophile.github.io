Standard Type Size Increments

May 20, 2007 • hanneman

I’m looking for some background on the traditional scale of sizes used for type sizes: 8 … 12, 14, 18, 24, 36, 48, 60, 72, 96 etc. I am interested in finding out what the progression is. Why is it so handy for creating contrasts of size, for example, 36 pt heading with either a 24 pt. or 18 pt. subhead? As opposed to something more arbitrary, like 43 pt with 21.5 pt subs? How would the progression extend to very large sizes (say, above 120).

I appreciate any thoughts from anyone less numerically challenged than myself.

Jim

The de facto standard list of point sizes used across many computer programs probably got its start with the original Macintosh, which, due to technical limitations, came with matching bitmapped screen and printer fonts for only certain point sizes. If I recall correctly, the list was 9, 10, 12, 14, 18, and 24 for most fonts. (There were a few oddball fonts that didn’t follow that rule; for example, I remember Venice being available in only 14, and Athens and San Francisco being available in only 18. But these were the exceptions.)

I believe that most of the standard point sizes in use today were chosen because they’re divisible by six, which makes them both easily halved and easily divided by three.

At smaller point sizes, you need finer granularity, which is why the list starts out with small steps. But beyond, say, 36 points, there is very little to be gained from offering default sizes in a smaller increment than six. If user wants something around 50 points high, he can just select 48 from the list, and then manually adjust it later if 48 turns out to be a bit too small.

Above a certain size (maybe at 48 points), it would probably be practical to increase the default increment to 12 points, and later to 24.

As an Alphatype shop - the progression was: 6, 12, 18, 24, 36, 48

When Alphatype created new typesetting equipment that went beyond 48 points - it continued wiht 64 and 72.

Some how there is some rhyme and reason to it - the point system itself in increments of 12.

If you ever look at old type speciman books, the ones designed to help designers do proper character counts for lines, these were set as standards.

I remember in the 1950s when artists were able to start using typositor for headlines - that started the change in headline sizes. Stat cameras were available to reducing and increasing type - most artist would ask that the Cap Height would be in “inches.”

It may also have to do with what was the most popular sizes being sold in hot metal. - This is only an educational guess and not fact. A typeshop had to buy type sizes in lead - and several ofr the same point size. (After all, how many lowercase Es do we use in a paragraph?)

When cold type came about - the manufacturers used standards to make their master fonts. Mergenthaler used 6 point type - and blew it up - which is why their type always “chunky.” Compugraphic used 24 point masters - creating all their point sizes from that. I was Alphatype, because they used several point sizes to create their masters, so the type set from them was closer in feeling, positioning, to hot metal type.

I just wonder if this was another one that the answer to the real question was money…

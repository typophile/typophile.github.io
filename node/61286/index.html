Hi Richard:

Stempel was a foundry that was acquired by Linotype in the early 80’s, but during that time very few were fonts were released as digital type. Bitstream was the first in 1981. Bitstream did many digitizations of Monotype and Linotype fonts, but—for example—Stempel Garamond could not be called Stempel Garamond by Bitstream because it was a copyrighted name, hence placeholder names like Original Garamond was used instead of Stempel Garamond.

Bitstream has licensed fonts to ParaType and Tilde. These foundries specialize in Cyrillic for Russian customers and have modified the fonts for that purpose. I recommend that you check the character set(s) to make sure that it has all the letters and symbols you need because many have been overwritten with Cyrillic.

If you plan on using this for serious text work. You need to skip the Bitstream, Tilde, and the Paratype versions— they don’t have the necessary small caps.

I too love Stempel Garamond, but I find it so compromised I use a contemporary alternative:

Merlo: http://www.felicianotypefoundry.com/main/?page_id=206 http://www.felicianotypefoundry.com/main/?page_id=64

If you go back to my link to myfonts, type the @ symbol— they all look wobbly or lousy or taken from Times Roman. The same goes for all the other analphabetic glyphs. Merlo was made from scratch an is entirely harmonious. So are the fabulous ones below.

Other excellent alternatives are:

FF Maiola: http://www.fontshop.com/fonts/downloads/fontfont/ff_maiola_pro/

Hoefler Text: http://www.typography.com/fonts/font_overview.php?productLineID=100010

Verdigris: http://www.mvbfonts.com/html/fonts/verdigris/pk_verdigris.htm

Sabon Next: http://www.linotype.com/53159/sabonnext-family.html

Hope I helped you,

Mike Diaz :-)

PS… It was/is often the case with the old Linotype and Monotype types that only the text version of the fonts were digitized. Many display versions that are sharper and prettier are still sitting in cabinets collecting dust. Maiola is the sharpest of all the alternatives yet it is good for text too.

Fontographer 5.1 Update Released

July 8, 2011 • bojev

Fontlab.com has released an upgrade for Fontographer to 5.1.It was mention here several weeks ago by Adam Twardoch

Yes. Fontographer 5.1 is a free update to all Fontographer 5 users: get it here and review the list of changes and improvements.

Also, during this summer, a Fontographer 5.1 + DTL OTMaster 2.4 bundle is available at US$449.00. We at Fontlab Ltd. are very exciting about this collaboration with Dutch Type Library and URW++.

The Mac OS X version of Fontographer 5.1 is compatible with Mac OS X 10.4 and higher, including 10.7 Lion. The Windows version is compatible with Windows XP, Vista and 7.

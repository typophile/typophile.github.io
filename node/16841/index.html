As long as the lettering is not a logotype (protectable by copyright) but simply lettering (not protectable by copyright), then you should only have to deal with the creator of the electronic version (aka, the font).
2005-12-19 14:09:36 • oldnick

I’d agree with oldnick, but you’d have to trust that the person offering the font actually drew it from scratch, and didn’t just open up an existing commercial version and re-name it.

Also think there are benefits in doing some research and trying to find a more original/professional version of the font if it exists. It will likely be better drawn and have better spacing than a re-digitization based on printed samples.

Is there a way to convert all type in an Illustrator file to outlines when saving to EPS in CS3? I thought that there was an option for it in older versions of Illustrator but it’s not there now. Select all and go to Type / Create Outlines (or Cmd-Shift-O). You should be able to create an action for this and batch process multiple files.

BTW are you adding Vietnamese accents to an existing font? If so, here are two things to consider:

  1. I think they’re exceedingly small - too small to be functional.
  2. Most fonts have a EULA (license agreement) that doesn’t allow modification; but even ones that do rarely allow you to redistribute the results - so you have to make sure you’re the only user of the modified font.

chimerical,

If we’re on the same page, the typeface you’re looking for is VAG Rounded. As far as I know, that’s still the typeface on all of the unibody “chicklet” keyboards too. I think they started using it some time after Steve Job returned to Apple – before that their keyboards were set in Univers.

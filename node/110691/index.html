I would like to use this text in my photography, does anyone know what fonts were used or if it can be purchased as a template?
As for a template, I suggest you check the eps files from Shutterstock and other major vendors. I trust the typophiles will either identify precise fonts or suggest good substitutes. I looked for the script but did not find the exact match. But I did like some alternatives that match the sensibility of the message, including Alejandro Paul’s Adios Script http://www.myfonts.com/fonts/sudtipos/adios-script-pro/ and his Feel Script http://www.myfonts.com/fonts/sudtipos/feel-script/. I also like several of Laura Worthington’s scripts http://www.myfonts.com/foundry/Laura_Worthington/ Don

Hi Courtney,

strictly speaking, not every [[Blackletter]] is a [[Fraktur]]. According to the DIN [[Typeface Classification Schemes|Scheme]], those typefaces could be classified into five subcategories.

Goudy Text is, as its name suggests, a [[Textura]], which is the oldest branch of Blackletter. Then again, this very typeface was designed as recently as in the 1920s.

So is Wilhelm-Klingspor-Gotisch, a Textura by [[Rudolf Koch]]. That one would be my suggestion: It is one the most beautiful Blackletter typefaces. And, IMHO, it is more representative in some details. Compare the letters ‘d’, ‘h’, ‘z’, ‘A’ etc. You’ll see that Goudy Text is more normalised, less ‘quintessentially Blackletter’. F

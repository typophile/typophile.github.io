September 5, 2013 • toddvirgil

So, I'm sure this is custom lettering, but if anyone knows a font (preferably free) that is similar, I'd love it. http://www.gilmourish.com/wp-content/images/electric_mistress.jpg
Oh, I just love that old analogue Electro-Harmonix gear, it used to be the main feature of my pedal board (before I went digital) - Switchblade channel selector, Small Stone Phaser, Deluxe Electric Mistress Flanger, Deluxe Memory Man Delay, Big Muff Pi Fuzz Box + Cryin' Tone Wha Pedal - Oh what joy! Anyhow - I digress - The Electric Mistress font. Well as I recall, EHX used to have a habit of taking a commercially available typeface and then modifying it for their own purpose. So, as Ryuk suggested above, you are gonna need to do some serious editing but the DeCoro font may be a good place to start. Slice by adding a dividing line through the text. You could modify the "r" by using the top part of the letter "s". You will have to replace the "t" - flip the top of the "e" and take the stroke from the "l". You get the general idea. Find it here: http://www.myfonts.com/fonts/john-moore/decoro/ Should you want to add the "electro-harmonix" text you will need the font Sidhe Noble: http://www.urbanfonts.com/fonts/Sidhe_Noble.htm Hope this helps - enjoy! Already asked at WTF Forum [[http://www.myfonts.com/WhatTheFont/forum/case/426991|here]] and [[http://www.myfonts.com/WhatTheFont/forum/case/386744|here]] but part of some very loosely similars from these links, nothing really interesting. 3 solutions: - you need an exact match: make it yourself using some vector editing software like Adobe Illustrator, CorelDraw or [[http://inkscape.org|Inkscape]] (free); not so hard to reproduce as it's only made with basic geometric shapes; you'd need only one square and one circle, simply duplicating, overlapping and cutting. - perfect client for [[http://fontstruct.com|Fontstruct]]: same work as previous but you could build, directly and freely, a keyable font - start from any "[[http://www.myfonts.com/search/blippo|Blippo]]" font and rework it: . commercial: [[http://www.myfonts.com/search/blippo|Blippo]] . free: [[http://www.myfonts.com/fonts/typodermic/knuckle-down|Knuckle Down]], [[http://paulw.deviantart.com/art/Consume-of-Die-Typeface-160925786|Consume of Die]], [[http://www.dafont.com/hff-modern-strand.font|Modern Strand HFF]] (have also a look to these 2 sections at DaFont to find more possibilities: [[http://www.dafont.com/theme.php?cat=115|Fancy > Retro]] and [[http://www.dafont.com/theme.php?cat=304|Techno > Various]])

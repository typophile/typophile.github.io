Creating a lowercase form that matches an uppercase's x-height and stroke width

September 21, 2005 • timfm

Hi Typos, I'd like to explore the possibility of some lowercase forms in what is essential an mostly uppercase logo. A few of the letters might lend themselves better to lowercase forms. I'm terrible with Illustrator's beziers. Is there an easy way to increase a lowercase form's size to match an uppercase's x-height and stroke width in Illustrator or FontLab? -t
One thing you can do--and this generally will work better with a sans serif--is to use the lowercase from a lighter weight. If you're lucky, the strokes will be a nearer match.

Another thing to try: Use Illustrator's "offset path" filter with a negative value to make the lowercase forms thinner.

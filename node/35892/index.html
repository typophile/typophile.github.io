Jan Tschichold: The New Typography - which version?

August 7, 2007 • W1

I’m looking to buy a copy of Tschichold’s The New Typography but am unsure which version on Amazon is the best one to get; there’s a version from 1998 and a more recent one from 2006. Anyone got any advice on this?

The more recent 2006 edition has a new foreword by Richard Hendel, plus a different cover design. Everything else is exactly the same in both editions.

Hendel’s foreword is good, but what you really don’t want to miss is Robin Kinross’s excellent introduction.

I did some tests about ink coverage when defining the look of the font I’m developing. My method is much less interesting than this demonstrated by Matthew Robinson –but save some pens and walls:

1. I selected a typical book page set in my font in InDesign. 2. Exported the page to a high resolution, 1200dpi JPG image. 3. In Photoshop, I did crop the margins and desatureted the image. 4. Using histogram, I selected the first 128 levels and take note of the resulting percentil.

Note that ID exports to a RGB JPG. I preferred not to change it to CMYK in Photoshop to avoid many changes in the image. The histogram selection was taken the first 128 levels to contain the range between absolute black and the mid-gray dots.

In order to get a more consistent result, I repeated this with other two random pages from the middle of the document.

To compare with other text serif fonts, I changed the type size in order to have exactly the same amount of text in the sample pages. And repeated the operation.

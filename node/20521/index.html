Adding Hyphenation dictionaries in Ind

June 7, 2006 • dezcom

Is it possible to add more hyphenation dictionaries to InD? I like to test type in foreign languages but Adobe does not supply all of the. They have Library files such as “rus.hyp” which I assume do the trick. Are there others available and if so where? Is it possible to just drop a new “.hyp” file in the library and have that language supported and show up in the InD “Language” pop-up?

ChrisL

“InDesign comes with 28 Proximity dictionaries for the following languages: Catalan, Czech*, Danish, Dutch, English (Canadian*, UK, USA, USA Legal, and USA Medical), Finnish, French, French Canadian, German (Reformed, Swiss, and Traditional), Greek*, Hungarian*, Italian, Norwegian (Bokmal and Nynorsk), Polish*, Portuguese, Brazilian Portuguese, Russian*, Slovak*, Spanish: Castilian, Swedish, and Turkish*. (Languages marked with an asterisk have been newly added to InDesign CS2.)” http://www.adobe.com/aboutadobe/pressroom/pressmaterials/creativesuite/pdfs/id/id_faq.pdf

4.0.2 update adds support for seven Central European languages. http://www.adobe.com/support/downloads/detail.jsp?ftpID=3168

It is possible to use older Proximity dictionaries with InDesign CS by placing them from “Adobe\Linguistics\Providers\Proximity" to “Indesign CS\Plug-Ins\Dictionaries\Proximity" directory as Paul described here: http://www.typophile.com/node/17681

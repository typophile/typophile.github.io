The best way, in most fonts, to get the x-height algorithmically. Get the bounding box of the lower-case "o"; take the height. Subtract the overshoot below the baseline. Cheers, T

2007-03-15 07:05:48 • Thomas Phinney
There are various software libraries that allow you to access font files directly (rather than calling the Windows API for some information on installed fonts). I think using FreeType 2 would be easiest since it provides object-oriented access to both OpenType/TrueType and Type 1 fonts: http://www.freetype.org/freetype2/index.html For TrueType or OpenType fonts, you can read out the x-height included in the font using http://freetype.sourceforge.net/freetype2/docs/reference/ft2-truetype_tables.html#TT_OS2 -- see sxHeight I recommend joining the FreeType discussion list to get more help. Regards, A.

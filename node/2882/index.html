Almost all TrueType or OpenType text fonts are Unicode encoded (ie contain a Unicode character map), so I suppose you're really asking for a font that covers all or most of the Unicode standard (currently version 4.0). Unfortunately TrueType and OpenType fonts have a limit of 64K glyphs and as Unicode contains many more than this you're out of luck. Are there specific languages you're looking to support? Is there any reason why you need all of these languages supported in a single font? Cheers, Si

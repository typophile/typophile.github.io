Have you looked at Droid when printed? (I haven’t, but I’m not all that hopeful about it’s potential as a print font.) If you want optimal presentation for both mediums - screen and print - include a print stylesheet that re-formats and uses a print font(s). Those are going to be much easier to find. Top-notch screen fonts that are readable at body text sizes are still a comparatively rare breed. You might want to look into Ascender’s new font service and use one of the ClearType fonts (and there are others to choose from too.) It’s inexpensive, but you can have the pleasure of paying for the ClearType fonts all over again.

If you are talking about using Droid, I am assuming you’re thinking of linking the fonts using @font-face. Or are you talking PDF? In which case Droid doesn’t make a lot of sense to begin with.

I think those samples are Caledonia, by W. A. Dwiggins, which has been around since the late 30's, and is a very popular text font.

Bitstream's Transitional 511 is their version of Caledonia.

There is also New Caledonia, which I suppose is a slightly modernized version that looks virtually identical.

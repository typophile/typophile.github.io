A Quandary: Designing a symbol for the Indian Currency

April 7, 2009 • indiakangaroo

Dear Everyone, The Indian government is holding a competition to design a symbol for the Indian Rupee: http://www.hindustantimes.com/StoryPage/StoryPage.aspx?sectionName=HomePage&id=c8097698-a806-4cc2-8c67-668d594057dc&Headline=India+seeks+global+symbol+for+rupee and http://www.labnol.org/india/currency-symbol-for-indian-rupee/7739/ Currently, just the rupee is written, just as "Rs". They'd like it to be a symbol that is enterable from a standard keyboard like the "$" since the rules mention, symbol should be applicable to the standard computer keyboard." In which case, I was wondering then: a. Does that mean one can't create a special "logo" for it? b. Doesn't that leave one with only the standard glyphs that are common to most typefaces on a standard keyboard? Are there any other options? Thanks for your help!
As an addendum, I'd like to find out can one create a symbol that can be easily added to the glyph set of standard typefaces such as arial?
2009-04-07 15:18:29 • indiakangaroo
a and b: No, creating a special character is the whole point. The rules are written poorly as the symbol will need to be adopted by Unicode and added to new keyboard layouts. addedum: No. Standards, licensing, etc.

I can only suggest some starting points: => For /l/d/w/l/i/n: Flange BQ (http://new.myfonts.com/search/Flange+bq/fonts/) or Llewelyn Heavy SF => For /a/s: EdwardianEF ExtraBold => For N: Articula Display SSi [Regular] I found the alternatives using Find my Font utility: http://www.findmyfont.com Fivos

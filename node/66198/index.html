The referenced thread notwithstanding, in my experience the message goes away if, on the “Most important font dimensions” page, simply make the Ascender and the Descender AS A POSITIVE value equal the UPM (e.g., 750+250=1,000), then check the “Copy values to TrueType metrics” box, then hit “Apply.” Don’t ask me why, but it works every time.

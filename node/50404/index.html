If it’s not possible to achieve this with a macro I’ll have to go down the road of using Metrics classes, but I know it would be alot quicker and simpler using a macro.

Mertics Class Method:

  1. Create the classes - a seperate class for most of the uppercase, lowercase and smallcaps.
  2. Open up each font and import the metrics classes
  3. In each font, Tools - Metrics Assistance.
  4. Select the classes, click Use the measurement line then Apply and save.
  5. Delete the metrics classes from the classes panel - I don’t see the need in leaving them there as they are just temporary

Macro Method (if it’s possible)

  1. Create the macro
  2. Call the macro in each font.

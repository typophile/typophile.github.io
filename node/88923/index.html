The display issue is likely related to font format, rather than hinting per se. The Windows GDI PostScript renderer is about fifteen years old and is generally very poor for rendering text size. So PostScript fonts, whether Type 1 or OpenType format, will tend to display badly at text sizes on screen in most Windows apps. [The newer DWrite rendering is much better, but only found in some recent applications, e.g. IE9.]

Both FF Ernestine and [[https://typekit.com/fonts/ff-tisa-web-pro|FF Tisa]] are humanist slab serifs offered by Fontshop in Truetype format, hinted for recent Windows versions.

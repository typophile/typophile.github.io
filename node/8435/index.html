Here's a logo that I'm working on, unfortunately I'm not budgeted to spend too much time on it, so I'd appreciate any feedback I receive. I've learned so much from reading Leslie Cabarga's Bible and I want to incorporate this into the logo; like points in extrema, stroke contrast, etc. This early work-in-progress version was created a few months back.

It's based on the Sho-Bud logo which you can see at:
http://www.shobud.org/

The extra letters are based on Loki Cola:
http://www.dafont.com/en/search.php?q=loki&#43;cola&amp;nb_ppp_old=10&amp;page=1&amp;text=Shot&#43;Jackson&amp;nb_ppp=10&amp;classt=alpha

So I've been tweaking these to make them blend together. The client wants the Shot Jackson logo to look like the Sho-Bud one (they own Sho-Bud). I've marked some changes that I'm going to start working on, and some changes that the client has requested. So I thought I'd ask for some feedback and ideas before I get started. Let the slaughter begin! Thanks!

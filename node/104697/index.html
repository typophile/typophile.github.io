Be careful here with the use of the word "Commercial". Generally, and IANAL, in End User License Agreements, the general terms for license are "Business and Personal use". This covers what most users expect, the ability to use the font software to create documents of various kinds; correspondences, advertisements, publications, posters etc. The term "Commercial" in an EULA generally means using the actual font software in a product that will be sold. Game, App, set-top box, etc. This usage is usually not allowed in a standard EULA and requires an additional license fee.

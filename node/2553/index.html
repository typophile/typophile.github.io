I've finished a project which I'd started a couple months ago (old thread) to reconstruct Crouwel's New Alphabet. I wasn't able to find a full sample of all the characters so I extrapolated from what I could find and came up with a complete ASCII set. I put it all together in a free Opentype font. You can find the font and more info at my weblog.

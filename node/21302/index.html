I was going to suggest Yubnub.org, which is “a (social) command line for the web.” Basically, using very simple variables you can create commands to search your favorite websites for whatever you want (ex: “gim porsche” searches Google Images for “porsche.” To speed up my own searching for fonts, I made the commands “myfonts” and “linotype,” which, of course search those sites for whatever you want. It’s not as comprehensive as Rollyo, but you can search from the search bar and/or address bar in most browsers (go to “Installing YubNub” at the bottom of the main page), so it’s a bit more convenient than Rollyo.

2006-07-16 00:15:33 • Reed Reibstein

In firefox, you can do the following:

  1. visit myfonts.com
  2. right-click the search-box, and select "add keyword for search" (or similar, I'm using the german version...)
  3. enter a name and a short(!) keyword, and save it as a new bookmark
  4. you can now just type to the urlbar: [keyword] [fontname] and press enter
  5. firefox will do a search on myfonts, and display the results directly

So, for me this means: I type “mf helvetica” and I get the same results immediately, as if I visited myfonts.com and did a search there. Imho this is even faster than using a search-plugin (because there, you often have to switch search engines (two clicks + typing) instead of just typing.

Of course, this works with every search-field, not only on myfonts.com. I have keywords for google (g), google image search (gp), amazon (am) and many more.

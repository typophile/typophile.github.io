2003-11-19 13:23:51 • Ignacio
> probably because as each character is repeated

I've come to the tentative conclusion that the necessity for letterform "randomness" is limited to foveation. What I mean is that as long as the font uses 3 or 4 variants of a glyph (which is the most we can see in the fovea at one time, and that's assuming 3 or 4 of the same letter ever actually happens in sequence...) it's OK, because short-term memory concerning exact glyph shapes probably doesn't survive a saccade. Interestingly, Letterror's FF Kosmik-Flipper uses 3 variants per character.

But I've also come to realize there's a much bigger problem: handwritten stuff typically has mistakes in it, which can either be corrected -but visibly- or just left in. This means you need some fancy software to replicate that in order for something to truly appear handwritten...

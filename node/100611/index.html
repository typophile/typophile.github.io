[img:sites/default/files/old-images/Porto-Editora_3625.jpg] I don't know if this is an existing font, or if it was made only for this cover. The dictionary itself spells things in weird ways, like «coöperation», «fœtal» and even «bæby»...
To my eye, this shows strong influences of Joan Trochut Blanchard. If it is, indeed. his work, the letterforms were assembled from pieces, so this particular typeface never existed as a discrete commercial product. If it’s not his work, it’s still unlikely that the typeface was ever digitized. BTW, the Oxford English Dictionary—the Ultimate Authority—spells these words in precisely the same way: «coöperation», «fœtal»—so, maybe it’s not the dictionaries that are weird. Just sayin’…

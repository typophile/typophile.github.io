[img:sites/default/files/old-images/nameplate_4336.gif] Any typeface combination can work, if different enough. Extremes compliment each other. Simple geometric fonts work great as a contrast to blackletter. Even a really fancy script can work in combinations. Remember, the more extreme and different the two, the better. Good luck.

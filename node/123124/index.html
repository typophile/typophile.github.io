Looking for the Outpost font. Not sure if it’s been customized.

Confirming your hunch, not a font to me as well, rather a custom logo work and possibly heavily customized. I’d probably start with [[http://www.myfonts.com/fonts/mti/sackers-roman|Sackers Roman]] Heavy:

  • adding an outline stroke to embolden it and to round corners
  • reconstruct /O (the only real custom part of this logo to me) from /U left spurred quarter, mirroring vertically once, merging the 2 parts, mirroring horizontally, merging, done
  • add a rough/grunge/eroded/vintage post-treatment: the 2 /T treatments are different meaning the lettering has been treated globally and does indicate the lettering does not come treated

I think the Russian support may be the hard bit here … you would probably have to get it custom made. You could start with a font that already supports the listed western European languages, like Dot Matrix OT and contact FontShop if they can add Cyrillic for you. (The additional letters needed for Czech, Polish and Hungarian aren’t too much work because they just have diacritic marks added to some of the existing latin letters.)

The fix is to modify the fonts. In most cases large families like that are from commercial type foundries whose license terms don’t allow end users to modify the fonts, so your only (legal) recourse is to contact the foundry and ask if they have a fixed version of the family, or are willing to fix it.

I know of one fairly large family that is making such a set of updates right now, so I can attest that it does happen.

2010-03-25 00:24:10 • misterpatrick

… Windows OTF …

No, that’s not it, because there is no such thing. OpenType CFF fonts are inherently cross-platform, and were so when the format was invented.

At a technical level, this requires that the fonts make correct use of various name IDs in the name table. The details are pretty well understood by most font vendors.

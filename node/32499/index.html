Corporate Ipsum

March 23, 2007 • cuttlefish

I ran across this clever tool for generating dummy text: < a href="http://macupdate.com/info.php/id/24454/Corporate%20Ipsum">Corporate Ipsum It is a Mac Dashboard Widget, requiring Mac OS X 10.4.3 or later. It's a little hard to describe what is special about what it does, so here is a sample of the generated text: Authoritatively unleash proactive best practices with installed base relationships. Rapidiously leverage existing process-centric methods of empowerment and client-based information. Appropriately transition parallel solutions for synergistic supply chains. Interactively transform premium methodologies before sticky potentialities. Uniquely integrate e-business alignments and cross-media meta-services. Compellingly reconceptualize global information vis-a-vis user friendly meta-services.
This is nice. I always struggle with what texts I should use for testing my fonts. This will be fun to use for that purpose.

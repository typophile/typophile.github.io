I have a need to create artwork for use with a CNC router that incorporates type. My router-guy wants it drawn with a monoline font like people use with AutoCAD for use with pen plotters. Thanks!

2010-07-29 01:54:32 • clean

Type 3 fonts supported almost anything you could do in PostScript, except for color and some other things that were added later. This would include stroked paths. Letterror’s Beowolf font, which randomly moved points around each time you printed, is an example of the sort of thing that was possible with Type 3. You could even make a “clip art” font that included different shades of gray, etc.

The problem is that this format is obsolete, as far as I know. If there is still software that supports it, I’m not aware of it. Of course, old software still supports it, if you can find it and run it.

2010-07-29 02:05:55 • Mark Simonson

A big problem with stroke fonts is that you need some way to express modulation of that stroke, or else you get pretty cruddy results. For example, it’s important to be able to have thinning where a round stroke meets a separate curved stroke (as in the lowercase “d”). The reason that there aren’t many formats around that support them is that unless one makes the format super-complicated, they produce rather poor results.

Okay, that was fun :-)

Here is a link to the script I wrote for Illustrator CS4 (it might or might not work for any other version): monolinetext.v1.0.zip. It’s a Javascript for Illustrator, and you can put it into the regular Illustrator Scripts folder to see it appear in the “Scripts” menu, or somewhere else (and you’d have to browse for it each time).

It shows a little tiny dialog (on which I spend most of the time) where you can enter a size in “points” (relative to the 29 units high Hershey characters) and enter some text. Use Shift+Enter to insert a ‘Hard Return’, or enter as much text as you want. The text is then drawn out in monoline characters in the center of the screen.

The script is intelligent, in that you can change a previously drawn text by simply selecting it and running the script. Note, however, that stuff like manual scaling and changed attributes (stroke width, color) will be lost. If that’s important to you, I might look into it …

It only supports a small subset of the Hershey Simplex set – just ASCII, space up to tilde. (And it draws traditional K shapes.)

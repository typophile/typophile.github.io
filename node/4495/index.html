Burweed is originally from the Image Club Typeface Library, which at one time was owned by Adobe. This weight of it was distributed along with a few other Image Club faces on a sampler disc included with many Adobe applications in the mid-90s. The Image Club library is now owned by Agfa via Getty Images via EyeWire.

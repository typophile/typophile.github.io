This from a 1950s catalog for the Danish furniture company Illums Bolighus. The types are Rockwell (or close to) and Monotype Grotesque Light.

[img:sites/default/files/old-images/Illums_6564.jpg] Note that “squaring off” (forced justification) was used for small elements of display type, and this was often the case for type used in an architectural context, as names — it wasn’t all rag right.

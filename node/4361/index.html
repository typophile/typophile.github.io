2004-04-20 19:20:50 • rjohnston
There's some hilarious stuff in Piet Schreuders' book
"Lay In - Lay Out" (which should be mandatory literature
for graphic design students BTW) about Crouwel's grid-
based experiments. He has a fair stab at the New Alphabet,
or 7/3/11/3/7 as it was originally called: (my translation)

"This text set in Crouwel's 7/3/11/3/7, a typeface originally
conceived to be read by optical text recognition, was
submitted to such an OCR program in 1996, which resulted
in the following:
Urn df ~ trut I ren Lno ~ L tc~r~L~n RAC) J ~ Er U ~ ~rtrn]
dr. ~ trrnL: UCnrc~cEr~ ~rtr Lu~drn Irl~Jr ULB Lol ~r rlrr L
~IXt~L L~ LLr~rn d~ NV dLnrm~ ~rCrLut nap h n q n L rrqF
(and so and so on I'm getting a headache - ed.)
How intriguing it might be, the general sense of this text still
is not very clear to me."

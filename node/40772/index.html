2008-01-10 22:25:17 • malbright

You might want to start by doing a search of “stencil” on MyFonts.com.

MyFonts has Ko and DoctorBob, which are a bit rougher looking than what you would get using a Rapidograph on one of those rulers… (Allied Engine is also in the same category.)

Isometrik comes close, but the stencil is different.

Same goes for Crisafulli.

Cargo is all-caps.

EF Helio comes pretty dang close, and has several weights. Some of them look like what you would get with a Rapidograph that was too thin for the shapes in the ruler.

Hello fellow type-aficionados I'm sure a lot - if not all - of you do it: Take pictures of type that you think is interesting, well-made or odd in some way. Well, I decided to put mine up on a blog, and you're all very much invited: [[http://www.madshougaard.dk/blog]] My idea is to tag each letter, so that you'll be able to hit (for example) 'e' in the sidebar and be presented with a lot of different ways to do e's. Geeky! I know. My hope is that this will turn out to be some sort of inspiration resource when you're designing type or logos. If you know of other sites that do the same thing I would be happy to hear from you. Of course, I'd be happy to hear from you altogether :) Best regards Mads

2006-02-09 16:40:56 • charmagad

yes, it’s quark’s way of controlling how many lines of type in a row are allowed to end in a hyphenated word. This information interacts with the word spacing and letter spacing allowances you set to create your setting.

Knowing how to use the h&j interface is critical to professional typography, and the way to learn is through trial and error. Different typefaces require different allowances, and type for heads and decks ideally requires different optimums than text type if you want to really come up with a well-worked through typographical system for your magazine. The graphic designer / art director is the gatekeeper here. Even if you don’t justify, you need to (re)consider the default optimums.

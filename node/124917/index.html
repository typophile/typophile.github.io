Is it possible to control font license?

Server-based font management software does this.

What’s usual font format for mobile devices?

WOFF and EOT (for older IE versions) for commercial fonts. But usually, you don’t really need to worry about that. If you buy a webfont license, you can download a webfont package, where everything is already prepared for web use. For free fonts, there is the Fontsquirrel webfont generator, which works quite good.

2015-04-25 23:03:16 • Typography.Guru
Server based font management software allows the end user to control their licensed fonts. It is not a solution for the foundry to control distribution/use of their fonts. This would be Digital Rights Management or “DRM” for fonts. There is no practical system for foundries to do this that is widely accepted, and what solutions exist impose a substantial burden on users.

2006-09-26 02:00:23 • SuperUltraFabulous
The diagonal stripes feel a bit hard-edged and garish. Consider getting rid of the black diagonal lines and using tweaking the greens so that they're closer to each other in hue and lightness -- more of a tone-on-tone approach. Or screen the black so that you end up with a grey that's the same lightness as your greens. Not crazy about the client's name in all caps and italics. But if you want to use that approach, consider making the type smaller and looser.

Charles,

information about the limitation has existed on the “Specifications” section of the FontLab Studio web page ever since the product was released: http://www.fontlab.com/font-editor/fontlab-studio/

and is also included on: http://www.fontlab.com/fontlab-products/compare/

It is also described on page 20 of the FontLab Studio manual that says: “Open, edit and generate OpenType PS, TrueType / OpenType TT and PostScript Type 1 formats with up to 6,400 characters”, as well as on page 152.

If you need to generate an OpenType PS font with more than 6,400 glyphs, you can either use TypeTool 3 (which is just $99) or AsiaFont Studio 4, which both can generate fonts with up to 65,535 glyphs.

Regards, Adam Twardoch Fontlab Ltd.

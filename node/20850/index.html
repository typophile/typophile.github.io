2006-06-28 19:41:36 • Miguel Sousa

There are two locations of the macro folder. On Mac, it’s /Library/Application Support/FontLab/Studio 5/Macros/ where the FontLab’s “factory” scripts should go and [user]/Library/Application Support/FontLab/Studio 5/Macros/ where the user’s scripts should be placed. The subfolder structure of the both folders is identical. If the user macro folder doesn’t have a particular folder that is present in the application macro folder, just create one.

The locations of the folders for Windows are described in the manual. Note that you can change the location of the user folder in FontLab Studio 5’s Options/Preferences dialog. For example, on my Windows machine, it’s D:\FL\Studio5 rather than the default C:\Documents and Settings\Adam Twardoch\Application Data\FontLab\Studio5.

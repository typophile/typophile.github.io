2009-08-25 19:21:54 • riccard0

Gill Sans is, although not by default, still on a vast number of Windows machines. Something like at least 50%, IIRC. I believe it comes with Word?

The Garamond on a Windows box isn’t the greatest typeface in the world for screen legibility, and Georgia is used practically everywhere, so if you want to be slightly different, Palatino makes a good choice as a companion.

An added benefit is that both Gill Sans and Palatino follow classical proportions, so you’ll have that going for you as well.

But then again, I find Georgia to be beautiful in many sizes because it is so well designed.

New free font manager with a twist

July 11, 2004 • mjf

I've been helping to beta test a new free font manager with some unique features - such as saving and restoring "snapshots" of your font folder, managing fonts, and "defrenzying" your fonts folder back to Windows defaults for when you need maximim speed and performance from your computer.

This is not a commercial program and no money is being, or will be made, from it.

If you'd like to check it out, you can find it at:

http://www.fontfrenzy.org

We'd greatly appreciate any feedback.

–mjf

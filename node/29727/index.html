Copying kerning from one glyph to another

November 26, 2006 • thisisgrey

How can I copy kerning values from any pair of glyphs in FontLab 5.0? For example, values from “a” to “á”?
From the glyphs window, “copy” and “paste special”, then you choose.
2006-11-26 23:52:50 • Alessandro Segalini

Alternatively, use class kerning. In the Classes panel, specify a kerning class that has “a” as the key glyph and “á” as the dependant glyph (with all the other accented “a"s there as well), and mark the class scope to cover both the left and the right side (using the checkmarks). The kerning for “á” and the other accented “a"s will from now on be automatically linked to the kerning of “a”.

Note that for Type 1 and OpenType TT/TrueType fonts, you need to expand your kerning before generating the final font while OpenType PS fonts use class kerning directly and there is no need to expand it.

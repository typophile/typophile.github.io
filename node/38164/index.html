what ever your line count, use a sharp squeegee, even pressure, steady speed for your pull and sufficient off-contact. A dull squeegee will cause dot gain, and uneven pressure or speed will result in an uneven ink deposit. If there is any subtle shading, you may want to introduce a second (spot) colour.

My experience with screen printing didn’t really allow for a lot of trial and error :)

2007-11-05 13:37:53 • Alessandro Segalini

if you’re just doing a one-color screenprint, the best thing to do is halftone it with 65 lpi (frequency/lines-per-inch). here’s a quick guide i put together. 01/start off with your photograph. like most things you print, it should be 300dpi. some silkscreeners debate this, but i also print at 300 and get good results.

02/convert it to grayscale using image>mode>grayscale. play with the levels if necessary.

03/when you’re all set, go into print preview. in the option box that says color management (i put it in red), select output.

04/in the output menu, select “screen.”

05/in the screen menu, make sure you uncheck your default printer settings.

for frequency, enter 65 lines/inch. since you’re using a 195 mesh, and i’m assuming that it’s not a dyed mesh, this is probably the highest frequency you can get away with. any larger, and the dots will get too small for your screen to pick up, and you’ll end up losing them.

for angle, i would put in 45. if this is just a one-color print, it won’t matter. once you start introducing more colors, angle becomes critical so you don’t develop any weird distortions. again, since this is just a one-color job, use the key angle of 45. if you want to do more, google “cmyk halftone screenprinting” or some combination of that.

when you get this high of a frequency, the shape of your half-tone doesn’t really matter, but i’ve always been told to do ellipse since it’s “the best.”

hope this helps!

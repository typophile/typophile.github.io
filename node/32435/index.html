Do you have any ideas on how to make an ugly word pretty? Don't set it in all caps in a Windows system font. Arial Bold is not a very warm and inviting typeface, especially set in all caps. Neither is TNR Bold. Try setting in all-lowercase, in a rounder, friendlier font. You might want to stack the two words on top of each other as well since they are both very long. As for your mark, scrap what you have and get out the pencil and paper. Doodle for a while, and see what you can come up with. I find that this is a much more productive sketching technique than using clipart. - Lex

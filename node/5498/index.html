I looked around a bit more and found 'Font Showcase'. I'm testing it right now and must say that I kinda like it. It workes much like FontGander. It will allow you to load an external folder which contains fonts. Or you can just load one single font. It allows you to use your own text string to see the characters. This is good, because mostly I need the font for a specific logo or name. You can change this into any text you want. If you want to see the whole character set, use the 'Character View' option, which surprisingly brings up an extra window. Here you can see all the glyph boxes with it's number and keyboard combination. This only works with installed font though. An external font will not be previewed. And that's just what I wanted. All toghether it workes fine for now, but Ithink I'll find something better. I'll have a serious look at the other suggestions and let you know. Thanks,

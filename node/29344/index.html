“if i’d get a penny everytime i search on myfonts…”

i just discovered that there’s a plugin for firefox that adds myfonts.com to the search-bar (upper right corner).

with ctrl+k, you can access the search-bar, with ctrl+arrow up/down you can chose the one you want. use the plugin “search engine ordering” to be able to change the order of the site-searches and delete the ones you dont want (right-click on an entry, “delete”)

Nice, Thierry. I always love adding new engines to the search bar. So much more convenient than actually visiting the site.

You can do that with Opera too. Any search box from any site can be added to the drop-down search menu. So you only need to enter the name of a font once to search Google, Myfonts, Identifont, Typophile and any others in a row. Just right-click the box you want + “create new search” (or something like that, I have it in French).

That’s what I’m doing, but breaking the stories up leaves a lot of extra overset copy that I want to get rid of at the end of each story.

Insert cursor at place where you want to cut the text. Hit Shift-Command-End (that’s the key with the arrow pointing from upper left to lower right) to select all the text after the insertion point. Delete. Alternate method: Use Shift-Command-Home (the key above the one mentioned before) to select all text that has to remain. Copy. Next select all text with Command-A, delete, paste copied text in place.

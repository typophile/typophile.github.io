How do I make additional weights and styles?

February 12, 2010 • Martin Silvertant

I recently came across some software with which you can easily make additional weights and styles for your font. I don’t know if you have to trace your font again in the program or that it just uses your original vector points. In any case, the software was incredibly expensive, there was no trial version and I couldn’t find a place to download it to try it out, so I forgot the name of it.

That aside, I’m actually wondering how you make your additional weights and styles. Do you draw them manually or do you use software for it? If it’s the latter, which? Is it also possible in Fontlab Studio 5? If it’s the former, is there a source that describes the process to do it appropriately? Manually making weights and styles seems an awful lot of work, while the process of making a single font is incredibly time consuming already.

There’s no way to avoid manual work. Computers cannot make aesthetic judgments, and combined with rounding errors generated letters get ugly quickly. Using tools like KLTF Glyph Tweaker will allow you to add or remove weight and width while maintaining contrast, but there are limits to what it can do. For example, you can go from a regular weight to a rough start on a bold weight, but if you try to jump to a black weight it’s going to take as much time to clean it up as it would to just draw it. It is also much easier to add weight than to take it away, because once you start pushing strokes in on themselves all the computer can do is overlap them.

Extrapolation/interpolation tools are a great way to flesh out a family after you already have a few fonts drawn. Font Remix Tools, Superpolator, and UFOStretch can really speed stuff up, but if you’re using them to create entire fonts they’re limited to expanding a family into a much bigger family.

2010-02-17 23:06:50 • Bert Vanderveen

Martin,

Superpolator is €250, not ‘incredibly expensive’. It literally pays for itself in a week. That said, Superpolator does not make interpolation possible, just easy.

If you have the incredibly expensive software FontLab Studio, you have a Python API for that. Install the incredibly free RoboFab and start writing your own interpolation code. It’s actually rather easy:

from robofab.world import NewFont
f1 = [[your one font]]
f2 = [[your other font]]
x = [[interpolation percentage]]
interpolationfactor = float(x) / 100
f3 = NewFont()
for n in range (0, len(f1)):
– c = f1[n].name
– g = f3.newGlyph(c)
– g.interpolate(interpolFactor, f1[c], f2[c])
– f3[c].mark = f1[c].mark
– g.update()
– bar.tick()
f3.update()
print “whee, free new font!”

Replace “– " with a tab.(What, no indentation? No syntax colouring?)

The Robofab website has a bunch of reference about Python programming, the RoboFab libraries, “glyph math” and interpolation.

Hope this helps, Kai

2010-03-02 13:47:49 • Tim Ahrens
I generally start with Regular. Then I make an Extra Bold by adding as much weight as I can with the “change weight” feature, without it looking too weird, and then manually adding more weight by a combination of nudging stems and dragging points and vectors. At the other end of the spectrum, I make a Hairline weight by drawing a path and adding a stroke value. Then I interpolate between these three “weight nodes” to create the other weights. It’s important to develop the metric parameters of all three base weights at the same time, especially the alignment zones, because it’s likely that you will want to have the alignment zones the same for all weights.

InDesign: Page/spread setup of self-published book

July 4, 2005 • Fredrik

Hi, I'm trying to find a good way of printing a little book project from a standard inkjet and A4 format. What I want to do is set up a layout in Indesign that lets me do the following: 1. book will be printed one side only 2. A4 divided in four to provide 4 pages 3. Pages 3,4...7,8...etc. rotated so I can fold the sheet an trim it at lower edge (Book will have the paper-fold towards reader) see attachment. I could of course just set up the A4 and do the layout there, but is there any way of setting up a Spread like this in Indesign? Or any other clever way of doing it?
As far as I can see, InDesign doesn't really provide much in the way of imposition support. Which is a little frustrating. There are various plugins of which InBooklet is the one I use if I'm doing something small like this. You can download a free trial from their web site. Alternatively, I remember seeing some free plugins for Adobe Acrobate which will turn an exported PDF into various predefined impositions. Have a look on Version Tracker. Hope this helps _ Nick

2005-07-04 20:22:11 • Dan Weaver
The imposer I use is Imposer Pro by ALAP, http://www.alap.com/ They also have a free trial download. If you end up buying it, its a bit pricey, but well worth if you are into publishing books in small runs. The only other way is to figure out what the pages positions would be in their imposition states and drag them manually to those spots. The best way I have found to do this is by making a dummy and visually matching up the pages. It can get tricky though. If you have pages numbers, you have to make sure they are not shifting around with any kind of automation. If you have images they run over the gutters, you have to duplicate each side for each page. But, it is possible. I have done it many times. It would be even trickier to do a 4-up situation like you want, but possible. It can be real annoying with a very long book though. And, save that file as another version before you start dragging pages around!

2005-07-26 23:59:53 • hrant
The smallest I've ever printed on a cd face has been 6 pt Franklin Gothic. I think the limit is probably around 5 pt, I would be really nervous about going any smaller than that. From my experience, this is small enough if you put the copyright information around the outside rim. A serif font would need to be printed a little bigger because the thins tend to fill in. If you're knocking the text out it also needs to be bigger because of the ink spread.

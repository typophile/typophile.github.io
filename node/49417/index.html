It may make sense to think about which glyph set you would like to offer, and then create your own .enc file, or derive one from existing ones. It is just a list of glyph names, so easy to write or modify in a text editor. It also helps you quickly bringing your glyphs into a desired standard order by selecting your encoding and then “sort glyphs”. (A note.)

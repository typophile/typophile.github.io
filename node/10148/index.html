Sorry to come late to the party. But as a parent can I make a plea. Please consider whether this is (1) a book to be read by children or (2) a book to be read to children. If it is a book to be read to children then remember you are actually setting type for adults to read.

In any event, I have observed that my 6 year old son seems to have very little trouble with "adult" letters (two-storey a, hourglass g), and even with script types. He gets puzzled if type is made to run in weird patterns, where it is physically hard to read (e.g., on dark backgrounds), where it is highly condensed. He seems to like large word-spaces, which I suppose makes sense since he definitely reads word-by-word still.

Whatever you do please, please, please do not use Bembo Schoolbook. My other pet hate is scribbly looking scripts, which get very monotonous. OK for a little display I suppose, but if people want a short story to look handwritten, could they not try ... writing it by hand?

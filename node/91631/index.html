Sorry for being so late. I have provided a updated version of the Nudge Tool in the [[http://forum.fontlab.com/downloads.html;sa=view;down=5|download section at the FontLab Forum]]. This version is MM aware and works always on the active master. It comes in two pieces: one tool script and a modified version of the InterpolateNode module. Other tools which depend on the InterpolateNode module will also benefit from the update, if you redirect the import statements to this new version (as far as I have tested).

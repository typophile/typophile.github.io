2007-04-11 15:18:58 • DuncanL

Duncan,

I recommend running:

  1. Microsoft Font Validator, a free Microsoft tool for Windows.
  2. Adobe CompareFamily, which is part of the free Adobe FDK for OpenType toolkit for Mac or Windows.
  3. Apple ftxvalidator, which is part of the free Apple Font Tools suite for Mac OS X.
  4. FontQA, developed at FontShop, which is a free tool written in Python, working within FontLab Studio.
  5. FontTools/TTX, developed by Just van Rossum, which is a free tool written in Python that converts OpenType and TrueType fonts into XML and back.
Adam

Al fín una solución para identificar la letra de una imagen y conseguir el nombre de una tipografía que se encuentra en una imagen. El programa se llama BUSCAFONT (versión en español, inglés y alemán) lo podéis ver y conseguir en la web http://www.buscafont.es He probado el programa y es una pasada, muy muy rápido, encuentra fuentes comparando la imagen seleccionada con las fuentes que tengas en tu ordenador o en

otros medios de almacenamiento y también en una base de datos online del programa. Tiene herramientas para mejorar la imagen, rotar, separar letras, aumentar la sensibilidad, contraste, etc… También tiene un gestor y navegador de fuentes para visualizar tus fuentes, instalar fuentes y una opción que viene muy bien, seleccionando una fuente que tienes te

busca otras parecidas. Un programa muy muy completo y que soluciona el problema que tenemos muchos diseñadores. Podéis ver las características del programa en la web de http://www.buscafont.es y también descargar una demo.

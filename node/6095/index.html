This has a few very unsual features, especially the 'f' that goes below the baseline. It also has a single-storey 'g', and a double-storey 'a' along with the Venetian-style 'e'. The only typeface I have found with all those attributes so far is M.F. Benton's 'Motto' from 1915 (see McGrew, p. 224). Motto is a calligraphic style, so is not a match.

I would say this is not a Fraktur, by the way.

Judging from the name of this sample, is this book from Romania? How old is it? We are probably looking at a European metal type, and once again my books have little to show of these types. What the world needs is a McGrew-style book for European typefaces.

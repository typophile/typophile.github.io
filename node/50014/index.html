September 30, 2008 • bendy

I’m using Fontlab Studio, wanting to have a handful of glyphs in the edit layer, but picking the contours of a particular shape is tricky when they overlap at low zoom, and tricky when you can’t see the whole contour at high zoom, so what I wanted was to change the colour of individual contours in the edit window so I know which one I’m picking up and moving…is this possible? The mask layer is just as confusing when there’s more than one outline in it.

For example, when I have a 3 and a 5 overlaid on the edit layer, perhaps with other glyphs in the mask layer, can I change the 3 to blue outline and the 5 to pink outline so I can see the difference when I zoom into the bowl?

Does that make sense?!

It makes sense but I don’t think you can do it. You will usually be selecting whichever is on top by clicking. What I might try is moving the outline you don’t want to change by 100 units using command/control arrow and then adjust the outline you want to edit. You can then move the other outline back 100 units by command arrow again. You can do this very quickly via keyboard.

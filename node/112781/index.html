"Associated Business" must be [[http://www.fontshop.com/search/?q=Albertus|Albertus]] by Berthold Wolpe [1940, Monotype] Note that the same design can be found in the latin subset of Hebrew font [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=MS-Windows-Fonts&ffam=Narkisim%20-%20Regular&fid=0b5daaee28ed20474e5b2f204add18f0&fstyle=i&fsize=60&text=Associated%20Business&wrap=2|Narkisim]] designed by Zvi Narkiss in 1978 and supplied with Microsoft Windows. See also an interesting discussion about the legal case against Microsoft some years ago, regarding this font. http://typophile.com/node/75064 "Services" looks like [[http://www.findmyfont.com/index.php/fonts/font-preview?fset=Berthold&ffam=Frutiger%20BQ%20-%20Bold%20Italic&fid=fa2f1a60f3e42e5df2f2b6234c9ebf30&fsize=60&text=Services&wrap=2|Frutiger (Bold Italic)]]

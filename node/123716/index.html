The closest I’ve been able to find is Helvetica LT Std Roman. Is this the same thing, or an updated version?

Helvetica is the general name that describes the typeface design. Like “Sebring” describes a car model. But not all Sebrings are the same because they are all called this way. There were versions of that model produced at different times, and you could have it with different extras if you wish. It’s the same with such fonts. There are old PostScript versions of Helvetica still floating around from the 1980s and 1990s and then there are “re-mastered” OpenType versions with the old name or the new name Helvetica Neue with new character designs and font styles. And some versions come with a basic character set, some with a larger character set. Some are OTF, some are TTF. It all depends. If you were told to use Helvetica, follow the trail back to the source where this specification comes from. They will know which Helvetica to use and they should also know about how this font was licensed and can be used.

For general information on Helvetica see http://en.wikipedia.org/wiki/Helvetica. To license Helvetica, you can have a look at http://www.myfonts.com/search/helvetica/fonts/. On my macintosh, if I give the following input file to XeLaTeX

\documentclass{article}
\usepackage{fontspec}
\setmainfont{Helvetica}

\begin{document}
Hello World! \textit{Hello World}, \textbf{Hello World}
\end{document}

I get a pdf file with Helvetica, Helvetica-Oblique and Helvetica Bold embedded. Notice that the file Helvetica.dfont contains fonts with AAT features (an Apple format) that not all applications can handle.

Is there no such thing as just “Helvetica”?

There is, but that typeface is around since the 1950s and exists in dozens of versions — both regarding the actual design as well as the font formats. So you might want to check with the design department (or whoever is responsible for the corporate design) to see which version you should use and you might also check the licensing for those fonts. There might be special requirements for embedding, server use and things like that.

Nick, I’ve successfully built Central European OpenType TT fonts with "massaged" encoding vectors (well, let’s call it "hacked"), where one and the same font works fine on QuarkXPress 4.x, 6.x and 7 on both Mac and Windows, as well as in Adobe Creative Suite applications, Macromedia applications and Microsoft applications. Otherwise, you may want just deliver Mac Type 1 fonts encoded as Mac CE and Windows Type 1 fonts encoded as Windows CE. Adam

Yeah. With these Font Info settings, you select "MS Windows 1250 Central European" in Names mode, generate a Windows Type 1 font, then select "MacOS Central Europe" and generate a Mac Type 1 suitcase. Regards, Adam
2006-07-22 09:59:26 • twardoch
BTW, if the Polish customers are complaining that they need a font in the "Western" script rather than in the "Central European" script on the Mac, you set Mac script and FOND ID to "Roman" rather than "Central European Roman" and generate the Mac Type 1 suitcases. Of course, you should also make sure that the FOND IDs are not 30970 in all the fonts. They should be different but still within the respective range (Central European Roman or Roman). Adam

Cool I like it.

Any way you do it is going to be a bit laborious. This is what I’d do:

• draw a line with the pen tool • duplicate, rotate and repeat the lines the same way you have done with your text. • Then use ‘type on a path’ tool to start writing on one line and write more information than can fit into the single text box. • Deselect the type box and select the white arrow tool • Select the text box again and you should see a little box with a red plus indicating there’s more info than can fit in the text box. • using the white arrow tool, click on the red arrow and the cursor will change to show what looks like a tiny block of text • select the start of the next line and the overflow will run onto the next line – you can just repeat this all the way round the circle and every line will adjust when you change one detail.

Hope that helps. If you’re really stuck send me your email address and I’ll send you an ai file showing what I mean… or check out ‘typing on a path’ and ‘linking path type’ in help files.

Good luck! Ed

The web developer toolbar for Firefox will do what you want. After you have installed it, click on the “Information” menu and select “Display Element Information”. When you mouse over text it will tell you (among other things) typeface and font size. I am still looking for a plug-in that displays all type info such as font weight, line height, word spacing, etc.

It looks like BMW's custom typeface BMW-Sans to me. This would make sense too as it is a key element of BMW's corporate identity. It was designed in 2002 by Dalton Maag for Interbrand, closely related to Helvetica but with subtle differences including angled stroke termini which aren't parallel with the baseline. http://www.daltonmaag.com/portfolio/2002-2003

Looking for a monoline script (think slightly-retro Swiss ski resort)

July 22, 2010 • nina

Can anyone recommend some fonts that echo some of the features/character of these hand-lettered Alpine ski resort logos? I know it’s a relatively wide field, but I’m trying to find something that matches the feel rather than any one exact feature… I’d prefer something monoline/low-contrast and relatively «stiff», but not too posh, and connected. A retro touch is totally OK. (So far I can say that Monoline Script MT seems not rigid enough, Kursivschrift is too cartographic, Roman Script has too much contrast, and Coquette seems too urban/elegant.) Also, in fonts that follow a different handwriting tradition than German/Swiss (for instance featuring things like the «double-storey» “e”), alternates for German/Swiss usage would be a definite plus, since I’m trying to get a «local» flavor. It’s for a headline/logo, so I’d be OK with some tweaking, I’d just love to find something good to start with – figured maybe somebody here has a spontaneous idea. Thanks!

Ah, no. It ought to work (probably), but you can first try manually changing the association. In HKEY_CLASSES_ROOT, ".ai" points to a default handler for "Adobe.Illustrator.13". This key, in turn, contains all necessary Explorer information -- eg., the icon to use, but also the Shell stuff (as in "Open","Print"). The Shell subkey there contains a subkey "open", which then has a subkey "command", and this is finally the one to edit. Its value "(Default)" should read -- including the quotes "C:\Program Files\Adobe\Adobe Illustrator CS3\Support Files\Contents\Windows\Illustrator.exe" "%1" -- but note that this is the path on my system, and yours is highly likely different.

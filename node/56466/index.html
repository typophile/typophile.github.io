Kerning tool

April 1, 2009 • apankrat

I just ran into the following idea and I wonder what you guys think of it. Create a tool that would allow tracking and kerning a text, but it would also allow sharing the final kerning values with other people. The typeface and the kerning pairs will be submitted into a central "kerning repository", and this repository is going to be openly accessible by everyone else. Then, when someone starts on kerning text set in the same type, he can use the repository entry as a starting point. Multiple entries for the same kerning pair can be order based on certain criteria, e.g. give a higher preference to the submissions of a certain user or that with a higher "popularity index". So it is in its essence a crowd-sourced kerning improvement project. I am a programmer by trade and this is something that is not going to be that hard to write, so the prototype can be thrown together in a matter of few weeks. Thoughts or comments ? Not considering the implementation details at the moment, does the idea have any practical value ? Or does it solve the problem that does not and will not exist ?
The problem with this is that kerning is relative to size (among other things) and most designers aren’t doing much kerning at text sizes. So the kerns I might output using Hypothetical Grotesk (I’m trademarking that name, nobody use it!) at 24 points probably aren’t relevant to Hypothetical Grotesk at 120 points.

Seriously, what's wrong with just learning how to kern? Most of the time we only kern anything obvious - the bigger the copy, the more obvious the kerning issues. It's a good idea - but only if we spent all our time kerning body copy. Which we don't. At least, I don't. The downside to automating every detail is that designers become complacent and altogether ignorant of spacing issues. Kerning type allows designers to fine tune their abilities to read the space and negative space between forms; Gain greater sensitivity towards the rhythm between shapes and relative color/value - useful skills for any designer.

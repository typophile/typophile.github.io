2015-01-11 20:34:26 • bojev

The backslash, indeed, was basically invented by Bob Bemer for the Model 33 Teletype, for which ASCII was invented. His idea was that /\ and / could be used to represent the math symbols for ‘and’ and ‘or’ by only adding one character instead of two.

AFAIK, no one ever used the backslash for that purpose.

As for @ and #?

Well, they were standard on manual typewriters. But they weren’t standard in fonts - a font typically had, as you can see in old specimen books,

ABCDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyz .,:;!?()-&$

…and the dash as well as the hyphen, and that was it. You could get accented characters for some typefaces as well. But @ and # were sorts, and so you would basically have one serif version and one sans-serif version, and that was it. If you were lucky, there would be a bit more stylistic variation - but basically, if a book needed an @, and it was set in Times Roman, you still used the @ you had, which was likely to look more at home with Bodoni (it could well have been designed for Scotch Roman of some sort… or even more likely, Century Expanded).

And you would use the same @ with Baskerville, with Caledonia, and so on.

Of course: in choosing how to write e-mail addresses, they used what were available.

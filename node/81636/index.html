I confess that there are mysterious things happening with that font; part of the problem must be due to the naming scheme but also, part of the problem seems to be coming from the sequence "ita" that occurs in the fontname and that the mac seems to interpret as meaning "italic"; indeed, even if I manage to install the two fonts and access them (after replacing the fontname "Auctoritas" with "AuctoritasItalic", saving the font file "Auctoritas_Italic.ttf" and installing it), the "option I" command does not work: here is a grab: [img:sites/default/files/old-images/auctoritas_5760.jpg] You can see, the regular is selected, and Italic stays checked in the font menu. If I furthermore replace every occurrence of "Auctoritas" with "Auctorjtas", then the font starts working properly. Weird.

I’ve been on a mac for about 6 months. I was raised in dos and windows. Here are some things I’ve done to help me adjust:

In the system preferences > appearance change the scroll arrows to appear at both ends of a scroll bar.

I’m always accidently clicking too far to the left or right of a window and losing focus of the application. In photoshop, always hit F to go to full screen mode. Also, try to minimize the windows you’re not using.

Control = Alt. It hurts at first, like learning a new guitar chord. I still get cramps in my left thumb from reaching it all the way under my palm to press that weird open-apple key.

Font Explorer X has helped a lot with font management. Definately use it.

I still don’t understand the mac file structure. I keep all my work on the desktop in job folders. The finder window is pretty neat though. I primarily use the column view, it’s the fastest and easiest. Kind of like the old 2 column windows explorer but a but more clumsy.

Now I just wish I could figure out how to get rid of these stupid jelly buttons, rounded corners, and brushed aluminum.

2006-03-03 13:55:39 • dezcom

If you double-click a folder icon while holding the command key (the apple key), it will open that folder in a new window while keeping the old one open.

Hey, I didn’t know that. Now that I do, here is one way to do the shortcut Adam is looking for:

With a Finder window open, double-click a folder in it while holding the command key. A new window (the folder) will open. Hold down the command key and click on the title of the new window. A pop-up menu showing the folder path will appear. Choose the enclosing folder (or volume). You now have two windows to the same file path. The only case when this wouldn’t work is when there are no folders in the window view you want to duplicate. There is probably a way to do this with AppleScript in a way that works regardless.

FYI, before OS X, it was not possible to have two windows showing the same path. This is a Next carry-over that confused long-time Mac users at first.

Control-click is the same as a right-click–handy if your mouse only has one button.

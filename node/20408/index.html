T1 outlines should run counter-clockwise.

Assuming you’re using FL5, go to Tools–>Action Set, and them from the Available actions chose ‘Contour direction’. Click the button with the arrow and select the correct option. Hit Run.

2006-06-02 22:12:38 • Miguel Sousa

I always remove the ot files and do a computer restart to clear everything out of the cache so it’s doubtful that’s the problem. Not nececarrily true, especially if you’re using Adobe apps, (which i assume you are). you need to find any files named adobefnt.lst and trash em.

Are the little grey arrows at the edge of the outlines showing the direction? What direction should a type 1 outline be going in, to the left or right? yes. counterclockwise for PS flavored fonts, clockwise for TT flavored fonts.

Isn’t there a button to push in fontlab to make this happen automatically. in FLS5 there is. or you can select all your glyphs and then go to Contour>Paths>Set PS direction.

Using the Convert to Type 1 tool doesn’t change the direction. True, this will just change your bezier outlines to quadratic beziers.

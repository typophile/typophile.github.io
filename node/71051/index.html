2010-06-04 22:50:06 • Theunis de Jong

Some sans typefaces have a large “beak” on the one. e.g. Helvetica. Others have a baseline crossbar on the one, e.g Franklin Gothic.

Nowadays, many typefaces have alternate designs for proportional and tabular “one"s. I usually put a baseline crossbar on my tab “one"s.

You need to use an OpenType layout palette to access the alternate.

I want to know how to determine if they’re faux or real. In general, this is pretty easy to spot. Given that they were well designed, true SC are better proportioned and have the same weight (e.g. aproximately the same stem width) as lowercase characters. If you're not sure, print out a passage of text with some embedded words typeset in SC and you'll see the difference. If you don't want to print and are using InDesign, just open the Glyph Palette and visually check for the presence of SC.

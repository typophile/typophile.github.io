Maybe [[http://www.motaitalic.com/fonts/vesper/weights-styles|Vesper]] from Motaitalic, an outstanding and recent piece of work. It is a different category, more a wedge serif, but the description “angular” fits, plus it is not associated with anything when it comes to branding as far as I know, so that is a plus. Many weights, many characters, great use of contextual alternatives to improve the appearance on the page. A typeface with great potential.

Other than that, Fontbureau has some text typefaces which are a closer match.

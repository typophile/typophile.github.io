Eric, a trick I do when developing fonts is to stick the date in the name, as well as a version number, if I’m going to be generating several fonts per day. So, a font generated today would be “Fontname-Test2006020301”. I also change the version number, and make sure that all of the name fields in all of the many FontLab windows are updated with the right name.

Cache-clearing is a good idea. Also, delete all of your AdobeFntXX.lst files, and Log Out and In, or restart. (Every once in a while. You don’t need to restart every time…)

If you’re working with OT, make sure that you don’t have conflicting kerns; it could be that you have a kern feature which is battling with a kern table.

Small Caps

September 13, 2007 • Sharon Van Lieu

I just finished a project using Dolly small caps. I thought they worked really well and it made me wonder what other great small caps I might not know about. I like Feijoa’s small caps also.

So if you have some small caps you would recommend, I would appreciate it.

Sharon

Iowan Old Style, by John Downer. Iowan is old style Venetian finish and characteristics built on present-day proportions (metrics, extender lengths). It has a large useful family including an excellent set of smallcaps.

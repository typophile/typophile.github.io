September 15, 2005 • jjshell

Hi, I was wondering how much a text block of 58 lines of a 10/11 pt type size/leading would make in centimeters? Does it change with each type or is it always the same? thanks in advance for your time :)
It will change with the type. Point size is just a relative measurement. There will be some relationship in size from font to font but not a precise one. Sometimes the difference can be huge. If you want to answer the question you have to choose a type & then measure. Why do you want to know? What is your purpose?

The height of the block of text from ascender of the top line to descender of the bottom line will be 22.43cm (636pts)* the top line won't have any leading above it so it isn't 22.50cm (638pts). This is only pedantry and won't really affect a mark-up for text with such tight leading and the difference will only be tenths of a millimetre. *This figure is likely too high since the actual height of a font rarely takes up the full point size it is designated.

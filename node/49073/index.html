It does. But not every OpenType feature. The same is true of all applications! For instance, Photoshop has no glyph palette. And now, many foundries are putting features in OpenType fonts (e.g. , , ) which are not yet supported in any application, AFAIK. So this is one area of fine typography where it’s important to be knowledgable of which features are in which fonts, and which applications support them.

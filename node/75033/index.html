Changing corner to curve point in FL (PostScript Type 1)

October 8, 2010 • charles ellertson

This has bugged me for years – ever since moving to FL from Fontographer. How to change an existing corner point to a curve point. The manual says:

Note: A quick way to change the connection type is to double-click on a node. [page 345 in the PDF version]

Not on my machine(s). Windows, both XP and Vista.

Example: Suppose I add a point on what is now a straight line. It will be a corner point, w/o handles. How do I change it to a curve point, with handles?

On a mac, you need to option-drag the node to pull out the handles. On a PC I assume some similar procedure exists, though I don’t know which modifier key is used when dragging.

Alt/click on the line segment and the handles will appear. Do this to both segments and then double click on the node to change it from a corner to a smooth (curve) node.

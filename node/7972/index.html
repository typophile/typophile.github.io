From Insider Software's Bob Leeds:

<!-quote-!>quote:

FontAgent Pro does, indeed, split out the suitcases into individual styles.
This is a big bonus and one of the most important reasons for selecting
FontAgent Pro. It has the following benefits:

1) Allows more precise organization
2) Permits documents to be smaller since not all fonts need to be loaded
3) Salvages unusable suitcases by allowing a user to retrieve the good fonts
from an otherwise damaged suitcase
4) Provides greater system stability by eliminating problem fonts

AND, he gets all of this without paying the price of losing the organization
that he wants! There is a check box in the libraries pane to group the fonts
by name or show them individually. He apparently has them set up to view
individually, rather than by family.

BTW, as far as Suitcase crashing, it sounds like he doesn't have the fonts
available for Suitcase to manage, but he shouldn't want to go back there
anyway. He could have tried copying fonts to do the trial, but if he just
takes the fonts out from under FontAgent Pro management, everything should
work as well as before.<!-/quote-!>

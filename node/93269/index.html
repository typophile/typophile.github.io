May 22, 2012 • docunagi

Hello, I am designing a font with stylistic features but when I use it in the glyphs window in Illustrator, my alternate glyphs appears one after an other. But when I use some other fonts, a menu appears when alternative glyphs are available. I don't really understand what I am doing wrong. Is it some kind of opentype feature that I am not doing right in Fontlab ? Any help would be much appreciate :) [img:sites/default/files/old-images/menu_5693.jpg]
Where in the glyph window an alternate will appear depends on the order of glyphs within the font file. The menus you see rely on the 'aalt' feature. This should be present if you want alternates to be presented in the glyph palette. André

Pieter, here’s what I would do…

Create classes for your numerals like this:

@class_0 = [zero zero.OP]; @class_1 = [one one.OP]; @class_2 = [two two.OP]; @class_3 = [three three.OP]; @class_4 = [four four.OP]; @class_5 = [five five.OP]; @class_6 = [six six.OP]; @class_7 = [seven seven.OP]; @class_8 = [eight eight.OP]; @class_9 = [nine nine.OP];

Then your frac will look like this, (also incorporating the perthousand and percent subs Mr Connare suggested):

feature frac { # Fractions

Latin

sub @class_0 @class_slash @class_0 @class_0 by perthousand;
sub @class_0 @class_slash @class_0 by percent;
sub @class_1 @class_slash @class_4 by onequarter;
sub @class_1 @class_slash @class_2 by onehalf;
sub @class_3 @class_slash @class_4 by threequarters;

} frac;

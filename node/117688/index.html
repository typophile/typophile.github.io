Walpurga - a hand-drawn display font

July 2, 2014 • dancinghorses

Hello everyone!

Walpurga is a witchy display font that comes in two variants - regular & bold - and with four glyphs for each letter. There is a set of ornaments and the in-built open type features produce a random, natural looking effect when activated through the “contextual alternates” feature. Walpurga was created with stylized brush ends that become visible at larger font sizes. Hope you enjoy!

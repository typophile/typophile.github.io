For something asymmetric like this its probably easier to edit an existing font. This lettering was made in quick order using FF Trademarker Light https://www.myfonts.com/fonts/fontfont/ff-trademarker/ How to: Condense to 80% Thicken stroke on rhs by reducing width of counter. To make R use B and cut out the bottom. To make A thicken the stoke on both the counter side and the outside. To make N use the wide stoke from A and stretch it a little wider. If you want to make a closer match, edit as much as you want. Don

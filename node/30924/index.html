Stopping runaway italics on Typophile

January 27, 2007 • Thomas Phinney

I know I just said this in another thread, but I thought it deserved its own discussion.

Sometimes one sees runaway italics in a message thread, where somebody forgets to close a “cite” or “em” tag and then all the rest of the thread ends up in italics. This happens because from an HTML point of view, it’s one continuous block of HTML, so the unclosed tag “infects” the rest of the thread.

But the good news is that for the same reason, any subsequent poster can stop the run of italics in their message! Just start your message with closing “cite” and “em” tags with no opening, and all should be well. I’ve used this successfully on several occasions now.

This public service announcement is brought to you by the letter “C” and the number “5.”

Altsoft FTMaster 1.3 released

March 4, 2005 • altsoft

FTMaster version 1.3 is released March 3, 2005 Altsoft N.V. Altsoft N.V. announces the release of FTMaster version 1.3, the font editing and conversion tool. The update contains: - improved encodings creation and editing; - glyph outlines normalization; - the batch font renaming tool; - font preview generation. The new version can be downloaded at: http://alt-soft.com/products_ftmaster_download.jsp It is free for all users of versions 1.x. The retail price of FTMaster is 199$. Additional details on FTMaster can be found at: http://www.alt-soft.com/products_ftmaster.jsp For any additional information please visit our web site http://alt-soft.com/ or e-mail to info@alt-soft.com Best Regards, Altsoft N.V.

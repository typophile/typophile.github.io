Alpha Chi is a very simplified Old English for A and C The C is very similar to this old set of iron on letters [img:sites/default/files/old-images/IronOnOldEnglishTypophile78988b_4689.jpg] Nothing digital that even approximates the look. One could try to simplify something like Engravers Old English http://www.myfonts.com/fonts/mti/engravers-old-english-mt/ For the lowercase one could modify something like Morris Gothic http://www.myfonts.com/fonts/hihretrofonts/morris/ or P22 Morris Troy http://www.myfonts.com/fonts/p22/morris/ A far from satisfactory answer, but this is as good as it gets… Don

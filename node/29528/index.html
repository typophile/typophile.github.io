2006-11-16 02:18:42 • Chris Dean

Pretty darn close it’s not.

Indeed, at first sight, the M, A and N remind of Helvetica and the D and E could pass off as modified versions of the Helvetica originals.

A closer comparison of the M, A and N reveals, though, that they’re not from Helvetica. As far as one can judge from the low quality sample on the pack shot, in Helvetica, the M is narrower, the stems of the A broaden more towards the baseline and the modulation (line width variance) is more distinct in each of the five characters.

That was enough to make me suspicious, so I went to the EA Sports site and looked at another pack shot and, well, there’s not much more Helvetica in the EA font than Comic Sans in the Coca Cola logo:

(Undeniably, there is a certain degree of kinship between these fonts in terms of overall anatomy and the EA font might have been designed with Helvetica as a template.)

2006-11-16 06:49:15 • Chris Dean

Preferably jpg, gif (or png) file format, not wider than 600 pixels (http://typophile.com/readme).

Image linking makes sense for big pics (that would not fit the comfortable line length of the text messages) and pics of bandwidth-critical file size (I don’t know if there’s a file size limit for the “Insert Image” function of this board).

For all other pictures (and most of the graphics that are typically posted on a type forum fall in this category) it’s more convenient to actually have the pics in the discussion thread.

Storing pictures outside of this board saves the visitor no bandwidth, as there’s rarely a point in reading a discussion that deals with the pictures posted by someone without seeing these pictures, so the visitor will click the links anyway.

PS: This thread is a request for a type face identification and, as such, would belong into the ID forum.

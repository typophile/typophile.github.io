I checked the Antique handwriting (Part 10) section in the Script Font ID Guide, and also checked MyFonts for ‘old script’. I think Gauguin was the most similar to me. At least for these letters. I agree with Florian that Secret Scrypt is another similar style.

That suggests that this is either an amateur script ( the joins are rather poor), and/or a custom font, because it looks like a font (based on identical-looking duplicate letters).

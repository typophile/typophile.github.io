[[Indicies]] : [[Typefaces]] : Linotype Univers During the 1990s, [[Adrian Frutiger]] worked together with [[Linotype]] to expand and update his classic [[Univers]] family. The result, [[Linotype Univers]], includes 63 different weights, each drawn carefully to ensure compatibility with all the others. The original Univers type family can be found online here. Linotype Univers can be found on this overview page.

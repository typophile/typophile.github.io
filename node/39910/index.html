The file they sent you is a type gauge ruler of the variety that was in common use in the pre-computer era. I have a drawer full of these in various sizes, most with the logo of some long dead type house.

If you’re truly interested in measuring type offscreen, I would not trust something created from a file and output on your home printer. A good type gauge would give you real sizes, not ask you to go perform math.

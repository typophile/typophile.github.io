Andron: new special fonts

January 20, 2012 • Andreas Stötzner

I hereby announce the release of some new Andron fonts. The popular Andron-2-English font is now completed by an Italic and a Bold version: [img:sites/default/files/old-images/ANDRON_G2_ENG_2_5971.gif] The Andron-2-Deutsch sort is now grown to a family of its own by three new and rather peculiar fonts. There is a new Italic and a new Scriptive, and Andron-2-Deutsch-Laendlich explores the possibility of a different approach to Capitals. This variation has been inspired by the base glyphs of £, € and $ which dwell in each font but hardly count for a model usually. [img:sites/default/files/old-images/AND-2-DEU-LAENDL_5449.jpg] [img:sites/default/files/old-images/ANDRON_G2_DEU_2_6235.gif] Moreover, the new Italic and Scriptive fonts of Andron 2 contain a splendid set of fancy classic ornaments: [img:sites/default/files/old-images/ANDRON_G2_ENG_orn_4545.gif] This ornament selection is also available separately, as “Andron Ornamente”. A scriptive version has also been added to the Andron-1-Latin family. All new Andron Fonts are available now at [[http://new.myfonts.com/foundry/SIAS/|MyFonts]].

Hi Lizy, The text is set using siFR, which utilizes a combination of JavaScript, CSS and Flash to render fonts on screen without the need to have the fonts installed on ones system. Pros: You can use any (properly licensed,although I am unsure whether or not most EULA's even allow this kind of thing) font. It is easy as pie. You can get crazy with typography on the web without using graphics. Cons: It uses Flash, therefore Flash must be installed on a browser to view it. It doesn't support kerning or OT features Also doesn't support most characters outside of a standard character set. Grot - putting this in the FAQ is a good idea. I'll work on getting it there soon.

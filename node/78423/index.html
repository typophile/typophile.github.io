Select All characters, then Glyph > Glyph Names > Generate Unicode. Look for any glyphs whose names appear against a red bar, which indicates that either the glyphname is incorrect, the Unicode is wrong or mssing, or multiple glyphs are mapped to the same codepoints.

Also, what program are you using to test your finished font?

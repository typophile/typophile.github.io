"Ooooooooh, nice one , Mike! Brilliant ID." Thanks, Yves, but it was really solved by a tool. All I did was use the a, b, and e in the Serif Font ID Guide and pick it out of a visual lineup of 28 fonts that had those features. It's my 'secret weapon' for Serif fonts. It's a bit tricky to use, because the 12 key letters are not always easy to match with the choices, but it often gets you into the right styles if you don't try to use the 'shaky' letter choices.

Try using PhotoShop levels to limit gray range and tighten them up - then run poster edges and then covert to one bit. This will give you a nice engraved look.

Process:

  1. Convert Photo to Grayscale.
  2. Run Artistic/Poster Edges Filter at 1 for Thickness, 6 for Intensity and 2 for Posterization.
  3. Use Image Adjust Threshold to fine tune your Sketch.
  4. Convert to Bitmap using Diffusion Dither.

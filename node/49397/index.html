USing CS2 FOnts in Other applications

September 12, 2008 • imagerg

Does anyone know of a way to install the fonts supplied with CS2, in the windows Font folder?

Sure, just move them from the Adobe common fonts folder at C:\Program Files\Common Files\Adobe\Fonts.

(If you look at the Adobe’s OpenType User Guide it explains how to install fonts in general, and covers the locations of the Adobe-specific folders.)

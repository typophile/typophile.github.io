Type 1 fonts are indeed supported, but the postscript version of Helvetica Neue conflicts with the version Apple supplies (which is a .dfont and can’t be gotten rid of). I ended up buying the weights of Helvetica Neue that I needed in OTF from Linotype.

Helvetica Neue was the only type 1 font I’ve ever had an issue with.

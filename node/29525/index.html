Chris is quite right as concenrs a Greek final sigma being the shape of a standard uppercase Sigma when setting text in small caps.

As concerns diacritics, this is where it’s easy (or perhaps complex, depending on your viewpoint and the coding you’ll use for Open Type subs*). The rule to remember is: Greek text set in all caps and/or small caps (and yes petite caps, too, if anyone’s curious) uses practically no diacritics. The sole ones you’ll need to retain are for the vowels with dialytika (diaeresis).

Easy enough? Good. That’s the way to go if one wants to “play by the rules.” And please note this does not change dependent on whether the Greek text being set is in Katharevousa (Purist), Archaic, Koine (think New Testament era Greek), or contemporary Demotiki.

Best regards, R.C.C.

Should the final sigma slot for small caps (or all caps for that matter) fonts be just a regular sigma?

Yes. Both σ (U+03C3) and ς (U+03C2) capitalize to Σ (U+03A3).

and for the lowercase diacritic characters, what should be done?

The short answer is, accents are dropped except the dialytika, as R.C.C. said. This means, in AllCaps/SmallCaps only context, lowercase capitalizes like this:

ά –> Α έ –> Ε ή –> Η ί –> Ι ό –> Ο ύ –> Υ ώ –> Ω

ϊ –> Ϊ ϋ –> Ϋ ΐ –> Ϊ ΰ –> Ϋ

and this is only monotonic…

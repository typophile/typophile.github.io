Most likely. Adobe CS installs fonts in that location.

With respect to jsmi’s comment, there’s nothing in the license of which I’m aware that reserves those fonts for use only in Adobe applications. You can move or copy them to another font folder if you want to access them elsewhere.

Anyone wanting to make fonts available only to their own applications would probably store them inside the application bundle rather than in /library/application support.

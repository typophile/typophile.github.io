2010-03-25 02:51:47 • John Lyttle

The original version of Avant Garde has a “k” like that in the lighter weights. You can still see it in some cuts (such as the ITC version Monotype carries).

Adobe did a multiple master version during the ’90s. This required that inconsistencies like this among weights be eliminated. The multiple master version is long gone, but the enforced consistencies still remain in their version. The cap “R” is another example (victim?).

So, yes, this is Avant Garde, but some versions available don’t match.

* The [[http://freetype.sourceforge.net/FTL.TXT|FreeType license]] was designed as a license for the [[http://www.freetype.org/|FreeType]] rendering engine (a software library) not as a license for fonts. * Recently [[http://perens.com/about/introduction/|Bruce Perens]] in his blog article [[http://perens.com/blog/2009/02/17/64/|Warning: SIL Open Font License]] has expressed some concern about SIL's [[http://scripts.sil.org/OFL|Open Font License]] claiming: "There appears to be an unintended loophole that would allow the conversion of any font under the license to public domain." * There is a "[[http://www.gnu.org/licenses/gpl-faq.html#FontException|Font Exception]]" which can be tagged onto the end of the [[http://www.gnu.org/copyleft/gpl.html|GPL]] to make it suitable for fonts ~ however the author of that subsequently wrote an article on [[http://www.fsf.org/blogs/licensing/20050425novalis|Font Licensing and the GPL]] in which he expressed some reservations. - Chris

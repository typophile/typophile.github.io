Typography research for a magazine design project

October 28, 2006 • gretelz

I’m doing a typography research for a magazine design project. The magazine is italian and it’s going to be distributed in Italy only. The core theme it’s architecture. I’ve been asked to do a type research.

BODY COPY SERIF I’ve been ask from my creative director to find the font you can see in the sample attached for the body copy. Yesterday I posted the sample in the Type ID board, were the font was identyfied as Adobe serif. I think this font is not a good example of typogaphy, and i want to propose something different. i need to find something similar, well design and possibly new. the font has to be serif and modern with light contrast of thikness.

BODY COPY and TITLING SANS SERIF I need to find a sans serif to combine with the serif. Letter forms must be “not cold” (not so geometric) but simple. Must work good with body copy and as title text (big size) as well. And it must have at least five style. If new better. For this one I was thinking of:

  1. Fedra Sans
  2. Relay (may be to geometric)
  3. Amplitude (not so good in big size)
  4. Tetria
  5. FF meta

any suggestion will help thank you in advance getel

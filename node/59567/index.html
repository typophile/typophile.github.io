La Pignata is Verona Script by Helmut Hans Matheis, 1958.

The only digital version I’m aware of was made by Mike Allard.

Unfortunately, Verona Script was one of Allard’s commercial fonts and TypOasis offers for download only Allard’s fonts that were originally free and shareware. The only legitimate way to get hold of Allard’s Verona Script font would be to contact him.

Note that Allard’s Juliet Script is simply a heavier version of Verona Script and you CAN download that.

The media might be newer, but I’m not sure the time required to do them in Illustrator would be all that different: you might get some time-saving by using AutoTrace, but you still have to tinker with that, or draw ‘em from scratch, which certainly wouldn’t be much different than using pen and paper, at least in the first iteration.

Where you’d really see a difference is if there are changes to be made – much cleaner on the computer, since you’d never have to start from square one.

Given where you are, $65/hour sounds about right….
